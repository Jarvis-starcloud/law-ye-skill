# 法律文书写作规范汇编

> 定位：律师执业常用12种文书的写作规范、结构要求、实操技巧
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块4交付物

---

## 一、总则——法律文书写作通用规范

### 1.1 五大铁律

| 铁律 | 说明 |
|------|------|
| **事实清楚** | 时间、地点、人物、事件经过，叙述客观不带情绪 |
| **依据充分** | 引用具体法条（法律名称+条款号），不用"根据有关法律" |
| **逻辑严密** | 论证从事实→法律→结论，三段论结构 |
| **格式规范** | 标题、当事人信息、正文、落款四部分齐全 |
| **语言专业** | 用法言法语，避免口语化表达 |

### 1.2 常见错误

| 错误 | 正确 |
|------|------|
| "众所周知""不言而喻" | 不用假设性表达，用事实和法条 |
| "情节恶劣""令人发指" | 客观描述事实，由法官判断性质 |
| "约""大概""可能" | 精确表述，用具体数字和日期 |
| "相关法律法规" | 写明具体法律名称和条款 |

## 二、答辩状

### 2.1 使用场景
被告收到起诉状副本后，针对原告诉讼请求和事实理由进行逐一反驳。

### 2.2 结构

```
                     民事答辩状

答辩人：[基本信息，同起诉状格式]

答辩人就原告XXX诉答辩人XXX [案由] 纠纷一案，
提出如下答辩意见：

一、关于原告第一项诉讼请求
[针对性反驳，包含事实+法律依据]

二、关于原告第二项诉讼请求
[针对性反驳]

三、关于原告陈述的事实
[指出事实错误或遗漏]

综上所述，原告的诉讼请求缺乏事实和法律依据，
请求法院依法驳回原告的全部诉讼请求。

此致
[XX]人民法院

                        答辩人：[签名]
                        日  期：[XXXX年XX月XX日]
```

### 2.3 写作要点

| 要点 | 说明 |
|------|------|
| **逐条反驳** | 对原告每一项请求都要回应，不能遗漏 |
| **先否后辩** | 先否定对方主张，再阐述己方理由 |
| **抓核心争点** | 把最有力的反驳放在前面 |
| **避免自认** | 不要在答辩中承认对己不利的事实 |
| **预留空间** | 不要把话说死，为庭审留余地 |

## 三、代理词

### 3.1 使用场景
代理律师在法庭辩论阶段提交的书面意见，是庭审的核心文书。

### 3.2 结构

```
                      代理词

尊敬的审判长、审判员：

[律所名称]接受[当事人]的委托，指派本律师担任
[案由]纠纷一案中[原告/被告]的诉讼代理人。

经过法庭调查和举证质证，现发表如下代理意见：

一、关于本案基本事实
[概述案件事实，强调对己有利的事实]

二、关于[争议焦点1]
[结合证据和法律分析]

三、关于[争议焦点2]
[结合证据和法律分析]

四、关于[争议焦点3]
[结合证据和法律分析]

综上所述，[总结性结论]，恳请法庭依法支持
[原告的诉讼请求/驳回原告的诉讼请求]。

                    [律所名称]
                    代理律师：[签名]
                    日  期：[XXXX年XX月XX日]
```

### 3.3 写作要点

| 要点 | 说明 |
|------|------|
| **围绕争议焦点** | 法庭归纳的争议焦点是代理词的骨架 |
| **证据+法律双线** | 每个论点都要有证据支撑和法律依据 |
| **回应对方观点** | 针对对方质证意见和辩论意见逐一反驳 |
| **引用庭审事实** | 引用对方当庭自认的事实特别有效 |
| **结论明确** | 最后一段清晰表达诉求 |

## 四、上诉状

### 4.1 使用场景
不服一审判决，在上诉期内向上一级法院提起上诉。

### 4.2 结构

```
                     民事上诉状

上诉人（一审[原告/被告]）：[基本信息]
被上诉人（一审[被告/原告]）：[基本信息]

上诉人不服[XX法院]于[日期]作出的[案号]
民事判决书，现依法提起上诉。

上诉请求：
一、撤销[案号]民事判决书第X项；
二、改判[具体请求] / 发回重审；
三、一、二审诉讼费由被上诉人承担。

上诉理由：

一、一审判决认定事实错误
[具体指出哪些事实认定有误，结合证据分析]

二、一审判决适用法律错误
[具体指出适用了什么法律、应当适用什么法律]

三、一审程序违法（如有）
[具体指出程序违法事项]

综上，恳请二审法院依法改判。

此致
[XX中级/高级人民法院]

                        上诉人：[签名]
                        日  期：[XXXX年XX月XX日]
```

### 4.3 写作要点

| 要点 | 说明 |
|------|------|
| **必须写清上诉理由** | 不能只说"不服"，必须具体说明哪里错了 |
| **紧扣法定改判事由** | 事实认定错误/法律适用错误/程序违法 |
| **引用一审判决原文** | 指出判决书的具体段落进行反驳 |
| **新证据要说明** | 有新证据的必须说明来源和发现时间 |
| **注意上诉期限** | 判决15日内/裁定10日内，过期不候 |

## 五、律师函

### 5.1 使用场景
以律师名义向对方发出的正式法律文书，用于催告、警告、通知、中断时效等。

### 5.2 结构

```
                      律 师 函

[函号：XX律函字(20XX)第XXX号]

致：[对方名称/姓名]

[律所名称]依法接受[委托人]的委托，
就[事项简述]事宜，发出本律师函：

一、基本事实
[简述委托人与对方之间的法律关系及争议事实]

二、法律分析
[指出对方行为违反的法律规定及法律后果]

三、律师意见
兹正式通知贵方：请于收到本律师函之日起[X]日内
[具体要求，如"支付拖欠款项XXX元""停止侵权行为"等]。

逾期未履行，本所将根据委托人的指示，依法采取
进一步法律行动，届时由此产生的一切法律后果
由贵方承担。

特此函告。

                        [律所名称]
                        经办律师：[签名/盖章]
                        日  期：[XXXX年XX月XX日]

附：授权委托书复印件
```

### 5.3 写作要点

| 要点 | 说明 |
|------|------|
| **语气专业克制** | 不威胁不恐吓，陈述法律后果即可 |
| **事实简明** | 突出核心争议，不展开过多细节 |
| **期限明确** | 给对方明确的行动期限 |
| **保留送达凭证** | EMS寄送+保留签收回执，可中断诉讼时效 |
| **注意保密** | 不在律师函中过多暴露委托人的底牌 |

## 六、法律意见书

### 6.1 使用场景
为委托人就某一法律问题或交易事项提供的专业法律分析和建议。

### 6.2 结构

```
                    法律意见书

致：[委托人/公司]

[律所名称]接受贵司/阁下委托，就[事项]出具
如下法律意见：

一、委托背景
[简述委托事项和需要分析的问题]

二、事实概述
[列明与法律分析相关的基本事实]

三、法律分析

（一）关于[问题1]
[法律规定+适用分析+结论]

（二）关于[问题2]
[法律规定+适用分析+结论]

四、法律风险提示
[列明可能的法律风险和不利后果]

五、律师建议
[根据分析结论给出具体可行的建议]

六、声明与限制
本法律意见书仅供委托人参考使用，不构成对
任何第三方的法律意见。本所对委托人提供的
事实材料的真实性不承担审核责任。

                        [律所名称]
                        经办律师：[签名/盖章]
                        日  期：[XXXX年XX月XX日]
```

## 七、强制执行申请书

### 7.1 结构

```
                  强制执行申请书

申请人（原审[原告/被告]）：[基本信息]
被执行人（原审[被告/原告]）：[基本信息]

执行依据：[XX法院][案号]民事判决书/调解书

申请执行请求：
一、[具体执行内容，如"依法强制被执行人支付
    款项人民币XXX元及利息"]
二、[迟延履行利息/加倍支付迟延履行期间的
    债务利息]
三、本案执行费由被执行人承担。

事实与理由：
[简述：什么判决→判了什么→到期没履行→申请执行]

被执行人财产线索：
1. 银行账户：[银行名称+账号]（如知道）
2. 不动产：[地址]（如知道）
3. 车辆：[车牌号]（如知道）
4. 其他：[股权、知识产权等]

此致
[XX]人民法院

                        申请人：[签名]
                        日  期：[XXXX年XX月XX日]

附：
1. 生效法律文书原件
2. 生效证明
3. 申请人身份证复印件
```

## 八、财产保全申请书

### 8.1 结构

```
                  财产保全申请书

申请人：[基本信息]
被申请人：[基本信息]

申请事项：
请求贵院依法对被申请人名下价值不超过人民币
[XXX]元的财产采取保全措施，包括但不限于：
1. 冻结被申请人在[XX银行XX支行]的银行存款
   人民币[XXX]元；
2. 查封被申请人名下位于[地址]的房产；
3. [其他财产]。

事实与理由：
[说明不保全会导致判决难以执行的紧迫性]

申请人愿以[担保方式：如XX保险公司保函/
现金/房产]提供等额担保。

此致
[XX]人民法院

                        申请人：[签名]
                        日  期：[XXXX年XX月XX日]
```

## 九、调解协议

### 9.1 结构

```
                     调解协议

甲方：[基本信息]
乙方：[基本信息]

甲、乙双方就[纠纷事项]，经友好协商，
达成如下协议：

一、[核心条款：如乙方于X日前支付甲方XXX元]

二、[支付方式：如分期/一次性/转账至XX账户]

三、[违约条款：如逾期支付，乙方额外承担X%违约金]

四、[权利放弃：如甲方收到全部款项后，放弃就
    本纠纷向乙方主张其他权利]

五、[保密条款：如双方对协议内容保密]

六、本协议自双方签字之日起生效。

七、本协议一式X份，各方各执X份，具同等效力。

甲方（签名/盖章）：       乙方（签名/盖章）：
日期：                     日期：
```

### 9.2 重要提醒

| 注意事项 | 说明 |
|----------|------|
| **可申请司法确认** | 调解协议可向法院申请司法确认，确认后有强制执行力 |
| **写清违约后果** | 否则对方不履行你只能另行起诉 |
| **保留原始证据** | 签了调解协议也要保留原始证据，防止对方不履行 |

## 十、其他常用文书速查

### 10.1 民事反诉状
- 格式同起诉状
- 在一审辩论终结前提出
- 标题写"民事反诉状"

### 10.2 管辖权异议申请
- 在提交答辩状期间提出
- 写明理由和应管辖法院
- 法院审查后作出裁定（可上诉）

### 10.3 申请回避
- 口头或书面
- 说明回避理由（与案件有利害关系/当事人近亲属等）
- 院长决定（审判人员）或审判长决定（其他人员）

### 10.4 举证期限延长申请
- 书面申请
- 说明客观原因
- 在举证期限届满前提出
- 法院审查批准后延长

## 十一、文书送达方式建议

| 目的 | 推荐方式 | 保留凭证 |
|------|----------|----------|
| 正式通知/催告 | **EMS**（写清楚文件名称） | 快递单+签收回执 |
| 解除通知 | **EMS + 当面送达** | 快递单+录像 |
| 日常沟通确认 | 微信/邮件 | 截图+录屏 |
| 需中断时效 | **EMS** / 律师函 | 快递单+签收回执+律师函存档 |

> **EMS是法律文书送达的黄金标准**——法院认可度最高，回执可作证据。
