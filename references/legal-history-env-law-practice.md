### 1.2 先秦法制（夏商周+春秋战国）🔑

#### 1.2.1 夏商西周的礼与刑

**礼刑关系**：

```
┌─────────────────────────────────────┐
│          "出礼入刑"                  │
│                                      │
│  礼 = 正面规范（道德+礼仪+制度）      │
│      ↓ 违反                         │
│  刑 = 负面制裁（刑罚惩罚）           │
│                                      │
│  礼不下庶人，刑不上大夫               │
└─────────────────────────────────────┘
```

**西周主要刑罚**：

| 刑罚类型 | 内容 | 说明 |
|----------|------|------|
| **五刑** | 墨(黥)、劓(割鼻)、刖(断足)、宫(破坏生殖器)、大辟(死刑) | 奴隶制五刑（肉刑为主）|
| **圜土之制** | 劳役拘禁 | 类似现代的徒刑 |
| **嘉石之制** | 强制劳役并羞辱 | 较轻处罚 |

#### 1.2.2 西周婚姻家庭制度 🔑

| 要素 | 内容 |
|------|------|
| **婚姻原则** | 一夫一妻多妾制；同姓不婚 |
| **结婚程序（六礼）** | **纳采→问名→纳吉→纳征→请期→亲迎** 🔑必背 |
| **七出（休妻理由）** | 无子、淫佚、不事舅姑、多言、盗窃、妒忌、恶疾 |
| **三不去（限制休妻）** | 有所取无所归（无家可归）/与更三年丧（守孝三年）/前贫贱后富贵 |

> 📌 **六礼口诀**：纳采问名纳吉礼，纳征请期迎亲娶

#### 1.2.3 春秋战国成文法运动

| 国家 | 人物 | 措施 | 意义 |
|------|------|------|------|
| **郑国** | 子产（子产） | **铸刑书于鼎**（前536年） | 中国历史上**第一次正式公布**成文法 |
| **郑国** | 邓析 | **作《竹刑》**（私撰） | 私家法学开端 |
| **晋国** | 赵鞅/荀寅 | **铸刑鼎**（前513年） | 第二次公布成文法 |

**李悝《法经》** 🔑（战国魏国，中国历史上**第一部比较系统的封建成文法典**）：

```
《法经》六篇结构：
├── 盗法（防盗/保护财产）     ← 最重要！
├── 贼法（防贼/保护人身/政权）
├── 网法（囚禁/审判，即"囚法"）
├── 捕法（逮捕/追捕罪犯）
├── 杂法（其他犯罪：淫/借/逾制）
└── 具法（加减刑罚的一般规定，即总则）← 放在最后！
```

**特点**：①以刑法为主 ②盗贼为核心（维护财产和统治秩序）③具法为总则但放在末尾

### 1.3 秦汉法制

#### 1.3.1 秦朝法制特色

| 方面 | 内容 |
|------|------|
| **立法思想** | **"缘法而治""法令由一统"**（法家思想全面贯彻）|
| **法律形式** | **律**（主干）、令（皇帝诏令）、式（格式）、法律答问（官方解释）、廷行事（判案范例）、程（章程/规章）|
| **刑罚特点** | **严刑重罚**；笞刑（用竹板打）、肉刑（黥劓斩左趾）、死刑种类繁多（戮/磔/腰斩/车裂/枭首/弃市/族诛等）|
| **司法制度** | 廷尉为中央最高司法官；皇帝掌握最高审判权；**公室告与非公室告**（前者必须受理，后者不予受理即"家罪"）|
| **监察制度** | 御史大夫（中央监察）；**监御史**（地方派驻监察）|

#### 1.3.2 汉朝法制发展 🔑

**立法思想转变**：

```
秦：专任刑罚（法家极端）→ 汉初：黄老之学（清静无为）→ 武帝后："德主刑辅，礼刑并用"
```

**汉朝法律形式**：

| 形式 | 含义 | 地位 |
|------|------|------|
| **律**（律） | 相对稳定的规范性文件 | 主干 |
| **令**（令） | 皇帝发布的诏令 | 数量最多、效力最高 |
| **科**（科） | 判事科条（针对具体事项的规定）| — |
| **比**（比） | 可以比照适用的判例 | — |

**主要法典**：

| 法典 | 制定者 | 内容 |
|------|--------|------|
| **《九章律》** | 萧何 | 在李悝《法经》六篇基础上增加户（户籍/赋税）、兴（征发徭役）、厩（牛马/驿传）三篇 → **九篇** 🔑 |
| **《傍章》** | 叔孙通 | 关于朝廷礼仪的法规（18篇）|
| **《越宫律》** | 张汤 | 宫廷警卫法规（27篇）|
| **《朝律》** | 赵禹 | 朝贺制度的法规（6篇）|

**文景废肉刑** 🔑（中国刑罚史上**重大转折**）：

| 皇帝 | 废除/改革内容 | 替代方式 |
|------|---------------|---------|
| **文帝** | 废除**黥**（墨刑）→ **髡钳（剃发+颈戴铁圈）为城旦舂**（服劳役）；废除**劓**（割鼻）→ **笞三百**；废除**斩左趾** → **笞五百** | 以笞刑替代部分肉刑 |
| **景帝** | 进一步减轻笞刑：笞三百→**笞一百**；笞五百→**笞二百**；制定**《箠令》**（规定笞具规格/行刑部位/中途不得换人）| 笞刑规范化 |

> 📌 **意义**：标志着奴隶制五刑向封建制五刑（笞杖徒流死）过渡的开始。

**春秋决狱** 🔑（董仲舒提倡，西汉武帝后盛行）：

| 要点 | 内容 |
|------|------|
| **含义** | 引用《春秋》等儒家经典的精神/原则来审理案件 |
| **核心原则** | **"原心定罪"**——根据行为人的主观动机（而非仅看客观结果）来定罪量刑 |
| **影响** | ①推动儒法合流 ②开启了引经注律的传统 ③但也导致擅断/随意性 |

**秋冬行刑** 🔑：
- **理论依据**：天人感应（春夏万物生长不宜杀戮/秋冬肃杀之气适合行刑）
- **实际规则**：一般案件在**秋冬季执行死刑**；**春季一般不行死刑**
- **例外**：**"立春后不决"**——立春之后不执行死刑

### 1.4 三国两晋南北朝法制（律学鼎盛期）🔑

这一时期是中华法系从初创走向成熟的关键过渡阶段。

#### 1.4.1 重要法典

| 朝代 | 法典 | 特点/贡献 |
|------|------|-----------|
| **曹魏** | **《新律》/《魏律》**（18篇） | ①**将"具律"改为"刑名"，移到篇首**（首次确立总则在前的体例🔑）；②八议正式入律；③区分五种刑名 |
| **晋朝** | **《晋律》/《泰始律》**（20篇）+ **《晋律注》**（张斐/杜预注释） | ①**"准五服以制罪"**（按亲属远近确定量刑轻重）🔑；②**"峻礼教之防，准五服以制罪"**；③**张斐/杜预注律具有同等法律效力**（律注并行）|
| **北魏** | **《北魏律》** | ①将**官当**（以官职抵罪）正式入律；②**存留养亲**制度 |
| **北齐** | **《北齐律》**（12篇） | ①**首创"重罪十条"**（十恶的前身）🔑；②确立**12篇结构**（后被隋唐沿用）；③**将"刑名"拆分为"名例"**（成为后世律典首篇名称）|

#### 1.4.2 本时期关键制度创新 🔑🔑🔑

**八议** 🔑（魏律正式确立，源于《周礼》）：

| 议的对象 | 含义 | 效果 |
|----------|------|------|
| **议亲** | 皇亲国戚 | 死罪需奏请皇帝裁决 |
| **议故** | 皇帝故旧 | — |
| **议贤** | 大德行的人 | — |
| **议能** | 有大才干的人 | — |
| **议功** | 有大功勋的人 | — |
| **议贵** | 三品以上高官/有一品爵位者 | — |
| **议勤** | 特别勤劳的人 | — |
| **议宾** | 前朝皇室后裔 | — |

> 📌 八议者犯死罪 → **奏请皇帝裁决**（一般减免）；流罪以下 → **依律减一等**。
> 
> 但犯**"十恶"**者不得适用八议！

**准五服以制罪** 🔑（晋律首创，影响深远至清代）：

```
五服（按丧服轻重排序）：

    斩衰（最重）→ 服三年 → 子为父母/妻为夫
    齐衰       → 服一年 → 孙为祖父母/夫为妻
    大功       → 服九月  → 为堂兄弟/已婚姑姐妹
    小功       → 服五月  → 为堂侄/外祖父母
    缌麻（最轻）→ 服三月  → 为远亲

核心原则：服制越近（血缘越近）→ 以尊犯卑处罚轻 / 以卑犯尊处罚重
         服制越远（血缘越远）→ 差异逐渐缩小
```

**重罪十条** 🔑（《北齐律》首创，**"十恶"的前身**）：

```
反逆（谋反）  大逆（毁坏宗庙/皇宫）  叛（背叛投敌）
降（投降敌人） 恶逆（殴/谋杀亲属） 不道（残忍杀人/肢解）
不敬（冒犯皇帝） 不孝（不赡养/诅咒父母） 不义（杀本属长官）
内乱（亲属间乱伦）

→ 后演变为唐代"十恶"
```

**其他制度创新**：

| 制度 | 来源 | 内容 |
|------|------|------|
| **官当** | 北魏律 | 以官职爵位抵消刑罚 |
| **存留养亲** | 北魏律 | 犯死罪者如家中老人无人供养，可暂缓执行/免死 |
| **死刑复奏** | 北魏/北魏太武帝 | 死刑须报请皇帝核准（**死刑复核制度起源**）|
| **登闻鼓** | 西晋 | 直诉制度（击鼓鸣冤）|

### 1.5 隋唐法制（中华法系巅峰）🔑🔑🔑

#### 1.5.1 《唐律疏议》——中华法系的代表作

| 项目 | 内容 |
|------|------|
| **全称** | 《永徽律疏》/《唐律疏议》 |
| **制定** | 唐太宗贞观年间制定《唐律》（12篇502条），唐高宗永徽年间命长孙无忌等人作**疏议**（法律解释） |
| **结构** | **12篇**（继承北齐12篇体例）：名例/卫禁/职制/户婚/厩库/擅兴/贼盗/斗讼/诈伪/杂律/捕亡/断狱 |
| **条文数** | **律文502条 + 疏议同具效力** |
| **地位** | **中国现存最早最完整的古代法典**；**中华法系的代表性法典**；东亚各国（日本/朝鲜/越南）仿效蓝本 |
| **特点** | ①"一准乎礼"（完全以礼为指导）②"德礼为政教之本，刑罚为政教之用" ③**"诸断罪皆须具引律令格式正文"**（依法判决原则）|

#### 1.5.2 十恶 🔑🔑🔑（法考绝对高频！）

> 源自《北齐律》"重罪十条"，隋《开皇律》正式定为"十恶"。犯十恶者：**不得适用八议/上请/减/赎/官当**；**遇常赦不原**（大赦也不赦免）；**处刑极重**。

| 序号 | 罪名 | 含义 | 对应现代概念 |
|------|------|------|------------|
| 一 | **谋反（谋反大逆）** | 图谋危**害社稷/国家** | 叛国/颠覆国家 |
| 二 | **谋大逆** | 图谋毁坏**宗庙/山陵/宫殿** | 破坏国家象征物 |
| 三 | **谋叛** | 图谋**背叛本国/投靠敌国** | 叛国/投敌 |
| 四 | **恶逆** | 殴/谋杀**缌麻以上亲属**（殴打谋杀近亲长辈）| 杀害/伤害近亲属 |
| 五 | **不道** | **残忍杀害**一家非死罪三人/肢解人/造畜蛊毒/厌魅 | 恶性杀人 |
| 六 | **大不敬** | 冒犯**皇帝尊严**（盗祭神物/不敬/伪造御宝等）| 侵犯皇权 |
| 七 | **不孝** | 控告/咒骂**父母**；居丧期间嫁娶作乐/释服从吉 | 不赡养/虐待父母 |
| 八 | **不睦** | 殴/谋杀/出卖**缌麻以上亲属**（亲属相残）| 杀害近亲/卖亲 |
| 九 | **不义** | 杀**本属府主/刺史/县令/现授业师** | 杀长官/老师 |
| 十 | **内乱** | **亲属之间乱伦** | 性侵/乱伦 |

> 📌 **记忆技巧**：**"反逆叛"三项涉及国家（前三）；"恶不孝不睦内乱"五项涉及家族伦理（四五六七八十）；"不道"是恶性杀人（五）；"不敬"是冒犯皇帝（六）**
>
> 📌 **十恶分类口诀**：**三谋（反大逆叛）+ 两不（不敬不义）+ 恶道不孝不睦内乱**
>
> 📌 更短口诀：**"谋反大逆叛，恶逆不道不敬不孝不睦不义内乱"**（10个词顺序背下来）

#### 1.5.3 唐代刑罚制度 🔑

**封建制五刑** 🔑（取代了奴隶制五刑中的肉刑，自北魏/北齐/北周→隋→唐最终定型）：

| 刑种 | 等级 | 内容 | 备注 |
|------|------|------|------|
| **笞** | 五等（10/20/30/40/50下）| 用小荆条抽打臀部 | 最轻刑 |
| **杖** | 五等（60/70/80/90/100下）| 用大竹板抽打背部/臀腿 | — |
| **徒** | 五等（1/1.5/2/2.5/3年）| 剥夺自由强制劳役 | — |
| **流** | 三等（2000/2500/3000里）| 流放到边远地区并服劳役 | **加重：加役流（流3000里+服役3年）** |
| **死** | 两等（绞/斩）| 绞较斩为轻（保留全尸）| 唐代死刑只有这两种！|

> 📌 **对比**：奴隶制五刑（墨劓刖宫大辟）→ **肉刑为主**；封建制五刑（笞杖徒流死）→ **身体刑为主（逐步废除肉刑）**

#### 1.5.4 唐代主要司法制度

| 制度 | 内容 |
|------|------|
| **保辜** | 伤人后在一定期限内（辜限），若被害人死亡则按杀人论；若未死或期限内因他因死亡则仍按伤人论 |
| **类推（举轻以明重）** | 法律无明文规定时，**允许有限度的类推**（"出入罪，举重以明轻；入人罪，举轻以明重"）|
| **自首减免** | 自首者可减免其罪（但**不适用于"谋反"等重罪**）|
| **化外人相犯** | 同国籍外国人适用本国法；不同国籍外国人适用唐法（或依习惯法）|
| **三司推事** | 重大案件由大理寺/刑部/御史台三大司法机关联合审理 |
| **死刑复奏** | 京师案件**五复奏**（三次/一日内）；外地案件**三复奏** |

#### 1.5.5 唐律的主要原则

| 原则 | 内容 |
|------|------|
| **贵族官僚特权** | **八议/请/减/赎/官当/免官**——完整的等级特权体系 |
| **良贱同罪异罚** | 良民（平民）与贱民（奴婢/部曲/官户）同罪不同罚 |
| **同居相隐** | **亲属之间可以隐匿犯罪而不追究**（但不适用于谋反/谋大逆/谋叛等危害国家的犯罪）|
| **罪刑法定倾向** | "诸断罪皆须**具引律令格式正文**，违者笞三十"（依法判决原则，但有类推例外）|

### 1.6 宋元明清法制

#### 1.6.1 宋朝法制特色

| 项目 | 内容 |
|------|------|
| **法典** | **《宋刑统》**（962年）— 中国历史上**第一部刊印颁行**的法典 🔑；体例上沿袭唐律但分213门 |
| **编敕** | 将皇帝发布的"敕"整理编纂，**敕的效力高于律**（宋代一大特色！）|
| **重法地法** | 在特定地区（京城/诸路州军等）**从重惩处强盗犯罪** |
| **折杖法** | 太祖建隆四年创立——用**杖刑**代替原来的**笞/徒/流**刑（减轻刑罚力度）|
| **凌迟** | 作为**正式死刑**列入法典（仁宗时用于惩治"凶恶犯"）— 极其残酷的死刑方式 |
| **翻异别勘** | **被告人在录问/行刑时推翻原口供（"翻异"）→ 交另一司法机构重新审理（"别勘"）** 🔑（防止冤假错案的机制）|
| **审刑院** | 太祖时设立（中央复核机构），神宗时撤销（权力回归刑部）|
| **提点刑狱司** | 中央派往各路的**司法监察机构**（监督地方司法）|

#### 1.6.2 元朝法制特色

| 项目 | 内容 |
|------|------|
| **指导思想** | **"附会汉法"/"参照唐宋"** + 保留蒙古传统 |
| **法律形式** | 条格（皇帝颁布的政令）/断例（判案先例）→ **"条格"地位最高** |
| **民族歧视** | **四等人民**：蒙古人（一等）→ 色目人（二等）→ 汉人（三等）→ 南人（四等）；**同罪不同罚** |
| **婚姻制度** | 允许**收继婚**（父死子娶庶母/兄死弟娶嫂）——蒙古风俗入律 |
| **《大元通制》** | 元英宗时期编制，元朝最重要的法典（共2000多条）|

#### 1.6.3 明朝法制特色 🔑

| 项目 | 内容 |
|------|------|
| **立法思想** | **"刑乱国用重典"** / **"重典治吏"**（重点惩治贪官污吏）|
| **法典** | **《大明律》**（30篇460条）— 体例改为**以"名例"为首 + 吏/户/礼/兵/刑/工六部为纲**的**六部结构**（创新！改变了以往以罪名/事项分类的方法）|
| **《明大诰》** 🔑 | 朱元璋亲自编纂的**案例汇编 + 法令**（四编236条），**效力高于《大明律》**！作为"法外用刑"的依据，极其严酷 |
| **奸党罪** | 明律首创——惩治**臣下结党营私/排挤陷害**的罪名（加强中央集权）|
| **充军刑** | 强制犯人流放边疆充当军人（明代新增的重刑，仅次于死刑）|
| **廷杖** | 在**殿庭之上杖责大臣**（明代特有，皇帝直接行使刑罚权）|
| **厂卫干预司法** | 锦衣卫/东厂/西厂/内行厂——**特务机构干预司法**（明代司法黑暗的重要标志）|

#### 1.6.4 清朝法制特色

| 项目 | 内容 |
|------|------|
| **法典** | **《大清律例》**（43卷/47卷不等）— 结构同《大明律》（名例+六部）；特点是**律文稳定 + 例（条例）不断增加更新** |
| **例的地位** | **"例"（条例/则例）可以补充/修改甚至废止"律"**；**例的数量远远超过律**（清后期达1800多条）|
| **少数民族立法** | **《蒙古律例》《回疆则例》《理藩院则例》《苗律》** 等——针对少数民族地区的特别法 |
| **文字狱** | 从思想/文字角度定罪（加强文化专制）|
| **秋审制度** 🔑 | 每年秋季对各省上报的死刑案件进行**复审**的制度——**秋审**（审理地方上报的案件）/ **朝审**（审理京师案件）|
| **死刑类型** | **斩立绞**（立即执行）/ **斩监绞**（缓期执行待秋审复核后决定是否执行）|

> 📌 **秋审结果四种处理**：🔑
> 1. **情实**（罪行属实/执行死刑）
> 2. **缓决**（缓期执行，下次再审）
> 3. **可矜**（案情虽属实但情有可原/免死减刑）
> 4. **留养承祀**（符合条件者可留养亲人/免除死刑）

### 1.7 清末修律（1840-1911）🔑🔑

这是中国法律**从传统走向近代的关键转折点**。

#### 1.7.1 修律背景与指导原则

| 项目 | 内容 |
|------|------|
| **背景** | 鸦片战争后列强入侵；**领事裁判权问题**（外国人不受中国法院管辖）刺激了修律需求 |
| **主持者** | **沈家本**（修订法律大臣）/ **伍廷芳**（留学英国的法学家）|
| **指导思想** | **"参考古今，博稽中外"** / **"中外通行"** |
| **基本方针** | **"折衷世界大同之良规，而仍不戾乎我国历世相传之礼教民情"**（既要借鉴西方又不违背中国礼教）|

#### 1.7.2 主要成果

| 法典/法律 | 制定年份 | 特点/意义 |
|-----------|---------|-----------|
| **《大清新刑律》** 🔑 | 1910年 | 中国历史上**第一部近代意义上的专门刑法典**；采用**罪刑法定原则**；废除"十恶"/"八议"；引入**缓刑/假释/正当防卫**等现代制度；但保留了**"暂行章程"5条**（维护礼教的妥协产物）|
| **《大清民律草案》** | 1911年 | 未及颁布清朝就灭亡了；**中国第一部民法典草案**；前3编（总则/债权/物权）委托日本人松冈正义起草（大陆法系风格）；后2编（亲属/继承）由中国修订法律馆/礼学馆起草（保留中国传统）|
| **《商人通例》《公司律》** | 1903-1904年 | 中国最早的商事立法 |
| **《各级审判厅试办章程》** | 1906年 | 规定了四级三审制等诉讼制度 |
| **《法院编制法》** | 1910年 | 确立了四级三审制的法院组织体系 |

#### 1.7.3 司法制度改革 🔑

| 改革 | 内容 |
|------|------|
| **司法独立原则** | 确立行政与司法分立的原则 |
| **四级三审制** | **乡谳局/地方审判厅 → 高等审判厅 → 大理院**（三级审判）+ 各级对应的检察厅 |
| **检察制度** | 引进西方检察制度（各级设检察厅，行使公诉/侦查等职权）|
| **律师制度** | 初步确立了律师制度（《律师暂行章程》）|
| **领事裁判权** | 通过修律试图收回（实际未能实现）|

#### 1.7.4 "礼法之争" 🔑🔑

```
"礼法之争"（1907-1910年修律过程中的重大争论）：

┌─────────────────┐        ┌─────────────────┐
│   礼教派（法理派）│        │   法理派（法典派）│
│                 │        │                 │
│ 代表：张之洞/劳乃 │   VS   │ 代表：沈家本/杨度 │
│ 宣              │        │                 │
│                 │        │                 │
│ 主张：          │        │ 主张：          │
│ • 保存礼教      │        │ • 全面引进西方法律│
│ • 维持纲常名教  │        │ • 罪刑法定      │
│ • 反对新刑律中  │        │ • 废除"十恶"    │
│   的某些条款    │        │ • 男女平等      │
│                 │        │ • 引进现代制度  │
│                 │        │                 │
│ 结果：妥协      │        │                 │
│ • 《大清新刑律》│ ←──→   │ 结果：《大清新刑律》│
│   后附5条"暂行  │        │   正文通过但附加  │
│   章程"        │        │   5条"暂行章程"  │
└─────────────────┘        └─────────────────┘
```

> 📌 **实质**：中国传统文化（礼教/纲常）与西方近代法律文明（自由/平等/罪刑法定）之间的冲突。最终结果是**妥协——《大清新刑律》正文采用了近代原则，但又附上了维护礼教的"暂行章程"5条**。

## 二、第二部分：环境资源法（Environmental Law，~7分）

### 2.1 环境保护法的基本原则 🔑

| 原则 | 含义 | 法律依据 |
|------|------|----------|
| **保护优先、预防为主、综合治理** | 把环境保护放在优先位置；强调预防优于治理；多种手段综合运用 | 环保法第5条 |
| **公众参与原则** | 公众有权参与环境保护决策和监督（信息公开+公众参与决策）| 环保法第5条 |
| **损害担责原则** | 造成环境污染/生态破坏的主体应当承担责任（民事赔偿/行政责任/刑事责任）| 环保法第5条 |
| **协调发展原则** | 经济发展与环境保护相协调（不能牺牲环境换取经济发展）| 环保法第4条 |

---

### 2.2 环境保护基本制度 🔑

#### 2.2.1 环境规划制度

| 要点 | 内容 |
|------|------|
| **主体** | 国务院环保主管部门会同有关部门编制 |
| **内容** | 国家环境保护规划 + 区域/流域/海域环境保护规划 |
| **效力** | 纳入**国民经济和社会发展规划** |

#### 2.2.2 环境标准制度

| 类型 | 含义 | 举例 |
|------|------|------|
| **环境质量标准** | 为保障人体健康/维持生态良性循环而规定的污染物容许量/环境质量要求 | 空气质量标准/地表水环境质量标准 |
| **污染物排放标准** | 对污染源排放污染物的浓度/总量作出的限制规定 | 大气污染物排放标准/水污染物排放标准 |

> 📌 **关键规则**：**环境质量标准和污染物排放标准均为强制性标准**，**必须执行**！
> 
> 📌 **地方标准 vs 国家标准**：**地方污染物排放标准可以严于国家标准**（但不能宽于国家标准）。有地方标准的**优先适用地方标准**。

#### 2.2.3 环境监测制度

| 要点 | 内容 |
|------|------|
| **监测主体** | 国务院环保主管部门统一规划/建立监测网络 |
| **信息共享** | 建立监测数据共享机制 |
| **公开义务** | **监测机构应当使用符合国家标准的监测设备**；监测数据应当**依法公开** |

#### 2.2.4 环境影响评价制度（环评）🔑

| 要点 | 内容 |
|------|------|
| **适用范围** | 编制有关开发利用规划的/建设对环境有影响的项目 |
| **分类管理** | ①可能造成重大环境影响 → 编制**环境影响报告书**（全面评价）<br>②可能造成轻度环境影响 → 编制**环境影响报告表**<br>③影响很小 → 填报**环境影响登记表** |
| **时间要求 | 应当在**开工建设前**完成环评审批/备案 |
| **效力** | 未依法进行环评的建设项目**不得开工建设** |

#### 2.2.5 "三同时"制度 🔑

| 要点 | 内容 |
|------|------|
| **含义** | 建设项目中防治污染的设施，必须与**主体工程同时设计、同时施工、同时投产使用** |
| **适用范围** | 新建/改建/扩建项目/技术改造项目 |
| **验收** | 防治污染的设施**经验收合格后方可投入生产或使用** |
| **违反后果** | 验收不合格/未经验收擅自投产 → 责令停止生产/使用，可罚款 |

#### 2.2.6 排污许可制度

| 要点 | content |
|------|---------|
| **含义** | 排污单位向生态环境主管部门申请排污许可证，**持证排污、按证排污** |
| **适用范围** | 依照法律规定实行排污许可管理的企事业单位和其他生产经营者 |
| **内容** | 许可证载明：**排污单位名称/地址/法定代表人/主要污染物种类/排放浓度/排放量/排放去向/排放方式**等 |
| **期限** | 有效期一般为**5年** |
| **违规后果** | **无证排污** → 责令改正/限制生产/停产整治/罚款（10万-100万元）；拒不改正的**按日连续处罚** |

#### 2.2.7 总量控制制度

| 要点 | 内容 |
|------|------|
| **对象** | 重点污染物（根据国家确定的重点污染物排放总量控制指标）|
| **措施** | 分解落实到**企业事业单位**；**对超过排放总量控制指标的地区**暂停审批新增该污染物排放量的环评文件 |

#### 2.2.8 生态保护补偿制度

| 要点 | 内容 |
|------|------|
| **含义** | 对从事生态保护/恢复活动的主体给予经济补偿 |
| **形式** | 财政转移支付/生态补偿基金/市场交易等方式 |

### 2.3 环境法律责任 🔑🔑

#### 2.3.1 行政责任

| 责任类型 | 内容 | 适用情形 |
|----------|------|----------|
| **警告** | 申诫罚 | 较轻微违法 |
| **罚款** | 财产罚 | 最常用的行政处罚 |
| **责令改正/限期治理** | 行为罚 | 可以并处罚款 |
| **责令停业/关闭** | 能力罚 | 严重违法行为 |
| **行政拘留** | 人身罚 | 尚不够刑事处罚的严重违法（由公安机关决定）|
| **按日连续处罚** 🔑 | **拒不改正的** → 原处罚数额**按日连续计算**（上不封顶！）| 经责令改正但拒不改正的 |

> 📌 **按日连续处罚** 🔑（法考超级重点）：2014年修订《环境保护法》引入的**重磅武器**——企业拒不改正违法排污行为的，自**责令改正之日的次日起**，按照原处罚数额**按日连续处罚**。这意味着罚款可以是**天文数字**！

#### 2.3.2 民事责任 🔑

**环境污染侵权责任的特殊规则** 🔑：

| 规则 | 内容 | 法律依据 |
|------|------|----------|
| **无过错责任原则** 🔑 | **无论排污者有无过错**，只要造成了污染损害就应承担侵权责任 | 民法典侵权责任编第1229条 / 环保法第64条 |
| **举证责任倒置** 🔑 | **受害人只需证明**排污行为+损害+因果关系初步证据；**排污者应当证明**其行为与损害**不存在因果关系**（以及存在免责事由）| 民法典第1230条 |
| **共同侵权** | **二人以上分别实施污染行为造成同一损害** → 各自承担**相应**的责任（按份责任）；如果**难以区分责任大小** → **平均分担** | 民法典第1231/1171条 |
| **第三人过错** | 因第三人过错造成污染的 → 受害人可向**排污者或第三人请求赔偿**；排污者赔偿后可向第三人追偿 | 民法典第1233条 |
| **诉讼时效** | **3年**（自当事人知道/应当知道受到损害之日起算）— 注意：比普通民事诉讼时效长！（普通是3年，但环境污染特殊在于**从"知道受损害"起算，且最长不超过20年**）| 民法典第188条 / 环保法第66条 |

**环境公益诉讼** 🔑：

| 要点 | 内容 |
|------|------|
| **原告资格** | ①**依法在设区的市级以上政府民政部门登记的社会组织**（专门从事环境保护公益活动连续5年以上且无违法记录）；②**检察机关** |
| **管辖法院** | **污染行为发生地/损害结果地/被告住所地**的中级人民法院 |
| **前置程序** | 法律规定的机关/社会组织 → **可以直接起诉**（无需前置程序）|
| **禁止重复起诉** | 已经提起公益诉讼的，其他机关/社会组织**不得就同一污染行为再起诉** |
| **和解/调解** | 可以达成调解协议/自行和解，但**应当公告** |

#### 2.3.3 刑事责任（环境犯罪）

| 罪名 | 构成要件 | 刑罚幅度 |
|------|----------|----------|
| **污染环境罪**（刑法第338条）🔑 | 违反国家规定/排放/倾置/处置有放射性的废物/含传染病病原体的废物/有毒物质/其他有害物质，**严重污染环境的** | 一般：3年以下有期徒刑或拘役+并处/单处罚金<br>后果特别严重的：**3-7年**有期徒刑+处罚金 |
| **非法进口固体废物罪** | 将境外固体废物进境倾倒/堆放/处置的 | 5年以下有期徒刑+处罚金<br>造成重大环境污染事故的：5年以上有期徒刑 |
| **擅自进口固体废物罪** | 未经许可擅自进口固体废物用作原料 | 造成重大环境污染事故的：5年以下或5-10年 |

> 📌 **污染环境罪要点** 🔑：
> - **行为方式**：排放/倾倒/处置（三种行为方式）
> - **对象**：放射性废物/含传染病病原体废物/有毒物质/其他有害物质
> - **结果要件**：**"严重污染环境"**（司法解释明确了14种认定标准，如：致使公私财产损失30万元以上/致使30人以上中毒等）
> - **主观方面**：**既可以是故意也可以是过失**

### 2.4 其他重要环境法律速查

#### 2.4.1 大气污染防治法

| 项目 | 内容 |
|------|------|
| **监管体制** | **统一监督管理 + 分部门负责**（生态环境部统筹，各部门分工）|
| **重点区域联防联控** | **重点区域**实行**区域统一规划/统一标准/统一监测/统一的防治措施** |
| **燃煤控制** | 推广清洁能源/淘汰落后燃煤设施 |
| **机动车船污染防治** | 新车/新船必须符合**排放标准**；禁止生产/销售/进口**不符合标准**的车船用燃料 |
| **重污染天气应对** | 建立**预警机制**；采取**应急措施**（限制行驶/停产/停工等）|

#### 2.4.2 水污染防治法

| 项目 | 内容 |
|------|------|
| **基本原则** | **预防为主/防治结合/综合治理** |
| **排放控制** | 直接/间接向水体排放污染物的 → **必须取得排污许可证** |
| **饮用水水源保护** | **一级保护区/二级保护区/准保护区**的分级保护制度 |
| **农业/农村水污染防治** | 推广科学施肥/农药/畜禽养殖废水处理 |
| **地下水污染防治** | 禁止利用渗井/渗坑/裂隙/溶洞**排放/倾倒含有毒污染物的废水** |

#### 2.4.3 固体废物污染环境防治法

| 项目 | 内容 |
|------|------|
| **"三化"原则** | **减量化/资源化/无害化** |
| **分类管理** | **工业固体废物** / **生活垃圾** / **建筑垃圾** / **农业/农村固体废物** / **危险废物** 分类管理 |
| **全过程管理** | 产生→收集→贮存→运输→利用/处置**全过程**控制 |
| **危险废物** 🔑 | ①**经营许可证制度**（收集/贮存/运输/利用/处置危险废物的必须取得许可证）；②**转移联单制度**；③禁止**混合收集/贮存/运输/处置**性质不相容的危险废物 |
| **生产者责任延伸** | 生产者应当承担产品废弃后的回收/处置责任 |

#### 2.4.4 土壤污染防治法（2018年通过，较新的法律）

| 项目 | 内容 |
|------|------|
| **风险管控/修复制度** | 建立土壤污染**风险管控标准**和**修复标准** |
| **土壤污染责任人** | **土壤污染责任人/土地使用权人**负有风险管控/修复义务 |
| **农用地分类管理** | **优先保护类/安全利用类/严格管控类**三类农用地分类管理 |
| **建设用地准入管理** | **土壤污染状况调查** → **风险评估** → **风险管控/修复** → **效果评估** → **方可开发利用** |

#### 2.4.5 海洋环境保护法 / 噪声污染防治法 / 清洁生产促进法

| 法律 | 核心内容 |
|------|----------|
| **海洋环境保护法** | 海洋生态保护/海洋倾废管制/海洋工程/海岸工程/船舶污染防治 |
| **噪声污染防治法** | 工业/交通/建筑施工/社会生活**四类噪声**的分类管理；**夜间施工噪声限制** |
| **清洁生产促进法** | 鼓励企业采用**清洁的能源/原材料/工艺/设备**；**强制性清洁生产审核**制度 |

### 2.5 环境法与其他部门法的交叉 🔑

| 场景 | 涉及法律 | 关键点 |
|------|----------|--------|
| 环境污染导致人身损害 | 环境法 + 民法典（侵权责任编）| **无过错责任 + 举证责任倒置** |
| 企业非法排污 | 环保法 + 行政法 + 刑法 | 可能同时承担**行政责任（罚款/按日计罚）+ 民事赔偿责任 + 刑事责任（污染环境罪）**——**三责同追** |
| 环境公益诉讼 | 环保法 + 民事诉讼法 | **社会组织/检察机关**作为原告 |
| 建设项目环评 | 环保法 + 行政许可法 | 环评属于**行政许可**范畴 |
| 跨区域污染纠纷 | 环保法 + 国际法 | 跨界污染可能涉及国际环境法（如酸雨/海洋污染）|

## 四、两大咨询速答模板

### 模板一：历史法制制度溯源咨询

```
📋 问题定性
这是一个关于中国古代法制制度的问题，属于【中国法制史】学科。
需要判断涉及哪个/哪些朝代，具体是什么制度。

⚖️ 分析步骤
1. 定位朝代（先秦/秦汉/三国两晋南北朝/隋唐/宋元明清/近代）
2. 识别制度类型（立法/刑罚/司法/行政/婚姻家庭/经济）
3. 梳理演变脉络：
   - 该制度何时产生？
   - 历朝历代如何发展变化？
   - 有何标志性事件/法典？
4. 对比异同（如有跨朝代比较需求）

📌 结论与建议
- 给出准确的制度定义和演变过程
- 标注法考高频考点（🔑标记）
- 提供记忆口诀/助记方法
⏰ 易混提醒
- 注意相似制度辨析（如：八议vs官当/十恶vs重罪十条/
  春秋决狱vs引经注律/《法经》vs《唐律疏议》）
```

### 模板二：环境污染纠纷咨询

```
📋 问题定性
这是一起环境污染/生态破坏纠纷，涉及【环境资源法】+【民法典（侵权责任编）】+可能的【刑法】。
需要判断：污染类型？损害程度？责任主体？救济途径？

⚖️ 分析步骤
1. 确认污染类型（大气/水/土壤/固废/噪声/辐射等）
2. 判断责任主体（排污者/第三人/多方共同）
3. 适用法律分析：
   - **行政责任层面**：是否取得环评/排污许可证？是否超标排放？
     → 可能面临的行政处罚（罚款/按日计罚/停产整顿等）
   - **民事责任层面**：受害人能否主张赔偿？
     → **无过错责任**（排污者不能以"合规排污"为由抗辩！）
     → **举证责任倒置**（排污者需证明无因果关系的）
     → 诉讼时效3年
   - **刑事责任层面**：是否达到"严重污染环境"标准？
     → 可能构成**污染环境罪**（刑法338条）
4. 公益诉讼可能性：是否有合格的社会组织/检察机关提起？

📌 结论与建议
- 明确各类责任的构成要件和法律依据
- 赔偿范围（人身损害/财产损失/生态修复费用/鉴定评估费等）
- 争议解决途径：协商 → 调解 → 诉讼（包括环境公益诉讼）
⚠️ 特别提示
- **"合规排放"不是免除民事赔偿责任的理由**！（无过错责任原则）
- **行政机关罚款不影响民事赔偿**（三责同追）
- 按日连续处罚可能导致天价罚款——企业务必重视整改
```
