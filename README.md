# law爷 — AI 法律顾问 Skill

精通法律体系的 AI 法律顾问技能包（WorkBuddy / OpenClaw 兼容）。

## ✨ v1.5.0 新特性：劳动者保护 + 公司经营 + 个体户 三大实务模块

从"知道法条"到"能解决实际问题"——新增三个高频实务场景模块：

### 👷 劳动者权益保护实务

- **劳动仲裁全流程**：四阶段七步法，关键时限汇总
- **经济补偿完整攻略**：N/N+1/2N 全情形一览，含0赔偿的6种情形
- **工伤认定与赔偿**：七种法定情形 + 视同工伤 + 2025年赔偿标准（工亡108万）
- **社保维权**：放弃社保约定无效（司法解释二第19条）
- **竞业限制最新规则**：不知密不生效（司法解释二第13-15条）
- **十大常见纠纷处理**：举证责任分配速查
- **司法解释（二）21条**：法释〔2025〕12号，2025年9月1日施行

### 🏢 公司设立与经营实务

- **新公司法十大变化**：2023修订，2024年7月1日施行
- **注册资本5年实缴制**：新设5年内 + 存量2027年前调整
- **三种治理模式**：传统双层/单层审计委员会/极简模式
- **股东权利与保护**：知情权扩大、双重代表诉讼、回购请求权
- **五大退出途径**：股权转让/异议回购/减资/解散/简易注销
- **十大法律风险**：注册资本虚高、一人公司陷阱等

### 🏪 个体工商户法律实务

- **无限责任警示**：个人经营 = 个人全部财产担责
- **登记注册新规**：变更经营者不再需要先注销
- **税务制度**：2026年查账征收趋势，税收优惠政策
- **雇工劳动关系**：必须签合同缴社保，否则违法
- **个体户 vs 公司**：全维度对比表 + 选择决策树
- **转型升级路径**：何时从个体户转公司

## 🔧 安装方法

将本仓库克隆或下载到 `~/.workbuddy/skills/law-ye/` 目录：

```bash
git clone https://github.com/Jarvis-starcloud/law-ye-skill.git ~/.workbuddy/skills/law-ye
```

或者手动下载文件到相应目录。
