## 一、商经法全景地图

### 商法 vs 经济法：两大家族的分野

```
商经法（法考第⑧⑨科，约50分）
│
├── 📗 商法（~30分）—— 组织法+交易法
│   ├── 公司法（262条）⭐ 最大块头
│   ├── 合伙企业法（106条）
│   ├── 企业破产法（136条）
│   └── 票据法/证券法（后续补充）
│
├── 📕 经济法（~20分）—— 市场规制+宏观调控
│   ├── 反垄断法（70条）⭐ 2022大修
│   ├── 反不正当竞争法（33条）← 已在阶段八
│   ├── 消费者权益保护法（63条）
│   ├── 票据法/财税法（后续补充）
│
└── 🔵 跨领域
    ├── 保险法（185条）→ 民商交叉
    └── 劳动与社会保障法 ← 已在早期完成 ✅
```

### 核心区别速查表

| 维度 | 商法 | 经济法 |
|------|------|--------|
| **调整对象** | 平等主体间的商事交易关系 | 国家对市场的干预管理关系 |
| **核心理念** | 意思自治、契约自由 | 国家干预、社会本位 |
| **典型场景** | 开公司、做生意、签合同 | 反垄断、打假、消费者维权、税收 |
| **责任形态** | 以民事责任为主 | 行政责任突出，民事+行政并行 |

## 二、公司法（2023修订版）— 262条

### 2.1 公司基本制度

#### 公司类型

| 类型 | 特点 | 股东人数上限 | 注册资本 | 责任形式 |
|------|------|------------|----------|----------|
| **有限责任公司** | 最常见 | 1-50人 | 认缴制（5年内缴足） | 有限 |
| **股份有限公司** | 可公开发行股份 | 发起人1-200人 | 发行股本实缴 | 有限 |
| **一人有限公司** | 单独股东 | 1人 | 同有限公司 | ⚠️ 可能连带（混同需自证独立） |
| **国有独资公司** | 全民所有 | — | 国资委出资 | 有限 |
| **国家出资公司** | 国有资本控股 | — | 国资代表履行出资人职责 | 有限 |

**关键变化（2023新公司法）**：
- **注册资本5年实缴制**（第47条）：全体股东认缴的出资额应自公司成立起**5年内缴足**
- **注册资本最低限额**：不再设法定最低限额（但金融等行业另有规定）
- **认缴加速到期**（第54条）：公司不能清偿到期债务时，债权人可要求未届期的股东**提前缴纳**

#### 公司设立

**有限责任公司设立条件**（第42-46条）：
1. **股东**：1-50个
2. **章程**：股东共同制定（载明10项事项）
3. **注册资本**：认缴制，5年内缴足
4. **名称**：必须标明"有限责任公司"或"有限公司"
5. **住所**：主要办事机构所在地

**出资方式**（第48条）：
- ✅ 货币、实物、知识产权、土地使用权、**股权**、**债权**
- ❌ 法律行政法规禁止的财产除外
- 非货币财产须评估作价

**发起人/设立时股东责任**（第44、50条）：
- 设立中民事活动后果 → 由成立后的公司承受；未成立的由发起人承担连带
- 设立时股东未足额缴纳出资的 → **其他设立时股东承担连带补缴责任**

#### 公司登记（第29-41条，新增专章！）

| 登记事项 | 内容 |
|----------|------|
| 名称 | 必须符合规定，受保护 |
| 住所 | 主要办事机构所在地 |
| 注册资本 | 全体股东认缴出资额 |
| 经营范围 | 章程规定，需批准的项目先批后登 |
| 法定代表人 | 董事或经理担任 |

**信息公示要求**（第40条）：公司必须通过国家企业信用信息公示系统公示：
1. 股东认缴和实缴出资额、方式、日期
2. 股权变更信息
3. 行政许可信息

### 2.2 公司组织架构

#### 股东会（权力机构）

| 维度 | 有限公司 | 股份公司 |
|------|----------|----------|
| 组成 | 全体股东 | 全体股东 |
| 召开频率 | 定期会议按章程 | **每年一次年会** |
| 临时会议触发 | ①1/10以上表决权股东 ②1/3以上董事 ③监事会 | ①董事不足2/3 ②亏损达股本1/3 ③10%以上股份股东 ④董事会认为 ⑤监事会提议 |
| 召集顺序 | 董事会→监事会→代表1/10表决权的股东 | 董事会→监事会→连续90日持有10%股份的股东 |
| 通知期限 | 章程规定或15日前（另有约定除外） | 年会20日前，临时15日前 |

**决议效力三分类**（第25-28条，2023新增！）：

| 类型 | 条件 | 效果 |
|------|------|------|
| **无效决议** | 内容违反法律、行政法规 | 自始无效 |
| **可撤销决议** | ①程序违法/违章 或②内容违反章程 | 60日内请求撤销（未通知的自知道起60日），**1年内不行使消灭** |
| **不成立决议** | ①未开会作出 ②未表决 ③出席不够数 ④通过不够数 | 请求法院确认不成立 |

**特别决议（2/3以上通过）**：
- 修改章程、增减注册资本
- 公司合并、分立、解散、变更形式

**普通决议（过半数通过）**：其他事项

#### 董事会（执行机构）

| 维度 | 有限公司 | 股份公司 |
|------|----------|----------|
| 成员 | ≥3人（可设职工董事） | — |
| 职工代表要求 | 职工300人以上的**应当有** | — |
| 任期 | ≤3年，可连选连任 | 同左 |
| 会议召开 | 过半数出席方可举行 | 每年至少2次，10日前通知 |
| 表决 | 全体董事过半数 | 全体董事过半数 |
| **审计委员会替代监事会**（第69条，2023新！） | 可以设置，行使监事会职权 | 同左（≥3名成员，过半非兼职独立） |

**规模较小或人数较少的公司**：
- 有限公司：可不设董事会，设**一名董事**
- 股份公司：可不设董事会，设**一名董事**

#### 监事会（监督机构）

| 维度 | 有限公司 | 股份公司 |
|------|----------|----------|
| 成员 | ≥3人（职工代表≥1/3） | ≥3人（职工代表≥1/3） |
| 主席 | 全体监事过半数选举 | 同左（可设副主席） |
| 会议频率 | 至少每年1次 | 每**6个月**至少1次 |
| 简化 | 小型可不设监事会，只设1名监事 | 同左 |
| **免设条件** | 经**全体股东一致同意**也可不设监事 | — |

#### 经理层

| 项目 | 规则 |
|------|------|
| 产生 | 董事会聘任或解聘 |
| 对谁负责 | 董事会 |
| 权限 | 主持生产经营管理工作、组织实施决议、拟订方案等（章程或授权确定） |
| 兼任 | 董事可以兼任经理 |

#### 上市公司特殊规则（第134-141条，2023大幅加强！）

| 制度 | 核心内容 |
|------|----------|
| **重大交易审批** | 一年内购买出售资产或担保超总资产30% → 股东会2/3通过 |
| **独立董事** | 必须设立，具体办法国务院证券监管机构定 |
| **审计委员会前置** | 聘请/解聘会计师事务所、聘任/解聘财务负责人、披露财报 → 审计委员会过半通过 |
| **董事会秘书** | 必须设，负责筹备会议、文件保管、信息披露 |
| **关联董事回避** | 关联董事不得表决，也不得代理；无关联董事<3人的提交股东会 |
| **信息披露** | 必须真实准确完整披露股东、实际控制人信息 |
| **禁止代持** | 禁止违反规定代持上市公司股票 |
| **子公司持股禁令** | 控股子公司不得取得母公司股份（合并质权等例外且不得行权） |

### 2.3 股权与股份

#### 有限责任公司股权转让（第84-90条）

| 场景 | 规则 |
|------|------|
| **内部转让** | 股东之间自由转让 |
| **外部转让** | 书面通知其他股东 → 其他股东在同等条件下有**优先购买权** → 30日不答复视为放弃 |
| **继承** | 自然人死亡后继承人可继承资格（章程另有约定除外） |
| **异议股东回购请求权**（第89条） | ①连续5年盈利但不分红 ②合并分立转让主要财产 ③章程存续期届满修改续存 → 投反对票的股东可请求按合理价格收购 |

**股权转让出资责任**（第88条，2023新！）：
- 未届出资期限的股权 → 受让人承担缴纳义务，转让人**补充责任**
- 出资瑕疵的股权转让 → 转让人+受让人**连带责任**（受让人不知情除外）

#### 股份公司股份转让限制（第160条，2023细化）

| 人员 | 限制 |
|------|------|
| **发起人** | 自上市日起**1年内**不得转让 |
| **董监高** | 就任时申报；每年转让≤所持25%；上市1年内不得转让；离职半年内不得转让 |
| **质权人** | 限制转让期内出质的，质权人**不得行使**质权 |

#### 股份回购（第162条，2023扩展！）

**原则上禁止回购**，例外情形：

| 情形 | 决议机关 | 后续处理 |
|------|----------|----------|
| 减少注册资本 | 股东会 | 10日内注销 |
| 与他司合并 | 股东会 | 6个月内转让或注销 |
| 员工持股计划/股权激励 | 董事会（授权） | 合计≤已发行10%，**3年内转让或注销** |
| 股东对合并分立异议 | — | 6个月内转让或注销 |
| 转换公司债券 | 董事会（授权） | 同上 |
| **上市公司维护价值必需** | 董事会（授权） | 同上；**须公开集中交易** |

**财务资助禁止**（第163条，2023新增！）：
- 不得为他人取得本公司/母公司股份提供赠与/借款/担保（员工持股除外）
- 为公司利益经股东会或董事会决议可提供资助，累计≤已发行股本总额10%
- 董事会决议须经**全体董事2/3以上通过**

### 2.4 董监高义务与责任（第178-193章）

#### 任职资格负面清单（第178条）

❌ 下列情形不得担任董监高：
1. 无/限制民事行为能力
2. 因贪污贿赂侵占挪用破坏市场经济秩序罪被处刑罚，执行期满**不满5年**
3. 破产企业负有个人责任的董事/厂长/经理，破产完结**不满3年**
4. 违法吊销营业执照企业的法定代表人，吊销**不满3年**
5. 个人负债较大到期未清偿被列为失信被执行人

#### 核心义务

| 义务类型 | 核心内容 |
|----------|----------|
| **忠实义务**（180条） | 避免利益冲突，不得利用职权牟取不当利益 |
| **勤勉义务**（180条） | 为公司最大利益尽到**管理者通常应有的合理注意** |
| **禁止行为**（181条） | 侵占财产/挪用资金/以个人名义存储公款/收受贿赂/泄露秘密/违反对忠实义务的其他行为 |
| **关联交易报告**（182条） | 直接/间接与公司订立合同或交易 → 向董事会或股东会报告并决议通过 |
| **商业机会**（183条） | 不得利用职务便利为自己/他人谋取属于公司的商业机会（报告并决议通过或公司不能用除外） |
| **竞业禁止**（184条） | 未经报告和决议通过，不得自营或为他人经营同类业务 |

#### 责任体系

| 责任类型 | 触发条件 | 承担方式 |
|----------|----------|----------|
| **公司赔偿责任** | 违反法律/章程给公司造成损失 | 赔偿公司损失 |
| **收入归入权** | 违反181-184条所得收入 | 归入公司 |
| **股东代位诉讼**（189条） | 董/高违规，监事会拒绝起诉 → 连续180日持有1%以上股份股东可**为公司利益以自己名义直接起诉** |
| **股东直接诉讼**（190条） | 董/高损害**股东利益** | 股东直接起诉 |
| **第三人赔偿责任**（191条） | 故意或重大过失致第三人损害 | 公司+董高连带赔偿 |
| **实际控制人指示责任**（192条） | 实控人指示董/高管从事损害行为 | **实控人与董高连带责任** |
| **责任保险**（193条，2023新！） | 可为董事投保责任保险，投保后向股东会报告 |

### 2.5 公司合并、分立、减资

| 操作 | 债权保护程序 | 债权人权利 |
|------|--------------|-----------|
| **合并**（218-221条） | 决议日起**10日通知**+**30日公告** | 30日内可要求清偿/担保 |
| **分立**（222-223条） | 10日通知+30日公告 | 分立前债务**连带责任**（协议另有约定除外） |
| **减资**（224-225条） | 10日通知+30日公告 | 45日内可要求清偿/担保 |
| **简易合并**（219条，2023新！） | 持股90%以上合并，小股东可要求收购 | 支付价款≤净资产10%可不经股东会 |

**亏损减资**（225条，2023新！）：
- 弥补亏损后仍有亏损 → 可减少注册资本弥补
- 减资弥补亏损期间：**不得分配利润**，**不免除出资义务**
- 法定公积金+任意公积金达注册资本50%前不得分红

### 2.6 公司解散与清算（第229-242条）

#### 解散原因（229条）

1. 章程规定的营业期限届满/解散事由出现
2. 股东会决议解散
3. 合并/分立需要
4. 吊销执照/责令关闭/撤销
5. **司法解散**（231条）：经营管理严重困难，继续存续会使股东利益受到**重大损失**，持有**10%以上表决权**股东可诉请解散

#### 清算程序

| 步骤 | 时间节点 | 关键内容 |
|------|----------|----------|
| 成立清算组 | 解散事由出现**15日内** | **董事为清算义务人**（2023新！） |
| 通知公告 | 成立**10日**内通知，**60日**公告 | 债权人30日/45日内申报债权 |
| 制定方案 | 清理财产后 | 报股东会或法院确认 |
| 清偿顺序 | 按法定顺序 | 见下表 |
| 注销登记 | 清算结束后 | 申请注销 |

**清偿顺序**（236条）：
1. 清算费用
2. 职工工资+社保费用+法定补偿金
3. 所欠税款
4. **剩余财产**：有限公司按**出资比例**，股份公司按**持股比例**

**简易注销**（240条，2023新！）：
- 无债务或已清偿全部债务 → 全体股东承诺 → 公告**20日** → 无异议 → **20日内申请注销**
- 承诺不实的 → 股东**连带责任**

**强制注销**（241条，2023新！）：
- 吊销/责令关闭/撤销满3年未注销 → 公告60日 → 无异议 → 强制注销
- **原股东、清算义务人责任不受影响**

### 2.7 法律责任重点条文（250-264条）

| 违法行为 | 处罚对象 | 处罚力度 |
|----------|----------|----------|
| 虚报注册资本/虚假材料骗取登记 | 公司：5%-15%虚报额罚款；个人：3万-30万 | 情重吊销执照 |
| 虚假出资/未按期交付 | 5万-20万罚款；情节重：虚报/未出资金额5%-15%罚款 | — |
| 抽逃出资 | 抽逃金额5%-15%罚款；董监高连带 | — |
| 另立账簿/虚假财务会计报告 | 依《会计法》处罚 | — |
| 不依法办理变更登记 | 责令限期登记；逾期：1万-10万罚款 | — |
| 冒用公司名义 | 责令改正/取缔，并可**10万元以下**罚款 | — |

**民事赔偿优先原则**（263条）：财产不足以支付时，**先承担民事赔偿责任**。

## 三、合伙企业法（106条）

### 3.1 三类合伙企业对比

| 维度 | **普通合伙(GP)** | **有限合伙(LP)** | **特殊的普通合伙** |
|------|-------------------|-------------------|-------------------|
| **合伙人构成** | 全部是普通合伙人 | GP+LP混合 | 全部GP（专业服务机构） |
| **责任形式** | 全员**无限连带** | LP**有限责任**（认缴出资为限）；GP无限连带 | 有故意/重大过失的GP**无限**；其他GP**以出资额为限** |
| **劳务出资** | ✅ 可以 | LP**不可以** | ✅ 可以 |
| **事务执行** | 全体GP平等权利 | 仅**GP执行事务**，LP不得对外代表 | 同普通合伙 |
| **竞业禁止** | ✅ 竞业绝对禁止 | LP可以竞业（除非约定） | 同普通合伙 |
| **自我交易** | 须经全体一致同意 | LP可以（除非约定） | 须经全体一致 |
| **利润分配** | 不得全部分配给部分合伙人 | 可以约定全部分配给部分 | 同普通合伙 |
| **典型适用** | 一般中小企业 | 私募基金/创投/PE | 律师事务所/会计师事务所 |

### 3.2 普通合伙企业

#### 设立条件（14条）

1. **2个以上**合伙人（自然人须完全民事行为能力）
2. **书面合伙协议**（18条须载明10项事项）
3. 认缴或实缴出资
4. 合伙名称+经营场所
5. 法律行政法规规定的其他条件

**禁止成为GP的主体**（3条）：
- ❌ 国有独资公司
- ❌ 国有企业
- ❌ 上市公司
- ❌ 公益性事业单位/社会团体

#### 合伙财产（20-25条）

| 规则 | 内容 |
|------|------|
| **财产范围** | 出资+以合伙名义取得的收益+依法取得的其他财产 |
| **清算前分割禁止** | 不得请求分割（本法另有规定除外） |
| **私自处分对抗力** | 私自转移/处分 → **不得对抗善意第三人** |
| **对外转让** | 须经**全体一致同意**；其他合伙人有**优先购买权** |
| **份额出质** | 须经**全体一致同意**，否则无效+赔偿 |

#### 事务执行（26-36条）

| 要点 | 规则 |
|------|------|
| **执行权平等** | 所有GP享有同等权利 |
| **委托执行** | 可委托1或数个GP对外代表，其他人监督 |
| **表决机制** | 约定优先 → 约定不明 → **一人一票+全体过半数** |
| **一致同意事项**（31条） | 改名/改范围改地址/处分不动产/处分IP/担保/聘任外人做管理人员 |
| **竞业绝对禁止** | 未经许可不得自营或合作经营与本合伙竞争的业务 |
| **利润/亏损分担** | 约定 → 协商 → 按**实缴出资比例** → 平均 |
| **禁止条款** | 不得约定将**全部利润**给部分人或**全部亏损**由部分人担 |

#### 入伙退伙

| 类型 | 条件 | 责任 |
|------|------|------|
| **入伙**（43-44条） | 除约定外须**全体一致同意**+书面协议；原GP如实告知 | 新合伙人对**入伙前债务**承担**无限连带** |
| **自愿退伙**（45-47条） | 约定期限：退伙事由出现/全体同意/难参加/对方违约 | 应提前**30日通知**（无约定期限）；违规退伙赔偿损失 |
| **当然退伙**（48条） | 死亡/宣告死亡、丧失偿债能力、被吊销执照/宣告破产、丧失必要资格、全部份额被执行 | 丧失行为能力的 → 经一致同意可转LP；否则退伙 |
| **除名退伙**（49条） | 未出资/故意或重大过失造成损失/执行事务不正当/约定事由 | **其他合伙人一致同意**决议，**书面通知**生效，30日内可诉 |
| **继承人权益**（50条） | 继承人可取得资格（同意）；或退还份额 | 无行为能力继承人 → 一致同意可转为LP |
| **退伙结算**（51-54条） | 按当时财产状况结算，退还份额（现金或实物） | 退伙人对**退伙前原因发生之债**承担**无限连带**；财产少于债务时分担亏损 |

#### 特殊的普通合伙（55-59条）

**适用对象**：以专业知识和专门技能为客户提供**有偿服务**的专业服务机构（律所、会计师事务所等）

**责任切割**（57条）：
- 一个/数个GP因**故意或重大过失**造成债务 → 该GP**无限/连带**
- 非因故意/过失造成的债务及其他债务 → **全体GP无限连带**
-有过错GP对合伙企业承担**赔偿责任**

**风险防控**（59条）：必须建立**执业风险基金**+办理**职业保险**

### 3.3 有限合伙企业（60-84条）

#### 核心规则速查

| 项目 | 规则 |
|------|------|
| 人数 | **2-50人**（至少**1个GP**） |
| 名称 | 必须标明**"有限合伙"** |
| LP出资 | 货币/实物/IP/土地/财产权利均可，**不得劳务出资** |
| 事务执行 | **仅GP执行**，LP**不得对外代表** |
| LP的"安全行为"（68条） | 参与GP入退伙决策、提出经营建议、参与选审计师、获取财务报告、查阅账簿、主张权利督促、起诉/应诉、为本企业提供担保 |
| 利润分配 | 不得将**全部利润**分配给部分LP（约定除外） |
| LP的交易自由 | 可与本企业交易（除非约定）、可竞业（除非约定）、可出质（除非约定）、可对外转让份额（**提前30日通知**） |
| 身份转换 | GP↔LP转换须经**全体一致同意** |
| 转换责任追溯 | GP转LP → 对GP期间债务仍**无限连带**；LP转GP → 对LP期间债务也**无限连带** |
| **表见普通合伙**（76条） | 第三人有理由相信LP为GP并交易的 → LP对该笔交易**承担GP同样的责任** |
| 退伙 | LP退伙后 → 对退伙前债务以**取回财产为限**承担责任 |

**自动形态变化**（75条）：
- 仅剩LP → 应**解散**
- 仅剩GP → 转**普通合伙**

## 四、企业破产法（136条）

### 4.1 破产三程序全景图

```
债务人不能清偿到期债务
    │
    ├─► 重整（拯救企业，恢复生机）→ 重整计划→ 执行→ 存续
    │     适用：有挽救价值的企业
    │
    ├─► 和解（债务人与债权人协商让步）→ 和解协议→ 履行→ 存续
    │     适用：各方愿意妥协的情形
    │
    └─► 破产清算（公平清偿，退出市场）→ 变价分配→ 注销→ 终结
          适用：无挽救价值/重整失败/和解失败
```

### 4.2 破产申请与受理

#### 申请条件（2条）

| 申请人 | 适用程序 | 条件 |
|--------|----------|------|
| 债务人 | 重整/和解/清算 | 不能清偿到期债务+资不抵债/明显缺乏清偿能力 |
| 债权人 | 重整/清算 | 债务人**不能清偿到期债务**即可 |
| **出资人**（占注册资本**1/10以上**） | **仅重整** | 在受理后至宣告前提出 |

**清算责任人**：已解散未清算或未清算完毕+资不抵债 → 依法负有清算责任的人应申请破产清算

#### 受理时限（10-12条）

| 情形 | 时限 |
|------|------|
| 债权人申请 → 通知债务人 | 收到申请**5日内** |
| 债务人有异议 | 通知之日起**7日内**提出 |
| 异议期满或无异议 | **15日内**裁定是否受理 |
| **特殊情况延长** | 上一级法院批准可再延**15日** |
| 裁定受理 → 送达 | **5日内**送达申请人+债务人 |
| **不服裁定** | **10日内**上诉（一审/二审均同） |

**管理人指定**（13条）：受理破产申请的同时**应当指定管理人**

#### 破产受理的法律效力（16-21条）

| 效力 | 内容 |
|------|------|
| **个别清偿无效**（16条） | 受理后对个别债权人清偿**一律无效** |
| **债务人财产保全**（17条） | 债务人的债务人/持有人向**管理人**清偿/交付 |
| **未履行合同选择权**（18条） | 管理**人决定**解除或继续履行（2个月内未通知=解除；催告30日未答=解除） |
| **保全措施解除**（19条） | 有关财产保全**应当解除** |
| **执行中止**（20条） | 尚未终结的诉讼/仲裁**应当中止**（接管后恢复） |
| **专属管辖**（21条） | 有关债务人的民事诉讼**只能**向受理法院提起 |

#### 有关人员义务（15条）

| 义务 | 内容 |
|------|------|
| 妥善保管财产印章账簿 | — |
| 配合工作+如实回答 | — |
| 列席债权人会议 | — |
| 未经许可**不得离开住所地** | — |
| **不得新任**其他企业的董监高 | — |

### 4.3 管理人（22-29条）

#### 任命与资格

| 项目 | 内容 |
|------|------|
| 任命 | **人民法院指定** |
| 更换 | 债权人会议认为不胜任 → 可申请法院更换 |
| 任职资格（消极） | ①因故意犯罪受过刑责 ②曾被吊销执业证 ③有利害关系 ④其他不宜情形 |
| **个人管理人须投职业责任保险** | — |

#### 九大职责（25条）

1. 接管财产/印章/账簿/文书
2. 调查财产状况+制作报告
3. 决定内部管理事务
4. 决定日常开支和其他必要开支
5. **首次债权人会议前决定继续/停止营业**
6. **管理和处分财产**
7. 代表参加诉讼/仲裁
8. 提议召开债权人会议
9. 法院认为应当履行的其他职责

### 4.4 债务人财产（30-39条）

#### 定义：受理时属债务人的全部财产+受理后至终结前取得的财产

**撤销权（31条）— 受理前1年内**的可撤销行为：
1. 无偿转让财产
2. 以明显不合理价格交易
3. 对没有担保的债务**提供财产担保**
4. 对未到期的债务**提前清偿**
5. 放弃债权

**撤销权（32条）— 受理前6个月内**的个别清偿（受益清偿除外）

**无效行为（33条）**：
1. 为逃避债务隐匿/转移财产
2. 虚构债务/承认不真实债务

**追回权适用（34条）**：上述行为取得的财产均可追回

**追缴出资权（35条）**：出资人尚未缴足的，**不受出资期限限制**

**取回权（38条）**：不属于债务人但被其占有的财产 → 权利人可通过管理人**取回**（出卖人在途货物取回权39条）

**抵销权（40条）— 四种禁止情形**：
1. 受理后**取得**对债务人的债权
2. 债权人**已知**债务人有破产事实而负担的债务（法律规定/破产申请1年前原因除外）
3. 债务人的债务人已知破产而取得债权（同样例外）
4. 以上三种情形的变体

### 4.5 破产费用与共益债务（41-43条）

| 类别 | 内容 | 清偿顺序 |
|------|------|----------|
| **破产费用**（41条） | ①诉讼费 ②管理/变价/分配费用 ③管理人报酬/聘用费用 | **先行清偿** |
| **共益债务**（42条） | ①未履行合同产生的债务 ②无因管理 ③不当得利 ④继续营业劳动报酬/社保等 ⑤管理人履职致人损害 ⑥财产致人损害 | 同上 |

**不足清偿时的顺序**：先清偿破产费用 → 再清偿共益债务 → 按比例清偿 → **不足以清偿费用的，提请终结程序**

### 4.6 债权申报（44-58条）

| 项目 | 规则 |
|------|------|
| **申报期限** | 最短**30日**，最长**3个月**（自公告之日计算） |
| **未到期债权** | 到期 |
| **附利息债权** | 停止计息 |
| 附条件/附期限/未决债权 | **均可申报** |
| **职工债权**（48条） | 工资/医疗伤残补助抚恤/养老保险/医疗保险/补偿金 → **不必申报**，管理人调查列示公示 |
| **保证人/连带债务人**（51条） | 已代偿 → 以求偿权申报；未代偿 → 将来求偿权申报 |
| **连带债务多人破产**（52条） | 各就**全部债权分别**在各案中申报 |
| **未申报后果**（56条） | 不得依照本法行使权利 |
| **补充申报**（56条） | 最后分配前可补充，此前分配不补，审查费自理 |
| **确认程序**（58条） | 债权表提交第一次债权人会议核查 → 无异议→法院裁定确认；有异议→**起诉** |

### 4.7 债权人会议（59-69条）

#### 一般规定

| 项目 | 规则 |
|------|------|
| 成员 | **依法申报债权的债权人** |
| 表决权 | 债权尚未确定→临时确定；担保债权人未放弃优先权的对部分事项无表决权 |
| 主席 | 法院从有表决权债权人中**指定** |
| **首次会议** | 债权申报期限届满**15日内**召开 |
| **后续会议** | 法院必要时或管理人/债权人委员会/1/4以上债权总额债权人提议时召开 |
| 提前通知 | **15日** |
| **决议通过标准** | 出席会议有表决权债权人**过半数**+所代表债权额占**无担保债权总额1/2以上** |
| **决议约束力** | 对**全体债权人**有效 |
| **决议救济**（64条） | 认为违法/损己利益的 → **15日内**请求法院**撤销** |
| **法院裁定**（65条） | 特定事项表决未通过的 → 法院裁定（1/2以上债权额不服可复议） |

#### 十一大职权（61条）

1. 核查债权
2. 申请更换管理人+审查报酬
3. 监督管理人
4. 选任/更换债权人委员会
5. 决定继续/停止营业
6. **通过重整计划**
7. **通过和解协议**
8. 通过财产管理方案
9. 通过财产变价方案
10. 通过财产分配方案
11. 法院认为应由债权人会议行使的其他职权

#### 债权人委员会（67-69条，可选）

| 项目 | 内容 |
|------|------|
| 组成 | 债权人代表（**≤9人**）+职工/工会代表 |
| 六大职权 | 监督管理和处分、监督分配、提议开会、委托其他职权 |
| **重大行为报告**（69条） | 土地房屋转让、知识产权转让、全部库存/营业转让、借款、设定担保、债权证券转让、未履行合同、放弃权利、担保物取回、其他重大影响 |

### 4.8 重整（70-94条）

#### 申请与期间（70-78条）

| 项目 | 规则 |
|------|------|
| 申请人 | 债务人/债权人（直接申请）/出资人占**1/10以上**（受理后至宣告前申请） |
| **重整期间** | 裁定重整→终止重整 |
| **自行管理**（73条） | 经债务人申请+法院批准 → 管理人**移交**财产和营业 |
| **担保权暂停行使**（75条） | 但担保物损坏/价值明显减少可能危害的 → 请求恢复 |
| **重整借款担保**（75条） | 可为借款设定担保 |
| **出资人**（77条） | 不得请求收益分配；董监高不得转让股权（法院同意除外） |
| **终止重整**（78条） | 经营恶化缺挽救可能/欺诈恶意减少财产/致使无法执行职务 → 法院裁定终止+宣告破产 |

#### 重整计划（79-88条）

| 项目 | 规则 |
|------|------|
| 制定时限 | 裁定重整起**6个月**（可延期**3个月**） |
| 制定主体 | 自行管理→债务人制定；管理人管理→管理人制定 |
| 八大内容 | 经营方案、债权分类、调整方案、受偿方案、执行期限、监督期限、其他方案 |
| **分组表决**（82条） | ①担保债权组 ②职工债权组 ③税款组 ④普通债权组（可设小额债权组） |
| **通过标准**（84条） | 同组**过半数**+债权额**2/3以上** |
| **批准标准**（86条） | 各组均通过 → 10日内申请批准 → 30日内裁定 |
| **强裁条件**（87条，部分组未通过时可申请） | ①担保债权全额清偿 ②职工/税款全额清偿或该组通过 ③普通债权清偿比不低于清算比例 ④出资人调整公平公正 ⑤对待各组公平 ⑥清偿顺序合法 ⑦经营方案可行 |
| **未通过且未强裁**（88条） | 终止重整+宣告破产 |

#### 执行（89-94条）

| 项目 | 规则 |
|------|------|
| 执行人 | **债务人负责执行** |
| **监督期** | 管理人监督执行（可延长） |
| 约束力 | 对债务人和**全体债权人均有约束力** |
| **未申报债权**（92条） | 执行期间不得行使；执行完毕后按同类条件行使 |
| 保证人/连带债务人权利 | 不受影响 |
| **终止执行**（93条） | 不能或不执行 → 终止+宣告破产；已受清偿有效，未清偿部分作为破产债权 |
| **减免债务效力**（94条） | 执行完毕后债务人**不再承担**减免部分的清偿责任 |

### 4.9 和解（95-106条）

| 项目 | 规则 |
|------|------|
| 申请人 | 仅**债务人**（可直接申请或在受理后至宣告前申请） |
| 和解协议草案 | 由债务人提出 |
| 债权人会议通过 | 出席有表决权**过半数**+债权额**2/3以上** |
| 法院认可 | 裁定认可 → 终止和解 → 移交财产和营业 |
| 和解协议效力 | 对**全体和解债权人**有约束力 |
| **和解欺诈无效**（103条） | 欺诈/违法行为成立的和解协议 → **无效**+宣告破产；已受清偿在同等比例内不予返还 |
| **终止和解**（104条） | 不能或不执行 → 终止+宣告破产；已受清偿有效，未清偿部分作为破产债权 |
| **减免效力**（106条） | 执行完毕后债务人**不再承担**减免部分的责任 |

### 4.10 破产清算（107-124条）

#### 破产宣告（107-110条）

| 项目 | 规则 |
|------|------|
| 裁定时限 | 作出裁定**5日内**送达+**10日**通知已知债权人并公告 |
| 宣告效果 | 债务人→**破产人**；财产→**破产财产**；债权→**破产债权** |
| **终结前免除**（108条） | 第三人足额担保/债务人清偿全部债务 → 终结 |
| **别除权**（109条） | 对特定财产享有担保权的权利人 → **优先受偿** |
| 别除权未足受偿 | 未受偿部分作为**普通债权**；放弃优先权的债权也作为普通债权 |

#### 变价与分配（111-119条）

| 项目 | 规则 |
|------|------|
| 变价方式 | 主要通过**拍卖**；债权人会议另有决议的除外 |
| **清偿顺序**（113条，最重要！） | ①破产费用+共益债务 ②**职工工资/医疗伤残补助抚恤/养老保险/医疗保险/补偿金** ③**欠缴社会保险（除前述）+税款** ④**普通破产债权** |
| 分配方式 | 以**货币分配**为主（另有决议除外） |
| 分配方案 | 管理人拟订 → 债权人会议讨论通过 → 法院认可 → **管理人执行** |
| 多次分配 | 每次公告本次分配的财产额和债权额 |
| **附条件/解除条件债权**（117条） | **提存**；最后公告日生效/解除条件成就与否决定归属 |
| **未领分配**（118条） | 提存；**满2个月**不领取 → 分配给其他债权人 |
| **未决债权**（119条） | 提存；**满2年**仍不能受领 → 分配给其他债权人 |

#### 程序终结（120-124条）

| 项目 | 规则 |
|------|------|
| 无财产可分配 | 提请法院**裁定终结** |
| 最后分配完 | 提交报告 → **15日内**裁定终结 |
| 注销登记 | 终结**10日内**办理注销 |
| 管理人终止 | 办理注销完毕次日终止（未决案件除外） |
| **追加分配**（123条） | 终结后**2年内**发现应追回财产或有遗漏财产 → 按分配方案追加（费用不足则上交国库） |
| 保证人/连带债务人责任 | 破产终结后，对未受清偿债权**依法继续承担**清偿责任 |

### 4.11 法律责任（125-131条）

| 行为 | 责任 |
|------|------|
| 董/监/高违反忠勤义务导致破产 | **民事责任**+**3年内**不得担任任何企业的董/监/高 |
| 拒不到庭/不作答/作虚假答 | **拘传**+**罚款** |
| 不提交/提交不真实资料 | 对直接责任人**罚款** |
| 隐匿/伪造证据 | 对直接责任人**罚款** |
| 管理人未尽勤勉尽责 | **罚款**+**赔偿损失** |
| 构成犯罪的 | **追究刑责** |

### 4.12 特别规定（132-136条）

**新旧衔接（132条）**：公布日前所欠职工工资/医疗伤残补助/养老保险/医保/补偿金 → 按113条清偿后不足以清偿的部分 → **优先于担保物权人受偿**（历史遗留问题特殊照顾）

**金融机构破产**（134条）：商业银行/证券公司/保险公司等 → 金融监督管理机构可申请**重整或破产清算**；国务院可制定**实施办法**

**参照适用**（135条）：非法人组织的清算属破产清算的 → **参照适用**

## 五、保险法（185条）

### 5.1 保险基本概念

```
保险 = 投保人付费 → 保险人承担赔偿/给付责任的商业行为
│
├── 人身保险（以人的寿命和身体为标的）
│   ├── 人寿保险（死亡/生存/生死两全）
│   ├── 健康保险
│   └── 意外伤害保险
│
├── 财产保险（以财产及其有关利益为标的）
│   ├── 财产损失保险
│   ├── 责任保险（以被保险人对第三者的赔偿责任为标的）
│   ├── 信用保险
│   └── 保证保险
│
└── 分业经营（95条）
    人身保险业务 ↔ 财产保险业务（一般不得兼营）
    例外：财产险公司经批准可经营短期健康险和意外伤害险
```

### 5.2 保险合同

#### 基本原则（5-9条）

| 原则 | 内容 |
|------|------|
| **最大诚信原则**（5条） | 诚实信用贯穿始终 |
| **合法经营原则**（6条） | 只有依法设立的保险公司及法律规定的组织才能经营 |
| **境内投保原则**（7条） | 境内法人/组织需要境内保险的 → 应当向**境内保险公司**投保 |
| **分业经营原则**（8条） | 保险业与银行业/证券业/信托业**分业经营分业管理**（国家另有规定除外） |
| **监管原则**（9条） | 国务院保险监督管理机构统一监管 |

#### 合同订立（10-14条）

| 要素 | 内容 |
|------|------|
| 当事人 | 投保人（付费方）+ 保险人（承担赔付责任的保险公司） |
| 生效 | 依法成立**即生效**（可附条件附期限） |
| 投保单/凭证 | 成立后及时签发 |
| **保险利益**（12条） | 人身险：**订立时**对被保险人有保险利益；财产险：**事故发生时**对保险标的有保险利益 |

**如实告知义务**（16条）— 核心！

| 情形 | 保险人权利 |
|------|-----------|
| 故意不告知+足以影响承保/费率 | 可解除合同，**不赔不退** |
| 重大过失不告知+严重影响 | 可解除合同，**不赔但退费** |
| 解除权行使期限 | 自知道之日起**超30日不行使消灭** |
| 不可解除 | 自合同成立**超2年**（不可抗辩条款） |
| 保险人已知不告知 | **不得解除**，事故发生**应当赔偿** |
| 格式条款免责说明 | **必须提示+明确说明**，否则该条款**不产生效力**（17条） |

**争议解释规则**（30条）：格式条款有两种以上解释的 → **作出有利于被保险人和受益人的解释**

#### 人身保险合同（31-47条）

**保险利益范围**（31条）：
1. 本人
2. 配偶、子女、父母
3. 有抚养/赡养/扶养关系的家庭成员和近亲属
4. **被保险人同意**投保的人（订立合同时不具有保险利益的 → **合同无效**）

**年龄误报**（32条）：
- 年龄不符限制 → 可解除合同+退还现金价值（适用解除权时效规则）
- 少缴保费 → 补交或多交退还/按比例支付

**死亡保险限制**（33-34条）：
- 不得为无民事行为能力人投保**以死亡为给付条件**的人身保险（父母为其未成年子女投保除外，但赔偿金总和受限）
- 以死亡为给付条件的合同 → 未经被保险人**同意并认可金额** → **无效**（父母为未成年子女除外）

**保费缴纳与宽限期**（35-37条）：
- 首期后逾期：催告**超30日**或约定期限**超60日**未付 → **效力中止**（可扣减欠交保费后赔付）
- 中止后补救：达成协议+补交保费 → 恢复效力
- 中止**超2年**未达成协议 → 保险人有权**解除**（退还现金价值）
- **人寿保险保费不可诉讼追讨**（38条）

**受益人制度**（39-42条）：

| 要点 | 规则 |
|------|------|
| 指定 | 由被保险人或投保人指定（投保人指定的须经**被保险人同意**） |
| 变更 | 被保险人或投保人**书面通知**保险人（变更投保人指定须经**被保险人同意**） |
| 受益人缺失 | ①无指定/不明 ②先于被保险人死亡 ③丧失/放弃受益权 → 保险金**作为遗产** |
| 同时死亡推定 | 同一事件不能确定先后 → **推定受益人死亡在先** |

**自杀条款**（44条）：合同成立/复效**2年内**自杀 → 不赔（无行为能力人除外）+退还现金价值

**故意制造事故**（43条）：投保人故意造成被保险人死亡/伤残/疾病 → **不赔**（交足2年以上保费的退还现金价值）

**受益人故意**（43条）：故意杀害/未遂 → 该受益人**丧失受益权**

**第三者责任**（46条）：被保险人因第三者行为伤亡 → 保险人赔付**不影响**被保险人/受益人对第三者的**请求赔偿权**

**投保人解除权**（47条）：随时可解除 → 保险人**30日内**退还**现金价值**

#### 财产保险合同（48-66条）

| 要点 | 规则 |
|------|------|
| 保险利益时间点 | **保险事故发生时**具有保险利益 |
| 标的转让（49条） | 受让人**承继**权利义务；应及时通知（货物运输保险除外） |
| 危险程度增加（52条） | 应通知 → 保险人可加费或解约（30日内行使） |
| 安全维护义务（51条） | 被保险人应遵守安全规定；保险人可检查并提出建议 |
| 保险价值降低（53条） | 明显降低 → **降低保费**（按日计算退还） |
| 解除权（50条） | 责任开始后**双方不得解除**（运输/航程保险） |
| **超额无效**（55条） | 保险金额 > 保险价值的 → **超过部分无效**，退还相应保费 |
| **不足额比例赔付**（55条） | 保险金额 < 价值的 → 按**保险金额/保险价值比例**承担 |
| **重复保险**（56条） | 应通知各保险人；各保险人赔偿总和**不得超过保险价值**；按**保险金额/总和比例**分担（可要求返还多余保费） |
| **施救费用**（57条） | 必要合理的施救费用由保险人承担（**另行计算，不超过保险金额**） |
| **代位求偿**（60条） | 保险人赔偿后在赔偿范围内**代位行使**对第三者的请求权 |
| 代位权限制（61-63条） | 赔偿前放弃对第三者权利 → **不赔**；赔偿后放弃 → **无效**；故/重大过失导致不能行使 → 可扣减或返还 |
| 家庭成员免责（62条） | 除非**故意**造成保险事故，**不得**对家庭成员/组成人员行使代位权 |
| **责任保险**（65-66条） | 保险人可直接向第三者赔偿；被保险人怠于请求 → 第三者可直接请求；被保险人未赔偿 → 保险人**不得**向被保险人赔偿 |

### 5.3 保险公司（67-94条）

#### 设立条件（68条）

| 条件 | 标准 |
|------|------|
| 主要股东 | 持续盈利、信誉良好、近3年无重大违法违规、净资产**≥2亿元** |
| 章程 | 符合公司法 |
| 注册资本 | 最低**2亿元**（必须**实缴货币资本**） |
| 董监高 | 专业知识和工作经验+任职资格 |
| 组织机构 | 健全 |
| 场所设施 | 符合要求 |

**设立流程**（70-77条）：
1. 提交申请书 → 国务院保险监管机构**6个月内**批准筹建
2. 筹建期**1年内**完成（不得经营）
3. 具备条件后开业申请 → **60日内**批准/不批准
4. 批准 → 颁发**经营保险业务许可证**
5. 许可证**6个月内**办工商登记（否则失效）

**分支机构**（74条）：须经保险监管机构批准；不具有**法人资格**，民事责任由保险公司承担

**董监高任职资格**（81-82条）：品行良好+专业知识+**任职资格核准**

**变更批准事项**（84条）：名称/注册资本/营业场所/撤销分支/合并分立/修改章程/**5%以上**股东/股份变动等 → 须经批准

#### 经营规则（95-116条）

**业务范围**（95条）：
- 人身险：人寿/健康/意外伤害
- 财产险：财产损失/责任/信用/保证
- **不得兼营人身险和财产险**（短期健康险/意外伤害险例外）
- **相关业务**：再保险（分出/分入）

**保证金**（97条）：注册资本的**20%**提取保证金，存入指定银行（**清算时才能动用**）

**责任准备金**（98条）：按保障被保险人利益+保证偿付能力提取

**偿付能力**（101条）：认可资产 - 认可负债 **≥** 规定数额；低于的采取**10项措施**直至停止接受新业务

**自留保险费限制**（102条）：当年自留保险费 ≤ 实有资本金+公积金的**4倍**

**危险单位限制**（103条）：每一危险单位责任 ≤ 资本金+公积金的**10%**；超出部分**必须再保险**

**资金运用**（106条）— 五种形式：
1. **银行存款**
2. 买卖债券/股票/证券投资基金等有价证券
3. **投资不动产**
4. 国务院规定其他形式
→ 必须**稳健、安全性第一**

**禁止行为**（116条，13项）：
1. 欺骗投保人/被保险人/受益人
2. 隐瞒重要情况
3. 阻碍如实告知/诱导不如实告知
4. 给予合同外回扣/利益
5. 拒绝依法履行赔付
6. 虚构/夸大事故骗取保险金
7. 挪用/截留/侵占保费
8. 委托无资格机构销售
9. 利用业务为他机构牟取不正当利益
10. 利用代理/经纪套取费用等违法活动
11. 虚假事实损害竞争对手信誉/不正当竞争
12. 泄露商业秘密
13. 其他违法违规

**整顿与接管**（140-148条）：

| 制度 | 触发条件 | 时限 |
|------|----------|------|
| **整顿**（140条） | 逾期不改 → 选派整顿组 | — |
| **接管**（144条） | 偿付能力**严重不足**或**严重危及**偿付能力 | 最长**2年**（可延长） |
| 接管后破产（148条） | 具备破产法第2条规定 → 申请重整或破产清算 | — |

**撤销**（149条）：违法经营吊销许可证或偿付能力低于标准 → **撤销并公告**+组织清算

### 5.4 保险代理人与经纪人（117-132条）

| 类型 | 定义 | 特征 |
|------|------|------|
| **保险代理人** | 根据保险人委托收取佣金代办业务 | 机构或个人；含**专业代理机构**和**兼业代理机构** |
| **保险经纪人** | 基于投保人利益提供中介服务 | 仅**机构**（非个人）；收取佣金 |

**代理责任**（127条）：代理行为 → 由**保险人承担责任**（无权代理/越权/终止后 → 表见代理有效 → 保险人追究越权责任）

**经纪人过错责任**（128条）：因过错造成损失的 → **依法赔偿**

**佣金支付**（130条）：**只能**向代理人/经纪人支付，**不得向其他人支付**

### 5.5 保险业监督管理（133-157条）

**监管原则**（133条）：依法、公开、公正

**条款费率管理**（135条）：
- 关系公众利益的险种/强制保险/新开发人寿险种 → **审批**
- 其他险种 → **备案**

**偿付能力监管**（137条）：健全**偿付能力监管体系**，实施监控

**对不足公司的措施**（138条，10项）：
增资/再保险、限业务范围、限分红、限固定资产购置/经营费用、限资金运用形式/比例、限增设分支、拍卖不良资产/转让业务、限薪酬、限广告、**停止接受新业务**

**监督检查措施**（154条，7项）：
现场检查/进入调查/询问/查阅复制/封存扣押/查询银行账户/冻结查封

### 5.6 法律责任（158-179条）

| 违法行为 | 处罚 |
|----------|------|
| 擅自设立保险机构/非法经营 | 取缔+没收违法所得+**1-5倍**罚款；无/不足20万的**20-100万** |
| 擅自设立代理机构/无证经营 | 取缔+没收+**1-5倍**；无/不足5万的**5-30万** |
| 超范围经营 | 责令改正+没收+**1-5倍**；不足10万的**10-50万**；停业整顿或吊证 |
| 116条违规行为（欺骗/隐瞒/拒赔/骗保等） | **5-30万**；情节重：限业务/停业/吊证 |
| 未按规定提存准备金/办理再保险/运用资金等 | **5-30万**；情节重：限业务/停业/吊证 |
| 代理/经纪131条违规 | **5-30万**；情节重：停业/吊证 |
| 聘任不合格人员 | **2-10万** |
| 转让/出租/出借许可证 | **1-10万** |
| 未报送/保管/披露 | 限期改正；**1-10万** |
| 编造虚假报告/资料 | **10-50万**；情节重：限业务/停业/吊证 |
| 保险诈骗（不构成犯罪） | 行政处罚 |
| 个人保险代理人违规 | 警告+**≤2万**；情节重**2-10万** |
| 外国机构擅自设立代表机构 | 取缔+**5-30万** |
| 保险从业人员违法 | 警告+**1-10万** |
| 构成犯罪 | 追究刑事责任 |

**民事赔偿优先**（175条）：民事赔偿+罚款/罚金同时 → 先民事

## 六、反垄断法（70条，2022修正）

### 6.1 三大垄断行为（3条）

```
垄断行为
│
├── 1. 垄断协议（横向+纵向）
│
├── 2. 滥用市场支配地位
│
└── 3. 具有排除/限制竞争效果的经营者集中
```

### 6.2 垄断协议（16-21条）

#### 横向垄断协议（17条）— 竞争者之间

| 行为 | 说明 |
|------|------|
| 固定/变更价格 | 价格同盟/串通投标 |
| 限制产量/销量 | 限产保价 |
| 分割市场 | 划分销售区域/客户/原材料采购 |
| 限制新技术/新产品 | 卡特尔式技术封锁 |
| 联合抵制 | 集体不与某方交易 |
| 其他认定 | — |

#### 纵向垄断协议（18条）— 产业链上下游

| 行为 | 说明 |
|------|------|
| 固定转售价格 | 限定下游售价 |
| 限制最低转售价格 | 限定最低售价（**转售价格维持RPM**） |

**安全港规则**（18条第二款，2022新增！）：
- 纵向协议能够证明**不具有排除限制竞争效果** → **不予禁止**
- 市场份额**低于规定标准**+符合其他条件 → **不予禁止**

**轴幅协议/帮助行为**（19条，2022新增！）：
- 经营者**不得组织**其他经营者达成垄断协议或提供**实质性帮助**

#### 豁免（20条）

| 类型 | 条件 | 额外证明 |
|------|------|----------|
| 技术改进/研发 | 为改进技术/研发新产品 | 不会严重限制竞争+消费者分享利益 |
| 质量/效率提升 | 提高质量/降低成本/专业化分工 | 同上 |
| 中小经营者 | 提高中小经营者竞争力 | 同上 |
| 社会公共利益 | 节能环保/救灾救助 | 同上 |
| 经济不景气 | 缓解销量下降/生产过剩 | 同上 |
| 对外贸易 | 保障正当利益 | 同上 |
| 法律/国务院其他 | — | — |

**行业协会**（21条）：**不得组织**本行业经营者从事本章禁止的垄断行为

### 6.3 滥用市场支配地位（22-24条）

#### 七大类滥用行为（22条）

| # | 行为 | 典型例子 |
|---|------|----------|
| 1 | **不公平高价/低价** | 暴利掠夺/低于成本倾销 |
| 2 | **拒绝交易** | 无正当理由拒绝供货/接入 |
| 3 | **限定交易**（二选一） | 只能与我交易或只能与我指定的人交易 |
| 4 | **搭售/附加不合理条件** | 买我的A必须买B |
| 5 | **差别待遇** | 同样条件不同待遇（杀熟） |
| 6 | **其他认定** | — |
| 7 | **数据/算法/平台规则滥用**（2022新！） | 大数据杀熟/算法合谋/平台封杀 |

#### 市场支配地位认定（23-24条）

**六大因素**（23条）：
1. 市场份额+竞争状况
2. 控制销售/采购市场的能力
3. 财力和技术条件
4. 其他经营者的**依赖程度**
5. 进入市场的**难易程度**
6. 其他因素

**推定标准**（24条）— 满足其一即推定：

| 情形 | 市场份额 |
|------|----------|
| 1个经营者 | **≥ 1/2** |
| 2个经营者合计 | **≥ 2/3** |
| 3个经营者合计 | **≥ 3/4** |

例外：其中有的**不足1/10**的不计入推定

**反驳权**：被推定的经营者有证据证明不具有支配地位的 → 不认定

### 6.4 经营者集中（25-38条）

#### 三种集中形式（25条）

1. 经营者**合并**
2. 取得**股权或资产**的控制权
3. 通过**合同**取得控制权/施加**决定性影响**

#### 申报标准与豁免（26-27条）

| 情形 | 要求 |
|------|------|
| 达到**国务院规定申报标准** | **事先申报**，未申报**不得实施** |
| **未达标准但有证据**排除限制竞争 | 反垄断执法机构可**要求申报** |
| **豁免申报**（27条） | ①参与方之一拥有其他每个参与者**50%以上**股份/资产 ②反之亦然 |

#### 审查流程与时限（30-32条）

| 阶段 | 时限 | 结果 |
|------|------|------|
| **初步审查** | 收到合格材料**30日内** | 不进一步审查/逾期 → **可实施** |
| **进一步审查** | 决定进一步审查**90日内** | 禁止/不禁止；审查期内**不得实施** |
| **延期** | 书面通知可延长**最长60日** | 逾期 → **可实施** |
| **中止** | 特定情形可**中止计算**审查期限 | 消除后**继续计算** |

#### 审查考量因素（33条，6项）

1. 参与者在相关市场**市场份额和控制力**
2. **市场集中度**
3. 对**市场进入/技术进步**的影响
4. 对**消费者和其他经营者**的影响
5. 对**国民经济发展**的影响
6. 其他因素

#### 决定类型（34-36条）

| 结果 | 条件 |
|------|------|
| **禁止** | 具有/可能有排除限制竞争效果 |
| **不予禁止+附条件** | 有利影响**明显大于**不利影响 或 符合社会公共利益 |
| **附限制性条件** | 减少不利影响的条件 |
| **分类分级审查**（37条，2022新！） | 加强对**国计民生重要领域**的审查 |

**国家安全审查**（38条）：外资并购/参与经营者集中涉及国家安全的 → 除反垄断审查外还需**国家安全审查**

### 6.5 滥用行政权力排除限制竞争（39-45条）

| # | 禁止行为 | 具体表现 |
|---|----------|----------|
| 1 | 强制交易（39条） | 限定/变相限定使用**指定经营者**的商品 |
| 2 | 妨碍准入（40条，2022新！） | 签订合作协议/备忘录妨碍进入或实行不平等待遇 |
| 3 | 地区封锁（41条） | ①歧视性收费/价格 ②歧视性技术标准 ③歧视性许可 ④设关卡阻碍流通 ⑤其他 |
| 4 | 招投标排斥（42条） | 歧视性资质/评审标准/不依法发布信息 |
| 5 | 投资限制（43条） | 不平等待遇排斥外地投资/设分支机构 |
| 6 | 强制垄断（44条） | 强制/变相强制经营者从事垄断行为 |
| 7 | 抽象规定（45条） | 制定含排除限制竞争内容的**规定** |

### 6.6 调查与法律责任（46-67条）

**调查措施**（47条，5项）：
1. 进入场所检查
2. 询问相关人员
3. 查阅复制单证/账簿/电子数据
4. **查封扣押**证据
5. **查询银行账户**（需主要负责人批准）

**承诺制度**（53条，2022完善！）：涉嫌垄断行为 → 承诺在指定期限采取措施消除后果 → **可中止调查**；履行承诺 → **可终止调查**

**法律责任**（56-67条）：

| 违法行为 | 经营者处罚 | 个人处罚 | 其他 |
|----------|-----------|----------|------|
| **垄断协议**（56条） | 停止违法+没收违法所得+销售额**1%-10%**；未实施的**≤300万** | **≤100万** | 组织/帮助的**同样处罚**；行业协会**≤300万**（情节重撤销登记） |
| **滥用支配地位**（57条） | 停止+没收+销售额**1%-10%** | — | — |
| **违法集中**（58条） | 停止+限期处分/转让/恢复原状+销售额**≤10%**；无不影响的**≤500万** | — | — |
| **民事责任**（60条） | 给他人造成损失的**依法赔偿** | — | 损害公益的 → **检察院可提起公益诉讼**（2022新！） |
| **行政垄断**（61条） | — | 上级机关**责令改正+处分** | 反垄断执法机构可建议上级机关处理 |
| **拒绝/阻碍调查**（62条） | 单位：销售额**≤1%**；无/难计算的**≤500万** | **≤50万** | — |
| **情节特别严重**（63条，2022新增！） | 在56/57/58/62条罚款基数上**2-5倍** | — | — |
| **信用记录**（64条，2022新！） | 记入**信用记录**并向社会**公示** | — | — |
| **安全港豁免** | 能证明不具有排除限制竞争效果的纵向协议 → **不予禁止** | — | — |
| **主动报告宽恕** | 主动报告并提供重要证据 → **酌情减轻或免除** | — | — |

**知识产权行使不适用**（68条）：依照知识产权法律行使的行为**不适用本法**；但**滥用**知识产权排除限制竞争的 → **适用本法**

**农业联合不适用**（69条）：农业生产者及农村经济组织在农产品**生产/加工/销售/运输/储存**中的联合/协同 → 不适用

### 6.7 2022修正要点总结

| 旧法(2008) | 2022新法 | 影响 |
|-------------|-----------|------|
| 无数字平台相关规定 | 第9条禁止利用数据和算法/技术/资本优势/平台规则从事垄断 | **平台经济纳入监管** |

| 无轴幅协议规定 | 第19条禁止组织和帮助垄断协议 | **堵住规避漏洞** |
| 无数字经济条款 | 第22条滥用支配地位增加数据算法平台规则 | **大数据杀熟入法** |
| 最高罚款倍数 | 第63条情节特别严重的**2-5倍** | **大幅提高威慑力** |
| 无信用惩戒 | 第64条记入信用记录+向社会公示 | **增加声誉成本** |

## 七、消费者权益保护法（63条）

### 7.1 消费者九大权利（7-15条）

| # | 权利 | 核心内容 |
|---|------|----------|
| 1 | **安全保障权**（7条） | 人身财产安全不受损害；商品服务符合保障安全的要求 |
| 2 | **知情权**（8条） | 知悉真实情况（价格/产地/生产者/用途/性能/规格/成分/生产日期/有效期/检验合格证明/使用方法/售后服务等） |
| 3 | **自主选择权**（9条） | 自主选择经营者/商品品种和服务方式/决定买不买；**比较/鉴别/挑选** |
| 4 | **公平交易权**（10条） | 获得质量保障/价格合理/计量正确；**拒绝强制交易** |
| 5 | **获得赔偿权**（11条） | 人身/财产损害的依法获赔权 |
| 6 | **结社权**（12条） | 依法成立维护自身合法权益的社会组织 |
| 7 | **受教育权**（13条） | 获得消费知识/提高自我保护意识 |
| 8 | **受人尊重权**（14条） | 人格尊严/民族风俗习惯得到尊重；**个人信息依法得到保护** |
| 9 | **监督权**（15条） | 检举/控告侵害行为/监督保护工作/批评建议 |

### 7.2 经营者义务（16-29条）

#### 核心义务一览

| 义务类别 | 具体内容 | 条文 |
|----------|----------|------|
| **依法履约** | 遵守法律法规+约定；不得设定不公平不合理条件；**不得强制交易** | 16条 |
| **听取意见** | 听取消费者意见+接受监督 | 17条 |
| **安全保障义务** | 商品服务符合保障安全要求；**真实说明+明确警示**+正确使用方法；宾馆商场等**安全保障义务** | 18条 |
| **缺陷召回**（19条） | 发现缺陷危及安全的 → **立即报告+告知**+**停止销售/警示/召回/无害化/销毁/停产停服**；召回费用由经营者承担 | 19条 |
| **真实信息披露**（20条） | 信息**真实全面**；**不得虚假或引人误解宣传**；**明码标价** | 20条 |
| **身份标识**（21条） | 标明真实名称标记（租赁柜台/场地须标明） | 21条 |
| **出具凭证**（22条） | 依规或惯例出具发票等凭证；**索要必须出具** | 22条 |
| **质量保证**（23条） | 正常使用应有质量/性能/用途/有效期；表明的质量状况须相符；**耐用商品/装修6个月内发现瑕疵→经营者举证** | 23条 |
| **退货/修理/更换**（24条） | 不符合质量要求 → 按国家规定/约定退货/换/修；无规定的**收到货7日内可退货**（7日后法定解除条件的及时退货）；**经营者承担运费** | 24条 |
| **七日无理由退货**（25条，网购/电视/电话/邮购！） | **7日内可无理由退货**；除外：①定作 ②鲜活易腐 ③下载/拆封数字化商品 ④报纸期刊；其他不宜退货经确认除外；**完好**；经营者**7日内退款**；**退货运费消费者承担**（另有约定除外） | 25条 |
| **格式条款限制**（26条） | 显著方式提请注意**重大利害关系**内容并说明；**不得**排除限制消费者权利/减轻免除经营者责任/加重消费者责任/借助技术手段强制交易；**含有上述内容的格式条款无效** | 26条 |
| **尊重人格尊严**（27条） | 不得侮辱诽谤/搜查身体及物品/侵犯人身自由 | 27条 |
| **网络等信息披露**（28条） | 网络/电视/电话/邮购+金融服务 → 提供经营地址/联系方式/数量质量/价款/费用/履行期限/安全注意事项/风险警示/售后/民事责任等信息 | 28条 |
| **个人信息保护**（29条） | **合法正当必要原则**+**明示目的方式和范围**+**经消费者同意**+公开收集使用规则；**严格保密不得泄露/出售/非法提供**；**采取技术措施确保信息安全**；**立即补救泄露丢失**；**未经同意/明确拒绝不得发送商业性信息** | 29条 |

### 7.3 争议解决途径（39-47条）

**五步解决路径**（39条）：

```
协商和解 → 调解（消协/调解组织）→ 行政投诉 → 仲裁（有协议）→ 诉讼
```

**特殊赔偿责任规则**：

| 场景 | 规则 | 条文 |
|------|------|------|
| **一般损害** | 销售者/服务者赔偿；追偿权 | 40条 |
| **企业分立合并** | 向变更后企业索赔 | 41条 |
| **借用营业执照** | 向违法经营者或执照持有人索赔 | 42条 |
| **展销会/租赁柜台** | 向销售者/服务者索赔；展销会结束/租赁期满后 → 向举办者/出租者索赔 | 43条 |
| **网络交易平台**（44条，2013重磅！） | 向销售者/服务者索赔；**平台不能提供真实信息的→平台赔偿**；平台**明知/应知利用平台侵权未采取措施的→连带责任** | 44条 |
| **虚假广告**（45条） | 向经营者索赔；广告经营者/发布者**不能提供经营者信息的→承担赔偿责任**；**关系生命健康虚假广告的→连带责任**；**推荐商品服务的→连带责任** | 45条 |
| **行政部门投诉**（46条） | **7个工作日**内予以处理告知 | 46条 |
| **公益诉讼**（47条，2013新增！） | 中消协及省消协可向法院**提起诉讼** | 47条 |

### 7.4 法律责任（48-61条）

#### 民事责任（48-52条）

| 情形 | 责任 |
|------|------|
| 一般侵权 | 依其他法律法规承担民事责任 |
| 人身伤害 | 医疗费+护理费+交通费+误工费+残疾赔偿金+丧葬费+死亡赔偿金 |
| 精神损害 | 停止侵害+恢复名誉+消除影响+赔礼道歉+**精神损害赔偿**（**严重精神损害**才可） |
| 财产损害 | 修理/重作/更换/退货/补足/退还/赔偿等 |

**惩罚性赔偿**（55条，核心武器！）：

| 情形 | 赔偿标准 |
|------|----------|
| **欺诈行为** | 增加**3倍**赔偿（**退一赔三**）；不足**500元**的按**500元**算 |
| **明知缺陷仍提供致人死亡/健康严重损害** | 损失赔偿+**≤2倍的惩罚性赔偿** |

#### 行政责任（56条，10项！）

| # | 违法行为 | 处罚 |
|---|----------|------|
| 1 | 不符合保障安全要求 | 没收违法所得+**1-10倍**罚款；无违法所得**≤50万**；情节重**停业/吊照** |
| 2 | 掺杂掺假/以假充真/以次充好/不合格冒充合格 | 同上 |
| 3 | 生产明令淘汰/销售失效变质 | 同上 |

## 九、咨询速答模板

### 模板1：「我想开家公司」

**Q：我想开一家公司，需要什么条件和流程？**

**A：** 开公司分以下几步：

1. **确定类型**：大多数人选有限责任公司（1-50个股东，5年内缴足注册资本）
2. **制定章程**：载明名称/住所/注册资本/股东/出资/机构/法定代表人等
3. **名称预先核准**：名称中必须包含"有限责任公司"/"有限公司"字样
4. **提交设立登记**：向公司登记机关提交申请书/章程/身份证明等
5. **领取营业执照**：签发日=成立日
6. **刻章/开户/税务登记**

**关键提醒**：
- 2023新公司法要求注册资本**5年内实缴**（不是以前的无期限认缴了）
- 如果是一人有限公司，要注意**财产独立性**（混同可能导致连带责任）
- 建议章程中对股权转让/议事规则/退出机制做详细约定

### 模板2：「公司经营不下去了怎么办？」

**Q：公司亏钱了想关门，怎么操作？**

**A：** 有三条路：

**路径一：解散清算（最常用）**
1. 召开股东会作解散决议（2/3以上表决权通过）
2. **15日内**成立清算组（2023新：**董事为清算义务人**）
3. 10日内通知+60日内公告债权人
4. 清理财产+清偿债务（职工工资→税款→普通债权）
5. 注销登记

**路径二：破产清算（资不抵债时）**
- 不能清偿到期债务 → 向法院申请破产
- 法院指定**管理人**接管
- 可能走重整（拯救）或和解（协商）或直接清算
- 清偿顺序：破产费用+共益债务→职工债权→税款→普通债权

**路径三：简易注销（无债务时）**
- 无债务/已清偿全部债务 → 全体股东承诺 → 公告**20日** → **20日内**申请注销（2023新！）

**注意**：不管走哪条路，**不能直接跑路**。不依法清算的，股东可能要对公司债务承担**连带责任**。

### 模板3：「买到假货/被骗了怎么办？**

**A：** 消费者维权的**五步走**：

1. **保留证据**：发票/购物记录/聊天记录/商品照片/包装
2. **协商**：先找商家沟通（微信/电话/现场）
3. **投诉**：协商不成 → **12315**（全国消费者投诉热线）或全国12315平台/小程序；行政部门**7个工作日内**处理
4. **求助消协**：向当地**消费者协会**投诉调解（免费）
5. **诉讼**：调解不成 → 向法院起诉（**标的额小的可适用简易程序**）

**如果遇到欺诈**（商家故意造假/虚假宣传/以次充好）：
- **退一赔三**！增加赔偿**3倍**（不足500元的按500元算）
- 例如买了2000元的假货 → 退2000 + 赔6000 = **拿回8000元**

**网购7天无理由退货**：大部分商品收到后**7天内**都可以无理由退（鲜活/定制/数字化/报刊除外），但要保证**商品完好**。
