> 核心法规：公司法（2023修订，2024年7月1日施行）+ 公司登记管理实施办法 + 注册资本登记管理制度规定

## 一、新公司法（2024年7月1日施行）十大核心变化

| 序号 | 变化 | 旧法 | 新法 | 条款 |
|------|------|------|------|------|
| 1 | **注册资本限期实缴** | 认缴制，无期限 | **5年内必须实缴** | 第47条 |
| 2 | **治理结构简化** | 必设监事会/监事 | 可不设监事，审计委员会替代 | 第69/76/83条 |
| 3 | **股东失权制度** | 无 | 催缴后宽限期满仍未缴 → 丧失股权 | 第52条 |
| 4 | **加速到期** | 无 | 公司不能清偿到期债务 → 股东提前缴资 | 第54条 |
| 5 | **出资形式扩展** | 货币+实物等 | 新增**股权、债权**出资 | 第48条 |
| 6 | **法代制度革新** | 执行董事/经理 | 执行公司事务的董事或经理 | 第10条 |
| 7 | **双重代表诉讼** | 无 | 股东可直接起诉全资子公司董监高 | 第189条 |
| 8 | **知情权扩大** | 查阅账簿 | 新增查阅**会计凭证** | 第57条 |
| 9 | **类别股** | 无 | 可发行优先股、特殊表决权股等 | 第144条 |
| 10 | **简易注销** | 无 | 无债务/已清偿 → 简易注销 | 第240条 |

## 二、有限责任公司设立全流程

### 2.1 设立条件

| 条件 | 具体要求 |
|------|----------|
| 股东 | 1-50人（含1人有限公司） |
| 注册资本 | 全体股东认缴出资额，5年内实缴 |
| 公司章程 | 全体股东制定并签名 |
| 公司名称 | 符合《企业名称登记管理规定》 |
| 住所 | 固定生产经营场所 |
| 组织机构 | 股东会 + 董事会（或执行董事）+ 可选监事 |

### 2.2 设立流程

```
Step 1: 公司名称核准
   └── 网上申请或现场核准

Step 2: 制定公司章程
   ├── 必备条款：名称、住所、经营范围、注册资本、出资方式及期限、
   │             组织机构、法定代表人、股东权利义务等
   └── 可选条款：分红规则、股权转让限制、解散事由等

Step 3: 出资
   ├── 货币出资：存入银行专用账户
   └── 非货币出资（实物/知识产权/土地/股权/债权）：需评估作价

Step 4: 登记注册
   ├── 向市场监管部门提交材料
   ├── 核准 → 领取营业执照
   └── 公司成立日 = 营业执照签发日

Step 5: 后续事项
   ├── 刻制公章
   ├── 开设银行账户
   ├── 税务登记
   └── 社保开户
```

## 三、注册资本五年限期实缴制

### 3.1 新规核心

| 适用对象 | 规则 |
|----------|------|
| 2024.7.1后新设公司 | 自成立之日起**5年内**缴足注册资本 |
| 2024.6.30前已设立公司 | 3年过渡期（至2027.6.30前调整），之后5年内缴足 |
| 股份有限公司 | 设立时全额缴足 |

### 3.2 不实缴的法律后果

| 后果 | 说明 | 法律依据 |
|------|------|----------|
| 催缴失权 | 董事会催缴 → 宽限期≥60日 → 仍未缴 → 丧失未缴部分股权 | 第52条 |
| 加速到期 | 公司不能清偿到期债务 → 认缴未到期的出资须提前缴纳 | 第54条 |
| 连带责任 | 设立时股东未全额缴纳 → 其他设立时股东承担连带责任 | 第50条 |
| 行政处罚 | 虚假出资/抽逃出资 → 市监部门罚款 | 第252条 |

### 3.3 实务建议

1. **别虚报注册资本**：现在必须5年内实缴，量力而行
2. **合理规划出资进度**：不一定要一次性缴清，可以分次缴纳
3. **存量公司抓紧调整**：2027年6月30日前必须调整
4. **非货币出资须评估**：用实物、知识产权等出资必须经依法评估作价

## 四、公司治理结构

### 4.1 三种治理模式

**模式一：传统双层制（保留监事会）**
```
股东会（最高权力机构）
    ├── 董事会（执行机构）
    │    ├── 董事长
    │    ├── 董事（300人以上公司须有职工代表）
    │    └── 经理（可选）
    └── 监事会/监事（监督机构）
```

**模式二：新型单层制（审计委员会替代监事会）**
```
股东会（最高权力机构）
    └── 董事会（执行+监督）
         ├── 董事长
         ├── 董事
         ├── 审计委员会（行使原监事会职权）
         └── 经理（可选）
```

**模式三：极简治理（小规模公司）**
```
股东会
    └── 1名执行董事（兼法定代表人、兼经理）
         └── 经全体股东一致同意 → 不设监事
```

### 4.2 法定代表人新规

| 要素 | 新规则 |
|------|--------|
| 人选范围 | 代表公司执行公司事务的董事或经理 |
| 责任归属 | 以公司名义活动 → 公司首责 → 过错可追偿 |
| 辞任 | 辞去董事/经理职务 = 同时辞去法代 |
| 补任期限 | **30日内**确定新法定代表人 |

## 五、股东权利与保护

### 5.1 股东核心权利

| 权利 | 内容 | 新法强化 |
|------|------|----------|
| 分红权 | 按实缴比例分红（章程另有规定除外） | — |
| 知情权 | 查阅章程、股东会记录、财务报告、会计账簿 | **新增查阅会计凭证** |
| 表决权 | 按出资比例行使（章程另有规定除外） | — |
| 优先购买权 | 其他股东转让股权时的优先购买 | — |
| 退出权 | 特定情形下请求公司回购股权 | **大幅扩展** |
| 诉讼权 | 直接诉讼 + 派生诉讼 | **新增双重代表诉讼** |

### 5.2 股东回购请求权（第89条）

| 情形 | 说明 |
|------|------|
| 公司连续5年盈利且符合分配条件但不分红 | 投反对票的股东可请求回购 |
| 公司合并、分立、转让主要财产 | 投反对票的股东可请求回购 |
| 章程规定的营业期限届满，决议续存 | 投反对票的股东可请求回购 |
| **控股股东滥用权利** | 🆕 新增情形 |

## 六、股东退出机制

### 五大退出途径

| 途径 | 适用场景 | 操作要点 |
|------|----------|----------|
| 1. 股权转让 | 最常用 | 内部自由；对外需通知其他股东，有优先购买权 |
| 2. 异议股东回购 | 对重大决议不满 | 第89条四种情形，协商不成60日内起诉 |
| 3. 减资退出 | 协商减少注册资本 | 需2/3以上表决权通过 + 通知债权人 |
| 4. 公司解散 | 经营严重困难 | 10%以上股权股东可请求法院解散 |
| 5. 简易注销 | 公司无债务 | 全体股东承诺 → 简易注销程序 |

## 七、公司注销与清算

### 7.1 解散事由（第229条）

1. 章程规定的营业期限届满/其他解散事由
2. 股东会决议解散
3. 因合并或分立需要解散
4. 被吊销营业执照、责令关闭或被撤销
5. 法院判决解散

### 7.2 清算流程

```
解散事由出现
    ↓
15日内成立清算组（逾期→利害关系人可请求法院指定）
    ↓
清算组接管公司
    ├── 清理公司财产
    ├── 通知/公告债权人（10日内通知+60日内公告）
    ├── 编制资产负债表和财产清单
    └── 制定清算方案（股东会确认）
    ↓
清偿顺序：清算费用 → 职工工资/社保/补偿金 → 欠税 → 普通债权
    ↓
剩余财产按实缴比例分配
    ↓
编制清算报告 → 股东会确认 → 注销登记
```

### 7.3 简易注销（第240条）

- 条件：公司存续期间未产生债务或已清偿全部债务
- 操作：全体股东承诺 → 简易程序注销
- ⚠️ 虚假承诺：全体股东对注销前的债务承担**连带责任**

## 八、开公司常见法律风险

### 十大风险清单

| 风险 | 说明 | 防范措施 |
|------|------|----------|
| 1. 注册资本虚高 | 5年实缴压力大 | 量力而行 |
| 2. 股东协议缺失 | 合伙没有书面约定 | 签署详细股东协议 |
| 3. 公私不分 | 个人与公司财产混同 | 严格分账 |
| 4. 一人公司风险 | 不能证明财产独立→连带责任 | 每年审计 |
| 5. 关联交易 | 控股股东利益输送 | 关联方回避表决 |
| 6. 合同风险 | 未审查合同条款 | 建立审查制度 |
| 7. 知识产权 | 员工作品归属不清 | 签署归属协议 |
| 8. 劳动用工 | 未签合同/未缴社保 | 规范用工管理 |
| 9. 税务风险 | 偷税漏税 | 合规纳税 |
| 10. 注销不清算 | 不清算直接注销 | 依法清算后注销 |

### 一人有限公司特别风险

> **第23条第3款**：一人公司股东不能证明公司财产独立于股东自己的财产的，应当对公司债务承担**连带责任**。

即：公私不分、账目混乱，"有限责任"的保护壳就没了。
