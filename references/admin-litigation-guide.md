# 行政诉讼流程概览

> 定位：民告官实操指南——从复议到诉讼到赔偿
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块1交付物
> 法律依据：《行政诉讼法》（2017修正）+《行政复议法》（2023修订）+相关司法解释

---

## 一、什么案件能告政府

### 1.1 受案范围（《行政诉讼法》第12条）

| 序号 | 可诉行政行为 | 典型场景 |
|------|-------------|----------|
| 1 | 行政处罚 | 罚款、吊销执照、行政拘留 |
| 2 | 行政强制 | 查封、扣押、冻结财产 |
| 3 | 行政许可 | 拒绝发证、不予审批 |
| 4 | 确认自然资源权利 | 土地确权、房屋权属争议 |
| 5 | 征收征用 | 房屋征收、土地征用 |
| 6 | 行政机关不履行法定职责 | 该办事不办、该保护不保护 |
| 7 | 侵犯经营自主权 | 违法限制经营 |
| 8 | 行政协议 | 政府不履行行政合同 |
| 9 | 滥用行政权力 | 排除/限制竞争 |
| 10 | 认为其他行政行为侵犯合法权益 | 兜底条款 |

### 1.2 不受理的案件（第13条）

| 不可诉类型 | 说明 |
|-----------|------|
| 国防、外交等国家行为 | 不属于司法审查范围 |
| 行政法规、规章等抽象行政行为 | 不能直接起诉"规定"本身（但可附带审查） |
| 内部行政行为 | 奖惩、任免等内部管理 |
| 刑事司法行为 | 公安依据刑事诉讼法采取的行为 |
| 行政调解/仲裁 | 非最终行政决定 |

## 二、行政复议 vs 行政诉讼

### 2.1 两条路怎么选

```
对行政行为不服
    ├── 路线A：直接起诉（大部分案件可选）
    │         → 法院（6个月内）
    │
    └── 路线B：先复议再起诉
              → 复议机关（60日内申请）
              → 不服复议决定 → 法院（15日内）
              → 复议机关逾期不作决定 → 法院（15日内）
```

### 2.2 哪些案件必须先复议

**复议前置**（不经复议不能直接起诉）：

| 情形 | 法律依据 |
|------|----------|
| 对自然资源权属确认不服 | 《行政复议法》第30条 |
| 纳税争议（先缴税/担保再复议） | 《税收征管法》第88条 |
| 对海关纳税决定不服 | 《海关法》 |
| 反垄断审查决定（部分） | 相关规定 |

> **2023年新行政复议法重大变化**：扩大了复议前置范围，将当场行政处罚、行政不作为等纳入复议前置。具体以最新规定为准。

### 2.3 复议 vs 诉讼对比

| 维度 | 行政复议 | 行政诉讼 |
|------|----------|----------|
| 审查机关 | 行政机关（上级或同级政府） | 人民法院 |
| 申请/起诉期限 | **60日** | **6个月** |
| 审理期限 | **60日**（可延长30日） | **6个月**（可延长） |
| 费用 | **免费** | 收诉讼费（50元） |
| 审查范围 | **合法性+合理性** | 主要审查**合法性** |
| 独立性 | 较低（行政系统内部） | 较高（司法独立） |

> **实操建议**：能直接起诉的，优先起诉（法院更独立）。必须先复议的，走完复议再起诉。

## 三、行政诉讼管辖

### 3.1 级别管辖

| 法院 | 受理案件 |
|------|----------|
| **基层法院** | 一般行政案件 |
| **中级法院** | 国务院部门/省级政府作出的行政行为；海关处理的案件；辖区内重大复杂案件 |
| 高级法院 | 辖区内重大复杂案件 |

### 3.2 地域管辖

| 规则 | 管辖法院 |
|------|----------|
| **一般原则** | **最初作出行政行为的行政机关所在地** |
| 经复议改变的 | 原行政机关所在地 或 复议机关所在地（原告选择） |
| 不动产案件 | **不动产所在地**（专属管辖） |
| 限制人身自由 | 被告所在地 或 原告所在地 |

## 四、起诉条件与期限

### 4.1 起诉条件（第49条）

1. 原告是行政行为的相对人或利害关系人
2. 有明确的被告（具体的行政机关）
3. 有具体的诉讼请求和事实根据
4. 属于受案范围和管辖法院

### 4.2 起诉期限

| 情形 | 期限 | 起算点 |
|------|------|--------|
| 直接起诉 | **6个月** | 知道或应当知道行政行为之日 |
| 经复议后起诉 | **15日** | 收到复议决定书之日 |
| 复议机关不作决定 | **15日** | 复议期满之日 |
| 不动产案件最长 | **20年** | 行政行为作出之日 |
| 其他案件最长 | **1年** | 行政行为作出之日 |
| 行政机关未告知起诉权 | **1年** | 知道行政行为之日 |

## 五、行政诉讼流程

### 5.1 起诉材料

| 材料 | 说明 |
|------|------|
| 行政起诉状 | 参照民事起诉状格式，增加"被诉行政行为"描述 |
| 原告身份证复印件 | — |

| 其他证据 | 支持诉讼请求的证据材料 |

### 5.3 举证责任倒置（最大特点！）

**行政诉讼最核心规则**：

> **被告（行政机关）对作出的行政行为负举证责任。**
> 应当提供作出行政行为的证据和所依据的规范性文件。（第34条）

翻译成白话：政府做了什么决定，政府自己证明这个决定是对的。证明不了就算政府输。

| 谁举证 | 证明什么 |
|--------|----------|
| **被告（行政机关）** | 行政行为合法——事实依据+法律依据+程序合法 |
| 原告 | 起诉符合条件（自己是利害关系人等）+ 损害事实（赔偿时） |

> **实务意义**：你只需要证明"政府做了这个事"+"这个事损害了你"，至于这个事做得对不对，由政府自己举证。

### 5.4 审理期限

| 程序 | 期限 |
|------|------|
| 一审 | **6个月**（可延长） |
| 简易程序 | **45日** |
| 二审 | **3个月**（可延长） |

## 六、判决类型

### 6.1 法院可以做出的判决

| 判决类型 | 适用情形 | 效果 |
|----------|----------|------|
| **撤销判决** | 行政行为违法 | 撤销行政决定，可要求重做 |
| **确认违法** | 行政行为违法但撤销无意义 | 确认违法，为赔偿铺路 |
| **确认无效** | 重大且明显违法 | 自始无效 |
| **履行判决** | 行政机关不作为 | 判令限期履行 |
| **变更判决** | 行政处罚明显不当 | 直接改变处罚内容 |
| **驳回诉讼请求** | 行政行为合法 | 原告败诉 |

### 6.2 行政赔偿

**《国家赔偿法》规定的赔偿范围**：

| 侵害类型 | 赔偿项目 |
|----------|----------|
| 侵犯人身权 | 医疗费+误工费+护理费+伤残赔偿金+精神损害抚慰金 |
| 侵犯财产权 | 返还财产/恢复原状/赔偿损失 |
| 违法拘留/逮捕 | 按**国家上年度职工日平均工资**×被羁押天数 |

**赔偿程序**：
1. 先向赔偿义务机关（作出违法行为的行政机关）申请赔偿
2. 2个月内不答复/不满意→向法院起诉

## 七、规范性文件附带审查

**2015年新增的重要制度**（第53条）：

- 对行政行为起诉时，可以一并请求审查该行为所依据的规范性文件
- 范围：国务院部门和地方政府及其部门的**规范性文件**（不含法律法规规章）
- 效果：法院认为不合法的，**不作为认定行政行为合法的依据**，并向制定机关提出司法建议

> **白话**：你不能直接告一个"红头文件"，但你可以在告具体行政行为时，顺带说"这个红头文件本身就不合法"。

## 八、常见行政诉讼场景实操

### 8.1 行政处罚（最常见）

| 步骤 | 操作 |
|------|------|
| 收到处罚决定 | 仔细看：处罚依据+事实认定+救济途径+期限 |
| 决定是否要告 | 处罚是否合法？金额是否合理？程序是否合规？ |
| 选择路径 | 复议（60日，免费）或 诉讼（6个月，50元） |
| 准备证据 | 处罚决定书+你的申辩材料+证明处罚不当的证据 |

**行政处罚常见违法点**（可作为胜诉理由）：
1. 未告知当事人陈述申辩权
2. 未举行听证（应当听证而未听证）
3. 事实认定证据不足
4. 适用法律错误
5. 处罚明显过重（不符合比例原则）
6. 超越职权
7. 违反法定程序

### 8.2 征收拆迁

| 要点 | 说明 |
|------|------|
| 征收决定 | 必须由市、县级人民政府作出 |
| 补偿标准 | 不低于被征收房屋类似房地产的市场价格 |
| 评估报告 | 有权申请复核评估+鉴定 |
| 强制拆除 | 必须经法院裁定，行政机关不能自行强拆 |
| 违法强拆 | 可起诉确认违法+申请国家赔偿 |

### 8.3 不履行法定职责（行政不作为）

| 步骤 | 操作 |
|------|------|
| 1. 先申请 | 向行政机关提出申请（书面+保留送达证据） |
| 2. 等回复 | 法定期限内不答复即构成不作为 |
| 3. 起诉 | 以行政机关不履行法定职责为由起诉 |
| 4. 判决 | 法院判令行政机关在一定期限内履行 |

## 九、常见问题速查

| 问题 | 答案 |
|------|------|
| 行政诉讼费多少？ | 每件**50元** |
| 能调解吗？ | 行政赔偿/补偿案件可以调解，其他行政案件**一般不适用调解** |
| 行政机关不到庭？ | 法院可以**缺席判决**（不影响审理） |
| 告了政府会不会被报复？ | 法律禁止行政机关对起诉人打击报复（第3条），实务中需留存证据 |
| 行政诉讼期间还要执行处罚吗？ | **一般不停止执行**，但可以申请法院裁定暂停 |
| 能告公务员个人吗？ | 不能，只能告行政机关（以机关名义起诉） |
| 能告村委会吗？ | 村委会不是行政机关，但受行政机关委托行使行政职能的可以告 |
