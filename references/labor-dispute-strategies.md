> 版本：v1.0 | Phase 2 模块2交付物

## 一、欠薪/拖欠工资

### 1.1 法律武器

| 法条 | 内容 | 用途 |
|------|------|------|
| 《劳动合同法》第30条 | 用人单位应当按时足额支付工资 | 基础依据 |
| 《劳动合同法》第38条第1款第2项 | 未及时足额支付→劳动者可解除+要经济补偿 | 主动离职也能拿N |
| 《劳动合同法》第85条 | 逾期不付→加付50%-100%赔偿金 | 追加赔偿 |
| 《工资支付暂行规定》第9条 | 解除后一次性付清工资 | 催促结清 |

### 1.2 实操策略

**第一步：固定证据**
1. 银行流水：打印近12个月工资流水（标注哪个月没发）
2. 工资条/工资单截图
3. 劳动合同中工资条款
4. 催要工资的微信/邮件记录（证明主张过权利→时效中断）

**第二步：选择路径**

| 路径 | 适用场景 | 优劣 |
|------|----------|------|
| **劳动监察投诉** | 欠薪事实清楚、公司还在正常经营 | 快（7日内立案），但执行力弱 |
| **劳动仲裁** | 需要强制执行、金额有争议 | 有法律效力，可强制执行 |
| **38条主动解除** | 公司持续欠薪，你想走人又想拿补偿 | 既能走人又能拿N，一举两得 |

**第三步（38条解除的关键操作）**：
1. 先发一份**书面通知**（EMS寄到公司注册地址，保留快递单+签收回执）
2. 通知内容："因贵司未及时足额支付劳动报酬（[月份]工资至今未发），依据《劳动合同法》第三十八条之规定，本人决定解除劳动合同。请于XX日前结清工资并支付经济补偿金。"
3. 同时申请劳动仲裁
4. **不要**先口头辞职再补发通知，顺序很重要！

### 1.3 赔偿计算

```
欠薪本金：拖欠月工资 × 月数
经济补偿金（N）：月均工资 × 工作年限（38条解除才有）
额外赔偿：欠薪本金 × 50%-100%（需先经劳动行政部门责令+逾期不付）
```

> **实务提醒**：部分地区仲裁委不支持50%-100%加付赔偿金，认为这属于行政执法范畴。建议先主张本金+经济补偿金，加付赔偿留到法院阶段。

## 二、违法解除劳动合同

### 2.1 什么算"违法解除"

| 情形 | 为什么违法 |
|------|-----------|
| 口头通知走人，没有书面文件 | 程序违法 |
| 没有合法理由就辞退（如"你态度不好"） | 无法定事由 |
| 孕期/产期/哺乳期辞退 | 《劳动合同法》第42条禁止 |
| 医疗期内辞退 | 《劳动合同法》第42条禁止 |
| 工伤停工留薪期辞退 | 《劳动合同法》第42条禁止 |
| 以"不能胜任工作"辞退但未经培训/调岗 | 程序不合法 |
| 经济性裁员不满足法定条件 | 《劳动合同法》第41条 |

### 2.2 实操策略

**核心思路**：先确认是否"违法"，然后选择要**继续上班**还是**要钱走人**。

| 选择 | 法律依据 | 效果 |
|------|----------|------|
| **恢复劳动关系** | 《劳动合同法》第48条 | 公司必须让你回去上班+补发期间工资 |
| **违法解除赔偿金（2N）** | 《劳动合同法》第87条 | 经济补偿的2倍 |

**选择建议**：
- 还想回去干→主张恢复劳动关系
- 不想回去了→主张2N赔偿金
- 二者只能选一个，不能同时主张

**取证要点**：
1. **解除通知**是核心证据——书面的留原件，口头的立刻录音+发微信确认
2. 微信确认话术："领导，你刚才说让我明天不用来了，是正式通知我解除劳动合同吗？"——诱导对方书面确认
3. 公司让你签"自愿离职申请"→**绝对不签！** 签了就变成你主动离职，拿不到赔偿

### 2.3 赔偿计算

```
2N赔偿金 = 月均工资 × 工作年限 × 2

月均工资 = 解除前12个月平均应发工资（含基本工资+奖金+补贴+加班费）

工作年限计算：
- 满1年 = 1个月
- 6个月以上不满1年 = 1个月
- 不满6个月 = 0.5个月

示例：月薪1万，干了3年7个月 → N=4 → 2N=8万
```

> **工资上限**：月均工资超过当地社平工资3倍的，按3倍计算，且补偿年限不超过12年。

## 三、加班费争议

### 3.1 举证责任分配

**关键规则**（《最高人民法院关于审理劳动争议案件适用法律问题的解释（一）》第42条）：

- **劳动者**：需提供加班事实**初步证据**（如考勤截图、微信加班通知、钉钉打卡记录）
- **用人单位**：劳动者已提供初步证据后，用人单位需提供反证（如考勤记录）——**考勤记录由用人单位保管，举证责任在公司**

### 3.2 实操策略

**取证方法**（按证据强度排序）：

| 证据 | 强度 | 获取方式 |
|------|------|----------|
| 公司考勤系统导出 | ⭐⭐⭐⭐⭐ | 在职时导出/截图 |
| 钉钉/企业微信打卡记录 | ⭐⭐⭐⭐ | 截图+录屏 |
| 加班审批单 | ⭐⭐⭐⭐ | 拍照/截图 |
| 微信工作群加班聊天记录 | ⭐⭐⭐ | 截图+原始载体录屏 |
| 加班时发送的工作邮件 | ⭐⭐⭐ | 截图（注意时间戳） |
| 同事证言 | ⭐⭐ | 书面证言+出庭 |

**特殊情况**：
- 公司实行"弹性工作制"→需查是否经劳动行政部门审批，未审批的按标准工时制算加班
- "综合计算工时制"→需有行政审批，超出总工时的部分仍算加班
- "不定时工作制"→法定节假日加班仍需支付3倍工资

### 3.3 加班费计算速查表

| 加班类型 | 倍率 | 计算公式 |
|----------|------|----------|
| 工作日延时 | **1.5倍** | 月工资÷21.75÷8 × 1.5 × 加班小时数 |
| 休息日（未补休）| **2倍** | 月工资÷21.75 × 2 × 加班天数 |
| 法定节假日 | **3倍** | 月工资÷21.75 × 3 × 加班天数 |

> **实务提醒**：休息日加班公司可以安排补休代替支付加班费，但法定节假日加班**不能用补休替代**，必须支付3倍工资。

## 四、未签劳动合同双倍工资

### 4.1 法律依据

《劳动合同法》第82条：
- 自用工之日起**超过1个月不满1年**未签书面合同→**每月二倍工资**
- 满1年仍未签→**视为已订立无固定期限劳动合同**

### 4.2 计算规则

```
双倍工资差额 = 月工资 × 未签合同月数（最多11个月）

起算：用工之日起第2个月
截止：补签之日 或 满1年之日（取早者）

示例：2025年1月1日入职，从未签合同
→ 2025年2月1日至2025年12月31日 = 11个月
→ 双倍工资差额 = 月工资 × 11
```

### 4.3 时效陷阱

⚠️ **最大坑**：双倍工资的仲裁时效**逐月起算**！

```
入职时间：2025-01-01
双倍工资起算：2025-02-01
2025年2月工资的时效：2025-02-01 → 2026-02-01（1年）
2025年3月工资的时效：2025-03-01 → 2026-03-01（1年）
...以此类推

如果你2026年6月才申请仲裁，
2025年2月-5月的双倍工资已经过了时效！
只能主张2025年6月-12月的（7个月）。
```

> **操作建议**：发现没签合同，尽早申请仲裁。每拖一个月，可能少拿一个月的双倍工资。

### 4.4 公司常见抗辩及应对

| 公司说法 | 应对 |
|----------|------|
| "我们发过合同你没签" | 要求公司举证（EMS送达记录等），口说无凭 |
| "你是管理层/HR，负责签合同" | 如果你不是人事负责人，不成立；即使是HR，公司也有义务与你签合同 |
| "试用期不用签" | 错！试用期包含在劳动合同期限内，必须签 |
| "你的电子合同在系统里" | 要求调取，如果你确实签过电子合同则不支持双倍工资 |

## 五、工伤赔偿

### 5.1 完整流程

```
受伤 → 工伤认定申请（30日/1年） → 工伤认定决定
→ 伤残鉴定（伤情稳定后） → 鉴定结论
→ 协商赔偿 / 劳动仲裁 → 获得赔偿
```

### 5.2 工伤认定时效

| 申请主体 | 时限 | 起算点 |
|----------|------|--------|
| 用人单位 | **30日** | 事故伤害发生之日/诊断之日 |
| 劳动者/近亲属/工会 | **1年** | 事故伤害发生之日/诊断之日 |

> **公司不申请怎么办**：公司30天不申请，劳动者自己在1年内申请。超过30天到公司申请之前产生的工伤待遇费用，由公司自行承担。

### 5.3 一次性赔偿速查（解除/终止劳动合同时）

| 伤残等级 | 一次性伤残补助金 | 一次性工伤医疗补助金 | 一次性伤残就业补助金 |
|----------|-----------------|--------------------|--------------------|
| 一级 | 27个月工资 | 不解除 | 不解除 |
| 二级 | 25个月工资 | 不解除 | 不解除 |
| 三级 | 23个月工资 | 不解除 | 不解除 |
| 四级 | 21个月工资 | 不解除 | 不解除 |
| **五级** | **18个月工资** | **各地标准不同** | **各地标准不同** |
| **六级** | **16个月工资** | **各地标准不同** | **各地标准不同** |
| **七级** | **13个月工资** | **各地标准不同** | **各地标准不同** |
| **八级** | **11个月工资** | **各地标准不同** | **各地标准不同** |
| **九级** | **9个月工资** | **各地标准不同** | **各地标准不同** |
| **十级** | **7个月工资** | **各地标准不同** | **各地标准不同** |

> **说明**：一次性工伤医疗补助金和一次性伤残就业补助金由各省自行规定标准，差异很大。具体金额需查当地规定。

### 5.4 公司没买工伤保险

所有工伤待遇**全部由用人单位自行承担**（《工伤保险条例》第62条第2款）。这是强有力的武器——公司不买保险，赔得更多。

## 六、竞业限制争议

### 6.1 核心规则速查

| 要素 | 规则 | 法律依据 |
|------|------|----------|
| 适用对象 | 高级管理人员/高级技术人员/其他负有保密义务人员 | 《劳动合同法》第24条 |
| 最长期限 | **2年** | 第24条第2款 |
| 补偿标准 | 月均工资**30%**（各地有差异，不低于当地最低工资） | 司法解释 |
| 违约金 | 协议约定（但不能畸高） | 第23条 |

### 6.2 劳动者可以解除竞业限制的情形

1. **用人单位3个月不付补偿金** → 劳动者可请求解除竞业限制（《劳动争议司法解释（一）》第38条）
2. **协议只约定义务没约定补偿** → 劳动者履行了→按月均工资30%补偿；劳动者不想履行→可主张解除
3. **用人单位主动解除** → 需额外支付3个月竞业限制补偿金

### 6.3 应对策略

**公司要求你履行竞业限制**：
- 先看协议里有没有约定补偿金
- 有约定→看公司是否按时足额支付
- 没约定/没支付→发函催要→3个月不付可解除
- 正在支付→必须遵守，违约代价很高

**公司告你违反竞业限制**：
- 核心争点：你是否属于"负有保密义务的人员"
- 如果只是普通员工，可以主张竞业限制协议无效/不合理
- 违约金畸高可以请求法院调低

## 七、社保争议

### 7.1 维权路径选择

| 争议类型 | 推荐路径 | 说明 |
|----------|----------|------|
| 公司未缴社保 | **社保稽核投诉**（12333） | 大部分仲裁委不受理"补缴"争议 |
| 因未缴社保导致损失 | **劳动仲裁** | 如医疗费自付/生育津贴损失/工伤待遇损失 |
| 缴费基数不足 | **社保稽核投诉** | 投诉到社保局查 |
| 以未缴社保为由解除合同+要补偿 | **劳动仲裁** | 依据《劳动合同法》第38条 |

### 7.2 实操步骤

**路径A：社保稽核投诉**
1. 拨打12333或到当地社保局窗口
2. 提交投诉材料（身份证+劳动关系证明+投诉书）
3. 社保局调查后责令公司补缴
4. 公司不补缴→社保局可强制征缴+罚款

**路径B：38条解除+仲裁**
1. 以公司未缴社保为由，书面通知解除劳动合同
2. 同时申请劳动仲裁，主张经济补偿金
3. ⚠️ 部分地区（如北京）对"已缴但基数不足"不支持38条解除，需查当地口径

### 7.3 社保损失赔偿（可仲裁）

| 损失类型 | 赔偿内容 |
|----------|----------|
| 未缴医保→自费看病 | 应由医保报销的部分，公司赔偿 |
| 未缴生育保险→自费生产 | 生育医疗费+生育津贴，公司赔偿 |
| 未缴工伤保险→工伤 | 全部工伤待遇由公司承担 |
| 未缴失业保险→失业 | 失业保险金损失，公司赔偿 |
| 未缴养老保险→退休 | 养老金差额，较难主张（多数走行政投诉） |

## 八、实战技巧总结

### 8.1 取证黄金期（在职时就要做）

| 时机 | 要固定的证据 |
|------|-------------|
| **入职第一天** | offer邮件/录用通知截图、入职登记表拍照 |
| **每月发工资** | 工资条截图、银行流水定期打印 |
| **日常工作** | 工牌拍照、工作邮件备份、钉钉/企微聊天记录定期导出 |
| **感觉要出事** | 加班记录截图、考勤记录导出、重要谈话录音 |
| **被通知解除** | 立刻录音、保存解除通知、不签任何文件 |

### 8.2 录音取证注意事项

| 规则 | 说明 |
|------|------|
| 你必须是对话参与者 | 偷录第三方对话不合法 |
| 不能剪辑 | 完整录制，不掐头去尾 |
| 不能诱导 | 正常对话中获取，不能设局 |
| 备份保存 | 原始录音文件+整理的文字版 |
| 录音设备 | 手机自带录音即可，不需要专业设备 |

### 8.3 离职交接注意事项

| 做 | 不做 |
|----|------|
| 签交接清单（留复印件） | 不签空白文件 |
| 要求出具离职证明 | 不签"自愿放弃一切权利"声明 |
| 结清工资确认单 | 不签金额不对的结算单 |
| 社保/公积金转移手续 | 不在压力下签任何协议 |
