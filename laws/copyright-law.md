# 中华人民共和国著作权法（2020修正版）

## 基本信息
- **最新修正**：2020年11月11日第三次修正
- **施行日期**：自**2021年6月1日**起施行
- **条文总数**：共67条
- **历次修订**：1990通过 → 2001修正 → 2010修正 → **2020修正（现行）**

---

## 第一章 总则

**第二条** 中国自然人、法人或者非法人组织的作品，不论是否发表，依照本法享有著作权。

**第三条** 作品是指文学、艺术和科学领域内具有独创性并能以一定形式表现的智力成果：
（一）文字作品；（二）口述作品；（三）音乐戏剧曲艺舞蹈杂技；（四）美术建筑作品；
（五）摄影作品；（六）视听作品；（七）工程设计图/产品设计图/地图/示意图等图形和模型；
（八）计算机软件；（九）符合特征的其他智力成果。

**第五条** 不适用：法律法规及官方译文、单纯事实消息、历法通用数表公式。

## 第二章 著作权

### 第一节 权利内容（第十条）

**人身权**（4项）：
1. 发表权 — 决定是否公之于众
2. 署名权 — 表明作者身份
3. 修改权 — 修改或授权修改
4. 保护作品完整权 — 防止歪曲篡改

**财产权**（13项）：
5. 复制权 6. 发行权 7. 出租权 8. 展览权 9. 表演权
10. 放映权 11. 广播权 12. 信息网络传播权 13. 摄制权
14. 改编权 15. 翻译权 16. 汇编权 17. 其他权利

> 财产权可以许可/转让；人身权不可转让（除发表权外）

### 第二节 权利归属
- **第十一条**：著作权属于作者（创作作品的自然人）
- **第十八条**：职务作品一般作者享有，单位优先使用2年
  - 特殊职务作品（工程设计图/计算机软件/新闻作品）：作者署名，其他权利归单位

### 第三节 保护期
- **署名权/修改权/完整权**：不受限制
- **自然人**：终生+死后50年（12月31日截止）
- **法人作品/特殊职务作品/视听作品**：首次发表后50年

### 第四节 权利的限制
- **第二十四条**：合理使用（13种情形，不需许可不付酬但需指明来源）
- **第二十五条**：法定许可——编写教科书可汇编已发表片段，需付报酬

---

## 第三章 许可使用和转让合同
- 第二十六条：许可合同主要内容包括权利种类/专有或非专有/地域期间/付酬标准/违约责任
- 第二十七条：转让必须订立书面合同
- 第二十九条：未明确许可/转让的权利，未经同意另一方不得行使

---

## 第四章 与著作权有关的权利（邻接权）
- 表演者权利（第39条）
- 录音录像制作者权利（第44条）
- 广播电台电视台权利（第47条）

---

## 第五章 保护

**技术措施**（第49条）：禁止故意避开/破坏

**权利管理信息**（第51条）：禁止删除/改变

**侵权民事责任**（第52条）：停止侵害/消除影响/赔礼道歉/赔偿损失

**惩罚性赔偿**（第53条）：故意侵犯+情节严重的，可在基数的**1-5倍**范围内赔偿

**赔偿计算**（第54条）：
1. 实际损失
2. 侵权人违法所得
3. 参照权利使用费
4. 都无法计算 → 法院判**500元~500万元**

## 📌 法考核心速记

| 考点 | 要点 |
|------|------|
| 作品要件 | 独创性 + 可复制性 |
| 人身权 vs 财产权 | 人身权不可转让（除发表）；财产权可许可转让 |
| 职务作品 | 一般作者有，单位先用2年；特殊（设计图/软件/新闻）归单位 |
| 合理使用 | 不需许可不付费，13种情形 |
| 法定许可 | 教科书编入已发表片段，需付费不需许可 |
| 侵权赔偿 | 实际损失/获利/使用费参照/法定500万上限；恶意+严重1-5倍 |
| 诉讼时效 | 2年（知道应知起算）|
