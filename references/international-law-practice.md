## 一、国际法（Public International Law，~6分）

### 1.1 国家与国际法主体

#### 国家的构成要素（四要素缺一不可）

| 要素 | 内容 | 说明 |
|------|------|------|
| **定居的居民** | 永久人口 | 数量不限，但必须有常住人口 |
| **确定的领土** | 固定的疆域 | 大小不限，但必须有明确边界 |
| **政府** | 有效统治的组织 | 必须能对其领土和人民行使有效控制 |
| **主权** | 对内最高、对外独立的权力 | 最核心要素——没有主权就不是国家 |

### 1.2 管辖权的四种类型

| 类型 | 含义 | 典型场景 |
|------|------|----------|
| **属地管辖** | 领域内一切人和事 | 在中国境内的犯罪/民事纠纷 |
| **属人管辖** | 对本国公民（无论在哪） | 中国公民在海外犯罪 |
| **保护管辖** | 侵害本国重大利益的行为 | 外国人在境外侵害中国国家利益 |
| **普遍管辖** | 严重国际罪行（海盗/战争罪/反人类罪/种族灭绝） | 任何国家对这类罪犯均有权管辖 |

> 📌 **记忆口诀**："地属保普"→ 地(属地)>属(属人)>保(保护)>普(普遍)

### 1.3 国家主权豁免

- **绝对豁免论（传统）**：国家的一切行为和财产都享有豁免
- **有限豁免论（现代趋势）**：区分**主权行为**享豁免 vs **商业行为**不享豁免
- **中国的立场演变**：传统坚持绝对豁免 → **2024年《外国国家豁免法》转向限制豁免**

### 1.4 引渡四原则

| 原则 | 内容 |
|------|------|
| **政治犯不引渡** | 政治犯罪不引渡 |
| **本国公民不引渡** | 多数国家不引渡本国公民 |
| **双重犯罪原则** | 引渡请求的行为在两国都构成犯罪 |
| **罪名特定原则** | 只能按引渡请求的罪名审判/处罚 |

### 1.5 海洋法（法考重中之重！）

```
基线
  │
  ├── 内水 ← 完全主权
  ├── 领海 ≤12海里 → 完全主权 + 无害通过权
  ├── 毗连区 ≤24海里 → 海关/财政/移民/卫生管制
  ├── 专属经济区(EEZ) ≤200海里 → 自然资源主权权利 + 航行飞越自由
  ├── 大陆架 自然延伸≤350海里 → 海床底土自然资源权利
  └── 公海 以上全部以外 → 公海自由 + 船旗国管辖
```

> 📌 **数字口诀**：12领海 24毗连 200专属 350大陆

### 1.6 外交关系法（维也纳外交关系公约，1961）

#### 外交人员特权与豁免

| 特权/豁免 | 内容 | 关键限制 |
|-----------|------|----------|
| 人身不可侵犯 | 不得逮捕/拘留/搜查 | 可制止犯罪或正当防卫 |
| 寓所/文书/财产不可侵犯 | 不得搜查/查封/扣押 | — |
| 管辖豁免（刑事） | **完全豁免** | 无例外！ |
| 管辖豁免（民事/行政） | 原则豁免，但有**4大例外** | 见下 |

📌 **外交人员民事管辖豁免4大例外**：
1. **私有不动产物权诉讼**
2. **以私人身份参与继承诉讼**
3. **以私人身份从事商务/专业活动**的诉讼
4. **主动起诉而被反诉**

### 1.7 国际法院（ICJ）

- 当事人资格：仅限**国家**（个人不能直接起诉）
- 管辖权基础：自愿管辖 / 协定管辖 / 选择条款
- 判决效力：**终局判决，不得上诉**

## 二、国际私法（Private International Law，~5分）

### 2.1 冲突规范（本学科核心！）

冲突规范结构：`[范围] + [系属] = [法律关系] 应适用 [某国法律]`

四种基本类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| 单边冲突规范 | 直接指定内国法 | "中国境内合同适用中国法律" |
| 双边冲突规范 | 抽象连接点 | "不动产适用物之所在地法" |
| 重叠适用 | 必须同时满足多条件 | "离婚须同时符合两国法" |
| 选择适用 | 选择其一即可 | "婚姻财产可选择任一法律" |

> 📌 **中国的态度**：《涉外民事关系法律适用法》明确规定**不接受反致**（第9条）

### 2.2 涉外民事关系连接点速查

| 法律关系 | 连接点优先级 |
|----------|-------------|
| **结婚** | 共同经常居所地法 → 共同国籍国法 → 婚姻缔结地法 |
| **夫妻财产关系** | **意思自治优先** → 共同经常居所地法 → 共同国籍国法 |
| **离婚** | **法院地法** |
| **扶养** | **有利于被扶养人**的一方经常居所地法/国籍国法 |
| **监护** | **被监护人经常居所地法** |
| **法定继承** | **被继承人死亡时经常居所地法** |
| **不动产** | **不动产所在地法**（绝对硬性！） |
| **合同之债** | **意思自治** → 特征履行方经常居所地法 → 最密切联系 |
| **一般侵权** | **侵权行为地法** |
| **产品责任** | **被侵权人有选择权**：经常居所地/侵权人主营业地/产品取得地 |

## 三、国际经济法（International Economic Law，~3分）

### 3.1 WTO基本原则

| 原则 | 内容 | 主要例外 |
|------|------|---------|
| **最惠国待遇（MFN）** | 给予一国好处自动给所有成员 | FTA/关税同盟/普惠制 |
| **国民待遇（NT）** | 外国人不低于本国人 | 政府采购/补贴 |
| **透明度原则** | 公布法规、公正实施 | 国家安全机密 |

### 3.2 贸易救济措施（三反一保）

| 措施 | 触发条件 | 措施形式 |
|------|----------|----------|
| **反倾销** | 出口价 < 正常价值 + 损害 + 因果关系 | 征收反倾销税 |
| **反补贴** | 专向性补贴 + 损害 | 征收反补贴税 |
| **保障措施** | 进口激增 + 严重损害（**不需证明不公平**） | 提高关税/数量限制，**针对所有来源国** |

> 📌 "反倾销比价格，反补贴查政府钱，保障措施看激增"

### 3.3 外商投资法（2019年，2020.1.1施行）

- **准入制度**：**准入前国民待遇 + 负面清单管理**
- **负面清单**：清单内限制/禁止；清单外内外资一致
- **投资保护**：征收补偿、利润自由汇出、知识产权保护

### 3.4 国际税收

| 方法 | 内容 |
|------|------|
| **免税法** | 居住国对境外已纳税所得免税（最优惠） |
| **抵免法（最常用）** | 境外税款从本国应纳税款中**抵免** |
| **税收饶让** | 视同已在来源国缴税（发展中国家争取的优惠）|

**中国采用**：**限额抵免法**

## 四、跨领域交叉场景速查

| 场景 | 涉及法律 | 处理要点 |
|------|----------|----------|
| 外国人犯罪 | 刑法（属地管辖）+ 引渡法 | 属地优先；可能涉及引渡 |
| 跨国婚姻纠纷 | 国际私法（涉外民事关系法律适用法） | 结婚/离婚分别找连接点 |
| 外商投资企业劳动争议 | 劳动法 + 涉外民诉 + 外商投资法 | 先走劳动仲裁（必经） |
| 海上碰撞事故 | 海洋法 + 侵权之债冲突规范 + 海商法 | 侵权行为地法（可选法院地法） |
| 国际货物买卖合同纠纷 | 国际私法 + Incoterms + CISG | 当事人可择法；选缔约国法则CISG自动适用 |
| 外国法院判决在中国执行 | 国际民事诉讼法（承认执行条件） | 中级法院审查6项条件 |
