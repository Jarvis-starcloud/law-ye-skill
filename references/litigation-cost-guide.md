# 诉讼费用计算指南

> 定位：打一场官司到底要花多少钱——费用速算+减免+承担规则
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块5交付物
> 依据：《诉讼费用交纳办法》（国务院令第481号）

---

## 一、诉讼费速算表

### 1.1 财产案件受理费

| 标的额（元） | 费率 | 速算公式 |
|-------------|------|----------|
| ≤ 10,000 | 固定 | **50元** |
| 10,001 - 100,000 | 2.5% | 标的额 × 2.5% - 200 |
| 100,001 - 200,000 | 2% | 标的额 × 2% + 300 |
| 200,001 - 500,000 | 1.5% | 标的额 × 1.5% + 1,300 |
| 500,001 - 1,000,000 | 1% | 标的额 × 1% + 3,800 |
| 1,000,001 - 2,000,000 | 0.9% | 标的额 × 0.9% + 4,800 |
| 2,000,001 - 5,000,000 | 0.8% | 标的额 × 0.8% + 6,800 |
| 5,000,001 - 10,000,000 | 0.7% | 标的额 × 0.7% + 11,800 |
| 10,000,001 - 20,000,000 | 0.6% | 标的额 × 0.6% + 21,800 |
| > 20,000,000 | 0.5% | 标的额 × 0.5% + 41,800 |

**快速计算示例**：
- 告5万：50,000 × 2.5% - 200 = **1,050元**
- 告20万：200,000 × 2% + 300 = **4,300元**
- 告50万：500,000 × 1.5% + 1,300 = **8,800元**
- 告100万：1,000,000 × 1% + 3,800 = **13,800元**

### 1.2 非财产案件受理费

| 案件类型 | 费用 |
|----------|------|
| 离婚案件 | **50-300元**（涉及财产分割超20万部分按0.5%） |
| 侵犯人身权（非财产）| **100-500元** |
| 知识产权（非财产）| **500-1,000元** |
| 行政案件 | **50元** |
| 劳动仲裁 | **免费** |
| 商事仲裁 | 按仲裁委标准（通常比法院贵） |

### 1.3 其他诉讼费

| 费用类型 | 标准 |
|----------|------|
| 申请保全 | 保全金额的1%，最低不少于30元 |
| 申请执行 | 执行金额1%-0.5%（≤1万免交），最高不超5万 |
| 上诉 | 按一审标准计算 |
| 申请再审 | 免交 |

## 二、律师费参考标准

### 2.1 收费方式

| 方式 | 适用场景 | 说明 |
|------|----------|------|
| **按件收费** | 简单案件 | 每件X元（通常3,000-30,000元） |
| **按标的额比例** | 财产类案件 | 标的额的3%-8%（各地不同） |
| **计时收费** | 非诉/咨询 | 200-5,000元/小时 |
| **风险代理** | 原告胜诉概率较高 | 胜诉后按回款比例（10%-30%） |

### 2.2 各地律师费参考范围

| 城市 | 简单案件 | 标的额10万 | 标的额100万 |
|------|----------|-----------|------------|
| 北京 | 5,000-30,000 | 5,000-12,000 | 30,000-80,000 |
| 上海 | 5,000-25,000 | 5,000-10,000 | 25,000-60,000 |
| 深圳 | 3,000-20,000 | 3,000-8,000 | 20,000-50,000 |
| 二三线城市 | 2,000-10,000 | 2,000-5,000 | 10,000-30,000 |

> **实务提醒**：律师费通常由委托人自行承担。部分案件（如知识产权侵权、不正当竞争）可以请求败诉方承担合理律师费。

## 四、诉讼费减免

### 4.1 免交条件

| 情形 | 依据 |
|------|------|
| 残疾人无固定收入 | 诉讼费用交纳办法第45条 |
| 享受最低生活保障 | 同上 |
| 社会福利机构/救助站 | 同上 |
| 因见义勇为引发的诉讼 | 同上 |
| 请求给付赡养费/抚养费/扶养费 | 同上 |

### 4.2 减交/缓交条件

- 经济确有困难（提供：低保证/困难证明/残疾证等）
- 向立案法院提交《诉讼费减免申请书》
- 法院审查后决定是否批准

### 4.3 法律援助

符合条件的当事人可以申请法律援助（免费律师）：
- 经济困难（低保户/低收入家庭）
- 符合法律援助事项范围
- 向当地法律援助中心申请

## 五、费用承担规则

### 5.1 一般规则

| 结果 | 诉讼费承担 |
|------|-----------|
| 原告全部胜诉 | **被告承担全部诉讼费** |
| 被告全部胜诉 | **原告承担全部诉讼费** |
| 部分胜诉 | **按比例分担** |
| 调解结案 | **双方协商承担** |
| 撤诉 | **原告承担**（减半） |

### 5.2 简易程序优惠

适用简易程序审理的案件，诉讼费**减半收取**。
