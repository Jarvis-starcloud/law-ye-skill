# 刑事诉讼流程概览

> 定位：刑事案件从报案到执行的操作指南（侧重被害人/当事人视角）
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块1交付物
> 法律依据：《刑事诉讼法》（2018修正）+相关司法解释

---

## 一、刑事诉讼全流程

```
报案/控告 → 公安机关立案侦查 → 检察院审查起诉 → 法院审判 → 执行
   (7日)      (2个月+延长)        (1个月+延长)     (2-3个月)
                ↓
          不予立案 → 申请复议/复核/监督
```

**三阶段对应机关**：
| 阶段 | 主管机关 | 主要职责 |
|------|----------|----------|
| **侦查** | 公安机关（普通刑案）/ 检察院（职务犯罪等） | 收集证据、抓获嫌疑人 |
| **审查起诉** | 人民检察院 | 决定起诉还是不起诉 |
| **审判** | 人民法院 | 定罪量刑 |

## 二、报案与立案

### 2.1 报案方式

| 方式 | 操作 | 注意事项 |
|------|------|----------|
| **110报警** | 电话报案 | 紧急情况首选 |
| **派出所报案** | 到案发地/居住地派出所 | 带身份证+证据材料 |
| **网上报案** | 各地公安局网站/APP | 非紧急案件 |
| **检察院控告** | 向检察院控告申诉部门 | 公安不立案时的救济途径 |

### 2.2 报案材料准备

| 材料 | 说明 |
|------|------|
| 身份证 | 报案人身份证明 |
| 报案材料/控告书 | 写清时间、地点、人物、经过、损失 |
| 证据材料 | 转账记录、聊天记录、合同、伤情照片等 |
| 嫌疑人信息 | 姓名、电话、地址（知道多少写多少） |

### 2.3 公安不立案怎么办

| 救济途径 | 操作 | 法律依据 |
|----------|------|----------|
| **申请复议** | 向作出不立案决定的公安机关申请 | 《刑事诉讼法》第113条 |
| **向上级复核** | 对复议不服，向上级公安机关申请复核 | 同上 |
| **检察监督** | 向检察院控告申诉部门反映 | 第113条 |
| **刑事自诉** | 部分案件可直接向法院起诉 | 第210条 |

### 2.4 自诉案件范围

以下三类案件可以不经公安直接到法院起诉：
1. **告诉才处理**的案件：侮辱/诽谤、暴力干涉婚姻自由、虐待、侵占
2. **被害人有证据证明的轻微刑事案件**：故意伤害（轻伤）、非法侵入住宅、重婚等
3. **公安/检察院不追诉的案件**：有证据证明对方有罪且公安不立案/检察院不起诉

## 三、侦查阶段

### 3.1 侦查期限

| 措施 | 期限 | 说明 |
|------|------|------|
| **拘留** | 一般**3日**+延长1-4日，流窜/多次/结伙最长**37日** | 到期必须释放或变更强制措施 |
| **逮捕后侦查羁押** | **2个月**，可延长至**7个月**（特殊案件） | 延长需上级检察院批准 |
| **取保候审** | 最长**12个月** | 不羁押，但限制自由 |
| **监视居住** | 最长**6个月** | 在指定住所执行 |

### 3.2 被害人在侦查阶段的权利

| 权利 | 内容 |
|------|------|
| 报案控告权 | 向公安报案、向检察院控告 |
| 委托诉讼代理人 | 可以委托律师作为代理人 |
| 了解案件进展 | 有权询问公安机关案件办理情况 |
| 申请补充侦查 | 认为公安遗漏重要证据的 |
| 申请复议/复核 | 公安不立案时的救济 |

### 3.3 犯罪嫌疑人的权利

| 权利 | 时间节点 |
|------|----------|
| **委托辩护人** | 自被侦查机关第一次讯问或采取强制措施之日起 |
| **申请取保候审** | 任何阶段均可申请 |
| **沉默权（有限）** | 有权拒绝回答与案件无关的问题 |
| **核对笔录** | 讯问笔录应允许核对，可以补充/改正 |
| **法律援助** | 经济困难或其他原因可申请 |

## 四、审查起诉阶段

### 4.1 基本流程

```
公安移送 → 检察院审查(1个月+延长半月) → 起诉/不起诉/退回补充侦查
```

### 4.2 审查期限

| 情形 | 期限 |
|------|------|
| 一般案件 | **1个月**，可延长**半个月** |
| 退回补充侦查 | 每次**1个月**，最多退回**2次** |
| 重新计算 | 补充侦查完毕移送后重新计算审查期限 |

### 4.3 处理结果

| 结果 | 条件 | 后续 |
|------|------|------|
| **提起公诉** | 犯罪事实清楚、证据确实充分 | 移送法院审判 |
| **不起诉** | 法定不起诉/酌定不起诉/存疑不起诉 | 释放嫌疑人 |
| **退回补充侦查** | 证据不足 | 退回公安补充调查 |

### 4.4 被害人对不起诉的救济

1. **向上一级检察院申诉**→上级检察院可以撤销不起诉决定
2. **向法院提起自诉**→对于法定范围内的案件

## 五、审判阶段

### 5.1 一审期限

| 程序 | 期限 | 延长 |
|------|------|------|
| 简易程序 | **20日**（可能判3年以上的延至1.5个月） | — |
| 速裁程序 | **10日**（可能判1年以上的延至15日） | — |
| 普通程序 | **2个月**（可延长至3个月） | 特殊案件经上级批准可再延长 |

### 5.2 庭审流程

| 阶段 | 内容 |
|------|------|
| **1. 开庭准备** | 核实身份、宣布合议庭成员、告知权利 |
| **2. 法庭调查** | 公诉人宣读起诉书→被告人陈述→讯问被告→询问证人→出示证据→质证 |
| **3. 法庭辩论** | 公诉人发言→被告/辩护人发言→被害人/代理人发言→互相辩论 |
| **4. 被告人最后陈述** | 被告人最后的发言机会 |
| **5. 宣判** | 当庭宣判或择日宣判 |

### 5.3 刑事附带民事诉讼

**被害人可以在刑事诉讼中一并提出民事赔偿请求**。

| 要素 | 规则 |
|------|------|
| 提起时间 | 在刑事案件立案后一审判决前 |
| 赔偿范围 | 因犯罪行为造成的**物质损失**（医疗费/误工费/丧葬费等） |
| 精神损害赔偿 | **不支持**（刑事附带民事不受理精神损害赔偿） |
| 优势 | 不用另行起诉，节省时间和诉讼费 |

> **实务提醒**：如果想主张精神损害赔偿，需要另行提起民事诉讼。

## 六、上诉与抗诉

### 6.1 上诉

| 要素 | 规则 |
|------|------|
| 上诉人 | 被告人/自诉人（被害人不能上诉，但可请求检察院抗诉） |
| 上诉期 | 判决后**10日**，裁定后**5日** |
| 上诉不加刑 | **仅被告方上诉的，二审不得加重刑罚** |

### 6.2 抗诉

| 要素 | 规则 |
|------|------|
| 抗诉人 | 人民检察院 |
| 二审抗诉 | 检察院认为一审判决确有错误 |
| 被害人申请 | 被害人不服一审判决→请求检察院抗诉→检察院决定是否抗诉 |

## 七、被害人维权实操指南

### 7.1 全流程行动清单

| 阶段 | 被害人应该做什么 |
|------|-----------------|
| **报案时** | 准备详细材料、保留报案回执单、记录接警民警信息 |
| **侦查时** | 委托律师代理、定期联系办案民警了解进展、补充提交证据 |
| **审查起诉时** | 向检察院表达意见、提交书面材料、商议附带民事赔偿 |
| **审判时** | 出庭参加诉讼、提起附带民事诉讼、对判决发表意见 |
| **判决后** | 不满意可请求检察院抗诉（10日内）、附带民事部分可上诉 |

### 7.2 谅解书与调解

- 被害人出具谅解书→可作为法定从轻情节
- **谅解≠放弃追诉**，仅影响量刑
- 谅解通常伴随经济赔偿
- 赔偿金额协商要点：
  - 实际损失（医疗费/误工费等）全额赔偿
  - 精神抚慰金：根据伤害程度协商
  - 谅解书中写明"请求从轻处理"的程度

## 八、常见问题速查

| 问题 | 答案 |
|------|------|
| 报案后多久立案？ | 公安应当**立即审查**，认为有犯罪事实需追诉的应当立案。实务中视案件复杂程度数日到数周不等 |
| 取保候审是不是没事了？ | 不是！取保候审只是不羁押，案件仍在办理中 |
| 朋友被抓了怎么办？ | 第一时间委托律师，律师有权在侦查阶段会见嫌疑人 |
| 被害人能看案卷吗？ | 侦查阶段不能。审查起诉阶段被害人的代理律师可以查阅案卷 |
| 刑事案件有调解吗？ | 部分轻微刑事案件适用刑事和解（《刑事诉讼法》第290条） |
| 能私了吗？ | 自诉案件可以撤诉。公诉案件不能"私了"不追究，但和解/谅解可影响量刑 |
| 二审一般多久？ | 2个月（可延长至2.5个月），实务中通常2-4个月 |
