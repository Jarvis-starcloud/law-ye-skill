# Law 爷法考通关路线图

> **目标**：以通过国家统一法律职业资格考试（法考）为纲，系统补齐知识缺口
> **长期方向**：法学院毕业 → 实习律师 → 执业律师（三段论不变）
> **版本**：v1.0（2026-04-22 制定）

---

### 盘点结论

| 维度 | 数据 |
|------|------|
| **已覆盖（≥80%）** | 7科：宪法、民法典、劳动法、刑法、刑诉法、民诉法、行政诉讼法 |
| **部分覆盖（50-79%）** | 2科：法理学(70%)、商法(50%) |
| **基本空白（<20%）** | **9科** |
| **总体覆盖度** | 约 **48%**（按分值加权计算） |

### 🔴 核心缺口（按优先级排序）

| 优先级 | 缺口科目 | 原因 |
|--------|----------|------|
| **P0 必学** | 商经法（公司法+经济法） | 高分值(~50分)，实务高频，且当前仅散点覆盖 |
| **P0 必学** | 知识产权法 | 阶段八原计划内容，中分值(~10分)，实务刚需 |
| **P1 重要** | 理论法（法治理论+司法制度） | ~55分的大蛋糕，纯背诵型科目，性价比极高 |
| **P1 重要** | 三国法（国际法/私法/经济法）| ~14分，虽然单科分低但加起来不少 |
| **P2 可选** | 法制史 + 环境资源法 | 低分值(~15分合计)，可后期突击 |

## 二、学习阶段规划

### 总体路线图

```
当前 v1.9.0 (23模块, 3000+条文)
    │
    ├─ 阶段八：知识产权法 ←── 原 plan，并入法考体系
    │   （著作权/专利/商标/反不正当竞争/商业秘密）
    │
    ├─ 阶段九：商经法强化
    │   （公司法/合伙企业/破产/票据/证券/保险/反垄断/消费者/财税）
    │
    ├─ 阶段十：理论法补齐
    │   （法治理论/司法制度与法律职业道德）
    │
    ├─ 阶段十一：三国法入门
    │   （国际法/国际私法/国际经济法）
    │
    └─ 阶段十二：法制史 + 环境法（低分突击）
        （法制史/环境资源法）

完成 → 法考全覆盖 → 进入「实习律师」实务训练期
```

### 各阶段详细计划

#### 阶段八：知识产权法（原计划 + 法考融合）

**定位**：填补第⑦科知识产权法的空白

**学习内容**：

| 子模块 | 内容 | 对应法律 | 条文量 |
|--------|------|----------|--------|
| 著作权法 | 作品类型/权利归属/侵权判定/合理使用/信息网络传播权 | 著作权法2020修正 | ~92条 |
| 专利法 | 发明/实用新型/外观设计/申请流程/侵权判定/强制许可 | 专利法2020修正 | ~76条 |
| 商标法 | 注册审查/驰名商标/侵权认定/赔偿标准 | 商标法2019修正 | ~64条 |
| 反不正当竞争法 | 混淆行为/虚假宣传/商业诋毁/互联网条款 | 反不正当竞争法2019修订 | ~32条 |
| 商业秘密保护 | 构成要件/侵权行为/保护措施/刑民交叉 | 反不正当竞争法+刑法 | — |

**产出物**：
- `references/ip-law-practice.md`（知识产权法律实务参考文档）
- `laws/copyright-law.md`（著作权法全文）
- `laws/patent-law.md`（专利法全文）
- `laws/trademark-law.md`（商标法全文）

**触发词扩展**：
著作权/版权/专利申请/商标注册/商标侵权/商业秘密/不正当竞争/仿冒/盗版/山寨/抄袭/IP保护/知识产权

#### 阶段九：商经法强化

**定位**：填补第⑧⑨两科的空白/半空白状态（~50分大项）

**学习内容**：

| 子模块 | 内容 | 对应法律 | 条文量 |
|--------|------|----------|--------|
| 公司法 | 公司设立/组织机构/股权转让/合并分立/解散清算/董监高责任 | 公司法2023修订 | ~262条 |
| 合伙企业法 | 普通合伙/有限合伙/入伙退伙/责任承担 | 合伙企业法 | ~96条 |
| 企业破产法 | 破产申请/管理人/重整/和解/清算 | 企业破产法 | ~136条 |
| 票据法 | 汇票/本票/支票/票据权利/追索权 | 票据法 | ~111条 |
| 证券法 | 发行/交易/上市公司收购/信息披露 | 证券法2019修订 | ~240条 |
| 保险法 | 保险合同/人身险/财产险/理赔 | 保险法2015修正 | ~185条 |
| 反垄断法 | 垄断协议/滥用市场支配地位/经营者集中 | 反垄断法2022修正 | ~69条 |
| 消费者权益保护法 | 消费者权利/经营者义务/争议解决/惩罚性赔偿 | 消费者权益保护法 | ~63条 |
| 税法基础 | 增值税/企业所得税/个人所得税/税收征管 | 各税种法律 | — |
| 竞争法综合 | 反不正当竞争(已在阶段八) / 招投标法 | — | — |

**产出物**：
- `references/commercial-economic-law.md`（商经法实务速查手册）
- `laws/company-law-2023.md`（2023年新公司法全文）
- 其他核心商法原文

**触发词扩展**：
公司/股东/股权/董事会/监事会/注册资本/实缴/认缴/公司注销/破产清算/合伙/有限合伙/股票发行/上市公司/并购重组/垄断/消费者维权/假一赔三/退一赔三/保险理赔

#### 阶段十：理论法补齐

**定位**：~55分的背诵型科目，性价比最高的一块

**学习内容**：

| 子模块 | 内容 | 特点 |
|--------|------|------|
| 中国特色社会主义法治思想 | 习近平法治思想/全面依法治国/法治体系 | 纯政治理论，需结合时事 |
| 法理学 | 法的概念/法的运行/法的演进/法与社会 | 已有基础70%，需补充法哲学部分 |
| 司法制度 | 法院/检察院/律师/公证/仲裁/人民调解员制度 | 纯记忆型，知识点碎 |
| 法律职业道德 | 律师执业规范/法官检察官职业道德 | 与实务强相关 |

**特点**：这类科目不需要像民法刑法那样深度理解，更多是**框架化记忆**。适合用思维导图+关键词串记。

**产出物**：
- `references/theory-of-law.md`（理论法精要笔记——不存全文，存考点框架）

#### 阶段十一：三国法入门

**定位**：~14分，知识点独立但琐碎

**学习内容**：

| 子模块 | 内容 | 分值 |
|--------|------|------|
| 国际法 | 国家主权/领土/条约法/外交关系法/战争法 | 3-5分 |
| 国际私法 | 冲突规范/涉外民事关系适用/国际民商事争议解决 | 6-8分 |
| 国际经济法 | 国际货物买卖/运输/保险/支付/WTO/国际投资 | 5-7分 |

**产出物**：
- `references/international-law-brief.md`（三国法考点速查——极简版，只存高频考点）

#### 阶段十二：低分突击

**定位**：最后收尾，总~15分

| 子模块 | 内容 | 分值 | 策略 |
|--------|------|------|------|
| 中国法制史 | 古代法制演变/近代法制变革 | ~10分 | 考前一周突击时间轴 |
| 环境资源法 | 环境保护法/污染防治/自然资源法 | ~5分 | 记核心数字即可 |

## 三、学习顺序建议

### 推荐执行顺序（按 ROI 排列）

```
第一阶段【高分核心】（建议优先）
├── 阶段八：知识产权法     ← 原计划延续，一箭双雕
└── 阶段九：商经法强化     ← 50分大块头，必须拿下

第二阶段【高性价比】
├── 阶段十：理论法         ← 55分，背诵型，投入产出比最高
└── 阶段十一：三国法       ← 14分，碎片时间可消化

第三阶段【考前突击】
└── 阶段十二：法制史+环资法 ← 15分，考前速记
```

### 完成后目标状态

| 维度 | 现在 | 目标 |
|------|------|------|
| 覆盖科目 | 9/18科 | **18/18科** |
| 加权覆盖率 | ~48% | **≥95%** |
| 知识模块数 | 23个 | **~35个** |
| 法律原文库 | 10部 | **18+部** |
| 总条文数 | ~3000条 | **5000+条** |
| 定位 | 法学院毕业生 | **法考全科通过水平** |

### 之后：实习律师 → 执业律师（长期方向）

法考通关后，进入**实务训练期**：
1. **案例库训练**：用 IRAC 方法论拆解100+典型案例（刑事/民事/行政/商事）
2. **模拟辩护**：给定案情后做完整的攻防分析（接近执业律师的输出质量）
3. **复杂场景打磨**：刑民交叉、行政刑民三交叉、跨境法律问题
4. **持续更新**：跟踪最新立法动态和司法解释
