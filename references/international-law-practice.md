核心区别：
- 国际法：主体是国家/国际组织 → 调整"公对公"关系 → 无超国家强制力
- 国际私法：主体是自然人/法人 → 解决"跨国私权纠纷用哪国法" → 最终靠国内法院
- 国际经济法：调整跨国经济交往 → 规范贸易/投资/金融/税收行为 → 国内法+国际协定结合
```

### 三科对比速查表

| 维度 | 国际法 | 国际私法 | 国际经济法 |
|------|--------|----------|------------|
| **主体** | 国家、国际组织为主 | 自然人、法人 | 国家+自然人+法人+国际组织 |
| **性质** | 公法 | 私法（程序+实体混合） | 经济法 |
| **核心问题** | 国家间权利义务 | 用什么法？谁管？怎么执行？ | 怎么做生意？怎么管钱？ |
| **强制力** | 无超国家法院（ICJ无强制） | 依靠各国国内法院 | WTO争端机制+国内执法 |
| **法考分值** | ~6分 | ~5分 | ~3分 |
| **特点** | 记忆型为主 | 理解型为主（冲突规范逻辑） | 散点记忆 |

## 二、第一部分：国际法（Public International Law，~6分）

### 2.1 国家与国际法主体

#### 2.1.1 国家的构成要素（四要素缺一不可）🔑

| 要素 | 内容 | 说明 |
|------|------|------|
| **定居的居民** | 永久人口 | 数量不限，但必须有常住人口 |
| **确定的领土** | 固定的疆域 | 大小不限，但必须有明确边界 |
| **政府** | 有效统治的组织 | 必须能对其领土和人民行使有效控制 |
| **主权** | 对内最高、对外独立的权力 | 最核心要素——没有主权就不是国家 |

> ⚠️ 法考陷阱：台湾是中国的一个地方行政区域，不是一个国家——缺少**独立主权**这一核心要素。

**国家的类型**：
- **单一制国家**：中国、日本、法国——中央统一立法、地方无主权
- **联邦制国家**：美国、德国、瑞士——中央+成员邦分享主权，各有宪法地位
- **邦联**：松散联盟（如欧盟早期），不是国家——成员国保留完全主权

#### 2.1.2 国家的基本权利

| 权利 | 含义 | 限制 |
|------|------|------|
| **独立权** | 不受他国干涉，自主决定内外政策 | 不得违反国际义务 |
| **平等权** | 各国法律地位一律平等 | — |
| **自保权** | 单独或集体防御外来侵略 | 禁止使用武力（除自卫和安理会授权） |
| **管辖权** | 对人和物的管理和支配权 | 受属地/属人/保护/普遍管辖限制 |

#### 2.1.3 管辖权的四种类型 🔑（高频考点）

| 类型 | 含义 | 典型场景 |
|------|------|----------|
| **属地管辖** | 领域内一切人和事 | 在中国境内的犯罪/民事纠纷 |
| **属人管辖** | 对本国公民（无论在哪） | 中国公民在海外犯罪 |
| **保护管辖** | 侵害本国重大利益的行为 | 外国人在境外侵害中国国家利益 |
| **普遍管辖** | 严重国际罪行（海盗/战争罪/反人类罪/种族灭绝） | 任何国家对这类罪犯均有权管辖 |

> 📌 **记忆口诀**："地属保普"→ 地(属地)>属(属人)>保(保护)>普(普遍)
>
> 适用顺序：**属地优先 > 属人 > 保护 > 普遍**
>
> 当多个国家都有管辖权时，通常由**最先控制嫌疑人**的国家行使。

#### 2.1.4 国家主权豁免

**概念**：国家的行为和财产未经其同意，不受外国法院管辖。

| 要素 | 内容 |
|------|------|
| **绝对豁免论（传统）** | 国家的一切行为和财产都享有豁免 |
| **有限豁免论（现代趋势）** | 区分**主权行为（公共/统治权行为）**享豁免 vs **商业行为（管理权行为）**不享豁免 |
| **中国的立场演变** | 传统坚持绝对豁免 → **2024年《外国国家豁免法》转向限制豁免** |
| **放弃方式** | 明示（协议/书面声明）或默示（应诉/反诉/介入诉讼程序） |

**2024年《外国国家豁免法》要点**（中国首次专门立法，2024.1.1施行）：

| 要点 | 内容 |
|------|------|
| 基本原则 | 从绝对豁免转向**限制豁免** |
| 商业活动定义 | 合同交易/提供服务/参与市场活动等 |
| 管辖例外 | 商业活动相关争议可在中国法院起诉 |
| 执行例外 | 用于商业目的财产可被强制执行 |
| 仍享豁免事项 | 主权行为（军事/外交/立法/行政） |

### 2.2 国际法上的个人

#### 2.2.1 国籍（Nationality）🔑

**国籍的取得方式**：

| 方式 | 含义 | 示例 |
|------|------|------|
| **出生取得（原始国籍）** | 因出生而自然获得 | 血统主义 / 出生地主义 / 混合制 |
| **加入取得（继有国籍）** | 因自愿申请而获得 | 归化 / 婚姻 / 收养等 |

**中国的国籍制度**：

| 规则 | 内容 |
|------|------|
| **基本原则** | **血统主义为主**——父母一方为中国公民→子女即有中国籍（无论出生地） |
| **双重国籍** | ❌ **不承认双重国籍**（《国籍法》第9条）：取得外国国籍 = 自动丧失中国国籍 |
| **退出条件** | ①外国近亲 ②定居国外 ③正当理由（需批准） |
| **恢复条件** | 曾有过中国籍 + 正当理由（需批准） |
| **特殊情形** | 定居外国的中国公民，自愿加入/取得外国国籍的 = **自动丧失**中国国籍（不保留） |

> 📌 法考重点：中国不承认双重国籍是硬性规定，没有例外。

#### 2.2.2 引渡（Extradition）🔑（高频高频！）

**引渡四原则**（法考必背）：

| 原则 | 内容 | 说明 |
|------|------|------|
| **政治犯不引渡** | 政治犯罪不引渡 | 但什么是"政治犯"无统一标准 |
| **本国公民不引渡** | 多数国家不引渡本国公民 | 大陆法系常见；英美法系可引渡 |
| **双重犯罪原则** | 引渡请求的行为在请求国和被请求国的法律都构成犯罪 | 必须同时触犯两国刑法 |
| **罪名特定原则** | 只能按引渡请求的罪名审判/处罚，不得转审其他罪名 | 防止"借引渡之名行追诉之实" |

**引渡的程序流程**：

```
请求国提出引渡请求
        ↓
被请求国行政机关初审（外交部）
        ↓
可能涉及司法机关审查（最高法/最高检复核）
        ↓
国务院最终决定
        ↓
决定引渡 or 拒绝
        ↓
移交被请求引渡人
```

**中国引渡制度要点**：

| 方向 | 审核流程 |
|------|----------|
| **向外引渡**（把人送出去） | 外交部审核 → 最高法/最高检复核 → **国务院决定** |
| **向内引渡**（把人要回来） | 通过双边引渡条约或互惠原则向外国提出请求 |

> 📌 **著名案例**：赖昌星案（加拿大→中国）、杨秀珠案（荷兰→中国）

#### 2.2.3 庇护（Asylum）

**引渡 vs 庇护对比**（常考点）：

| 维度 | 引渡 | 庇护 |
|------|------|------|
| 性质 | 把人**送出去** | 把人**收进来** |
| 主体 | 国家之间的合作 | 单个国家给予的保护 |
| 对象 | 在被请求国境内的人 | 进入领土/使领馆寻求保护的人 |
| 原因 | 该人在请求国犯了罪 | 因政治原因受到迫害 |
| 法律基础 | 双边引渡条约 | 国内法（无条约要求） |

**领域庇护 vs 使馆庇护**：

| 类型 | 是否合法 | 说明 |
|------|----------|------|
| **领域庇护（领土庇护）** | ✅ 合法 | 国家在本国领土内给予外国人政治庇护，受国际习惯法承认 |
| **使馆庇护** | ❌ 一般非法 | 利用驻外使馆庇护他国公民 = 违反接受国主权（拉丁美洲有区域惯例例外） |

### 2.3 领土法

#### 2.3.1 领土的取得与变更方式

**传统方式（五种）**：

| 方式 | 定义 | 现代合法性 |
|------|------|------------|
| **先占（Occupation）** | 占领**无主地** | 已基本不可能（地球无主地几乎不存在） |
| **时效（Prescription）** | 长期公开和平占有他国领土 | ❌ 不被现代国际法认可 |
| **添附（Accretion）** | 自然或人工增加领土 | ✅ 自然添附合法；人工添受限制 |
| **征服（Conquest）** | 武力占领后兼并 | ❌ **《联合国宪章》明文禁止** |
| **割让（Cession）** | 通过条约转让领土 | ✅ 自愿割让合法；强迫割让非法 |

**现代合法变更方式**：
- **民族自决**：殖民地/非自治领土独立
- **全民公投**：居民投票决定归属（须自由公正）
- **交换领土**：两国协商互换
- **收回失地**：收复被非法占领的本国领土

#### 2.3.2 边界与边境制度

**划界三步骤**：
1. **定界（Delimitation）**：签订边界条约确定界线位置
2. **标界（Demarcation）**：实地竖立界碑/界标
3. **管理**：建立边境管理制度

**边境制度核心规则**：
- **界河**：通常以**主航道中心线**为界（通航河流）或**河道中心线**（不通航）
- **界湖**：一般以**中间线**为界
- 边境建筑不得威胁对方安全
- 出入境各自依国内法管理
- 跨界污染需合作处理

### 2.4 海洋法 🔑（法考重中之重！）

#### 2.4.1 海洋区域划分全景图

```
内陆国/沿海国的基线（正常基线/直线基线）
        │
        ├── 内水（Internal Waters）← 完全主权
        │     └── 港口/海湾/河口/内海
        │
        ├── 领海（Territorial Sea）← ≤12海里，完全主权
        │     └── 无害通过权（外国船舶可无害通过）
        │
        ├── 毗连区（Contiguous Zone）← ≤24海里（=领海+最多12）
        │     └── 管辖权：海关/财政/移民/卫生
        │
        ├── 专属经济区（EEZ）← ≤200海里（=领海+最多188）
        │     └── 主权权利：勘探开发自然资源
        │     └── 管辖权：人工岛屿/科研环保
        │
        ├── 大陆架（Continental Shelf）← 自然延伸，最远350海里
        │     └── 上覆水域=EEZ法律地位
        │     └── 海床底土：勘探开发自然资源的主权权利
        │
        └── 公海（High Seas）← 专属经济区以外
              └── 公海自由（航行/飞越/铺设电缆/捕鱼/科研）
              └── 船旗国管辖
```

#### 2.4.2 各海域法律地位详解

**① 内水（Internal Waters）**

| 要点 | 内容 |
|------|------|
| **范围** | 领海基线向陆一侧的水域（港口、海湾、河口、内海） |
| **法律地位** | **完全的国家领土主权**——等同于陆地领土 |
| **外国船舶进入** | 需经沿海国批准（除非遇难/不可抗力等特殊情况） |
| **港口国管辖** | 停泊港内的外国船舶：刑事管辖有限制；民事管辖基本不管 |

**② 领海（Territorial Sea）** 🔑

| 要点 | 内容 |
|------|------|
| **宽度** | 从基线量起**不超过12海里**（中国主张12海里） |
| **法律地位** | **完全的国家主权**——但必须允许**无害通过** |
| **无害通过（Innocent Passage）** | 外国非军用船舶可不事先通知/获准连续不停地通过 |
| **无害通过的条件** | ①连续不停 ②不损害沿海国和平安宁良好秩序 ③水下潜艇需浮出水面并展示旗帜 |
| **军舰的无害通过** | 多数国家要求**事先通知或批准**（中国要求批准） |
| **管辖权例外** | 通过期间的民事管辖一般不行使；刑事案件仅限于扰乱安宁/涉及船员以外的人 |

**③ 毗连区（Contiguous Zone）**

| 要点 | 内容 |
|------|------|
| **范围** | 领海以外**不超过24海里**（即领海外延最多12海里） |
| **性质** | **不是**独立海域，是领海和EEZ之间的缓冲带 |
| **管辖内容** | 仅限四项：**海关、财政、移民、卫生** |
| **⚠️ 注意** | 毗连区**不享有完全主权**，只是可在这四方面行使管制权 |

**④ 专属经济区（Exclusive Economic Zone, EEZ）** 🔑

| 要点 | 内容 |
|------|------|
| **范围** | 领海以外**不超过200海里** |
| **法律地位** | **既非公海也非领海**——自成一类的特殊海域 |
| **沿海国权利** | ①**勘探开发养护管理**自然资源（生物+非生物）的主权权利；②人工岛屿/设施建造；③海洋科学研究管辖；④海洋环境保护管辖 |
| **他国权利** | **航行、飞越、铺设海底电缆管道的自由**（类似公海自由） |
| **⚠️ 核心区别于领海**：EEZ内他国有**航行飞越自由**，领海只有无害通过权 |

**⑤ 大陆架（Continental Shelf）**

| 要点 | 内容 |
|------|------|
| **范围** | 领海以外陆地领土的自然延伸，最远**不超过350海里**（或100海里+2500米等深线外100海里） |
| **权利内容** | 海床底土**自然资源**（海床+底土，不含上覆水域）的勘探开发主权权利 |
| **⚠️ 特点** | 大陆架权利是**固有的**，不需要宣布或占领 |
| **vs EEZ** | 如果自然延伸<200海里，大陆架=EEZ海床部分；如果>200海里，可超出EEZ |

**⑥ 公海（High Seas）**

| 要点 | 内容 |
|------|------|
| **定义** | 不包括在任何国家专属经济区/领海/内水/群岛水域的全部海域 |
| **公海自由六大项** | 航行自由 / 飞越自由 / 铺设海底电缆管道自由 / 建造人工岛屿自由 / 捕鱼自由 / 科研自由 |
| **管辖原则** | **船旗国管辖**——船舶悬挂哪国国旗就由哪国管辖 |
| **登临权（Right of Visit）** | 军舰/政府船可在公海上登临嫌疑船只（海盗/奴隶贩运/无国籍/冒挂国旗/广播海盗） |
| **紧追权（Hot Pursuit）** | 从内水/领海/毗连区/EEZ开始追逐，可追至公海（须连续不间断） |

#### 2.4.3 海洋法核心数字速查表 🔑

```
内水      ← 基线向陆一侧          → 完全主权
领海      ← 基线起≤12海里         → 完全主权 + 无害通过
毗连区    ← 基线起≤24海里         → 仅海关/财政/移民/卫生管制
EEZ      ← 基线起≤200海里        → 自然资源主权权利 + 他国航行飞越自由
大陆架    ← 自然延伸≤350海里      → 海床底土自然资源权利（固有）
公海      ← 以上全部海域以外       → 公海自由 + 船旗国管辖

记忆口诀：12领海 24毗连 200专属 350大陆
```

### 2.5 空间法

#### 2.5.1 国际航空法

**两大法律体系**：

| 体系 | 名称 | 核心内容 |
|------|------|----------|
| **民用航空** | 《国际民用航空公约》（《芝加哥公约》，1944） | 领空主权、航班制度、航空器国籍 |
| **空间活动** | 《外层空间条约》（1967）等五项公约 | 外空探索利用、责任制度、登记制度 |

**领空主权三原则**：
1. **完全排他的主权**：每一国对其领空享有完全的和排他的主权
2. **航空器国籍**：以注册国为国旗国
3. **航班制度**：
   - **定期航班**：需要对方国**许可**（协议/批准）
   - **不定期航班**：可不经许可**飞入/降落**（但需遵守规章）

#### 2.5.2 外层空间法

**核心原则**：
- 外空**不得据为己有**（包括月球等天体）
- **为全人类利益**探索利用
- **和平利用外空**（禁止部署大规模杀伤性武器）
- **国际责任**：发射国对其空间物体造成的损害承担**国际责任**

**损害赔偿责任制**：

| 情形 | 责任方 | 责任类型 |
|------|--------|----------|
| 发射国空间物体对他国地面/飞行中飞机造成损害 | **发射国绝对责任**（无论有无过错） | 无过错责任 |
| 两发射国空间物体在外空相撞 | **各自承担**（按过失比例分担） | 过错责任 |

### 2.6 条约法 🔑（维也纳条约法公约，1969/1986）

#### 2.6.1 条约的缔结程序

```
谈判 → 签署（认证）→ 批准 → 交存批准书 → 生效
        ↓
    可选：加入 / 接受 / 赞同
```

**关键概念辨析**：

| 概念 | 含义 | 法律效果 |
|------|------|----------|
| **签署（Signature）** | 代表在约文上签字 | 认证约文文本；**不必然产生拘束力** |
| **批准（Ratification）** | 国家权力机关确认条约 | 使条约对国家生效；签署后需经国内批准程序 |
| **加入（Accession）** | 未签署国表示同意受约束 | 效果等同于批准 |
| **保留（Reservation）** | 排除/修改某条款效力 | 单方面声明；其他缔约方可接受或反对 |

**保留的规则** 🔑：
- **不得保留的情形**：①条约本身禁止保留 ②仅允许特定保留而你的不在其中 ③保留与条约目的宗旨不符
- **保留的效果**：保留国与接受保留国之间→按保留后条文执行
- **反对保留**：提出保留的国家与反对保留国之间→该条款**不适用**于彼此

#### 2.6.2 条约的效力

| 要点 | 内容 |
|------|------|
| **时间效力** | 原则上**不溯及既往**（除非有相反规定/意图） |
| **空间效力** | 适用于全部领土（除非另有声明） |
| **条约与第三国** | 原则上**不及于第三方**（"约定对第三方无损益"） |
| **条约冲突解决** | ①同一国家先后缔结的条约：后约优于先约（不同主题除外）；②《联合国宪章》义务优先 |

#### 2.6.3 条约的解释

**解释规则**（第31-32条）🔑：

1. **善意解释原则**：按文字的通常含义、上下文、目的宗旨进行**善意的**解释
2. **上下文**：包括序言、附件、全体缔约国间相关协定
3. **补充解释资料**：以上方法仍模糊时，可参考**准备工作**（travaux préparatoires）
4. **多语言条约**：各种作准文本**同等效力**，推定含义相同

### 2.7 外交关系法 🔑（维也纳外交关系公约，1961）

#### 2.7.1 外交机关体系

```
中央外交机关                    驻外外交机关
┌──────────────┐              ┌──────────────────┐
│ 国家元首（最高）│              │ 常驻外交代表机关    │
│ 政府（行政首脑）│              │   （大使馆/公使馆）│
│ 外交部        │──派往/接受──→│ 特别使团          │
│ 派驻外国      │              │ 联合国代表团       │
│ 的外交部门    │              │                  │
└──────────────┘              └──────────────────┘
```

**外交人员的派遣与接受**：

| 程序 | 说明 |
|------|------|
| **派遣** | 派遣国提名 → 征求接受国**同意**（馆长需事先同意） |
| **到任** | 向接受国国家元首递交**国书**后才正式开始职务 |
| **拒绝（不受欢迎的人）** | 接受国**无需说明理由**即可拒绝任何人 |

#### 2.7.2 外交特权与豁免 🔑（必背！）

**使馆的特权与豁免**：

| 特权/豁免 | 内容 | 限制 |
|-----------|------|------|
| **馆舍不可侵犯** | 接受国官员非经馆长许可不得进入（火灾/灾害例外） | — |
| **档案文件不可侵犯** | 绝对保护 | — |
| **通讯自由** | 使用密码/外交信使；可设无线电台（须经同意） | — |
| **免纳捐税** | 使馆不动产和公务用品免税 | 服务性费用不免 |
| **行动自由** | 在接受国境内自由旅行 | 除禁区外 |

**外交人员的特权与豁免**：

| 特权/豁免 | 内容 | ⚠️ 关键限制 |
|-----------|------|-------------|
| **人身不可侵犯** | 不得逮捕/拘留/搜查 | 但可**制止犯罪**或**正当防卫** |
| **寓所/文书/财产不可侵犯** | 不得搜查/查封/扣押 | — |
| **管辖豁免（刑事）** | **完全豁免**——绝对不受当地刑事管辖 | 无例外！ |
| **管辖豁免（民事/行政）** | 原则豁免，但有**4大例外**⬇️ | 见下表 |
| **作证义务免除** | 无作证义务 | — |
| **免纳捐税** | 个人物品和收入免税 | 具体服务性费用不免 |
| **免除关税行李查验** | 入境私人用品免税 | 禁止/检疫物品可检查但应陪同 |

> 📌 **外交人员民事管辖豁免4大例外** 🔑（法考高频！）：
>
> 1. **私有不动产物权诉讼**
> 2. **以私人身份参与继承诉讼**
> 3. **以私人身份从事商务/专业活动**的诉讼
> 4. **主动起诉而被反诉**

**外交特权的期限**：
- 从**入境**开始（已在境内的从**得知任命**开始）
- 到**离境**结束（如因故停留也享有，但不限于原职）

#### 2.7.3 领事关系法（维也纳领事关系公约，1963）

**领事 vs 外交对比** 🔑：

| 维度 | 外交代表 | 领事代表 |
|------|----------|----------|
| **职能** | 代表国家整体利益 | 保护本国公民/商业利益 |
| **特权级别** | 高 | 较低 |
| **馆舍不可侵犯** | 绝对（非经馆长许可不得进入） | 相对（遇紧急情况可征得馆长同意进入） |
| **刑事豁免** | 完全豁免 | 仅对**执行职务行为**享豁免 |
| **民事豁免** | 有4大例外 | 仅对**职务行为**享豁免 |
| **人身不可侵犯** | 绝对 | 受严重犯罪时可逮捕（但应对其尊重） |

### 2.8 国家责任

#### 2.8.1 国际不法行为的构成要件

| 要件 | 内容 |
|------|------|
| **行为归因于国家** | 国家机关的行为/实际上按国家指示行事的人的行为/别国机关经本国授权的行为 |
| **违背国际义务** | 作为或不作为违反了该国有效的国际义务 |

**解除行为不法性的情形**：

| 事由 | 条件 |
|------|------|
| **同意** | 有效且不违反强行法 |
| **对抗措施** | 对抗他国的先前不法行为（须成比例） |
| **不可抗力** | 不可预见/无法控制/无法避免的外力事件 |
| **危难** | 别无选择以保全生命 |
| **紧急状态** | 保护根本利益/唯一手段/不严重损害他国 |

#### 2.8.2 国家责任的形式

| 形式 | 含义 |
|------|------|
| **终止不法行为** | 停止持续性的不法行为 |
| **承诺并保证不再重犯** | 承诺不再违反 |
| **恢复原状** | 尽可能恢复到不法行为发生前的状态 |
| **赔偿** | 弥补任何可用恢复原状方式弥补的损害（物质+精神） |
| **抵偿** | 正式承认错误/道歉/纪念仪式等 |

### 2.9 国际争端解决

#### 2.9.1 和平解决争端的方法

**强制性 vs 非强制性**：

| 类型 | 方法 | 特点 |
|------|------|------|
| **非强制** | 谈判/协商、斡旋/调停/调解、调查/和解 | 自愿、无约束力 |
| **强制** | 战争/武力（❌已禁止）、反报/报复（有限合法）、**司法解决** | — |

#### 2.9.2 国际法院（ICJ）🔑

**性质**：联合国主要司法机关，位于海牙。

| 要点 | 内容 |
|------|------|
| **当事人资格** | 仅限**国家**（个人/法人/国际组织不能直接起诉） |
| **管辖权基础** | ①自愿管辖（当事国提交"强制管辖声明"）；②协定管辖（条约中的管辖条款）；③选择条款（任择条款） |
| **判决效力** | **终局判决，不得上诉**，但对双方有拘束力 |
| **执行** | 如一方不履行 → 可向安理会申诉 |
| **法官回避制度** | 一方当事人是本国时，该国法官**不必回避**（不同于国内法） |

> 📌 法考陷阱：**个人不能直接去ICJ告国家**。必须通过本国政府提起。

## 三、第二部分：国际私法（Private International Law，~5分）

### 3.1 冲突规范（Conflict Rules）🔑（本学科核心！）

#### 3.1.1 冲突规范的结构

```
连接点（Point of Contact）
    ↓
[范围] + [系属] = [法律关系] 应适用 [某国法律]
    ↑                ↑
   调整对象         连接因素指引的法律
```

**示例**："不动产纠纷 → 适用不动产所在地法"
- **范围（对象）**：不动产纠纷
- **系属（指引）**：适用不动产所在地法
- **连接点**：不动产所在地

#### 3.1.2 四种基本类型

| 类型 | 结构 | 示例 |
|------|------|------|
| **单边冲突规范** | 直接指定适用内国法 | "中国境内履行的合同适用中国法律" |
| **双边冲突规范** | 抽象连接点，可能指向内国法也可能指向外国法 | "不动产适用物之所在地法" |
| **重叠适用的冲突规范** | 必须同时满足两个/多个条件 | "离婚必须同时符合夫之本国法和妻之本国法" |
| **选择适用的冲突规范** | 选择其一即可（无条件/有序） | "婚姻财产可选择丈夫住所地法或法院地法" |

#### 3.1.3 连接点分类

| 分类标准 | 类型 | 示例 |
|----------|------|------|
| **客观 vs 主观** | 客观连接点 | 国籍、住所、居所、物之所在地、行为地、法院地 |
| | 主观连接点 | 当事人意思自治（最常用！） |
| **静态 vs 动态** | 静态（不变） | 不动产所在地、过去的行为/事件、侵权行为地 |
| | 动态（可变） | 国籍、住所、居所、动产所在地 |

#### 3.1.4 准据法的确定 🔑

**识别（定性）**：判断案件事实属于哪一类法律关系 → 选择对应的冲突规范
**反致**：A国冲突规范指向B国法 → B国冲突规范又指回A国法（包括：
- **直接反致（一级反致）**：A → B（总括指向B法含其冲突法）→ B → A（最终用A实体法）
- **转致（二级反致）**：A → B → C（最终用C实体法）
- **间接反致**：A → B → A → B（最终用B实体法）

> 📌 **中国的态度**：《涉外民事关系法律适用法》明确规定**不接受反致**（第9条）

**先决问题**：主要问题依赖的前提问题（如离婚案中的婚姻效力问题）
**法律规避（Evasion of Law）**：故意制造连接点以避开本应适用的法律（中国不承认其效力）

**公共秩序保留**：外国法的适用结果将**严重违背**中国社会公共利益时 → 排除该外国法适用

**外国法的查明** 🔑：

| 方式 | 中国的规定 |
|------|-----------|
| **当事人提供** | 当事人选择适用的 → 当事人提供 |
| **人民法院/仲裁机构/行政机关查明** | 通过：①中外法律专家；②使领馆；③中外法律服务机构；④国际公约/条约途径；⑤互联网等 |
| **不能查明后果** | 不能查明 → **适用中华人民共和国法律** |

### 3.2 《涉外民事关系法律适用法》（2023修正）核心条文速查 🔑🔑🔑

> 这是国际私法部分的绝对重心，几乎每年都考！

#### 3.2.1 一般规定

| 条款 | 内容 |
|------|------|
| **原则** | 最密切联系原则（兜底）、意思自治原则（优先） |
| **强制性规定** | 涉及中国社会公共利益/劳动者权益/食品安全/环境安全等的，**必须直接适用中国法** |
| **公共秩序保留** | 外国法适用结果违背中国公共利益 → **排除** |
| **反致** | ❌ **不承认**（第9条） |
| **区际法律冲突** | 涉及港澳台的，参照涉外规定处理 |

#### 3.2.2 权利主体

| 法律关系 | 连接点 | 规则 |
|----------|--------|------|
| **自然人权利能力** | 经常居所地法 | （国籍+经常居所不一致时） |
| **自然人行为能力** | 经常居所地法 | （例外：从事民商活动的，可适用行为地法） |
| **宣告失踪/死亡** | 经常居所地法 | — |
| **法人及其分支机构** | **登记地法** | 经常居所 = 主营业地 |
| **人格权** | 经常居所地法 | — |

#### 3.2.3 家庭关系（📌高频考点！）

| 法律关系 | 连接点 | 特殊规则 |
|----------|--------|----------|
| **结婚** | 共同经常居所地法 → 共同国籍国法 → 婚姻缔结地法（一方/双方经常居所地在内地→**中国法**） |
| **夫妻人身关系** | 共同经常居所地法 → 共同国籍国法 | — |
| **夫妻财产关系** | **意思自治优先**（协议选择）→ 无选择的：共同经常居所地法 → 共同国籍国法 | — |
| **离婚** | 法院地法 | — |
| **亲子** | 共同经常居所地法 → 共同国籍国法 → 有利于保护被扶养人权益的法 | — |
| **收养成立/效力** | 收养时**经常居所地法** | 收养手续同时符合收养人和被收养人经常居所地法 |
| **扶养** | **有利于被扶养人**的一方经常居所地法/国籍国法/供给方住所地法 | （"有利原则"！） |
| **监护** | **被监护人经常居所地法** | — |

#### 3.2.4 继承（📌必考！）

| 类型 | 连接点 | 特殊规则 |
|------|--------|----------|
| **法定继承** | **被继承人死亡时经常居所地法** | （注意：不是国籍国法也不是遗产所在地法！） |
| **遗嘱方式** | 符合遗嘱人立遗嘱时/死亡时**经常居所地法/国籍国法/遗嘱行为地法之一即可** | （选择性连结，宽松！） |
| **遗嘱效力** | 遗嘱人立遗嘱时/死亡时**经常居所地法/国籍国法/遗嘱行为地法** | — |
| **遗产管理** | 遗产所在地法 | — |

#### 3.2.5 物权

| 类型 | 连接点 |
|------|--------|
| **不动产** | **不动产所在地法**（绝对的硬性规则！） |
| **动产** | **意思自治优先**（协议选择）→ 无选择：**法律事实发生时动产所在地法** |
| **运输中的动产** | 目的地法 |
| **有价证券** | 有价证券权利实现地法 / 最密切联系地法 |
| **权利质权** | 质权设立地法 |

> 📌 **口诀**：不动产地法死记；动产意思自治优先；运输中看目的地。

#### 3.2.6 债权（📌重中之重！）

**合同之债**：

| 层次 | 连接点 | 说明 |
|------|--------|------|
| **第一优先：意思自治** | 当事人**协议选择**适用的法律 | 最基本原则！可以明示也可以默示推断 |
| **第二优先：特征履行** | 无选择的：**特征履行方的经常居所地法** | 卖方/服务提供方为特征履行方 |
| **兜底：最密切联系** | 与合同有**最密切联系**的法律 | — |

**消费者合同的特殊保护** 🔑：
- 消费者**选择**商品/服务提供地法 → 用消费者选择的法律
- 消费者**没选** → **商品/服务提供地法**（保护弱者！）
- **限制**：经营者不得利用格式条款排除消费者的法定权利

**劳动合同**：
- 意思自治受限（**不得**选择适用法律排除劳动者保护）
- 无选择：**劳动者工作地法** → 无工作地的：**用人单位主营业地法**

**侵权之债**：

| 类型 | 连接点 |
|------|--------|
| **一般侵权** | **侵权行为地法**（包括行为实施地+结果发生地） |
| **产品责任** | **被侵权人经常居所地法** 或 **侵权人主营业地法/产品取得地法**（被侵权人可选择！） |
| **网络侵权** | **被侵权人经常居所地法** |
| **知识产权侵权** | **被请求保护地法** |
| **不正当竞争/垄断** | **相关市场竞争地/受损地法** |
| **侵权双方共同经常居所地** | 可适用（意思自治也可） |

**不当得利和无因管理**：
- 意思自治优先（协议选择）
- 无选择：**发生地法**

#### 3.2.7 知识产权

| 类型 | 连接点 |
|------|--------|
| **知识产权归属/内容** | **被请求保护地法** |
| **转让/许可** | **意思自治**（协议选择）→ 无选择：**相关最密切联系地法** |
| **侵权** | **被请求保护地法**（法院地/主张保护地） |

### 3.3 国际民事诉讼 🔑

#### 3.3.1 管辖权

**中国涉外民诉管辖权依据**：

| 类型 | 依据 | 说明 |
|------|------|------|
| **普通管辖** | 被告住所地 | 一般原则 |
| **特殊管辖** | 合同签订/履行地/侵权行为地/标的物所在地等 | 《民事诉讼法》相关条款 |
| **专属管辖** | 中外合资/合作经营企业合同、合作勘探开发自然资源合同 → **必须由中国法院管辖** | 排除外国法院管辖权！ |
| **协议管辖** | 当事人书面协议选择 | 限于与争议有实际联系的地点的法院 |
| **应诉管辖** | 被告不对管辖权提出异议并应诉答辩 | 视为接受管辖 |
| **便利原则** | 中国法院不便审理/外国法院更方便 | 可以**驳回起诉**告知向外起诉 |

**平行诉讼问题**（一事两诉）：
- **外国法院已受理**：中国仍可受理（一般不禁止平行诉讼）
- **外国法院已作出判决**：中国法院可以受理（是否承认另说）

#### 3.3.2 司法协助

| 类型 | 内容 |
|------|------|
| **送达** | 依《海牙送达公约》或双边条约；外交途径；领事途径 |
| **取证** | 依《海牙取证公约》；代为取证；领事取证 |
| **承认与执行外国判决** | 详见下节 |

#### 3.3.3 外国法院判决的承认与执行 🔑

**条件**（缺一不可）：

| # | 条件 |
|---|------|
| 1 | 判决已**发生法律效力** |
| 2 | 作出判决的法院具有**合格管辖权** |
| 3 | 诉讼程序**公正**（被告得到合法传唤/有机会陈述） |
| 4 | 不与中国正在进行的/已经终结的诉讼**相矛盾** |
| 5 | 判决不**违背**中国**公共秩序**（公共政策） |
| 6 | 存在**互惠关系**（条约或事实互惠）|

**不予承认的情形**：
- 缺乏管辖权 / 未出庭未获合法传讯 / 程序不公正 / 诈骗取得 / 尚未生效 / 已被其他国家承认 / 违背公共秩序 / 无互惠关系

**承认与执行的程序**：
- 向**中级人民法院**提出申请
- 审查后裁定承认 → 需要执行的另行申请**强制执行**

## 四、第三部分：国际经济法（International Economic Law，~3分）

### 4.1 国际贸易法

#### 4.1.1 WTO（世界贸易组织）框架

| 机构 | 功能 |
|------|------|
| **部长级会议** | 最高决策机构（每两年一次） |
| **总理事会** | 日常决策（兼争端解决机构和贸易政策审议机构） |
| **理事会** | 货物贸易理事会、服务贸易理事会、TRIPS理事会 |
| **委员会** | 各专门委员会 |
| **秘书处** | 总干事领导，日常行政 |

**WTO基本原则**：

| 原则 | 内容 | 例外 |
|------|------|------|
| **最惠国待遇（MFN）** | 给予一国的好处，自动给予所有成员 | FTA/关税同盟/普惠制/边境贸易/保障措施 |
| **国民待遇（NT）** | 给予外国人不低于本国人的待遇 | 政府采购/补贴/例外清单 |
| **关税减让/取消数量限制** | 逐步降低关税、取消配额 | 保障措施/例外情况 |
| **透明度原则** | 公布贸易法规、统一公正合理实施 | 国家安全机密 |

**三大贸易协定**：

| 协定 | 全称 | 核心内容 |
|------|------|----------|
| **GATT** | 关税与贸易总协定 | 货物贸易规则 |
| **GATS** | 服务贸易总协定 | 服务贸易四种模式 |
| **TRIPS** | 与贸易有关的知识产权协定 | 知识产权最低保护标准 |

#### 4.1.2 WTO争端解决机制（DSU）🔑

**程序流程**：

```
磋商（60天内） → 专家组审理（6-9个月） → 上诉机构审查（60-90天） → 通过报告 → 执行
                                                                              ↓
                                                              不执行？→ 授权报复（补偿/中止减让）
```

**关键特点**：
- **反向一致通过**（即"自动通过"）：专家组/上诉报告除非**全体一致反对**，否则通过
- **交叉报复**：可在不同领域（货物/服务/TRIPS）中止减让
- **无强制执行力**：最终靠成员国自愿遵守或授权报复

#### 4.1.3 贸易救济措施（三反一保）🔑

**① 反倾销（Anti-Dumping）**：

| 要素 | 内容 |
|------|------|
| **倾销定义** | 出口价格 < 正常价值（出口国国内售价或成本加利润） |
| **损害认定** | 实质性损害 / 实质性威胁 / 实质性阻碍产业建立 |
| **因果关系** | 倾销与损害之间必须有因果关系 |
| **措施** | 征收**反倾销税**（价格承诺替代） |
| **期限** | 一般5年（日落复审） |

**② 反补贴（Countervailing）**：

| 要素 | 内容 |
|------|------|
| **补贴定义** | 政府或公共机构提供的财政资助/收入/价格支持，使企业获得利益 |
| **专向性** | 补贴必须是**特定企业/行业/地区**才能采取反补贴措施（普遍性补贴不行） |
| **措施** | 征收**反补贴税** |

**③ 保障措施（Safeguard Measures）** 🔑：

| 要素 | 内容 |
|------|------|
| **触发条件** | 进口**激增**（数量增加）→ 造成**严重损害/严重威胁**（不需要不公平竞争！） |
| **vs 反倾销/反补贴的区别** | 不需要证明倾销/补贴，只需进口激增+损害 |
| **措施** | 提高关税 / 数量限制 |
| **特殊规则** | **针对所有来源国**（非歧视），不超过4年（最长10年） |
| **发展中国家优惠** | 进口占比≤3%的小份额发展中成员可豁免 |

> 📌 **记忆口诀**："反倾销比价格，反补贴查政府钱，保障措施看激增"

#### 4.1.4 国际贸易术语（Incoterms 2020）🔑

| 术语 | 全称 | 风险转移点 | 运输/保险 |
|------|------|-----------|-----------|
| **EXW** | 工厂交货 | 在卖方工厂交货 | 买方负责全部运输和保险 |
| **FCA** | 货交承运人 | 货交第一承运人 | 买方承担运费（卖方协助） |
| **CPT** | 运费付至 | 货交第一承运人 | 卖方付运费；保险买方自担 |
| **CIP** | 运费保险费付至 | 货交第一承运人 | 卖方付运费+保险费 |
| **DAP** | 目的地交货 | 到达指定目的地交货 | 卖方承担运至目的地费用 |
| **DPU** | 卸货地交货 | 在目的地卸货后交货 | 卖方承担全部运费+卸货费 |
| **DDP** | 完税后交货 | 进口国指定地点完税交货 | **卖方承担最大责任（含关税）** |

> 📌 **记忆规律**：E组最少（买方全包）→ F组（卖方管到装船前）→ C组（卖方管运费不管风险）→ D组（卖方管到目的地）。**E/F/D是实际交货；C/C是象征性交货（装运合同）**。

### 4.2 国际投资法

#### 4.2.1 外商投资法（2019年，2020.1.1施行）

| 要点 | 内容 |
|------|------|
| **立法模式转变** | 从"三资企业法"（合资/合作/外资）→ 统一的**《外商投资法》** |
| **准入制度** | **准入前国民待遇 + 负面清单管理**（重大改革！） |
| **负面清单** | 清单内的领域限制/禁止外商投资；清单外的**内外资一致管理** |
| **投资保护** | 征收补偿（公平+市场价值）、利润自由汇出、知识产权保护 |
| **投诉机制** | 外商投资企业投诉工作机制 |

**负面清单 vs 准入前国民待遇**：

| 概念 | 含义 |
|------|------|
| **准入前国民待遇** | 外商在**投资准入阶段**即享受与内资相同的待遇（不限于运营阶段）|
| **负面清单** | 明确列出不开放/限制开放的领域；未列入的领域**全面放开** |
| **正面清单（旧制度）** | 仅允许清单内的行业投资（已被取代）|

**外商投资类型**：
- **绿地投资**（新建企业）
- **并购投资**（收购现有企业股权/资产）
- **再投资**（用利润继续投资）

#### 4.2.2 双边/多边投资协定

| 类型 | 内容 |
|------|------|
| **双边投资协定（BIT）** | 两国之间签订的投资保护和促进协定（中国已签署100+个BIT）|
| **多边协定** | 如《解决国家与他国国民间投资争端公约》（ICSID）|

### 4.3 国际金融法

#### 4.3.1 国际贷款

| 类型 | 特点 |
|------|------|
| **国际商业银行贷款** | 纯商业性质，利率高，手续简便 |

|------|-----------|
| **经常项目** | **有条件自由兑换**（真实性审核原则）|
| **资本项目** | **逐步放开**（目前仍有管制）|
| **外汇管理机构** | 国家外汇管理局（中国人民银行管理）|
| **逃汇/套汇处罚** | 行政处罚（罚款/责令调回）/ 构成犯罪的追究刑责（逃汇罪）|

### 4.4 国际税收

#### 4.4.1 税收管辖权

| 类型 | 行使依据 | 原则 |
|------|----------|------|
| **居民税收管辖权** | 对本国居民/企业的全球所得征税 | 属人原则 |
| **来源地税收管辖权** | 对来源于本国的所得征税 | 属地原则 |

> 📌 大多数国家**同时行使两种管辖权** → 导致**国际重复征税**。

#### 4.4.2 国际重复征税的避免

| 方法 | 内容 |
|------|------|
| **免税法** | 居住国对境外已纳税所得免税（最优惠）|
| **扣除法** | 将境外已纳税款作为费用扣除 |
| **抵免法（最常用）** | 允许境外已纳税款从应纳本国税款中**抵免** |
| **税收饶让** | 视同已在来源国缴税（发展中国家争取到的优惠）|

**中国采用的方法**：**限额抵免法**——境外所得税款可在应纳中国税款中抵免，但有**抵免限额**（按中国税率计算的应纳税额）。

**双边税收协定**：通过划分征税权和消除双重征税的条约（中国已签署100+个双边税收协定）。

### 5.1 三国法内部交叉

| 场景 | 涉及法律 | 处理要点 |
|------|----------|----------|
| 外国人犯罪 | 刑法（属地管辖）+ 引渡法 | 中国有优先管辖权（属地）；同时可能涉及引渡 |

| 外国法院判决在中国执行 | 国际民事诉讼法（承认执行条件） | 中级法院审查6项条件 |

## 七、三大咨询速答模板

### 模板一：跨国婚姻家庭纠纷咨询

```
📋 问题定性
这是一起涉外婚姻/继承/收养家庭纠纷，属于【国际私法】调整范围。
涉及《中华人民共和国涉外民事关系法律适用法》的冲突规范指引。

⚖️ 法律分析
1. 首先判断当事人经常居所地/国籍国（确定连接点）
2. 根据具体事项匹配对应冲突规范：
   - [结婚/离婚/夫妻财产/亲子/扶养/继承] → [对应的连接点和顺序]
3. 确定准据法后，按该国实体法分析权利义务
4. 注意中国的强制性规定和公共秩序保留条款

📌 结论与建议
- 明确适用的准据法是什么
- 该准据法下当事人的权利义务
- 如需在中国诉讼：管辖法院级别/送达方式/证据要求
⏰ 时效提醒
涉外诉讼时效依准据法确定；注意诉讼文书送达时间较长（域外可达6个月）
```

### 模板二：国际贸易纠纷咨询

```
📋 问题定性
这是一起国际贸易纠纷，涉及【国际经济法】+【国际私法】。
需区分：合同纠纷？产品质量？贸易救济措施？

⚖️ 法律分析
1. 如属合同纠纷：
   - 是否有有效的法律选择条款？（意思自治优先）
   - 无选择时：特征履行方经常居所地法
   - 是否涉及CISG（《联合国国际货物销售合同公约》）？
   - Incoterms术语决定了风险转移点和费用分摊
2. 如属产品质量/侵权：
   - 产品责任：被侵权人有选择权（经常居所地/主营业地/取得地）
3. 如涉及贸易救济：
   - 反倾销/反补贴/保障措施的构成要件和程序

📌 结论与建议
- 明确责任归属和法律依据
- 争议解决途径：协商 → 仲裁（推荐）→ 诉讼
⚠️ 特别提示
- 国际仲裁裁决可通过《纽约公约》在170+个国家承认执行
- 诉讼判决的跨国执行则复杂得多
```

### 模板三：外国人在华权益/出境咨询

```
📋 问题定性
涉及【国际法】（外国人法律地位）+【行政法】（出入境管理）+可能的【刑法】【民法】。

⚖️ 法律分析
1. 确认当事人身份（国籍/签证类型/在华居留身份）
2. 区分问题性质：
   - 民事权益：适用《涉外民事关系法律适用法》

   - 外交/领事保护：是否享有外交特权与豁免？
3. 如涉及外交人员：注意《维也纳外交关系公约》的特殊保护
