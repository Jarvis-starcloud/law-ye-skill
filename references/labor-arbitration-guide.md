# 劳动仲裁实务完全指南

> 定位：实操手册——从准备到执行每一步怎么走
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块2交付物
> 法律依据：《劳动争议调解仲裁法》（2008）+《劳动人事争议仲裁办案规则》（2017修订）

---

## 一、受案范围——哪些事能申请劳动仲裁

### 1.1 法定受理范围（《劳动争议调解仲裁法》第2条）

| 序号 | 争议类型 | 典型场景 |
|------|----------|----------|
| 1 | 确认劳动关系 | 没签合同、挂靠、外包纠纷 |
| 2 | 劳动合同履行/变更/解除/终止 | 调岗降薪、违法解除、逼迫辞职 |
| 3 | 除名/辞退/辞职/离职 | 被开除、主动离职但公司不放人 |
| 4 | 工作时间/休息休假/社保/福利 | 加班费、年假、未缴社保 |
| 5 | 劳动报酬/工伤医疗费/经济补偿/赔偿金 | 欠薪、工伤赔偿、N+1/2N |
| 6 | 其他劳动争议 | 竞业限制、培训服务期违约金 |

### 1.2 不受理的情况

- 社保缴纳争议（部分地区仲裁不受理，需走社保稽核/行政投诉）
- 住房公积金争议（找住房公积金管理中心投诉）
- 劳务关系纠纷（非劳动关系，走民事诉讼）
- 公务员/参公事业编（走人事争议仲裁，不是劳动仲裁）

> **实务提醒**：社保争议各地口径不一致。北京/上海通常不受理补缴类争议，但赔偿社保损失可以受理。具体以当地仲裁委答复为准。

## 二、管辖——去哪个仲裁委

### 2.1 管辖规则

**基本原则**：劳动合同履行地 或 用人单位所在地 的劳动人事争议仲裁委员会。

**双方不一致时**：劳动合同履行地优先。

### 2.2 实操选择策略

| 情形 | 建议选择 | 理由 |
|------|----------|------|
| 人在公司所在城市上班 | 任选（一般同一地点） | 方便出庭 |
| 人被派遣到异地 | 实际工作地 | 对劳动者有利，方便举证 |
| 远程办公（人在A地，公司在B地）| 优先选劳动合同履行地 | 一般是实际办公地 |
| 公司注册地≠实际经营地 | 实际经营地 | 能找到人、送达方便 |

> **实务提醒**：申请前先打当地仲裁委电话确认是否受理。12333热线可查当地仲裁委地址和电话。

## 三、仲裁时效——别过期了

### 3.1 时效规则

| 时效类型 | 期限 | 起算点 | 法律依据 |
|----------|------|--------|----------|
| **一般时效** | **1年** | 知道或应当知道权利被侵害之日 | 《调解仲裁法》第27条 |
| **特殊时效（劳动报酬）** | **在职期间不受限** | 劳动关系终止之日起1年 | 第27条第4款 |
| 中断 | 重新起算 | 主张权利/对方同意/向有关部门请求救济 | 第27条第2款 |
| 中止 | 暂停计算 | 不可抗力/其他正当理由 | 第27条第3款 |

### 3.2 时效实操要点

1. **欠薪类**：在职期间不受1年限制，但离职后从最后工作日起算1年
2. **违法解除赔偿金**：从收到解除通知之日起算1年
3. **工伤待遇**：从工伤认定书/鉴定结论送达之日起算1年
4. **年假工资**：从应休未休年度的次年1月1日或离职日起算

> **保命操作**：如果时效快到了但材料没准备好，先交一份简单的申请书占住时效，后续再补充证据。

## 四、申请材料清单（逐项详解）

### 4.1 必备材料

| 序号 | 材料 | 份数 | 备注 |
|------|------|------|------|
| 1 | **仲裁申请书** | 一式三份（仲裁委1+对方当事人各1+自留1） | 最核心文件，见模板 |
| 2 | **身份证复印件** | 1份 | 带原件核验 |
| 3 | **劳动关系证明** | 至少1份 | 劳动合同/工资流水/工牌/社保记录等 |
| 4 | **用人单位信息** | 1份 | 企业信用信息公示报告（天眼查/企查查打印） |
| 5 | **送达地址确认书** | 1份 | 仲裁委现场填写 |

### 4.2 按争议类型的补充证据

| 争议类型 | 建议准备的证据 |
|----------|---------------|
| **欠薪** | 工资条/银行流水/考勤记录/劳动合同约定的工资标准 |
| **违法解除** | 解除通知（书面/微信/邮件截图）/工作年限证明/月均工资证据 |
| **加班费** | 考勤记录/加班审批单/微信工作群加班记录/钉钉打卡截图 |
| **未签合同双倍工资** | 入职时间证明（offer/入职登记表）/工资发放记录 |
| **工伤赔偿** | 工伤认定决定书/劳动能力鉴定结论/医疗费票据/工资证明 |
| **竞业限制** | 竞业限制协议/补偿金发放记录/新入职公司信息 |

### 4.3 没有劳动合同怎么办？

以下任一材料都能证明劳动关系（参照《关于确立劳动关系有关事项的通知》劳社部发〔2005〕12号）：

1. 工资支付凭证（银行转账记录，注意备注是否写"工资"）
2. 社保缴纳记录
3. 工作证/工牌/门禁卡
4. 考勤记录
5. 同事证言（至少2人）
6. 招聘录用记录（offer邮件/微信聊天记录）
7. 公司发的工作通知/邮件
8. 快递收发记录（显示公司名+你的名字）

> **实务提醒**：微信聊天记录截图前，先做一次**原始载体展示**——用手机打开微信，录屏展示对方头像→点击→显示微信号/手机号→返回聊天→翻到目标内容。必要时做公证。

## 五、仲裁流程全览（时间线）

### 5.1 完整流程图

```
准备材料 → 提交申请 → 受理(5日) → 送达+组庭(5日) → 开庭 → 裁决(45/60日)
                                         ↓
                                    调解(贯穿全程)
                                         ↓
                                调解成功→调解书(有执行力)
                                调解不成→裁决
```

### 5.2 各阶段时间节点

| 阶段 | 法定期限 | 起算点 | 实务耗时 |
|------|----------|--------|----------|
| **受理审查** | **5个工作日** | 收到申请之日 | 通常3-5天 |
| 不予受理（需出通知书） | 5个工作日 | 收到申请之日 | — |
| **被申请人答辩** | **10日** | 收到申请书副本之日 | — |
| **仲裁庭组成** | 受理后 | — | 通常与送达同步 |
| **开庭通知** | 开庭前**5日** | — | 仲裁委安排 |
| **裁决** | **45日**（可延长至60日） | 受理之日 | 一般1-3个月 |

### 5.3 实务时间预估

| 情形 | 预计总耗时 | 说明 |
|------|-----------|------|
| 简单案件（欠薪、有证据）| 1-2个月 | 可能庭前调解就解决 |
| 中等案件（违法解除、双方有争议）| 2-3个月 | 通常开1-2次庭 |
| 复杂案件（工伤+多项请求）| 3-6个月 | 可能涉及鉴定、多次开庭 |

## 六、申请流程操作步骤

### 第一步：确定管辖仲裁委
- 百度搜"XX区劳动人事争议仲裁委员会"
- 拨打12333确认地址、电话、工作时间
- 部分城市支持网上立案（如北京、上海、深圳）

### 第二步：准备并提交材料
- 按第四节清单准备
- 现场取号排队→递交材料→窗口初审→出具收件回执
- **保留收件回执！** 这是证明你申请过的凭证

### 第三步：等待受理通知
- 5个工作日内出《受理通知书》或《不予受理通知书》
- 收到受理通知后，记住案号

### 第四步：等待开庭
- 仲裁委送达《开庭通知书》，写明开庭时间、地点
- 收到后立刻准备开庭材料（详见第七节）

### 第五步：参加开庭
- 详见第七节开庭实操

### 第六步：等待裁决
- 45日内出裁决（可延长至60日）
- 裁决书送达后15日内如不服可起诉至法院（一裁终局的除外）

## 七、开庭实操指南

### 7.1 开庭前准备

**携带物品清单**：
- [ ] 身份证原件
- [ ] 所有证据原件+复印件3份
- [ ] 证据目录（列明序号、证据名称、证明目的、页数）
- [ ] 仲裁申请书备份
- [ ] 纸笔（记录对方陈述）
- [ ] 委托代理人的：授权委托书+代理人身份证

**穿着**：正装不强求，但干净整洁。不要穿拖鞋。

### 7.2 开庭流程

| 阶段 | 时长 | 你要做什么 |
|------|------|-----------|
| **1. 核实身份** | 5分钟 | 出示身份证，确认当事人信息 |
| **2. 宣布纪律** | 3分钟 | 听着就行，手机静音 |
| **3. 申请人陈述** | 10-15分钟 | **最关键！** 照着申请书念仲裁请求+事实理由，简洁有力 |
| **4. 被申请人答辩** | 10-15分钟 | **仔细听、做笔记**，记下对方承认和否认的要点 |
| **5. 举证质证** | 20-40分钟 | 逐一出示证据→对方质证→你回应。对方出示证据→你质证 |
| **6. 辩论** | 10-20分钟 | 针对对方答辩和质证意见反驳，强调核心证据 |
| **7. 最后陈述** | 5分钟 | 总结请求，表态是否同意调解 |
| **8. 调解** | 10-30分钟 | 仲裁员可能当庭组织调解 |
| **9. 签笔录** | 5分钟 | **仔细看！** 笔录有误要当场提出修改 |

### 7.3 陈述要点（模板话术）

```
尊敬的仲裁员：

我是申请人XXX。我于XX年XX月XX日入职XX公司，
担任XX职位，月工资XX元。

XX年XX月XX日，公司[具体侵权事实，如"以经营困难为由
口头通知我解除劳动合同，未提前30日通知，未支付经济补偿金"]。

我认为公司行为违反了[具体法律条文]，
现请求仲裁庭裁决：
1. [仲裁请求1]
2. [仲裁请求2]
...

以上陈述完毕。
```

### 7.4 质证技巧

**对方出示证据时，你质证的三个角度**：

| 角度 | 说什么 | 示例 |
|------|--------|------|
| **真实性** | 是否真实、是否伪造/变造 | "该考勤记录未经我签字确认，真实性不认可" |
| **关联性** | 与本案是否有关 | "该证据与本案争议焦点无关" |
| **合法性** | 取得方式是否合法 | "该录音系偷录且经过剪辑，合法性存疑" |

**常见质证话术**：
- 对方没原件："被申请人未提供原件，仅为复印件，真实性无法核实，不予认可"
- 考勤记录无签名："该考勤记录系公司单方制作，未经申请人签字确认，不能证明其主张"
- 对方证人是公司员工："该证人与被申请人存在利害关系，证言可信度存疑"

### 7.5 调解策略

| 公司出价 | 建议策略 |
|----------|----------|
| ≥ 法定标准的80% | 可以考虑接受（省时间+确定性） |
| 50%-80% | 看证据强弱，证据强则拒绝继续打 |
| < 50% | 拒绝，等裁决 |

> **调解书 vs 裁决书**：调解书双方签字即生效，不能上诉，有强制执行力。好处是快；坏处是一旦签了反悔不了。

## 八、一裁终局规则

### 8.1 哪些案件一裁终局

根据《调解仲裁法》第47条，以下两类案件劳动者一方可一裁终局：

1. **追索劳动报酬/工伤医疗费/经济补偿/赔偿金**，不超过**当地月最低工资标准12个月**金额
2. **因执行国家劳动标准**（工作时间、休息休假、社保）产生的争议

### 8.2 一裁终局的效果

| 主体 | 不服裁决的救济 |
|------|---------------|
| **劳动者** | 15日内可向法院起诉（等于有"两审"机会） |
| **用人单位** | **不能起诉**！只能向中院申请撤销裁决（条件极严格） |

### 8.3 用人单位申请撤销的6种法定情形

1. 适用法律、法规确有错误
2. 仲裁委无管辖权
3. 违反法定程序
4. 裁决所根据的证据系伪造
5. 对方当事人隐瞒了影响公正裁决的证据
6. 仲裁员有索贿受贿、徇私舞弊、枉法裁决行为

> **对劳动者的意义**：一裁终局是劳动者的保护伞。如果金额在范围内，公司基本只能接受裁决结果。

## 九、裁决后的执行

### 9.1 对方不执行裁决怎么办

1. 裁决书生效后，对方**逾期不履行**
2. 向**有管辖权的人民法院**申请强制执行
3. 申请执行时效：裁决书规定的履行期限届满后**2年**内

### 9.2 申请执行需要的材料

- 强制执行申请书
- 生效的仲裁裁决书/调解书原件
- 裁决书送达回证（证明已生效）
- 申请人身份证复印件
- 被执行人财产线索（银行账户、房产、车辆等信息）

### 9.3 执行小技巧

1. **提前查财产**：执行前在企查查/天眼查查对方名下公司、关联公司
2. **申请财产保全**：如果担心对方转移资产，可在诉讼/仲裁阶段就申请
3. **追加法人**：公司是空壳/注销的，可以申请追加法定代表人/股东为被执行人（需满足"一人公司/出资不实/抽逃出资"等条件）

## 十、特殊情况处理

### 10.1 公司找不到人/拒收文书

- 仲裁委可以公告送达（挂网60日视为送达）
- 对方不到庭→缺席裁决（对申请人有利）

### 10.2 公司注销了

- 可以追加**股东/清算组/实际控制人**为被申请人
- 如果已注销且无人承接债务，可能需要转民事诉讼途径

### 10.3 仲裁委不受理

- 5日内出《不予受理通知书》
- 收到后可在**15日内**向人民法院起诉（绕过仲裁直接进法院）
- 不给不受理通知书的→视为受理

### 10.4 需要做劳动能力鉴定（工伤案件）

- 向设区的市级劳动能力鉴定委员会申请
- 鉴定费由用人单位承担（工伤保险基金支付或用人单位自付）
- 鉴定期间仲裁时效中止

## 十一、常见问题速查

| 问题 | 答案 |
|------|------|
| 仲裁收费吗？ | **不收费**（《调解仲裁法》第53条） |
| 一定要请律师吗？ | 不是，可以自己代理。简单案件建议自己来 |
| 可以委托别人代理吗？ | 可以，近亲属/律师/工会/法律援助，需授权委托书 |
| 开庭能请假吗？ | 可以申请延期，需提前且有正当理由 |
| 申请人不到庭？ | 视为撤回申请 |
| 被申请人不到庭？ | 缺席裁决 |
| 可以追加请求吗？ | 开庭前可以增加/变更请求，开庭后通常不允许 |
| 仲裁期间能去上班吗？ | 能，仲裁不影响劳动关系存续 |
| 同事能当证人吗？ | 能，但在职同事一般不愿意作证 |
| 录音能当证据吗？ | 能，但需证明未经剪辑且取得方式不违法 |
