## 一、八种法定证据类型与收集方法

### 1.1 证据分类总览

| 序号 | 证据类型 | 典型形式 | 收集难度 | 证明力 |
|------|----------|----------|----------|--------|
| 1 | **当事人陈述** | 起诉状/答辩状/庭审陈述 | ⭐ | ⭐⭐ |
| 2 | **书证** | 合同/收据/通知/信件 | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| 3 | **物证** | 伪劣商品/损坏的物品 | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| 4 | **视听资料** | 录音/录像/监控 | ⭐⭐ | ⭐⭐⭐⭐ |
| 5 | **电子数据** | 微信聊天/邮件/网页/APP截图 | ⭐⭐ | ⭐⭐⭐ |
| 6 | **证人证言** | 证人出庭陈述/书面证言 | ⭐⭐⭐ | ⭐⭐⭐ |
| 7 | **鉴定意见** | 司法鉴定报告 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 8 | **勘验笔录** | 法院/行政机关现场勘验记录 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |

### 1.2 各类证据收集要点

#### 书证

| 收集要点 | 说明 |
|----------|------|
| 保留原件 | 复印件证明力低于原件 |
| 签字/盖章页 | 必须包含签名或盖章页 |
| 完整性 | 多页文件注明总页数，骑缝章/页码 |
| 形成时间 | 有日期戳比没有日期好 |

#### 视听资料

| 要素 | 要求 |
|------|------|
| 合法取得 | 你必须是对话参与者（录音）或有权录制（监控） |
| 完整性 | 不剪辑、不掐头去尾 |
| 原始载体 | 保留录制设备上的原始文件 |
| 辅助文字 | 整理文字稿，方便法官阅读 |

#### 电子数据

| 类型 | 收集方法 |
|------|----------|
| 微信聊天记录 | 原始载体录屏+截图+对方身份验证 |
| 邮件 | 截图+导出eml原始文件+邮箱服务商存档 |
| 网页内容 | 截图+网页存档/公证+URL记录 |
| APP数据 | 录屏操作过程+截图 |
| 转账记录 | 银行流水+支付宝/微信交易记录截图 |

#### 证人证言

| 要素 | 说明 |
|------|------|
| 证人资格 | 知道案件情况的人（不能是律师/法官） |
| 出庭优先 | 出庭证言的证明力远高于书面证言 |
| 书面证言格式 | 证人签名+身份证复印件+日期+"以上陈述属实" |
| 限制 | 未成年人、精神障碍者的证言效力受限 |
| 利害关系 | 与当事人有利害关系的证人证言需要其他证据佐证 |

## 二、电子证据固定（专题）

### 2.1 微信聊天记录固定方法（黄金标准）

**五步操作法**：

| 步骤 | 操作 | 目的 |
|------|------|------|
| **1. 验证身份** | 打开聊天→点对方头像→显示微信号/手机号→截图 | 证明对方是谁 |
| **2. 录屏展示** | 全程录屏：从打开微信→进入聊天→翻到目标内容 | 证明来源真实 |
| **3. 截图保存** | 逐屏截图所有相关聊天内容 | 便于阅读和提交 |
| **4. 文字整理** | 按时间顺序整理对话文字版 | 方便法官查阅 |
| **5. 公证（重要案件）** | 到公证处做保全证据公证 | 最高证明力 |

### 2.2 录音固定方法

```
录音前：
- 确认你是对话参与者
- 测试录音设备能正常工作
- 记录录音时间、地点

录音中：
- 开场自然引出话题（不要直接"我现在录音了"）
- 在对话中自然提及日期/地点/对方名字
- 让对方在对话中确认关键事实
- 不诱导、不威胁、不设局

录音后：
- 原始文件不做任何剪辑
- 备份到多个设备（手机+电脑+云端）
- 整理完整文字稿
- 标注关键时间点（如 "02:15处对方承认欠款"）
```

### 2.3 电子邮件固定

| 方法 | 证明力 | 操作 |
|------|--------|------|
| 截图 | ⭐⭐ | 截取邮件全文（含发件人/收件人/时间/主题/正文） |
| 导出eml | ⭐⭐⭐ | 从邮箱导出原始邮件文件（含邮件头信息） |
| 公证 | ⭐⭐⭐⭐⭐ | 公证员在场操作，登录邮箱+展示邮件+出具公证书 |

### 2.4 网页内容固定

| 方法 | 适用场景 |
|------|----------|
| 截图+URL | 一般用途 |
| 网页存档（Wayback Machine） | 防止对方修改删除 |
| **公证保全** | 重要案件必做——公证员操作电脑登录网页并截图出具公证书 |
| 区块链存证 | 部分平台支持（如至信链、天平链），法院认可度提高中 |

## 三、证据目录编制规范

### 3.1 标准格式

```
                       证据目录

案  号：[受理后填写]
提交人：[原告/被告] [姓名]

┌────┬──────────┬────┬──────┬──────────┬────┐
│序号│证据名称  │形式│页数  │证明目的  │备注│
├────┼──────────┼────┼──────┼──────────┼────┤
│ 1  │劳动合同  │原件│ 4页  │证明双方  │    │
│    │          │    │      │存在劳动  │    │
│    │          │    │      │关系及工  │    │
│    │          │    │      │资标准    │    │
├────┼──────────┼────┼──────┼──────────┼────┤
│ 2  │银行流水  │原件│ 6页  │证明被告  │XX银│
│    │          │    │      │拖欠X月   │行打│
│    │          │    │      │工资事实  │印  │
├────┼──────────┼────┼──────┼──────────┼────┤
│... │...       │... │...   │...       │... │
└────┴──────────┴────┴──────┴──────────┴────┘

共计证据 X 组，共 XX 页。

                          提交人：[签名]
                          日  期：[XXXX年XX月XX日]
```

### 3.2 编制要点

| 要点 | 说明 |
|------|------|
| **分组编号** | 按争议焦点或时间线分组，每组编号 |
| **证明目的具体** | 不写"证明案件事实"，写"证明被告于X日解除劳动合同" |
| **注明原件/复印件** | 法院重视原件，有原件的标注"原件" |
| **页数准确** | 逐页标注页码 |
| **附件说明** | 如有光盘/U盘等电子证据载体，单独说明 |

## 四、质证技巧全解

### 4.1 质证三个角度（核心方法论）

| 角度 | 含义 | 常用话术 |
|------|------|----------|
| **真实性** | 证据内容是否真实 | "对该证据的真实性不予认可，理由是..." |
| **关联性** | 与待证事实是否有关 | "该证据与本案争议焦点无关联" |
| **合法性** | 取得方式是否合法 | "该证据取得方式违法，不应采纳" |

### 4.2 常见证据的质证要点

#### 对方出示书证

| 攻击点 | 话术 |
|--------|------|
| 没有原件 | "被告仅提供复印件，未出示原件，真实性无法核实" |
| 签名存疑 | "该签名与我方当事人平时签名笔迹不一致，申请笔迹鉴定" |
| 内容被改 | "该文件存在明显涂改痕迹，真实性不予认可" |
| 单方制作 | "该文件系被告单方制作，未经我方确认签字，不予认可" |

#### 对方出示电子证据

| 攻击点 | 话术 |
|--------|------|
| 无原始载体 | "被告未展示手机/电脑原始载体，仅有打印件，真实性存疑" |
| 可能被篡改 | "电子数据易被修改，被告未做公证或区块链存证，无法排除篡改可能" |
| 身份不明 | "微信聊天记录中对方身份未经验证，无法确认系我方当事人" |
| 内容不完整 | "被告仅截取部分聊天内容，未展示完整对话，存在断章取义" |

#### 对方出示证人证言

| 攻击点 | 话术 |
|--------|------|
| 未出庭 | "证人未出庭接受质询，书面证言不应采纳" |
| 利害关系 | "证人系被告公司员工，与被告存在利害关系，证言可信度存疑" |
| 前后矛盾 | "证人在X处的陈述与其在Y处的陈述相互矛盾" |
| 传闻证据 | "证人称'听XX说'，属于传闻证据，不能作为定案依据" |

### 4.3 质证记录表（庭审中使用）

```
┌────┬──────────┬──────┬──────┬──────┬──────┐
│编号│对方证据  │真实性│关联性│合法性│备注  │
├────┼──────────┼──────┼──────┼──────┼──────┤
│ 1  │劳动合同  │认可  │认可  │认可  │      │
├────┼──────────┼──────┼──────┼──────┼──────┤
│ 2  │考勤记录  │不认可│认可  │认可  │无签字│
├────┼──────────┼──────┼──────┼──────┼──────┤
│ 3  │证人证言  │不认可│不认可│认可  │未出庭│
└────┴──────────┴──────┴──────┴──────┴──────┘
```

> **庭前准备**：提前预判对方可能提交的证据，准备好质证意见。

## 五、举证期限管理

### 5.1 民事诉讼举证期限

| 程序 | 法院指定期限 | 说明 |
|------|-------------|------|
| 普通程序 | 不少于**15日** | 以法院《举证通知书》为准 |
| 简易程序 | 由法院确定 | 通常较短 |
| 小额诉讼 | 由法院确定 | 通常7-15日 |

### 5.2 逾期举证的后果

| 情形 | 后果 |
|------|------|
| 有正当理由 | 说明理由后法院可以采纳 |
| 故意逾期/重大过失 | 法院可以不采纳 |
| 不影响审理+对方同意 | 法院可以采纳，但可能训诫或罚款 |

> **实务操作**：尽量在举证期限内提交所有证据。确实来不及的，书面申请延期并说明理由。

### 5.3 证据突袭应对

对方在庭审中突然出示未交换的证据：
1. **当场要求审查**："审判长，该证据未在举证期限内提交，也未经证据交换，我方申请核实"
2. **申请延期质证**："该证据内容复杂，我方需要时间核实，申请庭后提交书面质证意见"
3. **不要慌**：法官通常会给你准备时间

## 六、举证责任与证明标准

### 6.1 举证责任分配

| 诉讼类型 | 一般规则 | 特殊规则 |
|----------|----------|----------|
| **民事** | 谁主张谁举证 | 部分案件举证责任倒置（医疗/环境/产品等） |
| **刑事** | 公诉方承担举证责任 | 巨额财产来源不明等被告方举证 |
| **行政** | 行政机关对行政行为合法性举证 | 原告证明起诉条件+损害事实 |
| **劳动** | 谁主张谁举证 | 用人单位掌握的证据由用人单位提供 |

### 6.2 证明标准

| 诉讼类型 | 标准 | 含义 |
|----------|------|------|
| 民事 | **高度盖然性** | 证据让法官相信事实"很可能是这样"（>50%确信度） |
| 刑事 | **排除合理怀疑** | 最高标准，不能有合理怀疑 |
| 行政 | **明显优势** | 一方证据明显优于另一方 |

## 七、特殊取证方法

### 7.1 申请法院调查取证

**适用条件**（当事人自己取不到的证据）：
1. 由国家有关部门保存的需要由法院调取的
2. 涉及国家秘密、商业秘密、个人隐私的
3. 确因客观原因不能自行收集的

**操作**：
- 书面申请，说明证据名称+保管单位+证明目的+无法自行取得的原因
- 在举证期限届满前**7日**提出

### 7.2 公证取证

| 应用场景 | 费用（参考） | 证明力 |
|----------|-------------|--------|
| 微信/网页内容保全 | 500-2000元 | ⭐⭐⭐⭐⭐ |
| 现场取证（如侵权现场） | 1000-3000元 | ⭐⭐⭐⭐⭐ |
| 录音录像保全 | 500-1500元 | ⭐⭐⭐⭐⭐ |
| 送达文件保全 | 300-800元 | ⭐⭐⭐⭐ |

> **什么时候做公证**：标的额大（>5万）、证据容易灭失（对方可能删除）、对方很可能否认。

### 7.3 申请鉴定

| 鉴定类型 | 常见场景 | 费用（参考） |
|----------|----------|-------------|
| 笔迹鉴定 | 签名真伪 | 2000-5000元 |
| 伤残鉴定 | 人身损害赔偿 | 2000-4000元 |
| 工程造价鉴定 | 建设工程纠纷 | 标的额0.5-1% |
| 价格评估 | 财产损害赔偿 | 1000-5000元 |
| 电子数据鉴定 | 数据篡改/恢复 | 3000-20000元 |

## 八、证据规则速查表

| 规则 | 内容 | 实务意义 |
|------|------|----------|
| **最佳证据规则** | 书证应提交原件 | 有原件尽量交原件 |
| **自认规则** | 当事人对不利事实的承认不需举证 | 对方自认了就不用再证明 |
| **免证事实** | 众所周知/自然规律/已生效判决确认的事实 | 不需要举证 |
| **证据排除** | 非法手段取得的证据不予采纳 | 偷偷安装窃听器取得的录音=不合法 |
| **补强规则** | 单一证据（如仅有当事人陈述）不能定案 | 至少需要两个以上证据相互印证 |
| **禁反言** | 当事人不能作出前后矛盾的陈述 | 庭审说过的话不能反悔 |
