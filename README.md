# law爷 — AI 法律顾问 Skill

精通法律体系的 AI 法律顾问技能包（WorkBuddy / OpenClaw 兼容）。

## ✨ v1.3.0 新特性：问题定性工作流

新增「先判断法域再分流模块」工作流，让法律分析更精准：

```
用户问题
    ↓
┌─────────────────────────────────────────┐
│  Step 1: 关键词识别                      │
│  提取法律关键词，初步判断涉及领域         │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  Step 2: 法域判断                        │
│  民事/刑事/行政？是否存在刑民交叉？       │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  Step 3: 分流到专业模块                   │
│  合同→民法典 | 劳动→劳动法 | 犯罪→刑法    │
└─────────────────────────────────────────┘
    ↓
精准回答
```

### v1.3.0 核心更新

- 🆕 **问题定性工作流**：五步定性法（关键词→法律部门→法域交叉→细分领域→管辖）
- 🆕 **证据规则模块**：三大诉讼证明标准、举证责任分配、非法证据排除
- 🆕 **刑民交叉处理**：先刑后民、刑民并行、以民代刑三种模式判断
- 🆕 **请求权基础分析法**：从结果出发寻找可主张的请求权
- 📊 知识体系从 12 模块扩展到 **14 模块**

## 🔧 安装方法

将本仓库克隆或下载到 `~/.workbuddy/skills/law-ye/` 目录：

```bash
git clone https://github.com/Jarvis-starcloud/law-ye-skill.git ~/.workbuddy/skills/law-ye
```

或者手动下载文件到相应目录。
