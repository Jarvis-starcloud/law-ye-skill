## 🔑 2025年修订重点新增内容

### 总则部分：
- **第八条** 新增诚信原则
- **第十一条** 确认在线仲裁法律效力（与线下同等）
- **第十二条** 支持国际交流合作与规则制定

### 制度性重大突破：
- **第二十条** 建立信息公开制度
- **第四十五条** 仲裁员披露义务
- **第六十一条** 打击虚假仲裁（捏造事实/恶意串通）
- **第八十一条** 可书面约定仲裁地制度
- **第八十二条** ⭐ **临时仲裁制度**（自贸区/海南自贸港企业间涉外纠纷可选临时仲裁，组庭后3日内备案）
- **第八十六条** 支持境外设业务机构
- **第九十四条** 国际投资仲裁案件办理

### 法考核心考点：
- **第三十条** 仲裁协议独立性原则（合同效力不影响仲裁协议效力）
- **第八十一条** 仲裁地确定程序法及管辖法院
- **第八十八条** 域外裁决承认执行管辖法院扩展

## 核心要点速查

| 制度 | 要点 |
|------|------|
| **受案范围** | 合同及财产权益纠纷；排除婚姻/继承/行政争议 |
| **基本原则** | 自愿、独立、公平合理、诚信、一裁终局 |
| **仲裁协议** | 必须书面；独立于主合同 |
| **仲裁庭组成** | 三人庭（各选1人+首席）或独任 |
| **审理方式** | 开庭为原则，不公开为原则 |
| **裁决生效** | 作出之日生效 |
| **撤销期限** | 收到裁决书后3个月内 |
