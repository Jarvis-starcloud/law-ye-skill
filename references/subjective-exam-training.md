### 1.2 主观题 vs 客观题的本质差异

| 维度 | 客观题（选择） | 主观题（案例分析/论述） |
|------|---------------|----------------------|
| **考查能力** | 知识记忆+判断 | **知识运用+逻辑推理+表达** |
| **答案形态** | ABCD四个选项 | 开放式文字作答 |
| **得分关键** | 找到正确选项 | **论证过程本身就有分** |
| **容错率** | 错就是错 | **部分正确也能拿步骤分** |
| **时间压力** | 大（平均1分钟/题）| 极大（40分钟/大题）|
| **AI优势** | ★★★★★ 极强 | ★★★☆☆ 需专门训练 |

### 1.3 评分标准揭秘 🔑

```
主观题采分点（Scoring Rubric）：

满分答案 = 结论正确（20%）+ 论据充分（50%）+ 表达规范（20%）+ 结构清晰（10%）

具体拆解：
┌────────────────────────────────────────────────────┐
│ 📌 结论（约20%分值）                                │
│   ✅ 必须明确给出结论（是/否/有效/无效/有罪/无罪）  │
│   ❌ 只写分析不给结论 → 直接扣掉结论分              │
│                                                    │
│ 📌 论据（约50%分值）🔑🔑🔑                          │
│   ✅ 引用法条：引用具体法律+条+款                    │
│   ✅ 构成要件分析：逐一对照构成要件                  │
│   ✅ 法律关系梳理：厘清当事人之间的权利义务           │
│   ❌ 只有结论没有论据 → 最多得结论分                 │
│                                                    │
│ 📌 表达规范（约20%分值）                             │
│   ✅ 使用法言法语                                   │
│   ✅ 逻辑连接词使用恰当                              │
│   ❌ 口语化表达 → 扣表达分                          │
│                                                    │
│ 📌 结构（约10%分值）                                 │
│   ✅ 分点作答、标号清晰                              │
│   ✅ 先总后分、层次分明                              │
│   ❌ 一段到底、不分要点 → 扣结构分                   │
└────────────────────────────────────────────────────┘

⚠️ 关键认知转变：
客观题 = 选对答案（结果导向）
主观题 = 展示思维过程（过程导向）
→ "即使最终结论错了，前面的分析过程也可能拿到步骤分"
```

## 二、IRAC论证法——律师思维核心框架

### 2.1 什么是IRAC？

**IRAC（Issue-Rule-Application-Conclusion）** 是全球法学院通用的法律分析方法，也是法考主观题最核心的答题框架。

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│   I ── Issue     （争议焦点/问题是什么？）            │
│         ↓                                        │
│   R ── Rule      （适用什么法律规定？）               │
│         ↓                                        │
│   A ── Application（事实如何适用到法律规定上？）🔑     │
│         ↓                                        │
│   C ── Conclusion （得出什么结论？）                  │
│                                                      │
│   💡 记忆口诀：「疑律用结」                           │
│   ── 疑问→法律→应用→结论                            │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 2.2 IRAC完整示范（以合同纠纷为例）

**案情**：甲向乙借款100万元，约定月息3%，以自己名下房产做抵押，双方签订了抵押合同但未办理登记。后甲到期不还，乙起诉要求行使抵押权。

**❌ 差生回答**：
> 乙不能行使抵押权，因为没有登记。

**✅ IRAC标准回答**：

> **I（争议焦点）**：乙对甲的房屋是否享有抵押权？
>
> **R（法律依据）**：
> 根据《民法典》第215条规定："当事人之间订立有关设立、变更、转让和消灭不动产物权的合同，除法律另有规定或者当事人另有约定外，自合同成立时生效；未办理物权登记的，不影响合同效力。"第402条规定："以本法第三百九十五条第一款第一项至第三项规定的财产或者第五项规定的正在建造的建筑物抵押的，应当办理抵押登记。抵押权自登记时设立。"
>
> **A（事实适用）**：
> 本案中，甲乙双方虽已签订书面抵押合同，该合同依法成立并生效（第215条）。但涉案标的为房屋，属于不动产，应当办理抵押登记（第402条）。由于双方未办理抵押登记手续，根据上述规定，抵押权未能设立。
>
> **C（结论）**：
> 因此，乙对甲的房屋**不享有抵押权**，其主张行使抵押权的请求不应支持。但乙可基于有效的抵押合同主张甲承担违约责任。

### 2.3 IRAC的变体——多问题复合型

当一道题目包含多个问题时：

```
整体结构：总（案件概述）→ 分（逐个IRAC）→ 总（总结）

第1步：先列明所有需要回答的问题（Issues List）
第2步：对每个问题逐一进行IRAC分析
第3步：如有交叉影响（如一个问题的结论影响另一个），需说明关联
第4步：最后给出整体结论总结

示例格式：

【问题一】XXX是否构成YYY？
I: ...
R: ...
A: ...
C: ...

【问题二】ZZZ应当如何承担责任？
I: ...
R: ...
A: ...（引用问题一的结论作为前提）
C: ...

【综述】综上所述...
```

## 三、「法言法语」写作规范

### 3.1 核心词汇替换表 🔑

| ❌ 口语化表达 | ✅ 法言法语 | 备注 |
|-------------|-----------|------|
| 告/起诉 | 提起诉讼/提起...之诉 | 动宾搭配 |
| 打官司 | 通过司法途径解决争议 / 诉诸法院 | |
| 不算数 | 无效 / 不发生效力 | 视情形区分 |
| 赔钱 | 承担赔偿责任 / 支付违约金 | 具体化 |
| 没责任 | 不承担法律责任 / 免责 | 视情形 |
| 违法了 | 构成违法 / 违反了...规定 | 后者需接具体条文 |
| 有道理 | 于法有据 / 具有事实和法律依据 | |
| 应该 | 应当 / 必须 | "应当"=一般义务，"必须"=强制义务 |
| 可以 | 得 / 有权 | "得"=可以，"有权"=享有权利 |
| 如果 | 若 / 如 / 倘若 | 书面语 |
| 但是 | 但 / 然而 / 不过 | |
| 因为...所以 | 基于......故 / 由于......因此 | 因果链条 |
| 关于 | 就...而言 / 至于 / 就...问题 | |
| 根据 | 依据 / 按照 / 遵照 | 可互换 |
| 规定说 | ...法第X条第Y款规定 | 必须精确到条款 |
| 这个事情 | 该案 / 本案 / 诉争事项 | |
| 他们 | 各方当事人 / 双方 | |
| 东西 | 标的物 / 对象 / 客体 | |

### 3.2 高频句式模板 🔑🔑

#### （1）引出法律依据

```
标准句式A：
根据《XX法》第X条第X款规定："[直接引用原文]"。

标准句式B：
依据《XX法》第X条之规定，[概括法条核心内容]。

标准句式C：
[法律名称]+[条文内容]（第X条）。

标准句式D（多法条并列）：
结合《XX法》第X条及《YY法》第Y条之规定，......
```

#### （2）将事实映射到法律

```
句式A：本案中，[当事人]的[行为]符合[罪名/民事违法行为]的构成要件。
      即：（1）......；（2）......；（3）......

句式B：就本案事实而言，[具体事实描述]，该情形属于[法律概念]的范畴。

句式C：对比本案事实与[法条]规定的要件，（逐一匹配）......，
      故本案满足/不满足该条的适用条件。
```

#### （3）给出结论

```
句式A：综上/基于上述/鉴于此，[当事人][应当/不应当][承担/不承担][责任类型]。

句式B：故此，[原告的诉讼请求][应予支持/应予驳回/部分应予支持]。

句式C（有条件结论）：若[前提条件成立]，则[结论]；
                但若[另一情形]，则[替代结论]。
```

#### （4）转折与让步

```
句式A：诚然/不可否认，[对方观点有一定合理性]；但/然而，[反驳理由]。

句式B：虽然[承认部分事实]，但[该事实不影响结论]，因为......。

句式C：即便假定[不利前提]，[结论仍然成立]，理由如下：......
```

### 3.3 段落组织原则

```
每一段的标准结构：

【主题句】一句话表明本段要论证什么（如"关于甲是否构成盗窃罪的问题"）
    ↓
【法条引用】引用适用的法律规定（精确到条+款）
    ↓
【事实对应】将案件事实与法条要件逐一对照
    ↓
【分析推导】说明为什么事实符合/不符合要件
    ↓
【小结论】本段的结论

⚠️ 禁忌：
❌ 一段超过8行 → 必须分段
❌ 没有主题句 → 阅卷人找不到重点
❌ 只堆法条不做分析 → 变成了法条抄写员
❌ 只有分析不引法条 → 变成了瞎扯
```

### 4.1 民法案例分析模板

#### 民法答题万能框架 🔑🔑🔑

```markdown
## 【案件概览】（必写，占10-15行）

本案系一起[合同/侵权/物权/不当得利/无因管理/担保]纠纷。
基本事实脉络如下：
① [当事人A]与[当事人B]于[时间]签订了[合同名称]，约定......
② [中间发生了什么事]......
③ [产生争议的原因]......
④ [原告诉请]......，[被告抗辩]......
⑤ 第三人[C]参与情况（如有）：......

---

## 逐一分析

### 问题一：[合同效力问题]

**I（争议焦点）**：甲乙之间的[合同名称]是否有效？

**R（法律依据）**：
根据《民法典》第143条关于民事法律行为有效条件的规定：
"具备下列条件的民事法律行为有效：（一）行为人具有相应的民事行为能力；
（二）意思表示真实；（三）不违反法律、行政法规的强制性规定，不违背公序良俗。"
以及第153条："违反法律、行政法规的强制性规定的民事法律行为无效......"

**A（事实适用）**：
（1）行为能力方面：[分析各方当事人的行为能力]
（2）意思表示方面：[分析是否存在欺诈胁迫重大误解显失公平]
（3）合法性方面：[分析是否违反强制性规定/公序良俗]
（4）（如有其他特殊效力规则需讨论：如无权代理/无权处分/格式条款等）

**C（结论）**：
综上，甲乙之间的合同【有效/无效/可撤销/效力待定】。
若是无效/可撤销，还需补充：根据《民法典》第157条，[后果处理]。

---

### 问题二：[违约责任问题]

**I（争议焦点）**：[谁]是否构成违约？应承担何种责任？

**R（法律依据）**：
《民法典》第577条："当事人一方不履行合同义务或者履行合同义务不符合约定的，应当承担继续履行、采取补救措施或者赔偿损失等违约责任。"
第585条关于违约金的规定......
（如涉及不可抗力→第590条）
（如涉及同时抗辩→第526/527条）

**A（事实适用）**：
[分析违约事实]→[分析免责事由是否存在]→[损失计算]

**C（结论）**：
[被告]【构成/不构成】违约，【应当/无须】承担【继续履行/赔偿损失/支付违约金】责任，金额约为......。
```

#### 民法主观题特殊技巧

```
🔑 民法综合题（第4题）应对策略：

这道题通常融合民法+商法+民诉法，是最难的！

解题顺序建议：
Step 1：先通读全题，画出人物关系图
Step 2：列出所有当事人及其法律身份
Step 3：识别每一个法律关系（合同关系/担保关系/侵权关系/婚姻家庭关系等）
Step 4：按"时间线"或"法律关系线"逐一分析
Step 5：特别注意交叉点（如：一个行为同时触发多个法律责任）

人物关系图画法：
        甲（出卖人）
       /    \
   买卖合同  所有权转移
     /        \
  乙（买受人）←── 贷款合同 ── 丙（银行）
     \
   抵押合同（丙对乙的房屋有抵押权）
     \
   丁（承租人）── 租赁合同 ── 乙

💡 时间线分析法更适合复杂案件：
T1: 甲乙签约
T2: 交付房屋
T3: 乙丙签贷款+抵押
T4: 乙出租给丁
T5: 甲主张撤销合同
→ 在每个时间节点标注法律状态变化
```

### 4.2 刑法案例分析模板

#### 刑法答题万能框架 🔑🔑🔑

```markdown
## 【案件概览】

本案被告人：[姓名]。公诉机关指控：[罪名]。
案件基本事实：[按时间顺序简述犯罪经过]。

---

## 逐一分析

### 问题一：[被告人]的行为是否构成[罪名]？

**I（争议焦点）**：[被告人]的行为是否满足[罪名]的构成要件？

**R（法律依据）**：
《刑法》第[XXX]条规定："[罪名定义原文]"。
相关司法解释：[如有，引用具体解释条文]。

**A（事实适用）—— 四要件分析**：

（1）**客体要件**：
本罪的客体是[具体法益]。本案中，[被告人的行为]侵害了[何种法益]，符合客体要件。

（2）**客观方面**：
本罪客观方面要求[具体行为/结果/危险/因果关系]。
① 行为：[被告人实施了......行为]；
② 结果：[造成了......后果]；
③ 因果关系：[行为与结果之间存在刑法上的因果关系]（条件说/相当因果关系说均可论证）。
故客观方面要件成就。

（3）**主体要件**：
本罪的主体是一般/特殊主体（年龄16/14/18周岁以上）。[被告人]作案时已满[X]周岁，【符合/不符合】主体要件。（如为单位犯罪需另述）

（4）**主观方面**：
本罪的主观方面要求故意/过失。[被告人]明知......而希望/放任......发生/应当预见......但因疏忽大意/过于自信而没有预见，故为【直接故意/间接故意/疏忽大意的过失/过于自信的过失】。

**C（结论）**：
[被告人]的行为**构成[罪名]**（或者：**不构成[罪名]，理由是......**）
```

### 5.2 命题人的"陷阱设计"手法

```
⚠️ 这些陷阱每年都在用，必须警惕：

陷阱1："表面A罪，实际B罪"
  例：看起来像盗窃 → 其实是侵占（已经占有→拒不归还）
  例：看起来像诈骗 → 其实是盗窃（秘密窃取+被害人不知情）
  
陷阱2："前一步合法，后一步非法"
  例：持有假币（买入时不知假→之后发现仍持有=持有假币罪）
  例：收赃（收购时以为是正品→后发现是赃物仍转卖=掩饰隐瞒）
  
陷阱3："多人多罪，互相影响"
  例：甲教唆乙犯罪 → 乙实行过限 → 甲只对教唆范围内的罪负责
  例：共同犯罪中一人中止 → 中止效果是否及于其他人？
  
陷阱4:"程序违法导致实体错误"
  例：侦查阶段刑讯逼供 → 排除口供 → 全案证据不足以定罪
  例：送达程序违法 → 上诉期间计算错误 → 判决未生效
  
陷阱5："看起来有效，其实无效/反之"
  例：抵押合同签了但没登记 → 合同有效但抵押权未设立
  例：法定代表人签字超越权限 → 但构成表见代理 → 对外有效
  
陷阱6："时间差导致的法律状态变化"
  例：T1签约 T2交付 T3登记 T4转让给第三人 → 每个节点的法律状态不同
  例：刑法中行为时法和审判时法的交替适用
```

## 六、模拟真题演练（含标准答案）

### 模拟题1：民法综合（难度★★★☆）

**【案情】**

2024年3月1日，甲公司与乙公司签订《设备买卖合同》，约定：甲向乙出售一台精密机床，价款500万元；乙应于合同签订后10日内支付定金100万元，余款在交货后30日内付清；甲应在收到定金后60日内交货。合同还约定：任何一方违约，应向对方支付合同总价15%的违约金。

3月5日，乙支付定金100万元。4月10日，甲将该机床交付给乙，乙验收合格。但乙迟迟不付余款400万元。

5月1日，甲公司将该债权转让给丙公司，并通知了乙。

5月15日，丙公司向乙催款，乙称："甲交付的机床有一处隐蔽瑕疵，我方正在检验，暂不付款。"经查，该瑕疵确实存在但不影响主要功能使用，维修费用约5万元。此外，乙公司目前经营困难，账面资金仅剩50万元。

另查明：甲公司在交付机床之前，已于3月15日与丁银行签订抵押合同，将该机床抵押给丁银行，借款300万元用于生产经营。双方办理了抵押登记。丁银行的借款于5月20日到期，甲无力偿还。

**【问题】**

1. 定金的数额是否合法？如不合法，应如何处理？
2. 甲将债权转让给丙公司的行为是否有效？丙能否直接向乙主张权利？
3. 乙以机床有瑕疵为由拒付余款的理由是否成立？
4. 丙公司可以向乙主张哪些权利？金额如何计算？
5. 丁银行对机床是否享有抵押权？如有，其与丙公司的债权何者优先受偿？

---

**【参考答案】**

**问题一：定金数额的合法性**

**I**：乙支付的100万元定金是否符合法律规定？

**R**：《民法典》第586条第2款规定："定金的数额由当事人约定，但是不得超过主合同标的额的百分之二十，超过部分不产生定金的效力。"

**A**：本案主合同标的额为500万元，法定定金上限为500万×20%=100万元。乙支付的定金恰好为100万元，未超过上限。

**C**：定金数额**合法有效**。

---

**问题二：债权转让的效力**

**I**：甲将债权转让给丙公司的行为是否有效？丙能否直接向乙主张？

**R**：《民法典》第545条规定："债权人可以将债权的全部或者部分转让给第三人，但是有下列情形除外：（一）根据债权性质不得转让；（二）按照当事人约定不得转让；（三）依照法律规定不得转让。"第546条："债权人转让债权，未通知债务人对债务人不发生效力。"

**A**：本案中，（1）该金钱债权不属于依法不得转让的情形；（2）合同中未约定禁止转让；（3）甲已通知乙转让事宜。因此债权转让行为合法有效。

**C**：甲的债权转让行为**有效**，丙可以直接向乙主张债权。乙应当向丙公司履行还款义务。

---

**问题三：乙拒付余款的抗辩是否成立**

**I**：乙以机床存在隐蔽瑕疵为由拒付全部余款400万元，该抗辩是否成立？

**R**：《民法典》第582条："履行不符合约定的，应当按照当事人的约定承担违约责任......受损害方根据标的的性质以及损失的大小，可以合理选择请求对方承担修理、重作、更换、退货、减少价款或者报酬等违约责任。"第577条关于一般违约责任的规定。

**A**：本案中，机床确有隐蔽瑕疵（维修费约5万元），构成质量瑕疵违约。但该瑕疵不影响主要功能使用，属于轻微瑕疵。乙据此享有的权利是：（1）请求甲修理/减少价款（减少额度≈5万元）；（2）但不能**拒绝支付全部余款**。拒绝全部付款属于行使同时履行抗辩权的前提是对方"完全不履行"或"履行严重不符"，而本案仅为轻微瑕疵。

**C**：乙的抗辩**部分成立**。乙可以主张减少价款约5万元，但仍应支付剩余395万元（400-5）。乙无权以该为由拒付全部款项。

---

**问题四：丙公司的权利主张**

**I**：丙公司作为受让人，可向乙主张哪些权利？

**R**：《民法典》第547条："债权人转让债权的，受让人取得与债权有关的从权利，但是该从权利专属于债权人自身的除外。"关于违约金与定金的选择适用（第588条）。

**A**：
（1）**本金债权**：400万元（减去瑕疵折价5万元后为395万元）。
（2）**违约金**：根据合同约定为500万×15%=75万元。
（3）**定金**：100万元（但需注意定金与违约金的选择规则）。

根据第588条："当事人既约定违约金，又约定定金的，一方违约时，对方可以选择适用违约金或者定金条款。"丙公司可以在违约金（75万元）和定金（100万元）中选择其一。**定金100万元更有利**。

此外，丙还可以主张逾期付款利息。

**C**：丙公司可向乙主张：**货款395万元 + 定金100万元（选择适用定金条款而非违约金）+ 逾期利息**（自应付款日起至实际给付日止，按LPR计算）。

---

**问题五：抵押权与债权受偿顺位**

**I**：丁银行是否享有抵押权？如有，其与丙公司的债权何者优先？

**R**：《民法典》第403条（动产抵押自合同生效时设立）、第414条关于抵押权顺位的规定（已登记的抵押权按照登记的时间先后确定顺位）。

**A**：
（1）丁银行的抵押权：甲以机床设定动产抵押并办理了登记，抵押权**自登记时设立**（3月15日登记），丁银行**享有抵押权**。
（2）丙公司：受让的是一般金钱债权，**无担保物权**。
（3）同一财产上既有抵押权又有普通债权时，抵押权人**优先受偿**。

**C**：丁银行对机床**享有抵押权**，且**优先于丙公司的普通债权**受偿。机床拍卖/变卖所得价款，应首先清偿丁银行的300万元贷款本息，剩余部分才可用于偿还丙公司的债权。

### 模拟题2：刑法综合（难度★★★★）

**【案情】**

甲（男，17周岁）因手头拮据，计划搞钱。甲找到好友乙（男，25周岁，有抢劫罪前科）商量。乙提议："隔壁小区的王大爷家好像有钱，我们去'借'点钱花吧。"甲同意。

2024年6月5日晚11时许，甲携带一把水果刀，乙携带一根铁棍，来到王大爷家门口。甲负责敲门，乙在后接应。王大爷开门后，甲持刀威胁："把钱拿出来！"王大爷大声呼救。乙见状用铁棍击打王大爷头部（事后鉴定为轻伤一级），王大爷昏倒。二人进入屋内翻找，发现现金2万元及一块手表（价值3万元）。

逃跑途中，乙对甲说："这块手表归我了，你拿钱就行。"甲未置可否。

次日，甲得知公安机关正在调查此案，内心恐惧，主动到派出所投案，如实供述了自己和乙的全部罪行。乙被公安机关抓获后，也如实供述了罪行。

**【问题】**

1. 甲、乙构成何罪？犯罪形态如何？
2. 甲、乙在共同犯罪中的地位和作用如何？
3. 乙的前科是否影响本案量刑？是否构成累犯？
4. 甲的投案行为如何认定？对其量刑有何影响？
5. 手表的归属如何处理？

---

**【参考答案】**

**问题一：罪名与形态**

**I**：甲、乙的行为构成何罪？处于何种犯罪形态？

**R**：《刑法》第263条关于抢劫罪的规定："以暴力、胁迫或者其他方法抢劫公私财物的，处三年以上十年以下有期徒刑......"入户抢劫的，处十年以上......。第23条（未遂）、第24条（中止）关于犯罪停止形态的规定。

**A**：
（1）**罪名认定**：甲、乙以非法占有为目的，当场使用暴力（持刀威胁+铁棍击打致人轻伤），强行劫取他人财物，且发生在被害人家中（入户），构成**抢劫罪（入户抢劫）**。
（2）**犯罪形态**：二人已实施完毕暴力劫财行为，财物已到手，犯罪**既遂**。抢劫罪的既遂标准为取得财物或造成轻伤以上后果，二者均已满足。

**C**：甲、乙构成**抢劫罪（入户抢劫）的共同犯罪，且均为犯罪既遂**。

---

**问题二：共犯地位**

**I**：甲、乙在共同犯罪中的作用如何划分？

**R**：《刑法》第26条（主犯）、第27条（从犯）的规定。

**A**：
（1）甲：发起犯意、积极参与、持刀威胁、敲门入户，在整个犯罪过程中发挥了主要作用，应认定为**主犯**。
（2）乙：提议作案、准备凶器、实施暴力（致人轻伤）、事后分赃，同样发挥主要作用，也应认定为**主犯**。
（3）二人作用相当，**不区分首要分子与普通主犯**。

**C**：甲、乙**均为主犯**，应当按照其所参与的或者组织的全部犯罪处罚。

---

**问题三：前科与累犯**

**I**：乙的前科是否构成本案的法定从重情节？是否构成累犯？

**R**：《刑法》第65条（一般累犯）："被判处有期徒刑以上刑罚的犯罪分子，刑罚执行完毕或者赦免以后，在五年以内再犯应当判处有期徒刑以上刑罚之罪的，是累犯，应当从重处罚，但是过失犯罪和未成年人犯罪的除外。"

**A**：
（1）前科性质：乙曾有抢劫罪前科，说明其主观恶性较深，属于**酌定从重处罚情节**。
（2）累犯认定：需要知道前罪刑罚何时执行完毕。如果前罪刑罚执行完毕**不满5年**又犯本案（应判10年以上有期徒刑的入户抢劫），则**构成累犯**。如果已满5年，则不构成累犯。

**C**：乙的前科至少构成**酌定从重情节**；若前罪执行完毕不满5年，则同时**构成累犯，应当从重处罚**。

---

**问题四：甲的投案认定**

**I**：甲主动到派出所投案的行为如何定性？

**R**：《刑法》第67条关于自首的规定："犯罪以后自动投案，如实供述自己的罪行的，是自首。对于自首的犯罪分子，可以从轻或者减轻处罚。其中，犯罪较轻的，可以免除处罚。"

**A**：甲在犯罪后**主动投案**（非被抓获），并**如实供述了全部罪行**（包括自己和同案犯乙的罪行），完全符合自首的成立条件。此外，甲作案时**未满18周岁**（17周岁），依法应当**从轻或者减轻处罚**。

**C**：甲的行为**构成自首**，且系**未成年人犯罪**，依法应当**从轻或者减轻处罚**。

---

**问题五：赃物归属**

**I**：犯罪所得手表应如何处理？

**R**：共同犯罪中赃物处理的原则——赃物属于共同违法所得，应由司法机关依法追缴后返还被害人，而非犯罪人之间自行"分配"。

**A**：手表和现金均为抢劫所得，属于甲、乙**共同的违法所得**。乙单方面声称"手表归我"，不具有法律效力。根据《刑法》第64条："犯罪分子违法所得的一切财物，应当予以追缴或者责令退赔退赔被害人的合法财产，应当及时返还......"

**C**：手表（3万元）和现金（2万元）均应**依法追缴，返还被害人王大爷**。乙无权独占手表，其与甲的"内部约定"不受法律保护。

