# 劳动仲裁申请书撰写指南

> 版本：v1.0.0 | 更新日期：2026-04-08
> 法律依据：《劳动争议调解仲裁法》《劳动人事争议仲裁办案规则》

## 一、仲裁申请书基本结构

### 1.1 标准格式

```
                    劳动仲裁申请书

申请人：[姓名]，[性别]，[民族]，[出生日期]，
        住所地：[住所地址]
        联系电话：[手机号码]
        身份证号码：[身份证号]

被申请人：[单位全称]
        住所地：[注册地址]
        法定代表人：[姓名]，职务：[职务]
        联系电话：[电话]
        统一社会信用代码：[代码]

仲裁请求：
    一、……
    二、……
    三、……

事实与理由：
    [正文内容]

    此致
[XX市/区]劳动人事争议仲裁委员会

                                    申请人：[签名/手印]
                                    日  期：[年] 年 [月] 月 [日] 日

附：1. 申请人身份证复印件
    2. 被申请人工商信息
    3. 证据材料清单及证据副本
```

## 二、当事人信息填写规范

### 2.1 申请人（劳动者）信息

| 项目 | 要求 | 示例 |
|------|------|------|
| 姓名 | 身份证上的法定姓名 | 张三 |
| 性别 | 男/女 | 男 |
| 民族 | 身份证上的民族 | 汉族 |
| 出生日期 | 年月日 | 1990年5月15日 |
| 住所地 | 身份证地址或实际居住地 | 北京市朝阳区XX路XX号 |
| 联系电话 | 保持畅通的手机号 | 138XXXXXXXX |
| 身份证号码 | 18位 | 110105199005150XXX |

### 2.2 被申请人（用人单位）信息

| 项目 | 要求 | 获取途径 |
|------|------|----------|
| 单位全称 | 营业执照上的准确名称 | 国家企业信用信息公示系统 |
| 住所地 | 注册地址（非实际经营地址） | 工商登记信息 |
| 法定代表人 | 姓名+职务 | 工商登记信息 |
| 联系电话 | 单位电话 | 工商信息/官网/合同 |
| 统一社会信用代码 | 18位 | 工商登记信息 |

### 2.3 多被申请人的写法

当存在多个被申请人时：

```
被申请人一：[派遣单位全称]

           法定代表人：……
           统一社会信用代码：……

被申请人二：[用工单位全称]
           住所地：……
           法定代表人：……
           统一社会信用代码：……
```

## 三、仲裁请求撰写技巧

### 3.1 请求类型

| 类型 | 用语 | 示例 |
|------|------|------|
| **给付请求** | "裁决……支付……" | 裁决被申请人支付工资差额 10,000.00 元 |
| **确认请求** | "确认……" | 确认申请人与被申请人自2023年3月1日起存在劳动关系 |
| **连带请求** | "裁决……连带支付……" | 裁决被申请人一、被申请人二连带支付违法解除赔偿金 |

### 3.2 金额格式规范

✅ 正确格式：
- `10,000.00 元`（小数点前每三位加逗号，保留两位小数）
- `（大写：壹万元整）`（大额金额建议加大写）

❌ 错误格式：
- `10000元` ← 缺少千位分隔符和小数位
- `一万元` ← 不应使用中文数字

### 3.3 常见请求项模板

#### （一）确认劳动关系

```
一、确认申请人与被申请人自[入职日期]起至[离职日期]止存在劳动关系。
```

#### （二）未签劳动合同二倍工资

```
二、裁决被申请人支付[起始月]至[终止月]未签订书面劳动合同二倍工资差额[金额]元。
```

计算公式：
- 期间：入职满1个月后至满1年（最长11个月）
- 金额：月工资 × 应付月数

#### （三）拖欠工资

```
三、裁决被申请人支付[起始月]至[终止月]拖欠工资[金额]元。
```

#### （四）加班费

```
四、裁决被申请人支付[起始日期]至[终止日期]期间加班费[金额]元，其中：
    1. 工作日延时加班费[金额]元；
    2. 休息日加班费[金额]元；
    3. 法定节假日加班费[金额]元。
```

计算公式：
- 工作日加班：小时工资 × 1.5 × 加班小时数
- 休息日加班：小时工资 × 2 × 加班小时数（未安排补休）
- 法定节假日加班：小时工资 × 3 × 加班小时数

#### （五）经济补偿金

```
五、裁决被申请人支付解除劳动合同经济补偿金[金额]元。
```

计算公式：
- N = 工作年限（每满1年 = 1个月，6个月~1年 = 1个月，< 6个月 = 0.5个月）
- 月工资 = 离职前12个月平均工资
- 经济补偿 = N × 月工资
- 高薪封顶：月工资 > 当地社平工资3倍的，按3倍计算，最高12年

#### （六）违法解除赔偿金（2N）

```
六、裁决被申请人支付违法解除劳动合同赔偿金[金额]元。
```

计算公式：赔偿金 = 经济补偿 × 2 = 2N

#### （七）社保补缴（特殊）

```
七、裁决被申请人为申请人补缴[起始月]至[终止月]期间的社会保险。
```

⚠️ **注意**：社保补缴严格意义上不属于仲裁受理范围，但部分地区仲裁委会受理或调解。建议同时向社保稽核部门投诉。

#### （八）工伤待遇

```
八、裁决被申请人支付以下工伤保险待遇：
    1. 一次性伤残补助金[金额]元；
    2. 一次性工伤医疗补助金[金额]元；
    3. 一次性伤残就业补助金[金额]元；
    4. 停工留薪期工资[金额]元；
    5. 护理费[金额]元。
```

#### （九）连带责任（劳务派遣）

```
九、裁决被申请人一（派遣单位）、被申请人二（用工单位）连带支付违法解除劳动合同赔偿金[金额]元。
```

## 四、请求项的攻防逻辑

### 4.1 请求顺序策略

```
基础请求（确认性）
    │
    ▼
主体请求（给付性）
    │
    ▼
附带请求（连带/从属）
```

**示例（劳务派遣案件）**：

1. **先确认**：确认与用工单位存在事实劳动关系（穿透派遣）
2. **再主张**：支付未签合同二倍工资、违法解除赔偿金
3. **备选**：如第1项不被支持，则主张派遣单位与用工单位连带责任

### 4.2 主备请求写法

当对法律关系认定存在不确定性时，可写明主备请求：

```
仲裁请求：

【主请求】
一、确认申请人与被申请人二（用工单位）自2023年3月1日起存在事实劳动关系；
二、裁决被申请人二支付违法解除劳动合同赔偿金 60,000.00 元。

【备选请求】（如主请求一不被支持）
三、裁决被申请人一（派遣单位）支付违法解除劳动合同赔偿金 60,000.00 元；
四、裁决被申请人二（用工单位）对上述赔偿金承担连带责任。
```

### 4.3 请求过高/过低的风险

| 风险 | 后果 | 应对 |
|------|------|------|
| **请求过高** | 仲裁费用增加（部分地区）、对方反驳激烈、显得不专业 | 严格按法定标准计算，有依据的可适当上浮 |
| **请求过低** | 裁决不会超过请求金额、利益受损 | 仔细计算，不要遗漏任何应得项目 |
| **请求遗漏** | 未请求的项目无法获得裁决 | 逐项检查，使用清单确认 |

## 五、事实与理由撰写规范

### 5.1 结构模板

```
事实与理由：

一、劳动关系基本情况
    申请人于[入职日期]入职被申请人处，从事[岗位]工作，
    双方签订/未签订[合同类型]劳动合同，约定月工资[金额]元，
    劳动合同期限为[起始日期]至[终止日期]。

二、争议发生经过
    [按时间顺序，客观陈述争议发生的经过]
    [重点描述对方的违法行为]

三、法律依据
    被申请人的上述行为违反了《劳动合同法》第[条款]条之规定，
    依法应当承担[具体责任]。

四、申请仲裁
    综上所述，申请人的合法权益受到严重侵害。为维护申请人的
    合法权益，特向贵委提起仲裁申请，恳请依法裁决。
```

### 5.2 撰写要点

| 要点 | 正确做法 | 错误做法 |
|------|----------|----------|
| **客观中立** | 陈述事实，避免情绪化语言 | "公司简直太黑心了" |
| **时间明确** | 精确到年月日 | "去年入职的"、"前几天被辞退" |
| **逻辑清晰** | 按时间顺序或因果关系组织 | 想到哪写到哪，前后跳跃 |
| **引用法条** | 明确具体法律条款 | "根据劳动法规定" |
| **避免结论** | 由仲裁员认定 | "被申请人明显违法" |

### 5.3 违法解除案件的事实陈述示例

```
二、争议发生经过

    2026年3月15日，被申请人人事部门口头通知申请人，以"公司
业务调整"为由要求申请人于当日离职，未出具任何书面解除通知。

    申请人当即表示异议，要求被申请人出具书面解除通知并说明
解除依据。被申请人拒绝出具书面通知，并于2026年3月16日停止
申请人的门禁权限和办公系统账号。

    2026年3月18日，申请人通过EMS向被申请人注册地址邮寄
《要求出具解除劳动合同证明书》，被申请人签收后至今未予回复。

三、法律依据

    被申请人的上述行为存在以下违法情形：

    （一）解除程序违法。根据《劳动合同法》第四十三条规定，
用人单位单方解除劳动合同，应当事先将理由通知工会。被申请人
未履行通知工会程序。

    （二）解除理由不成立。"业务调整"不属于《劳动合同法》
第三十九条、第四十条规定的法定解除情形。即使构成"客观情况
发生重大变化"，被申请人也应履行第四十条规定的协商程序。

    综上，被申请人的解除行为属于违法解除，依据《劳动合同法》
第八十七条规定，应当支付赔偿金。
```

## 六、证据清单编制

### 6.1 证据清单格式

| 序号 | 证据名称 | 证据形式 | 份数 | 证明目的 |
|------|----------|----------|------|----------|
| 1 | 劳动合同 | 原件 | 1份 | 证明双方存在劳动关系及合同约定 |
| 2 | 工资银行流水 | 原件 | 1份 | 证明工资发放情况及工资标准 |
| 3 | 社保缴纳记录 | 打印件 | 1份 | 证明社保缴纳主体及期间 |
| 4 | 微信聊天记录 | 打印件 | 1份 | 证明被申请人单方解除劳动合同 |
| 5 | EMS快递单及签收记录 | 原件+复印件 | 1份 | 证明申请人已主张权利 |

### 6.2 劳动争议常用证据清单

#### 证明劳动关系：
- [ ] 劳动合同
- [ ] 入职登记表/入职通知书
- [ ] 工资银行流水（显示发放主体）
- [ ] 社保缴纳记录
- [ ] 工牌、工服照片
- [ ] 考勤记录
- [ ] 工作邮件/钉钉/企微聊天记录

#### 证明工资标准：
- [ ] 劳动合同工资条款
- [ ] Offer Letter
- [ ] 工资条/工资单
- [ ] 银行流水
- [ ] 完税证明

#### 证明解除经过：
- [ ] 解除通知书/离职证明
- [ ] 解除前后的沟通记录（微信、邮件、录音）
- [ ] 门禁/系统停用记录
- [ ] 证人证言

#### 证明加班事实：
- [ ] 考勤记录（需单位盖章）
- [ ] 加班审批单
- [ ] 工作群消息记录（显示加班时间）
- [ ] 加班期间的工作成果（邮件发送时间等）

## 七、常见错误与避坑指南

### 7.1 当事人信息错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 被申请人名称写错 | 可能被驳回或无法执行 | 以工商登记信息为准 |
| 只写简称/曾用名 | 可能被要求补正 | 写全称并核实 |
| 法定代表人写错 | 可能影响送达 | 查询最新工商信息 |

### 7.2 仲裁请求错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 请求项表述模糊 | 仲裁庭难以裁决 | 明确具体金额和依据 |
| 计算错误 | 可能只支持正确部分 | 提前核算，必要时咨询 |
| 遗漏请求项 | 无法获得裁决 | 逐项检查 |
| 超过仲裁时效 | 被驳回或不予受理 | 注意1年时效，中断情形 |

### 7.3 事实与理由错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 情绪化表述 | 显得不专业，削弱说服力 | 客观陈述事实 |
| 时间模糊 | 影响计算和事实认定 | 精确到年月日 |
| 法条引用错误 | 削弱专业性 | 仔细核对法条编号 |
| 自认不利事实 | 被对方利用 | 只陈述对己方有利的事实 |

## 八、提交与立案流程

### 8.1 立案材料清单

1. **仲裁申请书**（正本1份 + 副本N份，N = 被申请人数量）
2. **申请人身份证明**（身份证复印件）
3. **被申请人主体资格证明**（工商登记信息打印件）
4. **证据材料**（正本1套 + 副本N套）
5. **证据清单**
6. **授权委托书**（如有代理人）

### 8.2 管辖确定

一般规则：
- 劳动合同履行地仲裁委
- 用人单位所在地仲裁委

两者不一致时，由**劳动合同履行地**仲裁委管辖。

### 8.3 仲裁时效

| 时效类型 | 期限 | 起算点 |
|----------|------|--------|
| 一般时效 | **1年** | 知道或应当知道权利被侵害之日 |
| 劳动关系存续期间的工资争议 | 不受1年限制 | 劳动关系终止后1年内 |

中断情形：
- 向对方主张权利（留存证据！）
- 向有关部门投诉
- 调解

```
工作年限计算：
  ┌─────────────────────────────────────────┐
  │ 入职日期：_____年___月___日              │

  │                                         │

  │                                         │

  │                                         │
  │ N = _____ 个月                          │

  ┌─────────────────────────────────────────┐

  经济补偿金 = N × 月工资 = _____ × _____ = _________元

## 十、实战模板

### 10.1 违法解除赔偿仲裁申请书（完整模板）

```
                    劳动仲裁申请书

申请人：张三，男，汉族，1990年5月15日出生，
        住所地：北京市朝阳区XX路XX号
        联系电话：13800000000
        身份证号码：110105199005150000

被申请人：北京XX科技有限公司
        住所地：北京市海淀区XX路XX号XX大厦X层
        法定代表人：李四，职务：执行董事
        联系电话：010-00000000
        统一社会信用代码：91110108XXXXXXXXXX

仲裁请求：

一、裁决被申请人支付违法解除劳动合同赔偿金人民币60,000.00元
   （大写：陆万元整）；
二、裁决被申请人支付2026年1月至3月拖欠工资人民币15,000.00元
   （大写：壹万伍仟元整）；
三、裁决被申请人出具解除劳动合同证明书。

事实与理由：

一、劳动关系基本情况

    申请人于2023年3月1日入职被申请人处，担任产品经理职务。
双方签订了期限为2023年3月1日至2026年2月28日的固定期限劳动
合同，约定月工资为人民币10,000.00元（税前）。

二、争议发生经过

    2026年3月15日，被申请人人事部门负责人王五口头通知申请人，
以"公司业务调整、岗位撤销"为由，要求申请人于当日办理离职手续。
申请人当即表示异议，要求被申请人出具书面解除通知并说明法律依据，
