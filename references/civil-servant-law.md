# 中华人民共和国公务员法 学习笔记

> 学习时间：2026年4月2日 | 整理：贾维斯
> 版本：2018年12月29日修订（2005年通过，2019年6月1日起施行）

## 一、法律概览

| 项目 | 内容 |
|------|------|
| 全称 | 中华人民共和国公务员法 |
| 性质 | 行政法（公务员管理的基本法律） |
| 通过时间 | 2005年4月27日 |
| 修订历程 | 2017年修正→**2018年全面修订** |
| 施行时间 | 2019年6月1日 |
| 结构 | **18章113条** |
| 核心定位 | 规范公务员的"选、用、管、退"全流程 |

### 公务员的法律定义（第2条）

三个核心要素缺一不可：

| 要素 | 内容 |
|------|------|
| 依法履行公职 | 行使公权力 |
| 纳入国家行政编制 | 有正式"编制" |
| 由国家财政负担工资福利 | 财政供养 |

> 简记：**公职+编制+财政** = 公务员

| 3 | 忠于人民，全心全意为人民服务 | 忠于人民 |
| 4 | 忠于职守，服从上级依法决定命令 | 忠于职守 |

## 五、录用制度（第23-34条）

### 5.1 录用范围

一级主任科员以下及其他相当职级层次 → **公开考试、严格考察、平等竞争、择优录取**。

### 5.2 不得录用的5种情形（第26条）

| 序号 | 情形 |
|------|------|
| 1 | 因犯罪受过**刑事处罚** |
| 2 | 被**开除中国共产党党籍** |
| 3 | 被**开除公职** |
| 4 | 被列为**失信联合惩戒对象**（2018年新增） |
| 5 | 法律规定的其他情形 |

### 5.3 录用程序

```
发布招考公告 → 资格审查 → 笔试+面试 → 考察+体检 → 公示（≥5个工作日）→ 备案/审批 → 试用（1年）
```

## 六、考核制度（第35-39条）

### 6.1 考核内容：德能勤绩廉

| 维度 | 内容 | 重点 |
|------|------|------|
| 德 | 政治素质、道德品行 | **重点考核政治素质** |
| 能 | 工作能力、业务水平 | |
| 勤 | 工作态度、勤勉程度 | |
| 绩 | 工作实绩、完成情况 | **重点考核工作实绩** |
| 廉 | 廉洁自律、遵纪守法 | |

### 6.2 考核方式与等次

| 考核方式 | 说明 |
|---------|------|
| 平时考核 | 日常表现 |
| 专项考核 | 特定任务 |
| **定期考核**（年度） | 以平时+专项为基础 |

| 考核等次 | 含义 |
|---------|------|
| 优秀 | 表现突出 |
| 称职 | 胜任工作 |
| 基本称职 | 勉强胜任 |
| 不称职 | 不胜任 |

**后果**：优秀/称职→年终奖金；不称职→降低一个职务或职级层次。

## 七、奖励与惩戒

### 7.1 奖励五等级（第53条）

| 等级 | 从低到高 |
|------|---------|
| 1 | 嘉奖 |
| 2 | 记三等功 |
| 3 | 记二等功 |
| 4 | 记一等功 |
| 5 | 授予称号 |

原则：精神奖励与物质奖励结合，**以精神奖励为主**。

### 7.2 处分六等级（第62条）

| 等级 | 处分期 | 影响 |
|------|--------|------|
| 警告 | **6个月** | 不晋升职务/职级/级别 |
| 记过 | **12个月** | +不晋升工资档次 |
| 记大过 | **18个月** | +不晋升工资档次 |
| 降级 | **24个月** | +不晋升工资档次+降低级别 |
| 撤职 | **24个月** | +不晋升工资档次+降低级别 |
| 开除 | 永久 | 不得录用为公务员（第26条③） |

**处分解除**（第65条）：开除以外的处分→处分期间有悔改表现+无再违纪→**期满自动解除**。但降级/撤职解除后**不恢复原级别/原职务**。

### 7.3 18种禁止行为（第59条）

涵盖：散布有损言论、参加非法组织/罢工、破坏民族关系、不担当不作为、拒绝执行命令、打击报复、弄虚作假、贪污贿赂、浪费资财、滥用职权、泄密、损害国家荣誉、参与色情赌博、违反职业道德/社会公德/家庭美德、违规网络行为、违规营利活动/兼职、旷工/逾期不归等。

### 7.4 上级命令执行规则（第60条）⭐

```
认为上级决定/命令有错误
    → 可以向上级提出改正意见
    → 上级不改变或要求立即执行
    → 应当执行，后果由上级负责
    
⚠️ 但执行"明显违法"的决定/命令 → 应当依法承担相应责任
```

## 八、交流与回避（第69-78条）

### 8.1 交流方式

| 方式 | 说明 |
|------|------|
| 调任 | 从国企/事业单位→机关（担任领导职务或四级调研员以上） |
| 转任 | 公务员之间不同职位转换 |
| 挂职 | 选派承担重大项目（不改变人事关系） |

### 8.2 三种回避制度

| 类型 | 内容 | 条文 |
|------|------|------|
| **任职回避** | 夫妻/直系血亲/三代旁系/近姻亲→不得在同一机关直接上下级+不得从事组织人事纪检监察审计财务 | 第74条 |
| **地域回避** | 乡/县/设区市级主要领导→不在原籍任职 | 第75条 |
| **公务回避** | 涉及本人/亲属利害关系→执行公务时回避 | 第76条 |

## 九、辞职与辞退

### 9.1 辞职

| 项目 | 内容 |
|------|------|
| 程序 | 书面申请→任免机关审批 |
| 审批时限 | 一般**30日**，领导成员**90日** |

#### 不得辞职的5种情形（第86条）

| 序号 | 情形 |
|------|------|
| 1 | 未满最低服务年限 |
| 2 | 涉及国家秘密职位/脱密期未满 |
| 3 | 重要公务未处理完毕 |
| 4 | 正在接受审计/纪律审查/监察调查/司法程序未终结 |
| 5 | 法律规定的其他情形 |

#### 领导职务辞职四类型（第87条）

| 类型 | 说明 |
|------|------|
| 因公辞职 | 工作变动依法辞去现职 |
| 自愿辞职 | 个人原因提出 |
| **引咎辞职** | 工作严重失误/重大事故→**应当**引咎辞职 |
| **责令辞职** | 不适合继续担任/应引咎辞职而不辞→**责令辞职** |

### 9.2 辞退

#### 五种辞退情形（第88条）

| 序号 | 情形 |
|------|------|
| 1 | 连续**2年**考核不称职 |
| 2 | 不胜任现职且不接受其他安排 |
| 3 | 机关调整缩编，本人拒绝合理安排 |
| 4 | 不履行义务不遵守纪律，教育无效，不宜开除 |
| 5 | 旷工连续超**15天**，或一年累计超**30天** |

#### 不得辞退的4种情形（第89条）

| 序号 | 情形 |
|------|------|
| 1 | 因公致残，丧失/部分丧失工作能力 |
| 2 | 患病/负伤在医疗期内 |
| 3 | 女性公务员孕期、产假、哺乳期 |
| 4 | 法律规定的其他情形 |

> 对比劳动合同法第42条——保护逻辑高度一致：**工伤+医疗期+三期**。

## 十、退休制度（第92-94条）

### 10.1 法定退休

达到国家规定退休年龄**或**完全丧失工作能力→**应当退休**。

### 10.2 提前退休（第93条，自愿申请+批准）

| 条件 | 内容 |
|------|------|
| 条件一 | 工作年限满**30年** |
| 条件二 | 距退休不足**5年**且工作年限满**20年** |
| 条件三 | 其他法定情形 |

## 十一、申诉与控告（第95-99条）

### 11.1 申诉流程

```
人事处理不服
    → 30日内向原处理机关申请复核
    → 复核不服→15日内向公务员主管部门/上级机关申诉
    → 也可不经复核→30日内直接申诉
    → 省级以下申诉决定不服→可再申诉

复核期限：30日
申诉期限：60日（可延长30日）
```

**重要保护**：
- 复核、申诉期间**不停止执行**
- **不因申诉而被加重处理**（禁止报复）

### 11.2 可申诉的8种人事处理

处分、辞退、取消录用、降职、不称职考核、免职、辞职/提前退休未获批准、扣减工资福利保险。

## 十二、职位聘任制（第100-105条）

| 项目 | 内容 |
|------|------|
| 适用范围 | 专业性较强的职位+辅助性职位（涉密职位除外） |
| 审批 | 省级以上公务员主管部门批准 |
| 合同期限 | **1-5年** |
| 试用期 | **1-12个月** |
| 薪酬 | **协议工资制** |
| 争议解决 | 60日内申请仲裁→不服15日内起诉 |

---

## 十三、离职后限制（第107条）⭐

| 人员类别 | 限制期 | 限制内容 |
|---------|--------|---------|
| 领导成员/县处级以上领导 | **3年** | 不得到原业务相关企业任职/从事营利活动 |
| 其他公务员 | **2年** | 同上 |

违反后果：没收违法所得+接收单位罚款（违法所得1-5倍）。

## 十五、2018年修订重点变化

| 变化 | 内容 |
|------|------|
| **职务与职级并行** | 最核心制度变革，12级职级序列，不当领导也能晋升 |
| 强化政治要求 | 党管干部写入总则，政治素质列为考核重点 |
| 增设心理素质条件 | 第13条新增"心理素质" |
| 失信惩戒 | 被列为失信联合惩戒对象→不得录用（第26条④） |
| 监察衔接 | 涉嫌职务违法犯罪→移送监察机关（第57条） |
| 宪法宣誓 | 就职时公开进行宪法宣誓（第9条） |
| 新增禁止行为 | "不担当不作为"、违规网络行为等 |
| 公开遴选 | 上级机关注重从基层公开遴选（第71条） |
| 挂职制度 | 明确挂职安排（第72条） |

## 十六、与已学法律的交叉关联

| 关联法律 | 交叉点 |
|---------|--------|
| **宪法** | 公务员制度根据宪法制定（第1条）；宪法宣誓制度（第9条对应宪法第27条） |
| **劳动法/劳动合同法** | 公务员**不适用**劳动合同法（公务员走公务员法，企业员工走劳动法）；但不得辞退保护逻辑一致（工伤/医疗期/三期） |
| **刑法** | 贪污受贿（刑法第382-393条）；滥用职权/玩忽职守（刑法第397条）；公务员犯罪→不得录用（第26条①） |
| **治安管理处罚法** | 公务员违纪情节较轻→行政处分；情节严重→移送监察/司法 |
| **民法典** | 公务员侵权→国家赔偿+个人处分；错误人事处理→赔礼道歉+赔偿（第110条） |
| **民事诉讼法** | 聘任制公务员争议→仲裁→不服起诉（第105条→民诉法审判程序） |
| **网络安全法** | 公务员不得泄露国家秘密和工作秘密（第14条⑤+第59条⑪）；不得违规网络行为（第59条⑮） |

## 十七、实践知识点

### 17.1 小明场景：想考公务员需要知道什么？

> 小明大学毕业想考公务员。

**考公指南**（依据公务员法）：
1. ✅ 基本条件：中国国籍+年满18岁+拥护宪法和党的领导+学历达标（第13条）
2. ❌ 不得报考：犯罪记录/开除党籍/开除公职/失信惩戒对象（第26条）
3. 📝 考试流程：公告→报名→笔试→面试→考察→体检→公示（≥5天）→审批→试用1年
4. 📋 试用期1年，不合格取消录用（第34条）
5. ⚠️ 入职后：宪法宣誓（第9条）；18种行为红线不可触碰（第59条）
6. 💰 工资=基本工资+津贴+补贴+奖金，国家统一规定（第80条）
7. 📈 晋升通道：领导职务（10级）+ 职级序列（12级），不当领导也能升

### 17.2 对比：公务员 vs 企业员工

| 对比项 | 公务员 | 企业员工 |
|--------|--------|---------|
| 适用法律 | 公务员法 | 劳动法+劳动合同法 |
| 录用方式 | 公开考录 | 企业自主招聘 |
| 合同形式 | 委任制为主（聘任制为例外） | 劳动合同 |
| 薪酬制度 | 国家统一规定 | 企业自主+最低工资保障 |
| 处分 | 6种（警告→开除） | 劳动合同法无处分规定 |
| 辞职 | 需审批（30/90日） | 提前30日通知即可 |
| 辞退保护 | 工伤+医疗期+三期 | 工伤+医疗期+三期+职业病+老员工 |
| 争议解决 | 复核→申诉→再申诉 | 仲裁（前置）→诉讼 |
| 退休 | 可提前退休（满30年/距退休5年+满20年） | 依社会保险法 |
| 离职后限制 | 领导3年/其他2年 | 竞业限制≤2年 |
