## 二、核心定位：刑法的"前置法"

### 第3条——治安管理处罚法与刑法的分界线

> 扰乱公共秩序，妨害公共安全，侵犯人身权利、财产权利，妨害社会管理，具有社会危害性，**依照刑法的规定构成犯罪的，依法追究刑事责任；尚不够刑事处罚的，由公安机关依照本法给予治安管理处罚。**

核心逻辑：**同一行为，够罪走刑法，不够罪走治安管理处罚法**。

### 与《行政处罚法》的关系（第4条）
治安管理处罚的程序适用本法；本法没有规定的，适用《行政处罚法》和《行政强制法》。

## 三、处罚种类

| 处罚类型 | 性质 | 说明 |
|----------|------|------|
| **警告** | 最轻 | 申诫罚，记录在案 |
| **罚款** | 财产罚 | 各条具体规定金额 |
| **行政拘留** | 最重 | 限制人身自由，合并执行最长20日 |
| **吊销许可证件** | 资格罚 | 吊销公安机关发放的许可证 |
| 附加：限期出境/驱逐出境 | 仅外国人 | 可以附加适用 |

## 四、年龄与责任能力

| 年龄/状态 | 处理方式 |
|-----------|----------|
| **不满14周岁** | 不予处罚，责令监护人严加管教 |
| **14-18周岁** | 从轻或减轻处罚 |
| **14-16周岁** | 不执行行政拘留（情节严重/一年内二次以上除外） |
| **16-18周岁初次违法** | 不执行行政拘留（情节严重除外） |
| **70周岁以上** | 不执行行政拘留（情节严重/一年内二次以上除外） |
| **怀孕/哺乳不满1岁婴儿** | 不执行行政拘留 |
| 精神病人/智力残疾人（不能辨认控制） | 不予处罚 |
| 间歇性精神病人（正常时） | 应当处罚 |
| 盲人/又聋又哑 | 可以从轻、减轻或不予处罚 |
| **醉酒** | **应当处罚**（不减免！） |

## 五、从轻/从重/正当防卫

### 从轻、减轻或不予处罚（第20条）
1. 情节轻微
2. 主动消除或减轻后果
3. 取得被侵害人谅解
4. 出于他人胁迫或诱骗
5. 主动投案如实陈述
6. 有立功表现

### 从重处罚（第22条）
1. 有较严重后果
2. 教唆、胁迫、诱骗他人违法
3. 对报案人/控告人/举报人/证人打击报复
4. **一年以内曾受过治安管理处罚**

### 正当防卫（第19条）
- 为免受正在进行的不法侵害而采取的制止行为 → **不受处罚**
- 明显超过必要限度，造成较大损害 → 减轻处罚
- 情节较轻 → 不予处罚

## 六、追诉时效

**第25条：6个月**

违反治安管理行为在**6个月内**没有被公安机关发现的，不再处罚。从行为发生之日起计算；连续或继续状态的，从行为终了之日起计算。

> 对比刑法追诉时效（5年起步），治安管理处罚的时效极短。

## 八、处罚程序要点

### 调查阶段
| 要点 | 规定 |
|------|------|
| 传唤询问时限 | 一般≤8小时；人数众多/身份不明≤12小时；可能拘留≤24小时 |
| 全程录音录像 | 在执法办案场所询问**必须**全程同步录音录像（第97条） |
| 未成年人询问 | 必须通知父母/监护人到场（第98条） |
| 检查 | 警察不得少于2人，出示证件；检查住所须县级以上公安机关检查证 |

### 决定阶段
| 要点 | 规定 |
|------|------|
| 决定权限 | 县级以上公安机关决定；警告、1000元以下罚款可由派出所决定 |
| 办案期限 | 30日（可延长30日×2次，即最长90日） |
| 当场处罚 | 事实清楚+证据确凿+警告或500元以下罚款→可当场决定 |
| **听证权** | 吊销许可证/4000元以上罚款/可能拘留的未成年人→告知听证权 |
| 法制审核 | 重大公共利益/重大权益/疑难复杂→必须法制审核 |
| 救济途径 | 可依法申请行政复议或提起行政诉讼 |

### 执行阶段
| 要点 | 规定 |
|------|------|
| 拘留执行 | 送拘留所执行 |
| 罚款缴纳 | 15日内到银行或电子支付；200元以下+无异议可当场收缴 |
| **暂缓拘留** | 升学考试/子女出生/近亲属病危死亡→可提担保人或200元/日保证金 |
| 拘留出所 | 同上情形，出所时间不计入拘留期限 |

## 十一、与已学法律的交叉关联

### 与刑法的关系
- **核心衔接**：第3条——够罪走刑法，不够罪走本法
- **同一行为的不同量级**：盗窃→治安(第58条)/刑法(第264条)；伤害→治安(第51条)/刑法(第234条)
- **转化关系**：治安管理中发现涉嫌犯罪→移送追究刑事责任（第113条三）

### 与民法典的关系
- **第8条**：治安处罚不免除民事责任——"行为人或者其监护人还应当依法承担民事责任"
- 人身权利保护（侮辱/诽谤/隐私/个人信息）在两法中均有规定，层级不同

### 与劳动法的关系
- 第47条强迫劳动→治安处罚
- 劳动法第96条用人单位强迫劳动→行政处罚+刑事责任

### 与网络安全法的关系
- 第33条侵入计算机信息系统→治安处罚
- 第56条个人信息出售/获取→治安处罚
- 网络安全法中的违法行为不够罪的，可能落入本法处罚范围

### 与宪法的关系
- 第1条明确"根据宪法，制定本法"
- 保障公民人身权利、财产权利（宪法第37-40条的具体落实）

## 十二、实务知识点

### "小明"案例——治安管理处罚法视角

回顾之前分析的小明案（偷iPhone+打晕店员+不赔付+辱骂），如果情节更轻（比如只是推了一下店员没造成伤害），不够刑事立案：

| 行为 | 适用条款 | 处罚 |
|------|----------|------|
| 偷窃手机 | 第58条（盗窃） | 5-10日/2000元→10-15日+3000元 |
| 推搡店员 | 第51条（殴打他人） | 5-10日+500-1000元 |
| 辱骂店员 | 第50条(二)（公然侮辱） | 5日以下/1000元 |
| 合并执行 | 第16条 | 分别决定，合并执行，拘留最长20日 |

> **注意**：实际的小明案因暴力致人轻微脑震荡，已达到刑法转化型抢劫的门槛，所以走刑事程序而非治安管理。

### QQ阅读APP关联点
- **第33条**：APP如遭受黑客攻击（侵入信息系统），可报案由公安机关处理
- **第56条**：涉及用户个人信息泄露，出售/非法获取个人信息可治安处罚
- **第29条**：在APP上散布谣言/虚假信息，可治安处罚

### 日常生活高频条款
1. **噪声扰民**（第88条）→ 需先经调解，调解无效才处罚
2. **饲养动物伤人**（第89条）→ 驱使动物伤人按殴打他人论
3. **高空抛物**（第43条）→ 独立违法行为，不需要造成实际损害
4. **跟踪骚扰**（第50条）→ 2025新增禁止接触令制度
