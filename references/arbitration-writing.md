# 劳动仲裁申请书实战指南

> law爷 v1.7.0 新增 | 从"会写仲裁请求"到"会打赢仲裁"
>
> 核心法律依据：《劳动争议调解仲裁法》、《劳动合同法》、最新司法解释

---

## 一、仲裁申请前的准备

### 1.1 仲裁管辖确定

| 管辖规则 | 说明 |
|----------|------|
| **劳动合同履行地** | 实际工作地点的仲裁委（**优先**） |
| **用人单位所在地** | 注册地/主要营业地的仲裁委 |
| **两地不一致** | 由劳动合同履行地仲裁委管辖 |
| **多个工作地点** | 最后一个工作地点 |

> **实务建议**：优先选劳动合同履行地——离你近、取证方便、地方标准有利。

### 1.2 仲裁时效

| 时效 | 期限 | 起算 |
|------|------|------|
| 一般劳动争议 | **1年** | 知道或应当知道权利被侵害之日 |
| 拖欠劳动报酬 | **劳动关系存续期间不受1年限制** | 劳动关系终止之日起1年内 |
| 中断事由 | 主张权利/对方同意/向有关部门投诉 | 中断后重新计算 |
| 中止事由 | 不可抗力/丧失行为能力 | 中止原因消除后继续计算 |

> **关键**：**在职期间的欠薪不受1年限制！** 但离职后必须在1年内主张。很多人不知道这个规定，白白放弃了追索欠薪的权利。

### 1.3 仲裁受理范围

| 受理 | 不受理 |
|------|--------|
| 确认劳动关系 | 社保缴纳（应向社保部门投诉） |
| 劳动合同订立/履行/变更/解除/终止 | 住房公积金（向公积金管理部门投诉） |
| 除名/辞退/辞职/离职 | 劳动能力鉴定结论（向省级劳鉴委再鉴定） |
| 工作时间/休息休假/社保/福利/培训/劳动保护 | 工伤认定不服（走行政复议/行政诉讼） |
| 劳动报酬/工伤医疗费/经济补偿/赔偿金 | |

## 二、经济补偿/赔偿金计算器

### 2.1 经济补偿（N）计算公式

```
经济补偿 = 月工资 × 工作年限

月工资 = 解除/终止前12个月的平均工资（含奖金、津贴、补贴等）
```

| 工作年限 | 折算 |
|----------|------|
| 每满1年 | 支付1个月工资 |
| ≥6个月不满1年 | 按1年计算 |
| <6个月 | 支付半个月工资 |

### 2.2 月工资上限

| 情形 | 处理 |
|------|------|
| 月工资 > 本地区职工月平均工资3倍 | 按3倍封顶 |
| 封顶时的年限上限 | **最高12年** |
| 月工资 ≤ 3倍 | 不受12年限制 |

### 2.3 赔偿金（2N）计算

```
违法解除赔偿金 = 经济补偿 × 2 = 月工资 × 工作年限 × 2
```

### 2.4 代通知金（+1）

| 情形 | 金额 |
|------|------|
| 未提前30日书面通知解除（第40条三种情形） | 1个月工资（解除前1个月的工资标准） |

### 2.5 实战计算示例

**场景**：小张在A公司工作3年7个月，月工资8,000元，被违法辞退。

```
工作年限：3年7个月 → 4年（≥6个月按1年计）
月工资：8,000元（未超3倍上限）
经济补偿（N）：8,000 × 4 = 32,000元
违法解除赔偿金（2N）：32,000 × 2 = 64,000元

如果是N+1（合法解除但未提前通知）：32,000 + 8,000 = 40,000元
```

### 2.6 其他常见赔偿项目速算

| 项目 | 计算方式 | 法律依据 |
|------|----------|----------|
| 未签劳动合同双倍工资 | 月工资 × 最多11个月 | 第82条 |
| 未签无固定期限合同双倍工资 | 月工资 × 应签之日起每月 | 第82条第2款 |
| 加班费（工作日延时） | 时薪 × 1.5 | 第44条 |
| 加班费（休息日） | 时薪 × 2（不能补休时） | 第44条 |
| 加班费（法定假日） | 时薪 × 3 | 第44条 |
| 年假工资报酬 | 日工资 × 未休天数 × 3（含已付1倍） | 《职工带薪年休假条例》 |
| 违法约定试用期赔偿 | 试用期满后月工资 × 已履行超出法定部分的月数 | 第83条 |

## 三、仲裁申请书写作规范

### 3.1 格式模板

```
劳动仲裁申请书

申请人：[姓名]，性别：[男/女]，民族：[  ]，
        出生日期：____年__月__日，
        身份证号码：________________，
        住所：____________________，
        联系电话：________________，
        邮编：________。

被申请人（一）：[公司全称]，
        统一社会信用代码：________________，
        住所：____________________，
        法定代表人：________，职务：________，
        联系电话：________________。

被申请人（二）：[第二家公司全称]（如适用），
        [同上格式]

仲裁请求：

1. ________________________；
2. ________________________；
3. ________________________；
[逐项列明，金额精确到元]

事实与理由：

一、基本事实
[按时间线叙述：入职时间→岗位→工资标准→争议经过→解除/终止经过]

二、被申请人行为的违法性
[逐项对应仲裁请求分析，引用法条]

三、赔偿计算
[列明每项请求的计算公式和金额]

四、法律依据
[列明引用的法律条文]

综上，申请人依据《劳动争议调解仲裁法》第二条之规定，
提起劳动仲裁，请求裁如所请。

此致
____市____区劳动人事争议仲裁委员会

          申请人：（签名）
          ____年__月__日

附：证据目录及证据材料（一式__份）
```

### 3.2 仲裁请求写作铁律

| 铁律 | 说明 |
|------|------|
| **金额必须精确** | 不要写"约X万元"，要精确到元 |
| **请求可量化** | 每项请求对应一个明确的金额或具体行为 |
| **列明计算依据** | 在事实理由中列出计算公式 |
| **请求之间不重复** | N和2N不能同时请求（二选一） |
| **先确认关系后主张权利** | 有争议时，第一项请求确认劳动关系 |
| **包含利息/滞纳金** | 如适用，一并请求 |

### 3.3 常见仲裁请求组合（按场景）

#### 场景一：被违法辞退（最常见）

```
1. 确认被申请人于____年__月__日解除劳动合同的行为违法；
2. 被申请人向申请人支付违法解除劳动合同赔偿金____元
   （月工资____元 × 工作年限____年 × 2）；
3. 被申请人向申请人支付____年__月至__月拖欠工资____元；
4. 被申请人向申请人出具解除劳动合同证明。
```

#### 场景二：被迫辞职（单位有过错）

```
1. 确认申请人依据《劳动合同法》第三十八条解除劳动合同；
2. 被申请人向申请人支付经济补偿金____元
   （月工资____元 × 工作年限____年）；
3. 被申请人向申请人补缴____年__月至____年__月的社会保险；
4. 被申请人向申请人支付____年__月至__月拖欠的加班费____元。
```

#### 场景三：工伤待遇

```
1. 被申请人向申请人支付停工留薪期工资____元
   （月工资____元 × __个月）；
2. 被申请人向申请人支付一次性伤残补助金____元
   （月工资____元 × __个月）；
3. 被申请人向申请人支付一次性工伤医疗补助金____元；
4. 被申请人向申请人支付一次性伤残就业补助金____元；
5. 被申请人向申请人支付护理费____元。
```

#### 场景四：劳务派遣争议

```
1. 确认申请人与被申请人一[用工单位]自____年__月__日起存在
   事实劳动关系；（如主张假派遣穿透）
2. 被申请人一[派遣单位]向申请人支付违法解除赔偿金____元；
3. 被申请人二[用工单位]对上述赔偿金承担连带赔偿责任；
4. 两被申请人连带向申请人支付____年__月至__月拖欠工资差额____元
   （同工同酬差额）。
```

## 四、证据准备与组织

### 4.1 证据目录模板

| 序号 | 证据名称 | 证据形式 | 页数 | 证明目的 |
|------|----------|----------|------|----------|
| 1 | 劳动合同 | 原件/复印件 | X页 | 证明劳动关系建立及合同内容 |
| 2 | 工资银行流水 | 原件 | X页 | 证明月工资标准 |
| 3 | 解除通知/辞退通知 | 原件/截图 | X页 | 证明被申请人单方解除的事实 |
| ... | ... | ... | ... | ... |

### 4.2 各类争议的核心证据清单

#### 违法解除

| 序号 | 证据 | 获取方式 | 紧急度 |
|------|------|----------|--------|
| 1 | 劳动合同 | 自己手中 | ⭐⭐⭐ |
| 2 | 解除/辞退通知 | 保存原件/截图 | ⭐⭐⭐ |
| 3 | 工资银行流水（12个月） | 银行柜台打印 | ⭐⭐⭐ |
| 4 | 社保缴纳证明 | 社保局自助打印 | ⭐⭐ |
| 5 | 工作证/工牌 | 自己手中 | ⭐⭐ |
| 6 | 考勤记录 | 申请仲裁时要求对方举证 | ⭐⭐ |
| 7 | 工作微信/邮件聊天记录 | 截图+公证 | ⭐⭐ |

#### 未签合同双倍工资

| 序号 | 证据 | 证明目的 |
|------|------|----------|
| 1 | 工资发放记录 | 证明实际劳动关系存在 |
| 2 | 考勤记录/打卡记录 | 证明出勤 |
| 3 | 工作证/工牌/工作服 | 证明身份 |
| 4 | 同事证人证言 | 辅助证明 |
| 5 | 社保/公积金缴纳记录 | 强力证据 |

### 4.3 电子证据保全要点

| 类型 | 保全方式 | 注意事项 |
|------|----------|----------|
| 微信聊天记录 | 截图+录屏+导出（如有原始数据） | 必须显示对方实名/头像+完整上下文 |
| 邮件 | 导出原始邮件(.eml) + 截图 | 保留邮件头信息 |
| 钉钉/企微 | 截图+录屏 | 注意离职后可能被踢出 |
| 电话录音 | 手机自带录音（单方录音合法） | 开头确认对方身份 |
| 网页内容 | 公证处公证 / 区块链存证 | 防对方删除后否认 |

> **铁律**：**离职前24小时是证据保全的黄金时间。** 离职后很多系统权限会被收回。

## 五、庭审攻防实战

### 5.1 仲裁庭审流程

```
1. 庭前准备
   ├── 仲裁员核实身份
   ├── 宣布庭审纪律
   └── 询问是否申请回避

2. 陈述阶段
   ├── 申请人宣读仲裁请求
   └── 被申请人答辩

3. 举证质证
   ├── 申请人举证
   ├── 被申请人质证
   ├── 被申请人举证
   └── 申请人质证

4. 辩论阶段
   ├── 申请人发表辩论意见
   └── 被申请人发表辩论意见

5. 调解阶段（仲裁员会问双方是否接受调解）

6. 最后陈述

7. 庭审结束（签笔录）
```

### 5.2 常见攻防对策

| 被申请人（单位）抗辩 | 申请人（劳动者）反击 |
|---------------------|---------------------|
| "劳动者严重违纪" | 要求出示规章制度 + 民主程序 + 公示证据 + 违纪事实证据。规章制度未经民主程序/未公示 = 不能作为依据 |
| "劳动者不胜任工作" | 要求出示考核标准 + 考核结果 + 培训/调岗记录。不胜任必须先培训或调岗，直接辞退 = 违法 |
| "经济性裁员" | 查是否裁减≥20人或占10%以上 + 是否提前30日向工会说明 + 是否向劳动部门报告 + 优先留用对象 |
| "双方协商解除" | 要求出示协商记录。口头协商不算；被迫签的不算（查是否有胁迫证据） |
| "试用期不合格" | 要求出示录用条件 + 考核标准 + 不合格的具体证据。试用期辞退举证责任在单位 |
| "劳动者自动离职" | 提供考勤记录/请假记录证明正常出勤。"自动离职"不是法定解除理由 |
| "已过仲裁时效" | 主张时效中断（有催讨记录即可）或特殊时效（欠薪不受1年限制） |

### 5.3 质证话术模板

#### 对方出示的证据你不认可时：

```
对该证据的真实性/关联性/合法性有异议：
1. 真实性异议：[具体说明：如签名非本人签署/日期与事实不符/存在篡改痕迹]
2. 关联性异议：[该证据与本案争议焦点无关]
3. 合法性异议：[取证程序违法/未经合法程序制定]
综上，该证据不能达到被申请人的证明目的。
```

#### 对方出示的规章制度：

```
对该规章制度不予认可：
1. 请被申请人提供该制度经职工代表大会或全体职工讨论通过的会议记录；
2. 请被申请人提供与工会或职工代表协商确定的记录；
3. 请被申请人提供向申请人公示或告知的证据；
以上三项任一缺失，根据《劳动合同法》第四条，该制度不得作为处理劳动争议的依据。
```

## 六、仲裁后的路径选择

### 6.1 裁决类型与救济

| 裁决类型 | 适用情形 | 效力 | 不服救济 |
|----------|----------|------|----------|
| **终局裁决** | 追索劳动报酬/工伤医疗费/经济补偿/赔偿金（不超过当地月最低工资标准12个月的）/ 因执行国家标准产生的争议 | 自作出之日起生效 | 劳动者不服→15日内向法院起诉；**用人单位→30日内向中院申请撤销**（注意不是上诉） |
| **非终局裁决** | 其他案件 | 15日内不起诉生效 | 不服→15日内向基层法院起诉 |

### 6.2 裁决后的时间线

```
裁决书送达
    ↓
15日内（劳动者）/ 30日内（用人单位对终局裁决）
    ↓
┌───────────────────────────────────┐
│ 不起诉 → 裁决生效 → 申请法院执行  │
│ 起诉   → 进入一审（基层法院）      │
│         → 一审不服 → 二审（中院）    │
│         → 二审终审                   │
└───────────────────────────────────┘
```

### 6.3 申请法院执行

| 要点 | 内容 |
|------|------|
| 申请期限 | 裁决生效后 **2年内** |
| 申请法院 | 被执行人住所地或财产所在地**基层法院** |
| 需要材料 | 执行申请书 + 生效裁决书 + 身份证 + 送达证明 |
| 执行措施 | 查询/冻结/划拨存款、扣留收入、查封财物等 |

## 七、仲裁费用与时间预期

### 7.1 费用

| 项目 | 费用 |
|------|------|
| **仲裁费** | **免费**（劳动仲裁不收费） |
| 律师费 | 劳动者自行承担（一般不判对方承担） |
| 公证费 | 如需公证证据，按各地标准 |

### 7.2 时间预期

| 环节 | 时间 |
|------|------|
| 受理审查 | 收到申请5日内 |
| 仲裁审理期限 | 受理之日起 **45日内**（延长+15日） |
| 终局裁决 | 同上 |
| 法院一审（如起诉） | 6个月内 |
| 法院二审（如上诉） | 3个月内 |

> **整体预期**：从提交申请到拿到裁决，顺利的话 **1.5-2个月**。如果对方起诉进入法院阶段，可能需要 **6-12个月**。

## 八、分步引导工作流

当用户咨询劳动争议维权时，law爷 应按以下步骤引导：

### Step 1：了解基本情况
```
请告诉我以下信息：
1. 你的工作起止时间？（精确到月）
2. 月工资多少？（税前/到手？包含哪些项目？）
3. 签过劳动合同吗？签了几次？什么类型？
4. 发生了什么争议？（被辞退/欠薪/工伤/其他）
5. 有什么证据？（合同/工资条/解除通知/微信记录等）
```

### Step 2：法律定性
- 判断争议类型（违法解除/欠薪/工伤/未签合同等）
- 计算各项赔偿金额
- 评估证据强弱

### Step 3：方案建议
- 推荐维权路径（仲裁/投诉/协商）
- 列出需要准备的证据清单
- 给出时效提醒

### Step 4：文书协助
- 帮写仲裁申请书
- 整理证据目录
- 准备质证意见

### Step 5：庭审指导
- 模拟对方可能的抗辩
- 准备反驳方案
- 调解底线建议
