## 第一部分：中国法制史（~8分）

### 二、先秦法制（夏商周+春秋战国）

#### 礼刑关系
```
"出礼入刑"
礼 = 正面规范（道德+礼仪+制度）
    ↓ 违反
刑 = 负面制裁（刑罚惩罚）
```

#### 西周六礼（结婚程序）🔑必背
**纳采→问名→纳吉→纳征→请期→亲迎**
> 口诀：纳采问名纳吉礼，纳征请期迎亲娶

#### 西周七出三不去
- **七出**（休妻理由）：无子、淫佚、不事舅姑、多言、盗窃、妒忌、恶疾
- **三不去**（限制休妻）：有所取无所归/与更三年丧/前贫贱后富贵

#### 春秋战国成文法运动

| 国家 | 人物 | 措施 | 意义 |
|------|------|------|------|
| 郑国 | 子产 | **铸刑书于鼎**（前536年）| 中国历史上**第一次正式公布成文法** |
| 郑国 | 邓析 | 作《竹刑》（私撰）| 私家法学开端 |
| 晋国 | 赵鞅/荀寅 | **铸刑鼎**（前513年）| 第二次公布成文法 |

#### 李悝《法经》🔑
六篇结构：盗法 → 贼法 → 网法 → 捕法 → 杂法 → 具法（总则，放最后！）

### 三、秦汉法制

- **秦朝立法思想**："缘法而治""法令由一统"
- **汉朝立法思想转变**：秦专任刑罚 → 汉初黄老之学 → 武帝后"德主刑辅，礼刑并用"

#### 主要法典
| 法典 | 制定者 | 内容 |
|------|--------|------|
| **《九章律》** 🔑 | 萧何 | 在《法经》六篇基础上增加户、兴、厩三篇→**九篇** |
| 《傍章》| 叔孙通 | 朝廷礼仪法规（18篇）|
| 《越宫律》| 张汤 | 宫廷警卫法规（27篇）|
| 《朝律》| 赵禹 | 朝贺制度法规（6篇）|

#### 文景废肉刑 🔑（刑罚史上重大转折）
- 文帝废除黥/劓/斩左趾 → 以笞刑替代部分肉刑
- 景帝笞三百→**笞一百**；笞五百→**笞二百**；制定**《箠令》**

#### 春秋决狱 🔑
- 核心原则："原心定罪"——根据主观动机定罪量刑
- 推动儒法合流；开启引经注律传统

### 四、三国两晋南北朝法制（律学鼎盛期）🔑

| 朝代 | 法典 | 特点/贡献 |
|------|------|-----------|
| 曹魏 | **《魏律》** | ①将"具律"改为"刑名"，移到篇首🔑；②八议正式入律 |
| 晋朝 | **《晋律》/《泰始律》** | ①**"准五服以制罪"**🔑；②张斐/杜预注律具同等效力 |
| 北魏 | **《北魏律》** | ①**官当**入律；②**存留养亲**制度 |
| 北齐 | **《北齐律》** | ①**首创"重罪十条"**🔑；②确立12篇结构 |

#### 八议 🔑
亲、故、贤、能、功、贵、勤、宾 —— 死罪奏请皇帝裁决，流罪以下减一等。犯**"十恶"**者不得适用！

#### 准五服以制罪 🔑
```
五服（按丧服轻重排序）：
    斩衰（最重）→ 服三年 → 子为父母/妻为夫
    齐衰       → 服一年 → 孙为祖父母/夫为妻
    大功       → 服九月  → 为堂兄弟/已婚姑姐妹
    小功       → 服五月  → 为堂侄/外祖父母
    缌麻（最轻）→ 服三月  → 为远亲
核心原则：服制越近 → 以尊犯卑处罚轻 / 以卑犯尊处罚重
```

### 五、隋唐法制（中华法系巅峰）🔑🔑🔑

#### 《唐律疏议》——中华法系代表作
- 全称：《永徽律疏》
- 结构：**12篇**502条
- 地位：中国现存最早最完整的古代法典

#### 十恶 🔑🔑🔑（法考绝对高频！）

| 序号 | 罪名 | 含义 |
|------|------|------|
| 一 | **谋反** | 图谋危害社稷/国家 |
| 二 | **谋大逆** | 图谋毁坏宗庙/山陵/宫殿 |
| 三 | **谋叛** | 图谋背叛本国/投靠敌国 |
| 四 | **恶逆** | 殴/谋杀缌麻以上亲属 |
| 五 | **不道** | 残忍杀害一家非死罪三人等 |
| 六 | **大不敬** | 冒犯皇帝尊严 |
| 七 | **不孝** | 控告/咒骂父母等 |
| 八 | **不睦** | 殴/谋杀/出卖缌麻以上亲属 |
| 九 | **不义** | 杀本属长官/老师 |
| 十 | **内乱** | 亲属之间乱伦 |

> 口诀：**谋反大逆叛，恶逆不道不敬不孝不睦不义内乱**

#### 封建制五刑 🔑
| 刑种 | 等级 | 内容 |

### 六、宋元明清法制

#### 宋朝特色
- **《宋刑统》**：第一部刊印颁行的法典
- **翻异别勘**🔑：被告人翻供→交另一机构重新审理
- **凌迟**：作为正式死刑列入法典

#### 明朝特色 🔑
- 立法思想："刑乱国用重典"/"重典治吏"
- **《大明律》**：**六部结构**
- **《明大诰》**🔑：效力高于《大明律》
- 厂卫干预司法：锦衣卫/东厂/西厂

#### 清朝特色
- **秋审制度**🔑：每年秋季复审死刑案件
- 秋审结果四种：**情实/缓决/可矜/留养承祀**

### 七、清末修律（1840-1911）🔑🔑

- 主持者：**沈家本**/**伍廷芳**
- 指导思想："参考古今，博稽中外"
- **《大清新刑律》**🔑：中国历史上**第一部近代意义的专门刑法典**；采用罪刑法定原则
- **"礼法之争"**：礼教派(张之洞/劳乃宣) vs 法理派(沈家本/杨度)

## 第二部分：环境资源法（~7分）

### 一、环境保护法基本原则 🔑

| 原则 | 含义 |
|------|------|
| **保护优先、预防为主、综合治理** | 把环境保护放在优先位置 |
| **公众参与原则** | 公众有权参与环保决策和监督 |
| **损害担责原则** | 造成污染的主体应当承担责任 |
| **协调发展原则** | 经济发展与环境保护相协调 |

### 二、环境保护基本制度 🔑

#### 2.1 环境影响评价制度（环评）🔑
| 类型 | 适用情形 |
|------|----------|
| **环境影响报告书** | 可能造成重大环境影响 |
| **环境影响报告表** | 可能造成轻度环境影响 |
| **环境影响登记表** | 影响很小 |

未依法进行环评的建设项目**不得开工建设**

#### 2.2 "三同时"制度 🔑
- **含义**：防治污染的设施必须与**主体工程同时设计、同时施工、同时投产使用**

#### 2.3 排污许可制度
- 有效期一般为**5年**
- 无证排污→罚款（10万-100万元）；拒不改正的**按日连续处罚**

### 三、环境法律责任 🔑🔑

#### 行政责任
- **按日连续处罚**🔑：拒不改正的→原处罚数额**按日连续计算**（上不封顶！）

#### 民事责任 🔑
| 规则 | 内容 |
|------|------|
| **无过错责任原则**🔑 | 无论排污者有无过错，只要造成污染损害就应承担责任 |
| **举证责任倒置**🔑 | 排污者需证明其行为与损害**不存在因果关系** |
| **诉讼时效** | **3年**（自知道受损害之日起算）|

#### 刑事责任
**污染环境罪**（刑法第338条）🔑：一般3年以下；后果特别严重**3-7年**有期徒刑

#### 环境公益诉讼
- **原告资格**：符合条件的社会组织（5年以上且无违法记录）；**检察机关**
- 管辖法院：中级人民法院
