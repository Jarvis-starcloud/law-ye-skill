# 劳动者权益保护实务

> 来源：律师成长计划阶段四 · 模块一
> 学习日期：2026年4月7日
> 核心法规：劳动法 + 劳动合同法 + 调解仲裁法 + 工伤保险条例 + 劳动争议司法解释(一)(二)

---

## 一、劳动仲裁全流程

### 1.1 适用范围

根据《劳动争议调解仲裁法》第2条，以下争议适用劳动仲裁：

| 争议类型 | 典型场景 |
|----------|----------|
| 确认劳动关系 | 没签合同，要证明存在劳动关系 |
| 劳动合同争议 | 签订、履行、变更、解除、终止合同 |
| 辞退/辞职纠纷 | 被违法辞退、被迫辞职 |
| 工资报酬 | 拖欠工资、克扣加班费 |
| 工伤待遇 | 工伤认定后的赔偿争议 |
| 社保争议 | 未缴/少缴社保（部分地区受理） |
| 经济补偿 | 解除/终止合同后的补偿金争议 |

### 1.2 四阶段七步法

```
【第一阶段：申请】
   ├── Step 1: 确定管辖——劳动合同履行地 or 用人单位所在地
   ├── Step 2: 准备材料
   │    ├── 仲裁申请书（一式两份）
   │    ├── 身份证原件+复印件
   │    ├── 劳动关系证明（合同/工资条/社保记录/工牌等）
   │    └── 证据材料（转账记录/解除通知/考勤记录等）
   └── Step 3: 提交申请

【第二阶段：受理】
   ├── 5日内审查材料
   ├── 材料不全 → 补正通知
   └── 7日内出具受理/不受理通知

【第三阶段：审理】
   ├── Step 4: 7日内组庭（指定仲裁员）
   ├── Step 5: 开庭前4日送达通知
   └── Step 6: 庭审 → 陈述举证 → 质证辩论 → 先行调解 → 裁决

【第四阶段：裁决】
   ├── Step 7: 受理后45日内结案（复杂案件+15日）
   ├── 不服 → 15日内向法院起诉
   └── 期满未起诉 → 裁决生效，可申请强制执行
```

### 1.3 关键时限汇总

| 事项 | 时限 | 法律依据 |
|------|------|----------|
| **仲裁时效** | 1年（从知道或应当知道权利被侵害之日起） | 《调解仲裁法》第27条 |
| **欠薪仲裁时效** | 劳动关系存续期间不受1年限制；终止后1年内 | 第27条第4款 |
| 仲裁委受理审查 | 5日 | 第29条 |
| 仲裁委出具受理通知 | 7日 | 第29条 |
| 审理期限 | 45日（可延长15日） | 第43条 |
| 不服裁决起诉期限 | 15日 | 第50条 |

### 1.4 实务要点

1. **仲裁前置**：劳动争议必须先仲裁后诉讼，不能直接起诉
2. **一裁终局**：小额争议（≤当地月最低工资×12个月）和标准明确的争议，对用人单位一裁终局
3. **举证责任倒置**：与争议事项有关的证据属于用人单位掌握管理的，**用人单位举证**
4. **免费**：劳动仲裁不收费

## 二、经济补偿与赔偿金（N/N+1/2N）

### 2.1 核心概念

| 术语 | 含义 | 计算方式 |
|------|------|----------|
| **N** | 经济补偿 | 工作年限 × 月工资 |
| **N+1** | 经济补偿 + 代通知金 | N + 1个月工资（未提前30天通知时） |
| **2N** | 违法解除赔偿金 | 经济补偿的2倍 |
| **0** | 无需补偿 | 劳动者严重违纪等6种情形 |

### 2.2 N 的计算规则

```
计算公式：N = 工作年限 × 月工资

工作年限认定：
  ├── 每满1年 → 1个月
  ├── ≥6个月不满1年 → 按1年算
  └── ＜6个月 → 0.5个月

月工资认定：
  ├── 解除/终止前12个月平均工资（税前）
  ├── 包含：基本工资 + 奖金 + 津贴 + 补贴
  ├── 高薪封顶：＞当地3倍平均工资 → 按3倍算，且最高12年
  └── 兜底：低于最低工资 → 按最低工资算
```

### 2.3 各情形适用一览

**适用 N（经济补偿）**：
- 劳动者被迫解除（第38条）：未提供劳动保护/未足额付薪/未缴社保等
- 协商一致（单位提出，第36条）
- 单位提前30日通知解除（第40条）：医疗期满/不能胜任/客观情况变化
- 经济性裁员（第41条）
- 合同期满终止（第44条）
- 单位破产/吊照/关闭

**适用 N+1（仅3种情形）**：
1. 医疗期满不能工作
2. 不能胜任工作
3. 客观情况重大变化

> 如已提前30天书面通知，则只需付 N，不需付 +1

**适用 2N（违法解除赔偿金）**：
- 单位没有合法理由辞退劳动者

**适用 0（无需补偿）的6种情形**：
1. 试用期不符合录用条件
2. 严重违反规章制度
3. 严重失职、营私舞弊造成重大损害
4. 兼职影响本职或拒不改正
5. 欺诈/胁迫导致合同无效
6. 被追究刑事责任

### 2.4 常见误区

- ❌ "2N+1"——法律上不存在，违法解除只赔2N
- ❌ 劳动者主动辞职有补偿——除非被迫辞职（第38条）
- ❌ 试用期辞退无需理由——必须证明"不符合录用条件"

## 三、工伤认定与赔偿

### 3.1 工伤认定七种法定情形

根据《工伤保险条例》第14条：

| 序号 | 情形 | 要点 |
|------|------|------|
| 1 | 工作时间+工作场所+工作原因受伤 | 三要素同时满足 |
| 2 | 工作前后在工作场所从事预备性/收尾性工作受伤 | 如提前到岗检查设备 |
| 3 | 工作时间+工作场所因履职遭受暴力等意外伤害 | 如遭受客户暴力 |
| 4 | 患职业病 | 职业病诊断证明 |
| 5 | 因工外出期间受伤或事故下落不明 | 出差途中 |
| 6 | 上下班途中非本人主要责任的交通事故 | 不含主责/全责 |
| 7 | 法律法规规定的其他情形 | 兜底条款 |

### 3.2 视同工伤（第15条）

| 情形 | 说明 |
|------|------|
| 工作时间+工作岗位突发疾病**48小时内**死亡 | 超过48小时不认定 |
| 抢险救灾等维护公共利益受伤 | — |
| 复转军人旧伤复发 | — |

### 3.3 工伤认定流程

```
事故发生
    ↓
单位30日内申请工伤认定（单位不申请→劳动者1年内自行申请）
    ↓
社保部门15日内决定受理
    ↓
60日内作出认定（简单案件15日）
    ↓
20日内送达
    ↓
不服 → 行政复议 / 行政诉讼
```

### 3.4 赔偿标准（2025年）

**一次性伤残补助金**：

| 伤残等级 | 补助月数 | 后续安排 |
|----------|----------|----------|
| 一级 | 27个月 | 保留劳动关系，退出岗位，按月伤残津贴90% |
| 二级 | 25个月 | 同上85% |
| 三级 | 23个月 | 同上80% |
| 四级 | 21个月 | 同上75% |
| 五级 | 18个月 | 安排适当工作/按月伤残津贴70% |
| 六级 | 16个月 | 同上60% |
| 七级 | 13个月 | 合同期满/本人提出→一次性医疗+就业补助金 |
| 八级 | 11个月 | 同上 |
| 九级 | 9个月 | 同上 |
| 十级 | 7个月 | 同上 |

**工亡赔偿标准（2025年）**：
- 一次性工亡补助金：54,188元 × 20 = **1,083,760元**（全国统一）
- 丧葬补助金：6个月统筹地区上年度月平均工资
- 供养亲属抚恤金：配偶40%/其他亲属30%/孤老孤儿+10%，总计≤100%

### 3.5 实务要点

1. **未签合同也能认工伤**：存在事实劳动关系即可
2. **违法转包的工伤**：承包方承担工伤保险责任，不以劳动关系为前提
3. **超过48小时**：工作中突发疾病超48小时死亡的不认定为工伤
4. **单位不申请的后果**：30日内不申请，该期间的工伤待遇费用由单位自掏腰包

## 四、社保维权

### 4.1 五险一金框架

| 险种 | 单位比例（参考） | 个人比例 | 说明 |
|------|-----------------|----------|------|
| 养老保险 | 16% | 8% | 退休后领取养老金 |
| 医疗保险 | 8-10% | 2% | 门诊住院报销 |
| 失业保险 | 0.5-1% | 0.5% | 失业后领取失业金 |
| 工伤保险 | 0.2-1.9% | 0 | 工伤赔偿 |
| 生育保险 | 0.5-1% | 0 | 生育津贴 |
| 住房公积金 | 5-12% | 5-12% | 购房贷款 |

### 4.2 社保维权关键规则（2025年9月1日起）

根据**最高法劳动争议司法解释（二）第19条**：
- **"放弃社保"约定无效**
- **被迫辞职权**：单位未依法缴社保 → 劳动者可解除合同 → 单位支付经济补偿
- **补贴返还**：单位补缴后，可要求返还之前的"社保补贴"

### 4.3 维权途径

```
发现单位未缴/少缴社保
    ↓
路径一：向社保经办机构/劳动监察大队投诉 → 责令补缴
路径二：以未缴社保为由解除合同 → 劳动仲裁 → 主张经济补偿（N）
路径三：因未缴社保导致无法享受待遇 → 主张损失赔偿
```

## 五、竞业限制（最新规则）

### 5.1 基本框架

| 要素 | 规则 |
|------|------|
| 适用对象 | 高管、高级技术人员、其他知密人员 |
| 期限 | 最长2年 |
| 补偿标准 | ≥离职前12个月平均工资的30%（且≥当地最低工资） |
| 违约金 | 可约定，但须合理 |

### 5.2 司法解释（二）新规则

| 新规 | 内容 | 影响 |
|------|------|------|
| 不知密不生效 | 劳动者未接触商业秘密 → 竞业限制条款不生效 | 保护基层员工 |
| 范围须合理 | 范围、地域、期限须与接触的秘密事项相适应 | 防止滥用 |
| 在职期间有效 | 在职期间竞业限制有效，不得以"未支付补偿"主张无效 | 保护企业 |

### 5.3 实务要点

- 连续3个月未付补偿 → 劳动者可请求解除竞业限制
- 补偿过低 → 可主张按30%标准补足
- 单位可额外支付3个月补偿后单方解除竞业限制

## 六、常见劳动纠纷处理

### 6.1 十大常见纠纷

| 纠纷类型 | 劳动者对策 | 关键证据 |
|----------|-----------|----------|
| 拖欠工资 | 劳动监察投诉/仲裁 | 工资条、银行流水、考勤 |
| 未签劳动合同 | 主张双倍工资（第2-12个月） | 工牌、社保记录、微信记录 |
| 违法辞退 | 要求2N赔偿金或继续履行 | 解除通知、录音录像 |
| 加班不给加班费 | 仲裁主张加班费 | 考勤记录、加班通知 |
| 未缴社保 | 投诉补缴/被迫辞职要N | 工资条无社保扣缴项 |
| 工伤不认 | 自行申请工伤认定 | 医疗记录、现场证据 |
| 调岗降薪 | 不合理变更可拒绝 | 岗位变更通知、薪资对比 |
| 试用期违规 | 过长/重复试用可主张 | 合同条款 |
| 年假不给 | 按3倍工资折算 | 工龄证明、请假记录 |
| 竞业限制滥用 | 不知密→条款不生效 | 岗位职责、接触范围 |

### 6.2 举证责任分配

| 争议事项 | 举证责任 | 说明 |
|----------|----------|------|
| 解除合同合法性 | **用人单位** | 单位证明解除理由合法 |
| 工资支付记录 | **用人单位** | 举证倒置 |
| 考勤记录 | **用人单位** | 举证倒置 |
| 加班事实 | **劳动者**初步 + **单位**考勤 | 双方分担 |
| 劳动关系存在 | **劳动者** | 提供工资流水等 |

## 七、劳动争议司法解释（二）核心（法释〔2025〕12号）

> 2025年9月1日起施行

| 条款 | 主题 | 核心规则 |
|------|------|----------|
| 第1条 | 转包用工 | 承包方转包给无资质个人 → 承包方承担用工主体责任 |
| 第2条 | 挂靠用工 | 个人挂靠单位经营 → 被挂靠单位承担用工主体责任 |
| 第3条 | 混同用工 | 关联公司交替用工 → 劳动者可请求共同承担 |
| 第7条 | 未签合同豁免 | 因劳动者故意或重大过失未签 → 不支持双倍工资 |
| 第10条 | 无固定期限合同 | 变换主体等视为连续两次 |
| 第12条 | 服务期违约 | 提前离职须赔偿实际损失 |
| 第13-15条 | 竞业限制 | 不知密不生效 + 范围须合理 + 在职期间有效 |
| 第17条 | 职业健康检查 | 未做离岗前检查即解除 → 可主张继续履行 |
| 第18条 | 违法解除期间工资 | 判决继续履行 → 支付解除到恢复期间工资 |
| 第19条 | 社保约定无效 | "放弃社保"无效 + 被迫辞职有补偿 |
| 第20条 | 仲裁时效失权 | 仲裁期间未提时效抗辩 → 诉讼中不予支持 |
