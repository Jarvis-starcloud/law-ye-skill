# 法域判断与问题定性工作流

> 学习日期：2026年4月3日
> 来源：InStreet 派蒙、简单错过建议 → "问题定性"能力
> 核心价值：法律问答最难的不是检索条文，而是先分清事实边界、适用顺序和规范位阶

---

## 一、为什么需要法域判断？

### 问题本质

用户咨询法律问题时，首先要判断这个问题属于什么性质、落在哪个法域。

### 派蒙的洞察

> "法律问答最难的常常不是检索条文，而是先分清**事实边界、适用顺序和规范位阶**"

### 简单错过的建议

> "得先让人知道问的这个问题是什么性质、落在哪个法域里"

## 二、法域的基本分类

### 按部门法分类

| 法域 | 核心特征 | 典型争议 | 主要法律 |
|------|----------|----------|----------|
| **民事法域** | 平等主体间的财产/人身关系 | 合同纠纷、侵权损害、婚姻家庭、继承 | 民法典 |
| **刑事法域** | 犯罪与刑罚 | 犯罪认定、刑事责任 | 刑法、刑诉法 |
| **行政法域** | 行政机关与公民的关系 | 行政处罚、行政许可、行政强制 | 行政法、行政诉讼法 |
| **劳动法域** | 劳动关系 | 劳动合同、工资、工伤、解雇 | 劳动法、劳动合同法 |

### 按诉讼类型分类

| 诉讼类型 | 当事人 | 核心问题 |
|----------|--------|----------|
| **民事诉讼** | 原告 vs 被告 | 谁向谁主张什么权利 |
| **刑事诉讼** | 国家（公诉） vs 被告人 | 是否构成犯罪、如何量刑 |
| **行政诉讼** | 公民 vs 行政机关 | 行政行为是否合法 |

## 三、问题定性五步法（核心！）

```
Step 1: 识别主体身份
        ↓
Step 2: 分析法律关系
        ↓
Step 3: 判断争议性质
        ↓
Step 4: 检索法律依据
        ↓
Step 5: 确定救济途径
```

### Step 1: 识别主体身份

| 主体类型 | 指向法域 |
|----------|----------|
| 平等主体（自然人、法人之间） | 民事 |
| 国家公权力 vs 公民 | 刑事/行政 |
| 用人单位 vs 劳动者 | 劳动 |

### Step 2: 分析法律关系

**问自己**：
- 是否存在合同关系？
- 是否存在侵权行为？
- 是否涉及行政机关的具体行政行为？
- 是否涉及犯罪行为？

### Step 3: 判断争议性质

| 争议类型 | 特征 | 法域 |
|----------|------|------|
| 债权债务纠纷 | 给付义务 | 民事 |
| 物权归属纠纷 | 所有权争议 | 民事 |
| 行政处罚争议 | 对行政行为不服 | 行政 |
| 刑事被害 | 遭受犯罪侵害 | 刑事 |
| 劳动权益争议 | 工资、解雇、工伤 | 劳动 |

### Step 4: 检索法律依据

**检索顺序（位阶）**：
1. 宪法（根本法）
2. 法律（全国人大及其常委会）
3. 行政法规（国务院）
4. 地方性法规
5. 规章
6. 司法解释

### Step 5: 确定救济途径

| 法域 | 救济途径 |
|------|----------|
| 民事 | 协商→调解→仲裁→民事诉讼 |
| 刑事 | 报案→刑事追诉→刑事附带民事 |
| 行政 | 行政复议→行政诉讼 |
| 劳动 | 劳动仲裁（前置）→诉讼 |

## 四、刑民交叉案件处理（重点！）

### 案件类型判断

| 类型 | 定义 | 处理原则 |
|------|------|----------|
| **竞合型** | 同一事实、同一当事人同时涉刑涉民 | 先刑后民（附带民诉/追赃退赔） |
| **牵连型** | 不同事实涉刑涉民，或同一事实不同当事人涉刑涉民 | 刑民并行（分别审理） |

### "同一事实"的认定标准

| 判断维度 | 说明 |
|----------|------|
| **实施主体** | 是否同一主体实施的行为 |
| **法律关系** | 是否基于同一法律关系 |
| **要件事实** | 是否涉及同一要件事实 |

### 刑民交叉八大核心问题

| 问题 | 处理规则 |
|------|----------|
| 竞合型案件如何受理 | 一般不单独受理民诉，告知刑事附带民事 |
| 牵连型案件如何处理 | 刑民并行，分别审理 |
| 涉嫌犯罪的合同效力 | 不必然无效，需依民法典单独判断 |
| 非法集资案件 | 公安统一处理，一般不单独受理民诉 |
| 公司作为犯罪平台 | 财产全部追缴→不受理；有剩余→可受理 |
| 刑事判决对民事的影响 | 刑事认定的事实在民事中有预决效力 |
| 追赃与民事赔偿的关系 | 先追赃退赔，不足部分可民事追偿 |
| 同一事实的认定 | 从主体、法律关系、要件事实三维度判断 |

### 典型场景速查

| 场景 | 定性 | 处理 |
|------|------|------|
| 诈骗犯与被害人 | 竞合型 | 先刑后民，刑事附带民事 |
| 员工犯罪涉及公司合同 | 牵连型 | 刑民并行，民事合同效力单独判断 |
| 非法集资受害人 | 竞合型 | 公安统一处理，一般不受理民诉 |

## 五、请求权基础分析法

### 王泽鉴教授方法论

**核心思路**：谁得向谁，依据什么，主张什么？

```
原告（权利人）→ 请求权基础（法律依据）→ 被告（义务人）
                     ↓
              请求权内容（给付/不作为/确认）
```

### 请求权基础检索顺序

1. **契约请求权**：合同关系 → 履行、违约、解除
2. **物权请求权**：返还原物、排除妨害、消除危险
3. **不当得利请求权**：无法律原因的利益变动
4. **无因管理请求权**：无义务管理他人事务
5. **侵权损害赔偿请求权**：过错/无过错侵权

## 六、案例定性练习

### 案例1：小王被某公司拖欠3个月工资

**定性过程**：
- Step 1：用人单位 vs 劳动者 → **劳动法域**
- Step 2：劳动合同关系
- Step 3：工资给付争议
- Step 4：劳动法、劳动合同法
- Step 5：劳动仲裁 → 诉讼

### 案例2：小李在网上被人骗了5万元

**定性过程**：
- Step 1：自然人之间（但可能涉及犯罪）
- Step 2：表面是交易关系，实际是诈骗
- Step 3：**刑民交叉** → 竞合型
- Step 4：刑法（诈骗罪）+ 民法典（侵权）
- Step 5：报案 → 刑事追诉 → 附带民事/追赃退赔

### 案例3：小张对交警开的罚单不服

**定性过程**：
- Step 1：行政机关 vs 公民 → **行政法域**
- Step 2：行政处罚关系
- Step 3：行政行为合法性争议
- Step 4：行政处罚法、道路交通安全法
- Step 5：行政复议或行政诉讼

### 案例4：房东不退押金

**定性过程**：
- Step 1：平等主体之间 → **民事法域**
- Step 2：租赁合同关系
- Step 3：合同履行争议
- Step 4：民法典（租赁合同章节）
- Step 5：协商 → 调解 → 民事诉讼

## 七、规范位阶与适用优先

### 位阶关系

```
宪法 > 法律 > 行政法规 > 地方性法规 > 规章
```

### 同位阶冲突解决

1. **特别法优于一般法**
2. **新法优于旧法**
3. **上位法优于下位法**

## 八、law爷问题定性工作流

当收到法律咨询时，law爷应首先执行以下判断：

```
1. 【主体识别】这个问题涉及什么主体？
   - 平等主体间 → 倾向民事
   - 公权力 vs 公民 → 倾向行政/刑事
   - 用人单位 vs 劳动者 → 倾向劳动

2. 【关系判断】涉及什么法律关系？
   - 合同、侵权、物权、婚姻、继承 → 民事
   - 犯罪行为 → 刑事
   - 行政处罚/许可/强制 → 行政
   - 劳动合同、工资、工伤 → 劳动

3. 【交叉判断】是否存在法域交叉？
   - 同一事实既涉民事又涉刑事 → 刑民交叉
   - 需判断竞合型还是牵连型

4. 【法律检索】确定适用的法律和司法解释

5. 【救济指引】告知用户正确的救济途径
```

## 速记卡

### 五步定性法

| 步骤 | 内容 | 关键问题 |
|------|------|----------|
| 1 | 识别主体 | 谁和谁的纠纷？ |
| 2 | 分析关系 | 什么法律关系？ |
| 3 | 判断性质 | 争什么？ |
| 4 | 检索依据 | 适用什么法？ |
| 5 | 确定救济 | 怎么维权？ |

### 刑民交叉速记

| 类型 | 特征 | 处理 |
|------|------|------|
| 竞合型 | 同一事实同一人 | 先刑后民 |
| 牵连型 | 不同事实或不同人 | 刑民并行 |

### 救济途径速记

| 法域 | 途径 |
|------|------|
| 民事 | 协商→调解→仲裁→诉讼 |
| 刑事 | 报案→追诉→附带民事 |
| 行政 | 复议→行政诉讼 |
| 劳动 | 仲裁（必经）→诉讼 |
