# 中华人民共和国民法典 学习笔记

> 2020年5月28日第十三届全国人民代表大会第三次会议通过
> 自2021年1月1日起施行
> 学习整理：贾维斯 | 2026-04-02

---

## 一、全文结构总览

民法典被称为"社会生活的百科全书"，是新中国成立以来第一部以"法典"命名的法律，共**7编+附则，1260条**。

| 编 | 名称 | 条文范围 | 核心内容 |
|---|------|---------|---------|
| 第一编 | 总则 | 第1-204条 | 基本规定、自然人、法人、民事权利、法律行为、代理、民事责任、诉讼时效 |
| 第二编 | 物权 | 第205-462条 | 所有权、用益物权、担保物权、占有 |
| 第三编 | 合同 | 第463-988条 | 合同通则+19种典型合同+准合同 |
| 第四编 | 人格权 | 第989-1039条 | 生命权、姓名权、肖像权、名誉权、隐私权、个人信息保护 |
| 第五编 | 婚姻家庭 | 第1040-1118条 | 结婚、家庭关系、离婚、收养 |
| 第六编 | 继承 | 第1119-1163条 | 法定继承、遗嘱继承、遗产处理 |
| 第七编 | 侵权责任 | 第1164-1258条 | 一般侵权+10类特殊侵权责任 |
| 附则 | - | 第1259条 | 术语解释 |

## 二、第一编 总则（第1-204条）

### 2.1 基本规定（第1-12条）

**调整对象**：平等主体的自然人、法人和非法人组织之间的**人身关系**和**财产关系**（第2条）

**七大基本原则**：
1. **平等原则**（第4条）——法律地位一律平等
2. **自愿原则**（第5条）——按自己意思设立/变更/终止法律关系
3. **公平原则**（第6条）——合理确定权利义务
4. **诚信原则**（第7条）——秉持诚实，恪守承诺（"帝王条款"）
5. **守法原则**（第8条）——不违法，不违背公序良俗
6. **绿色原则**（第9条）——节约资源、保护生态环境（**民法典新增亮点**）
7. **法律适用**（第10条）——法律→习惯（不违背公序良俗）

### 2.2 自然人（第13-56条）

**民事行为能力三级分类**：

| 类型 | 年龄/条件 | 法律行为效力 |
|------|----------|-------------|
| 完全民事行为能力人 | ≥18周岁（或≥16周岁以劳动收入为主要生活来源） | 独立实施所有民事法律行为 |
| 限制民事行为能力人 | ≥8周岁未成年人；不能完全辨认行为的成年人 | 纯获利或与年龄智力相适应→有效；其他→需法定代理人同意/追认 |
| 无民事行为能力人 | <8周岁；不能辨认自己行为的成年人 | 由法定代理人代理 |

**关键年龄节点**：8岁（限制行为能力起点）、16岁（可视为完全行为能力）、18岁（成年）

**监护制度**：
- 未成年人监护顺序：父母→祖父母外祖父母→兄姐→其他个人组织
- 成年人监护顺序：配偶→父母子女→其他近亲属→其他个人组织
- **成年监护制度**（第33条）：可预先以书面形式确定自己的监护人（新增亮点）

**宣告失踪与宣告死亡**：
- 失踪：下落不明满**2年**
- 死亡：下落不明满**4年**（意外事件满2年；证明不可能生存的不受时间限制）

### 2.3 法人（第57-101条）

**法人四分类**：
1. **营利法人**（第76-86条）——有限公司、股份公司等
2. **非营利法人**（第87-95条）——事业单位、社会团体、基金会等
3. **特别法人**（第96-101条）——机关法人、农村集体经济组织法人、居委会/村委会
4. **非法人组织**（第102-108条）——个人独资企业、合伙企业等

**法人人格否认**（第83条）：滥用法人独立地位和有限责任逃避债务→对法人债务承担**连带责任**

### 2.4 民事权利（第109-132条）

涵盖：人身权利（人格权+身份权）、财产权利（物权+债权+知识产权+股权等）

**亮点条文**：
- **第111条**：个人信息保护（首次在民法中明确）
- **第127条**：数据和网络虚拟财产保护

### 2.5 民事法律行为（第133-160条）

**有效条件**（第143条）：
1. 行为人有相应的民事行为能力
2. 意思表示真实
3. 不违反强制性规定，不违背公序良俗

**无效情形**：无民事行为能力人实施、虚假意思表示、违法违背公序良俗、恶意串通

**可撤销情形**：重大误解、欺诈、胁迫、显失公平
- 撤销权除斥期间：一般1年，重大误解90日，最长5年

### 2.6 代理（第161-175条）

- **委托代理** vs **法定代理**
- **表见代理**（第172条）：无权代理但相对人有理由相信有代理权→代理行为有效
- **自己代理和双方代理**（第168条）：原则禁止，经同意或追认除外

### 2.7 民事责任（第176-187条）

**承担方式**（第179条）：11种方式（停止侵害、排除妨碍、返还财产、赔偿损失、赔礼道歉等），可单独或合并适用

**免责/减责事由**：不可抗力、正当防卫、紧急避险

**重要规则**：
- **好人法**（第184条）：自愿紧急救助造成受助人损害→不担责（鼓励见义勇为）
- 民事责任优先于行政/刑事责任中的财产支付（第187条）

### 2.8 诉讼时效（第188-199条）

| 时效类型 | 期间 |
|---------|------|
| 普通诉讼时效 | **3年** |
| 最长保护期 | **20年** |
| 不适用诉讼时效 | 请求停止侵害/排除妨碍/消除危险、不动产物权返还请求、抚养费/赡养费请求 |
| 未成年人性侵害 | 自受害人**年满18周岁**之日起算 |

**诉讼时效中止**：最后6个月内遇障碍；**诉讼时效中断**：权利人请求、义务人承认、起诉/仲裁

## 三、第二编 物权（第205-462条）

### 3.1 物权通则

- **物权法定**（第116条）：物权种类和内容由法律规定
- **不动产**：登记生效（第209条）；**动产**：交付生效（第224条）
- 不动产登记簿 > 权属证书（第217条）

### 3.2 所有权

**三级所有权**：国家所有权、集体所有权、私人所有权

**业主的建筑物区分所有权**（第271-287条）：
- 专有部分→所有权；共有部分→共有权+共同管理权
- 重大事项表决：2/3面积+2/3人数参与，其中特别事项需3/4同意

**善意取得**（第311条）：无处分权人转让→善意+合理价格+已登记/交付→受让人取得所有权

### 3.3 用益物权

| 物权类型 | 核心内容 |
|---------|---------|
| 土地承包经营权 | 耕地30年/草地30-50年/林地30-70年，到期可继续承包 |
| 建设用地使用权 | **住宅用地到期自动续期**（第359条，重大民生条款） |
| 宅基地使用权 | 集体所有，农民依法享有 |
| **居住权**（新增） | 无偿设立，不得转让/继承，登记设立（第366-371条） |
| 地役权 | 利用他人不动产提高自己不动产效益 |

### 3.4 担保物权

| 类型 | 特征 | 设立方式 |
|------|------|---------|
| 抵押权 | 不转移占有 | 不动产→登记生效；动产→合同生效（登记对抗） |
| 质权 | 转移占有 | 交付质押财产时设立 |
| 留置权 | 合法占有的动产 | 法定产生（同一法律关系） |

**清偿顺序**：留置权 > 抵押权/质权（按登记/交付先后）

**禁止抵押**：土地所有权、宅基地/自留地、公益设施等

## 四、第三编 合同（第463-988条）

### 4.1 合同通则

**合同订立**：要约→承诺
- 要约：内容具体确定+受约束的意思
- 实质性变更（标的/数量/质量/价款/违约责任/争议解决）→新要约

**格式条款**（第496-498条）：
- 提供方须合理提示、说明重大利害条款
- 不合理免除己方责任/加重对方责任/限制主要权利→无效
- 两种以上解释→不利于提供方解释

**合同解除**（第562-567条）：
- 协议解除、约定解除、法定解除（不可抗力/预期违约/根本违约）
- 通知到达对方时解除

**违约责任**（第577-594条）：
- 继续履行、补救措施、赔偿损失（可预见规则）
- **违约金**：过低可请求增加，过高可请求减少
- **定金**：不超过主合同标的额的**20%**；给付方违约→无权要求返还；收受方违约→**双倍返还**
- 违约金和定金只能**择一适用**

**情势变更**（第533条，新增）：合同基础条件重大变化，非商业风险→可请求变更/解除

### 4.2 十九种典型合同

| 合同类型 | 核心要点 |
|---------|---------|
| 买卖合同 | 风险转移以交付为界；分期付款未付达1/5→出卖人可解除 |
| 赠与合同 | 一般赠与可撤销（权利转移前）；公证/公益赠与不可撤销 |
| 借款合同 | **禁止高利放贷**（第680条）；自然人间无约定=无息 |
| **保证合同** | 默认**一般保证**（第686条，重大变更：旧法默认连带保证）；保证期间默认6个月 |
| 租赁合同 | 最长**20年**；"买卖不破租赁"（第725条）；承租人优先购买权 |
| **物业服务合同**（新增） | 不得以停水停电催交物业费（第944条） |
| **保理合同**（新增） | 应收账款转让融资 |
| 合伙合同 | 合伙人对合伙债务承担连带责任 |

### 4.3 准合同

- **无因管理**（第979-984条）：无义务为避免他人损失而管理→可请求偿还必要费用
- **不当得利**（第985-988条）：无法律根据取得利益→应返还

## 五、第四编 人格权（第989-1039条）

> 人格权独立成编是民法典的**最大创新亮点**

### 5.1 人格权体系

| 权利 | 核心保护内容 |
|------|-------------|
| 生命权 | 生命安全和生命尊严 |
| 身体权 | 身体完整和行动自由 |
| 健康权 | 身心健康 |
| 姓名权/名称权 | 决定、使用、变更、许可使用 |
| 肖像权 | 制作、使用、公开（需同意）；**AI换脸也是侵权**（第1019条） |
| 名誉权 | 品德、声望、信用等社会评价 |
| 荣誉权 | 荣誉称号不得非法剥夺 |
| **隐私权** | 私人生活安宁+私密空间+私密活动+私密信息 |

### 5.2 个人信息保护（第1034-1039条）

- 处理原则：**合法、正当、必要**，不得过度处理
- 需要**征得同意**（或法律法规另有规定）
- 信息主体有权：查阅、复制、异议更正、请求删除
- 信息处理者义务：安全保障、泄露时及时补救并报告

### 5.3 重要亮点条款

- **性骚扰条款**（第1010条）：机关、企业、学校应预防、受理投诉、调查处置
- **声音保护**（第1023条）：自然人声音参照肖像权保护
- **人格权禁令**（第997条）：紧急情况下可申请法院责令停止侵害

## 六、第五编 婚姻家庭（第1040-1118条）

### 6.1 结婚

- 法定婚龄：男**22**周岁，女**20**周岁
- 禁止结婚：直系血亲、三代以内旁系血亲
- 婚姻无效：重婚、禁止结婚的亲属关系、未到法定婚龄
- 可撤销婚姻：胁迫（1年内提出）；**隐瞒重大疾病**（第1053条，新增）

### 6.2 夫妻财产

| 分类 | 内容 |
|------|------|
| **共同财产** | 工资奖金劳务报酬、经营投资收益、知识产权收益、继承或受赠的财产（除指定归一方的） |
| **个人财产** | 婚前财产、人身损害赔偿/补偿、遗嘱或赠与确定归一方的、一方专用生活用品 |
| **共同债务** | 双方共同签名/一方事后追认、一方以个人名义为家庭日常生活所负债务 |

### 6.3 离婚

- **离婚冷静期**（第1077条，新增）：自申请之日起**30天**内任何一方可撤回；期满后30天内亲自申请发证，否则视为撤回
- 诉讼离婚法定情形：重婚/同居、家暴、赌博吸毒、分居满2年等
- **判决不准离婚后又分居满1年**→再次起诉应当准予（第1079条第5款，新增）
- 离婚后子女抚养：<2岁→母亲；≥2岁→最有利于未成年子女原则；≥8岁→尊重其意愿

### 6.4 离婚救济

- **离婚经济补偿**（第1088条）：因抚育子女/照料老人/协助工作负担较多义务→有权请求补偿
- **离婚损害赔偿**（第1091条）：重婚/同居/家暴/虐待遗弃/**其他重大过错**（新增兜底条款）
- **转移财产惩罚**（第1092条）：隐藏、转移、变卖、毁损、挥霍共同财产→可少分或不分

### 6.5 收养

- 收养人条件：无子女或只有1名子女、年满30周岁、有能力、无违法犯罪记录（新增）、无不宜收养的疾病
- 收养应**登记**成立；收养评估制度（新增）

## 七、第六编 继承（第1119-1163条）

### 7.1 继承顺序

| 顺序 | 继承人 |
|------|-------|
| 第一顺序 | 配偶、子女、父母 |
| 第二顺序 | 兄弟姐妹、祖父母、外祖父母 |

- **代位继承**扩大（第1128条）：被继承人的兄弟姐妹先死亡→其子女可代位继承（新增侄甥代位）
- 丧偶儿媳/女婿尽了主要赡养义务→第一顺序继承人

### 7.2 遗嘱形式

| 形式 | 要求 |
|------|------|
| 自书遗嘱 | 亲笔书写+签名+注明年月日 |
| 代书遗嘱 | 2名以上见证人+代书人+签名+年月日 |
| **打印遗嘱**（新增） | 2名以上见证人+每页签名+年月日 |
| **录音录像遗嘱**（新增） | 2名以上见证人+记录姓名/肖像+年月日 |
| 口头遗嘱 | 危急情况+2名以上见证人（危急消除后→无效） |
| 公证遗嘱 | 公证机构办理 |

**重大变更**：取消了公证遗嘱的优先效力，**以最后的遗嘱为准**（第1142条）

### 7.3 遗产管理人制度（新增）

- 遗嘱执行人→继承人推选→继承人共同→民政部门/村委会
- 职责：清理遗产、报告、防止毁损、处理债权债务、分割遗产

## 八、第七编 侵权责任（第1164-1258条）

### 8.1 归责原则

| 原则 | 适用情形 |
|------|---------|
| 过错责任 | 一般侵权（第1165条第1款） |
| 过错推定 | 法律规定推定有过错（第1165条第2款） |
| 无过错责任 | 法律规定不论有无过错（第1166条） |

### 8.2 十类特殊侵权责任

| 类型 | 责任规则要点 |
|------|-------------|
| 产品责任 | 无过错责任；生产者/销售者→消费者可择一请求；**惩罚性赔偿**（明知缺陷） |
| 机动车交通事故 | 强制保险→商业保险→侵权人；**好意同乘**减轻责任（第1217条，新增） |
| 医疗损害 | 过错责任为主；三种情形推定医方过错 |
| 环境污染和生态破坏 | 举证责任倒置；**惩罚性赔偿**（故意污染+严重后果） |
| 高度危险责任 | 无过错责任（核设施、航空器、危险物、高空高压） |
| 饲养动物损害 | 一般无过错；违规饲养→仅故意可减轻；**禁养烈性犬→绝对责任** |
| 建筑物和物件损害 | **高空抛物条款**（第1254条）：禁止抛掷物品；难以确定侵权人→可能加害人补偿；物业须安全保障；公安须调查 |
| 网络侵权 | **通知-删除**规则（第1195-1197条） |
| 用人单位责任 | 用人单位承担→可向故意/重大过失的员工追偿 |
| 安全保障义务 | 公共场所经营者/管理者/活动组织者→补充责任 |

### 8.3 损害赔偿

- 人身损害：医疗费+护理费+交通费+营养费+误工费；残疾→残疾赔偿金；死亡→丧葬费+死亡赔偿金
- **精神损害赔偿**（第1183条）：人身权益严重损害可主张；**人身意义特定物**毁损也可主张（新增）
- 知识产权**惩罚性赔偿**（第1185条，新增）

## 九、关键数字速记表

| 数字 | 对应内容 |
|------|---------|
| 8周岁 | 限制行为能力起点 |
| 16周岁 | 以劳动收入为主→视为完全行为能力 |
| 18周岁 | 成年 |

## 十、民法典十大创新亮点

1. **绿色原则**（第9条）——民事活动应有利于节约资源、保护生态环境
2. **人格权独立成编**——全球首创，系统保护人格权
3. **个人信息保护**入法——合法、正当、必要原则
4. **居住权**制度——新型用益物权，保障居住需要
5. **离婚冷静期**——申请离婚后30天内可撤回
6. **高空抛物**治理——禁止抛掷+物业安保+公安调查
7. **好人法/好意同乘**——鼓励见义勇为和善意搭载
8. **隐瞒重大疾病婚姻可撤销**——保障知情权
9. **打印遗嘱和录音录像遗嘱**——适应数字时代
10. **保证方式默认变更**——未约定或约定不明→一般保证（原为连带保证）

## 十一、与已学法律知识体系的关联

### 与《宪法》的关系
- 民法典第1条明确"根据宪法，制定本法"
- 宪法规定的公民基本权利（人身自由、人格尊严、私有财产保护等）在民法典中具体化

### 与《劳动法》的关系
- 民法典调整平等主体间民事关系，劳动法调整劳动关系（管理与被管理）
- 二者交叉领域：劳务关系（个人间劳务适用民法典第1192条）、竞业限制、知识产权归属

### 与法律概念体系的关系
- 民法典是**私法**的核心（调整平等主体）
- 属于**实体法**（规定权利义务内容）
- 属于**成文法**（大陆法系典型特征）
- 在法律效力层级中位于"法律"级别（仅次于宪法）

## 十二、实务中最值钱的知识点

### 打工人必知
1. 用人单位员工执行工作任务造成他人损害→**单位担责**，单位可向故意/重大过失的员工追偿（第1191条）
2. 个人劳务关系中受损害→根据双方过错分担（第1192条）

### 消费者必知
1. 产品缺陷造成损害→可选择向**生产者或销售者**索赔（第1203条）
2. 明知缺陷仍生产销售→可主张**惩罚性赔偿**（第1207条）
3. 格式条款中不合理免责→无效（第497条）

### 房产相关
1. **住宅建设用地使用权到期自动续期**（第359条）
2. 业主共有部分的收入归业主共有（第282条）
3. 物业不得以停水停电催交物业费（第944条）
4. 买房可以**预告登记**保护权利（第221条）

### 借贷相关
1. **禁止高利放贷**（第680条）
2. 自然人间借款未约定利息→无息（第680条第2款）
3. 利息不得预先在本金中扣除（第670条）

### 婚姻家庭
1. 离婚冷静期30天（第1077条）
2. 夫妻共同债务认定：共签/追认+家庭日常生活需要（第1064条）
3. 婚前个人财产不会因婚姻自动变为共同财产
4. 离婚时对家务劳动较多方应给予补偿（第1088条）

### 继承
1. 遗嘱以**最后一份**为准，不再以公证遗嘱优先
2. 遗产继承优先级：遗赠扶养协议 > 遗嘱/遗赠 > 法定继承
3. 继承人以所得遗产实际价值为限清偿债务（第1161条）
