## 一、法律概览

| 项目 | 内容 |
|------|------|
| 全称 | 中华人民共和国劳动合同法 |
| 性质 | 社会法（劳动法律体系核心法律） |
| 通过时间 | 2007年6月29日 |
| 施行时间 | 2008年1月1日 |
| 修正 | 2012年修正（主要修改劳务派遣相关条款） |
| 结构 | **8章98条** |
| 核心定位 | 专门规范劳动合同的订立、履行、变更、解除和终止 |

### 八章结构一览

| 章 | 名称 | 条文范围 | 核心内容 |
|---|------|---------|---------|
| 第一章 | 总则 | 第1-6条 | 立法目的、适用范围、基本原则 |
| 第二章 | 劳动合同的订立 | 第7-28条 | 合同类型、必备条款、试用期、竞业限制 |
| 第三章 | 劳动合同的履行和变更 | 第29-35条 | 全面履行、拒绝违章指挥权、变更规则 |
| 第四章 | 劳动合同的解除和终止 | 第36-50条 | 解除/终止情形、不得解除保护、经济补偿 |
| **第五章** | **特别规定** | 第51-72条 | **集体合同+劳务派遣+非全日制用工** |
| 第六章 | 监督检查 | 第73-79条 | 劳动行政部门监督 |
| 第七章 | 法律责任 | 第80-95条 | 违法后果 |
| 第八章 | 附则 | 第96-98条 | 事业单位适用、过渡规定 |

### 与《劳动法》的关系

| 对比 | 劳动法（1994） | 劳动合同法（2007） |
|------|---------------|-------------------|
| 定位 | 劳动领域的"基本法" | 劳动合同的"专门法" |
| 范围 | 劳动关系全覆盖（工时、工资、安全等） | 聚焦劳动合同制度 |
| 关系 | 一般法 | 特别法（优先适用） |
| 特点 | 原则性规定较多 | 更具体、更具操作性 |

## 二、适用范围（第2条）

| 主体 | 适用方式 |
|------|---------|
| 企业、个体经济组织、民办非企业单位 | 直接适用 |
| 国家机关、事业单位、社会团体 | 与其建立**劳动关系**的劳动者→依照本法执行 |
| 事业单位聘用制工作人员 | 有特别规定从其规定，无规定从本法（第96条） |

## 三、劳动合同的订立（第7-28条）

### 3.1 劳动关系建立

| 规则 | 内容 |
|------|------|
| 建立时点 | **用工之日**起建立劳动关系（第7条） |
| 书面合同 | 必须订立书面劳动合同（第10条） |
| 签订期限 | 用工之日起**1个月内** |
| 不签后果 | 超过1个月不满1年→**每月二倍工资**（第82条） |
| 满1年不签 | **视为已订立无固定期限劳动合同**（第14条） |

### 3.2 三种合同类型

| 类型 | 定义 | 特点 |
|------|------|------|
| 固定期限 | 约定终止时间 | 最常见 |
| 无固定期限 | 不约定终止时间 | 稳定性最强 |
| 完成一定任务期限 | 以某项工作完成为终止 | 项目制常用 |

### 3.3 必须签无固定期限的三种情形（第14条）

1. 连续工作满**10年**
2. 初次实行合同制/国企改制时连续工作满**10年**且距退休不足**10年**
3. 连续订立**二次**固定期限合同后续订（无第39条/第40条第1、2项情形）

### 3.4 合同九大必备条款（第17条）

| 序号 | 条款 |
|------|------|
| 1 | 用人单位名称、住所、法定代表人 |
| 2 | 劳动者姓名、住址、身份证号 |
| 3 | 劳动合同期限 |
| 4 | 工作内容和工作地点 |
| 5 | 工作时间和休息休假 |
| 6 | 劳动报酬 |
| 7 | 社会保险 |
| 8 | 劳动保护、劳动条件和职业危害防护 |
| 9 | 法律法规规定的其他事项 |

可选约定：试用期、培训、保守秘密、补充保险和福利等。

### 3.5 试用期规则（第19-21条）⭐ 超高频考点

| 合同期限 | 试用期上限 |
|---------|-----------|
| < 3个月 / 完成一定任务期限 | **不得约定试用期** |
| 3个月 ~ 1年 | ≤ **1个月** |
| 1年 ~ 3年 | ≤ **2个月** |
| ≥ 3年 / 无固定期限 | ≤ **6个月** |

**重要限制**：
- 同一用人单位同一劳动者只能约定**一次**试用期
- 仅约定试用期的→试用期不成立，该期限即为合同期限
- 试用期工资 ≥ 本单位同岗最低档/**合同约定**的**80%**，且 ≥ 最低工资标准
- 试用期内用人单位解除合同→必须说明理由

### 3.6 服务期与违约金（第22-25条）

| 项目 | 服务期（第22条） | 竞业限制（第23-24条） |
|------|-----------------|---------------------|
| 前提 | 提供专项培训费用 | 负有保密义务 |
| 违约金 | ≤ 培训费用（按未服务期分摊） | 按约定支付 |
| 适用人员 | 所有培训人员 | 高管、高技、保密人员 |
| 期限 | 按约定 | ≤ **2年** |
| 补偿 | 不影响正常工资调整 | 离职后按月支付经济补偿 |

> ⚠️ **第25条核心规则**：除服务期和竞业限制外，**用人单位不得与劳动者约定违约金**。

### 3.7 劳动合同无效（第26条）

三种无效情形：
1. 欺诈、胁迫、乘人之危
2. 免除单位法定责任、排除劳动者权利
3. 违反法律、行政法规强制性规定

无效但已付出劳动→仍应支付劳动报酬（第28条）。

## 四、劳动合同的解除（第36-43条）

### 4.1 解除方式全景图

```
                    劳动合同解除
                        │
            ┌───────────┼───────────┐
        协商解除        劳动者解除     用人单位解除
        (第36条)          │              │
                    ┌────┴────┐    ┌────┴────┐
                  预告解除  即时解除  过错性解除 非过错性解除 经济性裁员
                  (第37条) (第38条②) (第39条)  (第40条)    (第41条)
```

### 4.2 劳动者解除合同

| 方式 | 条件 | 条文 |
|------|------|------|
| 预告解除 | 提前**30日书面通知**（试用期**3日**） | 第37条 |
| 即时解除（用人单位有过错） | 未提供劳动保护/未足额支付工资/未缴社保/规章违法/合同无效等6种情形 | 第38条第1款 |
| 紧急解除 | 暴力强迫劳动/违章指挥危及人身安全→**不需事先告知** | 第38条第2款 |

### 4.3 用人单位解除合同

#### 过错性解除（无需补偿）（第39条）

| 序号 | 情形 |
|------|------|
| 1 | 试用期不符合录用条件 |
| 2 | 严重违反规章制度 |
| 3 | 严重失职、营私舞弊，造成重大损害 |
| 4 | 兼职影响工作或经提出拒不改正 |
| 5 | 欺诈导致合同无效 |
| 6 | 被追究刑事责任 |

#### 非过错性解除（需补偿）（第40条）

**前置条件**：提前30日书面通知**或**额外支付1个月工资（代通知金）

| 序号 | 情形 |
|------|------|
| 1 | 医疗期满不能从事原工作也不能另行安排 |
| 2 | 不能胜任→培训/调岗后仍不能胜任 |
| 3 | 客观情况重大变化→协商不成 |

#### 经济性裁员（第41条）

**条件**：裁减≥20人**或**≥职工总数10%

**程序**：提前30日向工会/全体职工说明→听取意见→报劳动行政部门

**四种法定情形**：破产重整、经营严重困难、转产/技术革新/经营方式调整、客观经济情况重大变化

**优先留用**：长期合同员工 > 无固定期限员工 > 家庭困难员工

**6个月内重新招聘→优先录用被裁人员**

### 4.4 不得解除的六种情形（第42条）⭐

| 序号 | 情形 | 记忆口诀 |
|------|------|---------|
| 1 | 职业病危害未检查/疑似职业病诊断中 | 职业病 |
| 2 | 本单位患职业病/工伤丧失劳动能力 | 工伤 |
| 3 | 患病/非因工负伤在医疗期内 | 医疗期 |
| 4 | 女职工孕期、产期、哺乳期 | 三期 |
| 5 | 连续工作满**15年**且距退休不足**5年** | 老员工 |
| 6 | 法律法规规定的其他情形 | 兜底 |

> 口诀：**职工医三老**（职业病+工伤+医疗期+三期+老员工）

### 4.5 工会参与（第43条）

用人单位单方解除合同→**必须事先通知工会**→工会有权要求纠正→用人单位应研究并书面回复。

## 五、劳动合同的终止（第44-50条）

### 5.1 终止情形（第44条）

| 序号 | 情形 |
|------|------|
| 1 | 合同期满 |
| 2 | 劳动者开始享受基本养老保险待遇 |
| 3 | 劳动者死亡/宣告死亡/宣告失踪 |
| 4 | 用人单位被宣告破产 |
| 5 | 用人单位被吊销执照/责令关闭/撤销/提前解散 |
| 6 | 其他法定情形 |

### 5.2 终止后的义务（第50条）

| 义务 | 时限 |
|------|------|
| 出具解除/终止证明 | 解除/终止时 |
| 办理档案和社保转移 | **15日内** |
| 合同文本保存 | 至少**2年**备查 |

## 六、经济补偿制度（第46-47条）⭐ 核心制度

### 6.1 应当支付经济补偿的7种情形

| 序号 | 情形 | 简记 |
|------|------|------|
| 1 | 劳动者因用人单位过错解除（第38条） | 被迫辞职 |
| 2 | 用人单位提出协商解除（第36条） | 单位提出 |
| 3 | 非过错性解除（第40条） | N+1 |
| 4 | 经济性裁员（第41条） | 裁员 |
| 5 | 合同期满（单位降低条件或不续签） | 到期不续 |
| 6 | 破产/吊销/关闭终止（第44条④⑤） | 单位消亡 |
| 7 | 其他法定情形 | 兜底 |

### 6.2 经济补偿计算公式（第47条）

```
经济补偿 = 工作年限 × 月工资

年限规则：
  - 每满1年 = 1个月工资
  - 6个月 ~ 1年 = 1个月工资
  - < 6个月 = 0.5个月工资

月工资 = 解除/终止前12个月平均工资

高薪封顶：
  - 月工资 > 当地职工月平均工资3倍 → 按3倍计算
  - 且补偿年限最高12年
```

### 6.3 违法解除的赔偿金（第48条+第87条）

违法解除/终止 → 劳动者可选择：
1. 要求继续履行合同
2. 不要求继续履行（或已不能继续） → 支付经济补偿标准的**2倍**赔偿金（2N）

## 七、特别规定

### 7.1 集体合同（第51-56条）

| 项目 | 内容 |
|------|------|
| 主体 | 工会代表职工 vs 用人单位 |
| 内容 | 劳动报酬、工时、休假、安全卫生、保险福利等 |
| 程序 | 草案→职工代表大会/全体职工讨论通过 |
| 生效 | 报送劳动行政部门→**15日内**未提异议→生效 |
| 效力 | 劳动合同标准 ≥ 集体合同标准 ≥ 当地最低标准 |

### 7.2 劳务派遣（第57-67条）⭐ 2012年修正重点

| 项目 | 内容 |
|------|------|
| 派遣单位资质 | 注册资本 ≥ **200万元**+行政许可 |
| 合同要求 | 必须订立 ≥ **2年**固定期限合同 |
| 无工作期间 | 按当地最低工资标准**按月支付** |
| 适用范围 | **临时性**（≤6个月）/**辅助性**/**替代性**岗位 |
| 同工同酬 | 与用工单位同类岗位劳动者**同工同酬** |
| 禁止 | 不得再派遣+不得自设派遣单位向本单位派遣 |
| 用工比例 | 不得超过用工总量一定比例（国务院规定） |
| 连带责任 | 用工单位造成损害→派遣单位与用工单位**连带赔偿** |

### 7.3 非全日制用工（第68-72条）

| 项目 | 内容 |
|------|------|
| 工时限制 | 日均 ≤ **4小时**，周累计 ≤ **24小时** |
| 合同形式 | 可以订立**口头协议** |
| 多重劳动关系 | 允许与多家单位建立劳动关系 |
| 试用期 | **不得约定** |
| 终止 | 任何一方可**随时通知**终止，**无需支付经济补偿** |
| 工资支付 | ≥ 最低小时工资标准，结算周期 ≤ **15日** |

## 八、法律责任速查表

| 违法行为 | 法律后果 | 条文 |
|---------|---------|------|
| 超过1个月未签书面合同 | **二倍工资** | 第82条 |
| 应签无固定期限而不签 | 自应签之日起**二倍工资** | 第82条 |
| 违法约定试用期已履行 | 按试用期满工资标准支付赔偿金 | 第83条 |
| 扣押证件 | 责令退还+处罚 | 第84条 |

| 派遣单位/用工单位违规 | 每人**5000-10000元**罚款+吊销许可证 | 第92条 |

## 十、关键数字速查表

| 数字 | 含义 |
|------|------|
| **3日** | 试用期内劳动者提前通知解除合同 |
| **15日** | 离职后办理档案社保转移/集体合同生效等待期/非全日制工资结算周期上限/答辩状提交期 |
| **30日** | 劳动者预告解除/用人单位非过错性解除通知期/经济性裁员提前通知期 |
| **1个月** | 签订书面合同的最长期限/试用期上限（3个月～1年合同） |
| **2个月** | 试用期上限（1年～3年合同） |
| **6个月** | 试用期上限（≥3年合同）/经济裁员后重新招聘优先录用期/劳务派遣临时性岗位上限 |
| **1年** | 不签书面合同→视为无固定期限 |
| **2年** | 竞业限制最长期限/派遣单位最低合同期限/合同文本保存期限 |
| **10年** | 连续工作→可签无固定期限 |
| **15年+5年** | 连续工作15年+距退休不足5年→不得解除 |
| **80%** | 试用期最低工资比例 |
| **200万元** | 劳务派遣单位最低注册资本 |
| **4小时/24小时** | 非全日制用工日/周工时上限 |

## 十一、与已学法律的交叉关联

| 关联法律 | 交叉点 |
|---------|--------|
| **劳动法** | 劳动合同法是劳动法的**特别法**，劳动合同专门条款优先适用；劳动法第16-35条的原则性规定→劳动合同法细化 |
| **民法典** | 合同订立原则（诚实信用、公平）一致；劳动合同的特殊性：不完全适用民法典合同编（劳动者是弱势方→倾斜保护） |
| **刑法** | 强迫劳动→刑法第244条（强迫劳动罪）；扣押证件→可能涉及非法拘禁罪 |
| **民事诉讼法** | 劳动报酬可申请支付令（第30条→民诉法督促程序）；劳动争议仲裁→不服起诉→民诉法审判程序 |
| **治安管理处罚法** | 强迫劳动情节较轻→治安处罚；扣押证件→治安处罚 |
| **网络安全法** | 用人单位保密义务→竞业限制（涉及技术秘密、商业秘密保护） |
| **宪法** | 劳动权是宪法基本权利（第42条），劳动合同法是具体落实 |

## 十二、实践知识点

### 12.1 小明场景：入职第一天该注意什么？

> 小明拿到offer准备入职新公司。

**入职清单**（依据劳动合同法）：
1. ✅ 签书面劳动合同（第10条，1个月内必须签）
2. ✅ 检查九大必备条款是否齐全（第17条）
3. ✅ 确认试用期是否合法（第19条，不能超上限）
4. ✅ 试用期工资≥合同工资80%（第20条）
5. ❌ 拒绝扣押身份证（第9条，违法）
6. ❌ 拒绝缴纳"入职保证金"（第9条，违法）
7. ❌ 拒绝不合理的违约金条款（第25条，只有服务期和竞业限制可约定）
8. ✅ 了解公司规章制度（第4条，公司有义务公示/告知）

| 被口头辞退 | 要求书面证明+经济补偿（违法辞退→2N） | 第50条+第87条 |
| 试用期被莫名开除 | 公司必须证明"不符合录用条件" | 第21条+第39条① |
| 怀孕被辞退 | 违法！可要求继续履行或2N赔偿 | 第42条④+第48条 |
| 公司要求交保证金 | 违法！投诉+退还+罚款 | 第9条+第84条 |
| 被迫签竞业限制但离职后没给补偿 | 竞业限制可解除 | 第23条（司法解释） |
