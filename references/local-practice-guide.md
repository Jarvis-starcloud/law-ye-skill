## 一、各地裁判口径差异速查

### 1.1 劳动争议领域

| 争议问题 | 北京 | 上海 | 深圳 | 广州 |
|----------|------|------|------|------|
| **未签合同双倍工资时效** | 逐月起算 | 逐月起算 | 逐月起算 | 逐月起算 |
| **年终奖争议** | 有约定从约定，无约定看惯例 | 同北京 | 偏向劳动者 | 有约定从约定 |
| **加班费举证** | 劳动者提供初步证据后举证责任转移 | 同北京 | 同北京 | 同北京 |
| **社保缴费基数不足能否38条解除** | 一般不支持 | 一般不支持 | 较宽松，部分支持 | 一般不支持 |
| **经济补偿基数含加班费？** | 含 | 含 | 含 | 含 |
| **竞业限制补偿标准** | 月均工资30% | 月均工资30% | 月均工资2/3 | 月均工资30% |

### 1.2 民商事领域

| 争议问题 | 北京 | 上海 | 深圳 | 广州 |
|----------|------|------|------|------|
| **民间借贷利率** | 统一适用LPR4倍 | 同 | 同 | 同 |
| **夫妻共同债务认定** | 较严格（需共同签名或事后追认） | 同 | 同 | 同 |
| **房屋买卖违约金标准** | 通常支持总价款20%以内 | 类似 | 类似 | 类似 |

> **重要提醒**：裁判口径随时可能因新司法解释或上级法院指导意见而变化。以上信息供参考，具体以最新规定和当地最新判例为准。

## 二、四大城市法院特点

### 2.1 北京

| 维度 | 特点 |
|------|------|
| **审理效率** | 案件量大，排期较长，简易程序约2-3个月 |
| **法官专业度** | 高，尤其知识产权、涉外商事领域 |
| **调解偏好** | 中等，法官会组织但不强制 |
| **网上立案** | 成熟，北京法院电子诉讼平台 |
| **特殊规定** | 知识产权案件集中管辖（海淀/朝阳） |
| **劳动仲裁** | 区仲裁委各有特点，朝阳/海淀案件量最大 |

### 2.2 上海

| 维度 | 特点 |
|------|------|
| **审理效率** | 相对较高，信息化程度最好 |
| **法官专业度** | 高，商事领域尤其突出 |
| **调解偏好** | 较强，法官倾向促成调解 |
| **网上立案** | 非常成熟，上海法院12368平台 |
| **特殊规定** | 金融法院处理金融案件 |
| **劳动仲裁** | 各区仲裁委效率较高 |

### 2.3 深圳

| 维度 | 特点 |
|------|------|
| **审理效率** | 较高，改革试点城市 |
| **法官专业度** | 高，劳动争议处理经验丰富 |
| **调解偏好** | 较强，尤其劳动争议案件 |
| **网上立案** | 成熟，深圳法院网上诉讼服务 |
| **特殊规定** | 劳动争议裁判口径相对宽松（偏向劳动者） |
| **劳动仲裁** | 南山/福田/宝安案件量最大 |

### 2.4 广州

| 维度 | 特点 |
|------|------|
| **审理效率** | 中等，案件量大 |
| **法官专业度** | 高，互联网法院处理涉网案件 |
| **调解偏好** | 中等 |
| **网上立案** | 成熟，广东法院诉讼服务网 |
| **特殊规定** | 广州互联网法院（涉网合同/侵权） |
| **劳动仲裁** | 天河/越秀/番禺案件量较大 |

## 三、网上立案操作

### 3.1 各地网上立案平台

| 城市 | 平台 | 网址/入口 |
|------|------|----------|
| **全国通用** | 中国移动微法院 | 微信小程序搜索"中国移动微法院" |
| **北京** | 北京法院电子诉讼平台 | sspt.bjcourt.gov.cn |
| **上海** | 上海法院12368 | 12368.court.gov.cn/sh |
| **深圳** | 深圳法院网上诉讼服务 | 微信搜"深圳法院" |
| **广州** | 广东法院诉讼服务网 | ssfw.gdcourts.gov.cn |

### 3.2 网上立案通用流程

```
Step 1: 注册+实名认证
├── 注册账号（手机号）
├── 实名认证（身份证+人脸识别）
└── 律师另需上传律师证

Step 2: 选择法院+案由
├── 选择管辖法院
├── 选择案由（系统有案由列表）
└── 选择程序（简易/普通/小额）

Step 3: 上传材料
├── 起诉状（PDF/图片）
├── 身份证明
├── 证据材料
└── 其他附件

Step 4: 提交+等待审核
├── 确认信息无误后提交
├── 法院7日内审核
├── 通过→缴费通知
└── 不通过→退回补正

Step 5: 缴费
├── 收到缴费通知
├── 网上缴费（微信/支付宝/银行卡）
└── 缴费成功→正式受理
```

### 3.3 网上立案注意事项

| 注意事项 | 说明 |
|----------|------|
| 文件格式 | 通常支持PDF/JPG/PNG，单文件≤20MB |
| 起诉状签名 | 需手写签名后拍照/扫描上传 |
| 材料清晰度 | 拍照要清晰，法官要能看清内容 |
| 补正期限 | 退回补正通常5个工作日内要补交 |
| 备用方案 | 网上立案被退回可以转现场立案 |

## 四、各地劳动仲裁委联系方式（查询方法）

### 4.1 统一查询方式

1. **12333热线**：全国统一劳动保障热线，可查当地仲裁委地址和电话
2. **当地人社局官网**：搜索"XX市/XX区 人力资源和社会保障局"→劳动仲裁
3. **百度地图**：搜索"XX区劳动人事争议仲裁委员会"

### 4.2 主要城市仲裁委分布

| 城市 | 特点 | 提示 |
|------|------|------|
| **北京** | 各区各有仲裁委，按用人单位注册地或合同履行地确定 | 朝阳/海淀案件最多，排队较久 |
| **上海** | 各区各有仲裁委 | 效率相对较高 |
| **深圳** | 各区仲裁委 + 市仲裁院 | 福田/南山/宝安最忙 |
| **广州** | 各区仲裁委 | 天河/越秀最忙 |

## 五、地方性特殊规定提示

### 5.1 需要特别关注的地方规定

| 城市 | 文件 | 内容 |
|------|------|------|
| 北京 | 京高法发/京劳仲委会议纪要 | 劳动争议疑难问题解答 |
| 上海 | 沪高法民一/劳动争议口径 | 上海特有的裁判标准 |
| 深圳 | 深中法会议纪要 | 深圳劳动争议特殊规则 |
| 广州 | 粤高法/广州仲裁会议纪要 | 广东省裁判口径 |

### 5.2 查找方法

1. 北大法宝 → 地方法规 → 选择城市 → 搜索关键词
2. 各地高院/中院官网 → 司法文件/审判指引
3. 搜索引擎："XX市 劳动争议 会议纪要 2024/2025/2026"

> **实务提醒**：地方会议纪要虽然不是法律，但实务中法官高度参照。打官司前一定要查当地的最新会议纪要。

## 六、跨地域诉讼注意事项

| 事项 | 建议 |
|------|------|
| **管辖权选择** | 如果合同有管辖约定，从约定；没有约定，选择对自己有利的法院 |
| **当地律师** | 跨地域诉讼建议在当地请律师或与当地律师合作 |
| **地方规定** | 提前研究受理法院所在地的裁判口径 |
| **出庭安排** | 跨地域出庭需提前安排交通住宿，预留充足时间 |
| **文书送达** | 确认送达地址准确，避免因送达问题影响案件进度 |
