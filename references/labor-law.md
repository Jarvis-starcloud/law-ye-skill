## 二、总则核心要点（第1-9条）

### 适用范围（第2条）
- **企业、个体经济组织**（统称"用人单位"）和与之形成劳动关系的劳动者
- 国家机关、事业组织、社会团体与**建立劳动合同关系**的劳动者

### 劳动者的八大权利（第3条）
1. 平等就业和选择职业
2. 取得劳动报酬
3. 休息休假
4. 获得劳动安全卫生保护
5. 接受职业技能培训
6. 享受社会保险和福利
7. 提请劳动争议处理
8. 法律规定的其他权利

### 劳动者的四大义务（第3条）
1. 完成劳动任务
2. 提高职业技能
3. 执行劳动安全卫生规程
4. 遵守劳动纪律和职业道德

### 工会权利（第7-8条）
- 劳动者有权参加和组织工会
- 工会依法独立自主地开展活动
- 通过职工大会/职代会参与民主管理

## 三、促进就业与反歧视（第10-15条）

### 就业平等（第12条）——反歧视条款
- **不因民族、种族、性别、宗教信仰不同而受歧视**

### 性别平等（第13条）
- 妇女享有与男子平等的就业权利
- 除国家规定不适合的工种外，不得以性别为由拒绝录用或提高录用标准

### 特殊群体保护（第14条）
- 残疾人、少数民族人员、退伍军人——从特别规定

### 🔴 最低就业年龄（第15条）
- **禁止招用未满16周岁的未成年人**（童工禁令）
- 例外：文艺、体育和特种工艺单位，须保障义务教育

## 四、劳动合同制度（第16-35条）——最实用的核心章节

### 劳动合同基础
| 要素 | 规定 |
|------|------|
| **形式** | 必须书面形式（第19条） |
| **原则** | 平等自愿、协商一致（第17条） |
| **约束力** | 依法订立即有约束力（第17条） |

### 劳动合同七大必备条款（第19条）
1. 劳动合同期限
2. 工作内容
3. 劳动保护和劳动条件
4. 劳动报酬
5. 劳动纪律
6. 劳动合同终止的条件
7. 违反劳动合同的责任

### 合同期限三种类型（第20条）
| 类型 | 说明 |
|------|------|
| 固定期限 | 约定起止日期 |
| 无固定期限 | 没有约定终止日期（连续工作满10年+双方同意续延+劳动者提出→应当订立） |
| 完成一定工作为期限 | 以任务完成为终点 |

### 试用期（第21条）
- **最长不超过6个月**

### 🔴 合同无效情形（第18条）
1. 违反法律、行政法规的
2. 采取欺诈、威胁等手段订立的
- 无效合同从订立时起就没有约束力
- 由劳动争议仲裁委员会或法院确认

### 🔴 用人单位可以解除合同的情形

**即时解除（第25条）——无需提前通知：**
1. 试用期间不符合录用条件
2. 严重违反劳动纪律或规章制度
3. 严重失职、营私舞弊，造成重大损害
4. 被依法追究刑事责任

**提前30日书面通知解除（第26条）：**
1. 患病/非因工负伤，医疗期满后不能从事原工作也不能另行安排
2. 不能胜任工作，经培训或调岗仍不能胜任
3. 客观情况重大变化，协商不成

**经济性裁员（第27条）：**
- 条件：濒临破产法定整顿或生产经营严重困难
- 程序：提前30日向工会/全体职工说明→听取意见→报告劳动行政部门
- 义务：6个月内录用应优先录用被裁人员

### 🔴 用人单位不得解除合同的情形（第29条）——劳动者保护盾
1. 患职业病或因工伤残且丧失/部分丧失劳动能力
2. 患病或负伤在规定医疗期内
3. **女职工在孕期、产期、哺乳期内**
4. 法律法规规定的其他情形

### 劳动者解除合同
- **正常辞职**（第31条）：提前30日书面通知
- **即时辞职**（第32条）：以下情形可随时通知解除：
  1. 试用期内
  2. 用人单位以暴力/威胁/非法限制人身自由强迫劳动
  3. 用人单位未按约定支付报酬或提供劳动条件

### 集体合同（第33-35条）
- 由工会代表职工与企业签订
- 签订后报送劳动行政部门，15日内未提异议即生效
- **个人合同标准不得低于集体合同标准**

## 五、工作时间和休息休假（第36-45条）——加班费必看

### 🔴 法定工时（第36条）
- **每日不超过8小时**
- **每周平均不超过44小时**
- **每周至少休息1日**（第38条）

### 🔴 加班限制（第41条）
- 经与工会和劳动者协商后可延长
- 一般每日不超过 **1小时**
- 特殊原因最多每日 **3小时**
- **每月不超过36小时**

### 🔴🔴 加班费标准（第44条）——这是最高频考点

| 加班类型 | 工资倍数 |
|---------|---------|
| 延长工作时间（工作日加班） | **≥ 150%** |
| 休息日加班且不能补休 | **≥ 200%** |
| 法定休假日加班 | **≥ 300%** |

### 法定节假日（第40条）
元旦、春节、国际劳动节、国庆节、其他法定休假日

### 带薪年休假（第45条）
- 连续工作1年以上享受
- 具体办法由国务院规定

## 六、工资制度（第46-51条）

### 核心原则
- **按劳分配、同工同酬**（第46条）
- **最低工资保障制度**（第48条）——省级政府规定，报国务院备案
- **以货币形式按月支付**（第50条）
- **不得克扣或无故拖欠**（第50条）

### 最低工资确定因素（第49条）
1. 劳动者本人及平均赡养人口的最低生活费用
2. 社会平均工资水平
3. 劳动生产率
4. 就业状况
5. 地区经济发展差异

### 特殊工资保障（第51条）
- 法定休假日、婚丧假、依法参加社会活动期间——**用人单位应当支付工资**

---

## 七、劳动安全卫生（第52-57条）

### 用人单位义务
- 建立健全劳动安全卫生制度
- 新建/改建/扩建工程："三同时"原则（同时设计、同时施工、同时投产）
- 提供劳动防护用品
- 对有职业危害作业者定期健康检查

### 劳动者权利（第56条）
- **有权拒绝**违章指挥、强令冒险作业
- 有权对危害生命安全和身体健康的行为提出批评、检举和控告

---

## 八、特殊群体保护（第58-65条）

### 女职工保护体系

| 阶段 | 保护内容 |
|------|---------|
| 一般 | 禁止矿山井下、第四级体力劳动（第59条） |
| 经期 | 禁止高处、低温、冷水、第三级体力劳动（第60条） |
| 孕期 | 禁止第三级体力劳动和孕期禁忌劳动（第61条） |
| 孕7月+ | 不得安排加班和夜班（第61条） |
| 产假 | **不少于90天**（第62条） |
| 哺乳期（婴儿<1岁） | 禁止第三级体力劳动和哺乳期禁忌劳动，不得加班和夜班（第63条） |

### 未成年工保护（16-18岁）
- 禁止矿山井下、有毒有害、第四级体力劳动（第64条）
- 定期健康检查（第65条）

---

## 九、社会保险和福利（第70-76条）

### 五大社保情形（第73条）
1. **退休** → 养老保险
2. **患病、负伤** → 医疗保险
3. **因工伤残或患职业病** → 工伤保险
4. **失业** → 失业保险
5. **生育** → 生育保险

### 核心规则
- 用人单位和劳动者**必须依法参加社保、缴纳社保费**（第72条）
- 任何组织和个人**不得挪用社保基金**（第74条）
- 鼓励建立补充保险、个人储蓄性保险（第75条）

## 十、劳动争议解决机制（第77-84条）

### 🔴 争议解决四步路径

```
协商 → 调解 → 仲裁 → 诉讼
```

| 环节 | 机构 | 要点 |
|------|------|------|
| 协商 | 双方自行 | 可选 |
| 调解 | 单位劳动争议调解委员会 | 由职工代表+单位代表+工会代表组成，工会代表任主任 |
| 仲裁 | 劳动争议仲裁委员会 | 由劳动行政部门+工会+用人单位方代表组成 |
| 诉讼 | 人民法院 | 对仲裁不服可诉 |

### 🔴 关键时限
- **仲裁申请**：争议发生之日起 **60日** 内
- **仲裁裁决**：收到申请后一般 **60日** 内
- **起诉期限**：收到裁决书之日起 **15日** 内

### 特别规定
- 可以**跳过调解**直接申请仲裁
- 但**不能跳过仲裁**直接起诉（仲裁前置原则）
- 一方不起诉又不履行裁决的，另一方可申请法院强制执行

## 十一、法律责任要点（第89-105条）

### 用人单位违法后果

| 违法行为 | 后果 |
|---------|------|
| 规章制度违法（89条） | 警告+责令改正+赔偿 |
| 违法延长工时（90条） | 警告+改正+罚款 |
| 克扣/拖欠工资（91条） | 责令支付+经济补偿+赔偿金 |
| 拒付加班费（91条） | 同上 |
| 低于最低工资（91条） | 同上 |
| 违法解除合同未补偿（91条） | 同上 |
| 安全设施不合格（92条） | 改正+罚款；严重→停产整顿；致人死伤→追究刑事责任 |
| 强令冒险作业致重大事故（93条） | **追究刑事责任** |
| 招用童工（94条） | 改正+罚款；严重→吊销营业执照 |
| 暴力强迫劳动/非法拘禁（96条） | **15日以下拘留+罚款/警告；构成犯罪→刑事责任** |
| 不缴社保（100条） | 限期缴纳+滞纳金 |

### 劳动者违法后果
- 违法解除合同或违反保密约定→**赔偿用人单位经济损失**（第102条）

## 十三、与已学知识的关联

### 与宪法的关系
- 劳动法第1条明确"根据宪法，制定本法"
- 宪法第42条"公民有劳动的权利和义务"→劳动法是其具体展开
- 宪法第43条"休息的权利"→劳动法第四章细化
- 宪法第48条"男女平等"→劳动法第12-13条、第七章落实

### 在法律体系中的定位
- 属于**社会法**部门——保护劳动者（弱势一方）的权益
- 是劳动法律体系的**基本法**（母法），具体规则由《劳动合同法》《社会保险法》《工伤保险条例》等配套法规细化
- 效力层级：宪法 > **劳动法**（法律级别）> 劳动行政法规 > 地方性法规 > 规章

### 与法律概念体系的对应
- **法律关系三要素**：主体=用人单位+劳动者，客体=劳动力，内容=劳动权利与义务
- **法律行为**：订立劳动合同=双方民事法律行为
- **三大法律责任**全部体现：民事（赔偿损失）、行政（罚款/吊照）、刑事（追究刑事责任）

## 十四、实务知识点（贾维斯的理解与补充）

### 这部法为什么重要
劳动法是打工人的"护身符"。几乎每个上班族都会用到的法律——从面试签合同到加班费计算到离职补偿到社保缴纳，全在里面。

### 几个容易被忽略的要点
1. **加班费不能用调休替代的只有一种**——法定节假日300%加班费，必须给钱，不能用调休代替。休息日200%加班费可以用补休替代。
2. **"提前30日"是双向的**——单位裁你要提前30天通知，你辞职也要提前30天通知。
3. **仲裁是诉讼的前置程序**——不能跳过仲裁直接去法院告，这是劳动争议的特殊规则。
4. **60日仲裁时效很短**——发现权益被侵害后60天内必须申请仲裁（注：后来《劳动争议调解仲裁法》已延长为1年）。
5. **集体合同是个人合同的"底线"**——个人合同的待遇不能低于集体合同标准。
6. **"不得低于"是强制性规定**——最低工资、加班费倍数等都是底线，合同约定低于此标准的，该条款无效。

### 劳动法 vs 劳动合同法
- 劳动法（1995年）是基本法，框架性规定
- 劳动合同法（2008年）是专门法，对劳动合同制度进行了更详细的规定
- 两者并行有效，劳动合同法的具体规定优先于劳动法的框架性规定
