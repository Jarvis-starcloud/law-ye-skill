## 二、5种常见仲裁请求写法

### 类型1：违法解除——要2N赔偿金

```
仲裁请求：
一、确认被申请人于XXXX年XX月XX日解除与申请人的
    劳动合同行为违法；
二、裁决被申请人向申请人支付违法解除劳动合同
    赔偿金人民币XXXXX元
    （月工资XXXX元 × 工作年限X年 × 2倍 = XXXXX元）；
三、裁决被申请人向申请人支付XXXX年XX月至XX月
    拖欠工资人民币XXXX元（如有欠薪一并主张）。
```

**计算说明**：
- 经济补偿金（N）= 月均工资 × 工作年限（每满1年算1个月，6个月以上不满1年按1年，不满6个月算半个月）
- 赔偿金（2N）= 经济补偿金 × 2
- 月均工资 = 解除前12个月的平均工资（含奖金、补贴、加班费）

### 类型2：欠薪

```
仲裁请求：
一、裁决被申请人向申请人支付XXXX年XX月至XX月
    拖欠工资人民币XXXXX元；
二、裁决被申请人向申请人支付拖欠工资的经济补偿金
    人民币XXXX元（拖欠工资总额的25%）。
```

> **注意**：加付赔偿金（25%/50%/100%）各地规定不同，部分仲裁委不支持在仲裁阶段主张。建议先只主张本金，如不足再诉讼阶段追加。

### 类型3：未签劳动合同——要双倍工资

```
仲裁请求：
一、裁决被申请人向申请人支付XXXX年XX月至XX月
    未签订书面劳动合同的二倍工资差额人民币XXXXX元
    （月工资XXXX元 × X个月 = XXXXX元）。
```

**计算说明**：
- 用工之日起超过1个月不满1年未签合同→支付**双倍工资**
- 最长支持11个月（第2个月到第12个月）
- 超过1年视为已签无固定期限合同
- **仲裁时效**：从应签之日的第2个月起算，逐月计算

### 类型4：加班费

```
仲裁请求：
一、裁决被申请人向申请人支付XXXX年XX月至XX月
    延时加班工资人民币XXXX元
    （日工资XXX元 ÷ 8 × 1.5倍 × 加班小时数）；
二、裁决被申请人向申请人支付XXXX年XX月至XX月
    休息日加班工资人民币XXXX元
    （日工资XXX元 × 2倍 × 加班天数）；
三、裁决被申请人向申请人支付XXXX年XX月XX日
    法定节假日加班工资人民币XXXX元
    （日工资XXX元 × 3倍 × 加班天数）。
```

**计算说明**：
- 工作日延时加班：1.5倍
- 休息日加班（未安排补休）：2倍
- 法定节假日加班：3倍
- 日工资 = 月工资 ÷ 21.75天

### 类型5：工伤赔偿（已有工伤认定+鉴定）

```
仲裁请求：
一、裁决被申请人向申请人支付一次性伤残补助金
    人民币XXXXX元（本人工资XXXX元 × X个月）；
二、裁决被申请人向申请人支付一次性工伤医疗补助金
    人民币XXXXX元；
三、裁决被申请人向申请人支付一次性伤残就业补助金
    人民币XXXXX元；
四、裁决被申请人向申请人支付停工留薪期工资
    人民币XXXXX元（月工资XXXX元 × X个月）；
五、裁决被申请人支付医疗费、住院伙食补助费等
    共计人民币XXXX元。
```

## 三、事实与理由写作要点

### 3.1 结构模板

```
第一段：劳动关系基本信息
- 入职时间、岗位、工资、合同期限

第二段：争议事实（按时间线）
- 什么时间+公司做了什么+对你的影响
- 用客观描述，不用情绪化语言

第三段：法律依据
- 引用具体法条，说明公司行为违反了什么规定

第四段：结尾
- "综上，为维护合法权益，特提出仲裁申请"
```

### 3.2 写作铁律

| 规则 | 说明 |
|------|------|
| **金额必须精确** | 写具体数字+计算过程，不写"若干"、"大约" |
| **时间必须精确** | 写"2026年3月15日"，不写"去年""前几天" |
| **法条必须引用** | 写"《劳动合同法》第四十七条"，不写"根据有关法律" |
| **事实必须客观** | 写"公司口头通知解除"，不写"公司恶意开除" |
| **请求必须明确** | 每一项请求单独列，编号清晰 |

### 3.3 常见错误

| 错误 | 正确 |
|------|------|
| "公司不给我发工资" | "被申请人自2026年1月起未向申请人支付工资" |
| "要求赔偿所有损失" | "裁决被申请人支付违法解除赔偿金38,000元" |
| "根据劳动法" | "根据《劳动合同法》第八十七条" |
| "我在这干了好几年" | "申请人于2022年3月入职，工作年限4年零1个月" |

## 五、代理人相关

### 5.1 授权委托书模板

```
                     授权委托书

委托人：[姓名]，身份证号：[XXXXXXXXXXXXXXXXXX]，
联系电话：[手机号]。

受委托人：[代理人姓名]，[律师事务所名称]律师/
[与委托人关系]，联系电话：[手机号]。

现委托上述受委托人作为我与[公司名称]劳动争议
仲裁一案中我方的代理人。

代理权限：
□ 一般代理
☑ 特别授权（代为承认、变更、放弃仲裁请求，
  进行和解、调解，代收法律文书）

委托期限：自本委托书签署之日起至本案终结之日止。

                              委托人：[签名]
                              日  期：[XXXX年XX月XX日]
```

### 5.2 谁可以当代理人

| 代理人类型 | 需要的材料 |
|-----------|-----------|
| 律师 | 律所函+律师证复印件+授权委托书 |
| 近亲属（父母/配偶/子女/兄弟姐妹） | 户口本/结婚证+身份证+授权委托书 |
| 工会推荐 | 工会推荐函+身份证+授权委托书 |
| 法律援助 | 法律援助函+身份证+授权委托书 |
