## 一、首次接待问诊清单

### 1.1 开场（2分钟）

```
"您好，我是[姓名/law爷]。先简单了解一下您的情况，
我会做些记录方便后续分析。您先大概说一下遇到了什么问题？"
```

**原则**：先让当事人自由陈述3-5分钟，不打断。记下关键词后再逐一追问。

### 1.2 核心信息采集表

| 类别 | 必须问的问题 | 目的 |
|------|-------------|------|
| **当事人信息** | 姓名/联系方式/身份证号 | 主体确认 |
| **对方信息** | 对方是谁/公司还是个人/联系方式 | 确定被告 |
| **时间线** | 事情是什么时候发生的？按时间顺序说 | 梳理事实 |
| **核心诉求** | 您想要什么结果？（赔钱/道歉/恢复关系/其他）| 明确目标 |
| **已有证据** | 手上有什么材料？（合同/收据/聊天记录/录音）| 评估证据 |
| **对方态度** | 有没有协商过？对方怎么说？ | 了解对方立场 |
| **时效检查** | 事情发生多久了？有没有向谁反映过？ | 确认时效 |
| **经济状况** | 对方有没有资产？公司还在正常经营吗？ | 执行可能性 |

### 1.3 分类型追问

#### 劳动争议追问

| 问题 | 为什么问 |
|------|----------|
| 入职时间？签了几年合同？ | 计算工龄和补偿 |
| 月工资多少？怎么发的？ | 确定赔偿基数 |
| 怎么被辞退的？有书面通知吗？ | 判断是否违法解除 |
| 社保交了吗？公积金呢？ | 是否有额外索赔空间 |
| 签过竞业限制吗？ | 可能的对方反诉风险 |

#### 合同纠纷追问

| 问题 | 为什么问 |
|------|----------|
| 合同在手上吗？还是原件？ | 证据准备 |
| 对方违约的具体表现？ | 确定违约事实 |
| 合同里有没有违约金/管辖条款？ | 法律适用和管辖 |
| 你这边有没有违约的地方？ | 评估对方可能的抗辩 |
| 还有没有其他协议/补充协议？ | 完整了解合同关系 |

#### 侵权纠纷追问

| 问题 | 为什么问 |
|------|----------|
| 受伤情况？去医院了吗？ | 伤情和医疗费 |
| 有没有报警？交通事故认定书？ | 责任划分 |
| 误工多久？月收入多少？ | 计算损失 |
| 有没有做伤残鉴定？ | 赔偿计算依据 |

## 二、案情分析汇报

### 2.1 汇报结构（5分钟内讲清楚）

```
第一步：复述确认（1分钟）
"我来确认一下：您是[时间]入职[公司]，[月薪]，
[时间]被公司[具体行为]，现在想要[诉求]。
我理解得对吗？"

第二步：法律分析（2分钟）
"根据您描述的情况，从法律角度来看——
[一句话结论]。
具体来说，[法律依据+分析]。"

第三步：方案建议（1分钟）
"我建议[方案A/B]。
方案A是[具体方案]，优点是[XXX]，缺点是[XXX]。
方案B是[具体方案]，优点是[XXX]，缺点是[XXX]。"

第四步：风险提示（1分钟）
"需要提醒您的风险是：[具体风险]。
最好/最坏的结果分别是[XXX]。"
```

### 2.2 分析汇报模板

```
案情分析摘要

一、基本事实
- 当事人：[姓名] vs [对方]
- 案由：[XX纠纷]
- 核心诉求：[具体金额/具体要求]

二、法律分析
- 法律依据：[法律名称+条款]
- 有利因素：1. [XXX] 2. [XXX]
- 不利因素：1. [XXX] 2. [XXX]

三、胜诉评估
- 综合评分：[XX/100]
- 风险等级：[低/中/高]

四、方案建议
- 推荐方案：[协商/仲裁/诉讼]
- 预估成本：[XXX]元
- 预估周期：[X]个月
- 预期结果：[XXX]

五、下一步行动
1. [需要当事人做什么]
2. [需要律师做什么]
3. [时间节点]
```

## 三、预期管理

### 3.1 核心原则

| 原则 | 说明 |
|------|------|
| **诚实第一** | 不为了接案夸大胜算 |
| **讲概率不讲必然** | "胜诉概率约70%"而非"一定能赢" |
| **最好最坏都说** | 让当事人有心理准备 |
| **成本透明** | 所有费用提前说清楚 |
| **时间透明** | 告知合理的时间预期 |

### 3.2 预期管理话术

#### 胜诉概率较高时

```
"根据目前的证据和法律规定，您这个案子胜诉概率比较大，
我初步评估在70%-80%左右。

但我必须提醒您：
1. 法律诉讼没有100%的事，对方可能有我们不知道的证据
2. 赢了官司和拿到钱是两回事，执行还需要一段时间
3. 预计整个流程需要[X]个月

成本方面，诉讼费约[X]元，律师费约[X]元。
胜诉后诉讼费由对方承担。"
```

#### 胜诉概率一般时

```
"坦率说，这个案子有机会但也有风险。

有利的方面：[1. XXX 2. XXX]
风险方面：[1. XXX 2. XXX]

我建议先尝试协商/调解。如果协商不成，
再考虑诉讼——但您需要做好可能败诉的准备。

败诉的话，您需要承担自己的律师费和诉讼费，
大约[X]元。"
```

#### 胜诉概率较低时

```
"我需要跟您说实话——基于目前的情况，
这个案子打赢的难度比较大。主要困难是：
1. [核心困难]
2. [核心困难]

我的建议是：
首选：尝试协商解决，接受一个合理但可能不完美的方案
次选：补充[XXX]方面的证据后再重新评估
不建议：在目前证据条件下直接起诉

这是我的专业判断，最终决定权在您。"
```

## 四、坏消息传达

### 4.1 坏消息类型

| 类型 | 示例 |
|------|------|
| 案件评估不乐观 | 胜诉概率低 |
| 败诉通知 | 一审判我们输了 |
| 执行困难 | 对方没有可执行财产 |
| 费用超预期 | 需要做鉴定，额外增加费用 |
| 时间超预期 | 对方拖延，案件延期 |

### 4.2 SPIKES沟通法（医学领域借鉴）

| 步骤 | 操作 | 话术示例 |
|------|------|----------|
| **S**etting | 选合适的时间和场景 | 约面谈或电话，不发微信 |
| **P**erception | 先了解对方的预期 | "您觉得这个案子进展怎么样？" |
| **I**nvitation | 预告有不好的消息 | "有件事需要跟您沟通一下" |
| **K**nowledge | 清楚说明情况 | "法院判决下来了，结果不太理想..." |
| **E**mpathy | 表达理解 | "我理解这个结果不是您想要的" |
| **S**trategy | 提出下一步方案 | "但我们还有上诉的机会，我建议..." |

### 4.3 绝不能做的事

| ❌ 不能做 | 说明 |
|-----------|------|
| 微信发"败诉了" | 太冷冰冰，必须语音或面谈 |
| 推卸责任 | "法官不公正"——不专业 |
| 过度承诺 | "上诉一定能翻"——不真实 |
| 消失不回应 | 当事人最怕的是联系不上律师 |
| 用专业术语绕圈子 | 直接说结果，再解释原因 |

## 五、委托签约要点

### 5.1 签约前确认清单

| 确认事项 | 说明 |
|----------|------|
| 委托范围 | 仲裁/一审/二审/执行——哪些阶段 |
| 代理权限 | 一般代理 vs 特别授权 |
| 律师费 | 金额/支付方式/支付时间 |
| 办案费用 | 诉讼费/鉴定费等额外费用谁垫付 |
| 风险代理条款 | 如果是风险代理，胜诉分成比例 |
| 案件预估 | 再次确认风险评估和时间预期 |

### 5.2 委托代理协议核心条款

| 条款 | 内容 |
|------|------|
| 委托事项 | 具体案件+代理阶段 |
| 代理权限 | 一般/特别授权 |
| 律师费 | 金额+支付方式+发票 |
| 办案费用 | 谁承担+报销标准 |
| 保密义务 | 双方互相保密 |
| 解约条件 | 什么情况可以解除 |
| 争议解决 | 律所所在地法院/仲裁 |

## 六、沟通频率建议

| 阶段 | 建议频率 | 方式 |
|------|----------|------|
| 准备阶段 | 每周1次 | 微信/电话 |
| 开庭前 | 开庭前3天必须联系 | 面谈/电话 |
| 开庭后 | 当天或次日通报 | 电话 |
| 等待判决 | 每2周1次 | 微信 |
| 判决后 | 收到后立即通知 | 电话+面谈 |
| 执行阶段 | 有进展随时通报 | 微信/电话 |

> **铁律**：当事人的消息48小时内必须回复。哪怕暂时没有进展，也要回复"目前在等法院通知，有新消息第一时间告诉您"。
