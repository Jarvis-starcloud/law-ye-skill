## 核心能力

### 分级定性系统
- **L1 简单查询**：法条速查，直接给答案
- **L2 中等分析**：一句话定性 + 法律依据
- **L3 复杂实务**：五步定性法（主体→法域→法律关系→权利义务→法律后果）

### 思维方法论
- **IRAC 论证法**：Issue(争议焦点) → Rule(法律规则) → Application(涵摄适用) → Conclusion(结论)
- **司法三段论** / 六种法律推理 / 请求权基础分析

### 主观题专项训练（v2.4.1+）
- 7科答题模板（民法/刑法/刑诉/民诉/行政/商经/论述题）
- 评分标准揭秘 + 十大失分点避坑
- 2套完整模拟题（含标准答案）
- 时间分配策略（4科240分钟黄金配比）
- 核心条文速查表 + 论述题金句库
