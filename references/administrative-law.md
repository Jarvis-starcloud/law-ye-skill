> 学习时间：2026年4月8日
> 覆盖法律：行政诉讼法(2017修正)、行政复议法(2023修订)、行政处罚法(2021修订)、行政许可法(2019修正)、行政强制法(2011)、国家赔偿法(2012修正)、政府信息公开条例(2019修订)

## 一、行政法基本原理

### 六大基本原则

| 原则 | 核心内容 | 记忆关键词 |
|------|---------|-----------|
| **合法行政**（首要） | 法律优先（不得违反）+ 法律保留（必须有依据） | "法无授权不可为" |
| **合理行政** | 公平公正 + 考虑相关因素 + **比例原则**（适当性+必要性+均衡性） | 比例原则="帝王条款" |
| **程序正当** | 行政公开 + 公众参与 + 回避制度 | "看得见的正义" |
| **高效便民** | 遵守时限 + 提供便捷服务 | 不拖延 |
| **诚实守信** | 行政信息真实 + **信赖保护**（不得随意撤销变更已生效行为） | 信赖保护 |
| **权责统一** | 有权必有责 + 用权受监督 + 违法须追究 + 侵权须赔偿 | 权责一致 |

### 行政主体分类

```
行政主体
├── 行政机关（中央/地方/派出机关/派出机构）
├── 法律法规授权的组织（事业单位/社会团体/企业/自治组织）
└── 受委托组织（不是行政主体！以委托机关名义行使权力）
```

**授权 vs 委托**：

| 维度 | 法律法规授权 | 行政委托 |
|------|-------------|---------|
| 法律地位 | 获得行政主体资格 | 不获得 |
| 以谁名义 | 自己 | 委托机关 |
| 法律后果 | 自己承担 | 委托机关承担 |
| 被告认定 | 被授权组织 | 委托机关 |

## 二、行政行为体系

### 行政行为分类

```
行政行为
├── 抽象行政行为（制定规范性文件——不可直接诉讼，可附带审查）
│   ├── 行政法规（国务院）
│   ├── 行政规章（部门/地方）
│   └── 其他规范性文件（"红头文件"）
└── 具体行政行为（针对特定对象——可以诉讼）
    ├── 依申请：行政许可、行政给付
    ├── 依职权：行政处罚、行政强制、行政征收/征用、行政检查
    └── 双方行为：行政协议
```

### 行政行为的效力四要素

| 效力 | 含义 |
|------|------|
| 公定力 | 一经作出推定合法有效 |
| 确定力 | 非经法定程序不得变更撤销 |
| 拘束力 | 相对人必须遵守执行 |
| 执行力 | 可强制执行 |

### 无效、撤销、废止

| 类型 | 原因 | 效力 |
|------|------|------|
| 无效 | 重大且明显违法 | 自始无效 |
| 撤销 | 违法（非重大明显） | 溯及既往，补偿信赖利益 |
| 废止 | 情况变化/已完成 | 向后失效 |

## 三、行政许可（行政许可法，2003年通过，2019年修正）

**本质**：一般禁止 + 个别解禁

### 可以不设许可的四种情形

1. 公民能自主决定
2. 市场竞争能有效调节
3. 行业组织能自律管理
4. 事后监督能解决

### 许可设定权层级

| 层级 | 权限 |
|------|------|
| 法律 | 可设定所有类型许可 |
| 行政法规 | 可设定（法律未设定事项） |
| 国务院决定 | 临时性许可 |
| 地方性法规 | 可设定（法律、行政法规未设定事项） |
| 省级政府规章 | 临时性许可（1年后须立法） |
| 部门规章/市级规章 | ❌ 不得设定 |

### 撤回、撤销、注销、吊销

| 类型 | 许可本身 | 原因 | 性质 |
|------|---------|------|------|
| 撤回 | 合法 | 客观情况变化 | 补偿 |
| 撤销 | 违法 | 许可过程违法 | 赔偿 |
| 注销 | 效力消灭 | 到期/放弃/死亡 | 登记手续 |
| 吊销 | 合法 | 被许可人事后违法 | 行政处罚 |

## 四、行政处罚（行政处罚法，2021年全面修订）

### 处罚种类

| 类别 | 种类 |
|------|------|
| 声誉罚 | 警告、**通报批评**（新增） |
| 财产罚 | 罚款、没收违法所得/非法财物 |
| 资格罚 | 暂扣/吊销许可证执照、**降低资质等级**（新增） |
| 行为罚 | 责令停产停业、**限制开展经营活动**（新增）、**限制从业**（新增） |
| 人身罚 | 行政拘留（只能由法律设定，只有公安执行） |

### 2021年修订十大核心变化

| 变化 | 要点 |
|------|------|
| 明确概念 | 首次定义"行政处罚" |
| 新增种类 | 通报批评、降级、限制经营、限制从业 |
| 下沉执法 | 乡镇/街道可获委托处罚权 |
| 延长时效 | 一般2年；涉及生命健康/金融安全5年 |
| **首违不罚** | 初次违法+危害轻微+及时改正→可不处罚 |
| **无过错不罚** | 有证据证明无主观过错→不处罚 |
| 听证扩大 | 新增没收大额/降级/关闭/限制从业 |
| 非现场执法 | 电子监控须经法制和技术审核 |
| 法制审核 | 重大案件必经法制审核 |
| 办案期限 | 立案起90日内 |

### 处罚程序

```
发现违法→立案→调查取证(2人以上)→告知权利→法制审核(重大案件)→负责人审批→作出决定→送达
```

简易程序：事实清楚+证据确凿+个人≤200元/单位≤3000元罚款或警告

## 五、行政强制（行政强制法，2011年）

### 强制措施 vs 强制执行

| 维度 | 行政强制措施 | 行政强制执行 |
|------|------------|------------|
| 目的 | 制止违法/防止证据损毁 | 迫使履行已生效行政决定 |
| 前提 | 不以行政决定为前提 | 以不履行生效决定为前提 |
| 性质 | 暂时性、即时性 | 终局性、强制性 |
| 种类 | 限制人身自由、查封/扣押/冻结 | 加处罚款/滞纳金、划拨、拍卖、代履行 |

### 设定权

| 设定主体 | 强制措施权限 |
|----------|------------|
| 法律 | 所有类型 |
| 行政法规 | 查封、扣押（不能冻结、不能限制人身自由） |
| 地方性法规 | 查封、扣押 |
| 规章及以下 | ❌ 不得设定 |

> "冻人只能法律定"——冻结存款和限制人身自由只能由法律设定

### 有直接强制执行权的机关

公安、国安、税务、海关、县级以上政府（强拆违建）——其他机关须申请法院执行

## 六、行政诉讼法（2017年修正，10章103条）

### 基本原则

| 原则 | 条文 |
|------|------|
| **合法性审查** | 第6条 |
| **被告举证** | 第34条 |
| 负责人出庭应诉 | 第3条 |
| 不停止执行（原则） | 第56条 |
| 有限调解 | 第60条 |

### 受案范围（第12条，12类）

1. 行政处罚不服
2. 行政强制措施/执行不服
3. 行政许可决定不服
4. 自然资源权属决定不服
5. 征收/征用决定及补偿不服
6. 申请履行职责被拒/不答复
7. 侵犯经营自主权/土地经营权
8. 滥用行政权力排除限制竞争
9. 违法集资/摊派/要求履行义务
10. 未依法支付抚恤金/低保/社保
11. 不履行/违法变更政府协议
12. 侵犯其他人身权财产权（兜底）

### 不予受理（第13条，4类）

国家行为、抽象行政行为、内部人事行为、终局裁决

> 抽象行政行为不可直接诉，但可**附带审查**规范性文件（不含规章）

### 管辖

| 规则 | 管辖法院 |
|------|---------|
| 一般 | 最初作出行政行为的机关所在地 |
| 经复议 | 原机关所在地或复议机关所在地（原告选） |
| 限制人身自由 | 被告所在地或原告所在地 |
| 不动产 | 不动产所在地（专属管辖） |

### 被告确定

| 情形 | 被告 |
|------|------|
| 作出行政行为的机关 | 该机关 |
| 复议维持 | 原机关+复议机关（共同被告） |
| 复议改变 | 复议机关 |
| 委托组织作出 | 委托机关 |

### 举证责任（核心区别！）

| 事项 | 举证方 |
|------|--------|
| **行政行为合法性** | **被告**（行政机关） |
| 提出过申请（不作为案件） | 原告 |
| 损害事实（赔偿案件） | 原告 |

> 与民事诉讼最大区别——行政诉讼**被告举证**，行政机关需证明自己行为合法

### 起诉期限

| 情形 | 期限 |
|------|------|
| 一般 | 知道之日起**6个月** |
| 不动产 | 行为作出之日起**20年** |
| 其他 | 行为作出之日起**5年** |
| 复议后 | 收到复议决定**15日** |

### 六种判决类型

| 判决 | 适用条件 |
|------|---------|
| 驳回诉讼请求 | 行政行为合法 |
| **撤销/部分撤销** | 六种情形：证据不足/法律错误/程序违法/超越职权/滥用职权/明显不当 |
| 判决履行 | 被告不履行法定职责 |
| 判决变更 | 处罚明显不当或金额确定 |
| 确认违法 | 应撤销但会损害国家利益；程序轻微违法 |
| 确认无效 | 重大且明显违法 |

### 审理期限

| 程序 | 期限 |
|------|------|
| 一审普通 | 6个月 |
| 简易 | 45日 |
| 二审 | 3个月 |

## 七、行政复议法（2023年修订，2024年1月1日施行，7章89条）

### 定位

行政复议 = 行政机关内部自我纠错 + **化解行政争议的主渠道**

### 复议 vs 诉讼

| 维度 | 行政复议 | 行政诉讼 |
|------|---------|---------|
| 性质 | 行政救济 | 司法救济 |
| 审查范围 | 合法性+**合理性** | 原则上只审合法性 |
| 费用 | 免费 | 收诉讼费 |
| 期限 | 60日(简易30日) | 6个月 |
| 终局 | 不服可诉讼 | 二审终审 |

### 2023年八大核心变化

1. **管辖集中**：统一由县级以上地方政府管辖（垂直领导机关除外）
2. **受案范围扩大**：新增行政协议、信息公开、工伤认定等；首次制定负面清单
3. **前置范围扩大**：当场处罚、不履行职责、信息公开必须先复议
4. **申请制度完善**：60日申请期限，可书面/邮寄/互联网提交
5. **受理程序规范**：5日内审查，10日内补正
6. **双轨审理**：普通程序(60日)+简易程序(30日)
7. **附带审查完善**：可附带审查规范性文件，新增专节5条
8. **决定类型丰富**：新增确认无效、责令履行、返还财产等

### 复议前置事项（必须先复议）

- 当场作出的行政处罚
- 未履行法定职责
- 政府信息公开决定
- 其他法定前置

## 八、国家赔偿法（行政赔偿，1994年通过，2010/2012年修正）

### 核心前提：**违法→赔偿；合法→补偿**

### 赔偿范围

**人身权**：违法拘留、非法拘禁、殴打虐待致伤亡、违法使用武器警械致伤亡

**财产权**：违法罚款/没收/征收、违法查封扣押冻结、违法征收征用

### 赔偿标准

| 损害类型 | 标准 |
|---------|------|
| 人身自由 | 每日 = 国家上年度职工日平均工资 |
| 造成伤害 | 医疗费+护理费+误工收入（最高5倍年均工资） |
| 丧失劳动能力 | +残疾赔偿金（最高**20倍**年均工资） |
| 造成死亡 | 死亡赔偿金+丧葬费（合计**20倍**年均工资） |
| 精神损害 | 严重后果→精神损害抚慰金（2010年新增） |
| 财产损害 | 返还/恢复原状/赔偿直接损失 |

### 赔偿程序

```
违法行为致损→向赔偿义务机关提出→2个月内决定→不赔/逾期→3个月内向法院起诉
```

### 特殊举证

被限制人身自由期间死亡/丧失行为能力 → **赔偿义务机关**举证因果关系

## 九、行政协议

### 概念

行政机关为公共管理/服务目的与相对人协商订立的协议

典型类型：特许经营(BOT/PPP)、征收补偿、国有土地出让、自然资源出让

### 双重属性

| 属性 | 规则 |
|------|------|
| 行政性 | 行政机关有行政优益权（单方变更/解除） |
| 协议性 | 双方合意，可参照民事合同法 |

> **合法性优先于合约性**——行政机关基于公共利益行使优益权可单方解除，但必须依法补偿

## 十、政府信息公开（条例，2007年制定，2019年修订）

### 原则

**以公开为常态、不公开为例外**

### 不予公开

国家秘密、法定禁止、危及"三安全一稳定"、商业秘密、个人隐私、过程性信息、执法案卷

### 公开方式

- 主动公开：涉及公共利益的信息
- 依申请公开：20个工作日内答复（可延长20日）

## 十一、行政诉讼实务要点

### 起诉前战略思考

| 要素 | 内容 |
|------|------|
| 复议还是诉讼？ | 复议免费+审查合理性+更快；诉讼更独立 |
| 是否复议前置？ | 2023年新增前置情形要熟悉 |
| 起诉期限 | 6个月一般期限 |
| 被告对不对？ | 复议维持→共同被告 |
| 受案范围 | 抽象行政行为不可诉但可附带审查 |

### 常见案件攻防要点

| 案件类型 | 原告攻击点 | 被告防御点 |
|---------|-----------|-----------|
| 征收补偿 | 标准过低/程序违法/未评估 | 符合征收条例/已评估听证 |
| 行政处罚 | 事实不清/程序违法/处罚过重 | 证据充分/程序合法/裁量适当 |
| 行政不作为 | 已申请+期限已过+未答复 | 已办理/不属职责范围 |
| 信息公开 | 申请合法+应公开范围 | 国秘/第三方权益/过程性信息 |
| 行政协议 | 违约/单方解除无依据 | 公共利益优益权/已补偿 |

### 行政协议典型裁判规则

| 规则 | 要点 |
|------|------|
| 政府守信践诺 | 行政协议有公信力，签了就要兑现 |
| 合法性优先 | 协议与法律冲突时依法变更 |
| 单方解除权 | 相对人根本违约→行政机关可单方解除 |
| 行政优益权 | 公共利益+政策调整→可解除但须补偿 |
| 违约金调整 | 可参照民事合同法调整 |
| 附条件协议 | 条件未成就→协议未生效 |
