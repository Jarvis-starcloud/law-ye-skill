## 二、立案阶段（第109-114条）

### 2.1 立案条件

| 条件 | 内容 |
|------|------|
| 事实条件 | 有犯罪事实 |
| 法律条件 | 需要追究刑事责任 |

**不立案的情形**（《刑诉法》第16条）：
1. 情节显著轻微、危害不大，不认为是犯罪的
2. 犯罪已过追诉时效期限的
3. 经特赦令免除刑罚的
4. 告诉才处理的犯罪，没有告诉或撤回告诉的
5. 犯罪嫌疑人、被告人死亡的
6. 其他法律规定免予追究刑事责任的

### 2.2 职能管辖分工

| 机关 | 管辖范围 |
|------|----------|
| **公安机关** | 绝大多数刑事案件（普通侦查权） |
| **人民检察院** | 职务犯罪（司法工作人员利用职权实施的侵犯公民权利、损害司法公正的犯罪）；经省级以上检察决定的重大国家工作人员犯罪 |
| **人民法院** | 自诉案件（直接受理） |
| **国家安全机关** | 危害国家安全案件 |
| **军队保卫部门** | 军队内部案件 |
| **中国海警局** | 海上案件 |
| **监狱** | 监狱内罪犯犯罪 |

### 2.3 立案监督（第113条）

> **被害人维权关键点**！如果公安机关对应当立案的案件不立案：

**救济路径**：
```
公安机关不立案 → 向人民检察院提出 → 检察院要求说明理由
→ 认为理由不成立 → 通知公安机关立案 → 公安必须立案
```

## 三、侦查阶段（第115-168条）

### 3.1 强制措施一览表

| 措施 | 执行机关 | 适用对象 | 最长期限 | 核心条件 |
|------|----------|----------|----------|----------|
| **拘传** | 公安/检/法 | 嫌疑人/被告 | ≤12小时（重大复杂≤24h） | 到案接受讯问 |
| **取保候审** | 公安 | 嫌疑人/被告 | **最长12个月** | 不致发生社会危险性 |
| **监视居住** | 公安 | 符合逮捕条件的嫌疑人 | **最长6个月** | 有特殊情况不便羁押 |
| **拘留** | 公安 | 现行犯/重大嫌疑分子 | 一般14天+7天=**21天**；流窜等可**37天** | 紧急情况先行控制 |
| **逮捕** | 检察院批准/法院决定，公安执行 | 嫌疑人/被告 | 2个月起（可延长） | 有证据证明有罪+可能判徒刑+有社会危险性 |

#### 取保候审 vs 监视居住 vs 逮捕——怎么选？

| 维度 | 取保候审 | 监视居住 | 逮捕 |
|------|----------|----------|------|
| 自由度 | 较高（限制出境/离开市县） | 低（不得离开执行处所） | 无（关押在看守所） |
| 适用门槛 | 可能判轻刑+无危险性 | 符合逮捕条件但不宜羁押 | 有社会危险性必须羁押 |
| 折抵刑期 | ❌ 不折抵 | ✅ 1:1或2:1折抵 | ❌ 不折抵（已实际执行） |

### 3.2 侦查羁押期限（核心时间线）

```
逮捕后侦查羁押：
├── 基础期限：2个月
├── 第一次延长（上级检察院批准）：+1个月 = 3个月
├── 第二次延长（4种情形，省检批准）：+2个月 = 5个月
├── 第三次延长（可能判10年以上，省检批准）：+2个月 = 7个月
└── 特殊情况（全国人大常委会批准）：无限期延长
    （发现新罪→重新计算！）
    
拘留转逮捕的时间线：
├── 拘留后：24小时内讯问
├── 提请批捕：3日内（一般）/ 7日内（流窜等可延至30日）
├── 检察院批捕审查：7日内决定
└── 总计最长时间：37天（30+7），俗称"黄金37天"
```

> ⚠️ **"黄金37天"**：这是刑事辩护最关键的窗口期。律师在37天内介入，可以申请取保候审、提出不批捕意见。一旦批捕后取保难度大幅上升。

### 3.3 律师在侦查阶段的权利（第34-49条）

| 时间节点 | 权利内容 |
|----------|----------|
| **第一次讯问/采取强制措施起** | 可委托律师辩护（仅限律师） |
| **持三证** | 会见在押嫌疑人（至迟48小时内安排） |
| **危害国家安全/恐怖活动** | 会见需经侦查机关许可 |
| **移送审查起诉日起** | 阅卷（查阅摘抄复制案卷材料） |
| **侦查期间** | 提供法律帮助、代理申诉控告、申请变更强制措施 |
| **核实证据** | 移送审查起诉后可向嫌疑人核实证据 |

### 3.4 证据收集规则（第50-65条）

#### 八种法定证据类型

| 序号 | 类型 | 说明 |
|------|------|------|
| 1 | 物证 | 犯罪工具、赃物、痕迹等实物 |
| 2 | 书证 | 文件、信件、账簿等书面材料 |
| 3 | 证人证言 | 目击者/知情人的陈述 |
| 4 | 被害人陈述 | 受害人对案情的描述 |
| 5 | 犯罪嫌疑人供述和辩解 | 口供 |
| 6 | 鉴定意见 | 司法鉴定机构的专业意见 |
| 7 | 勘验、检查、辨认、侦查实验笔录 | 侦查活动的记录 |
| 8 | 视听资料、电子数据 | 录音录像、聊天记录、数据日志等 |

#### 非法证据排除规则（第56条）

| 证据类型 | 排除标准 | 排除方式 |
|----------|----------|----------|
| **言词证据（口供/证言/陈述）** | 采用刑讯逼供/暴力/威胁/非法拘禁等方法取得 | **绝对排除**（一律排除） |
| **实物证据（物证/书证）** | 收集程序违法，可能严重影响司法公正 | **相对排除**（补正或合理解释则不排） |

> **"重证据，重调查研究，不轻信口供"原则（第55条）**：
> - 只有口供没有其他证据 → 不能定罪
> - 没有口供但证据确实充分 → 可以定罪
> - "确实充分"三要件：①定罪量刑事实都有证据证明 ②据以定案的证据经法定程序查实 ③全案证据综合认定排除合理怀疑

### 3.5 技术侦查措施（第150-154条）

| 要素 | 规则 |
|------|------|
| 适用案件 | 危害国家安全/恐怖活动/黑社会/重大毒品/严重危害社会的犯罪 |
| 批准手续 | 严格批准（公安负责人/检察长） |
| 有效期 | 每次**最长3个月**，可延期（每次≤3月） |
| 保密要求 | 知悉的国家秘密/商业秘密/隐私须保密，无关材料及时销毁 |
| 使用限制 | 只能用于侦查起诉审判，不得他用 |
| 证据效力 | 可作为刑事诉讼证据使用（必要时庭外核实保护安全） |

## 四、审查起诉阶段（第169-182条）

### 4.1 审查起诉五项内容（第171条）

1. 犯罪事实、情节是否清楚，证据是否确实、充分，犯罪性质和罪名认定是否正确
2. 有无遗漏罪行和其他应当追究刑事责任的人
3. 是否属于不应追究刑事责任的
4. 有无附带民事诉讼
5. 侦查活动是否合法

### 4.2 审查期限

| 情形 | 期限 |
|------|------|
| 一般案件 | **1个月** |
| 重大/复杂案件 | 可延长**15日**（共45天） |
| 认罪认罚速裁 | **10日**（可能判>1年可延至15日） |
| 改变管辖 | 从收到案件之日起重新计算 |
| 补充侦查 | 退回补侦**1个月内**完成，以**2次**为限 |

### 4.3 四种不起诉类型

| 类型 | 适用条件 | 效果 |
|------|----------|------|
| **法定不起诉**（第177条第1款） | 无犯罪事实 / 第16条六种不追诉情形之一 | **应当**不起诉（没得选） |
| **酌定不起诉**（第177条第2款） | 犯罪情节轻微 + 不需要判处刑罚或者免除刑罚 | **可以**不起诉（检察官裁量权） |
| **存疑不起诉**（第175条第4款） | 二次补充侦查后仍然证据不足 | 应当不起诉 |
| **特殊不起诉**（第182条） | 自首+重大立功/涉及国家重大利益，经**最高检核准** | 可以撤销案件/不起诉（极少数） |

### 4.4 认罪认罚从宽制度（第15条、174-176条、201条）

> 这是2018年刑诉法修改的最大亮点，也是当前刑事案件处理的主流路径。

**构成要素**：
| 要素 | 含义 |
|------|------|
| **认罪** | 自愿如实供述自己的罪行，承认指控的犯罪事实 |
| **认罚** | 真诚悔罪，愿意接受处罚 |
| **具结书** | 在辩护人或值班律师在场情况下签署《认罪认罚具结书》 |
| **量刑建议** | 检察院就主刑、附加刑、是否缓刑等提出建议 |

**法律效果**：
- 一般应当从轻处罚
- 符合条件的适用非监禁刑
- 法院一般应采纳检察院指控的罪名和量刑建议（201条）

**例外情况**（法院可不采纳量刑建议）：
1. 行为不构成犯罪或不追责
2. 违背意愿认罪认罚
3. 否认指控事实
4. 起诉罪名与审理认定不一致
5. 其他影响公正审判的情形

**不需要签署具结书的例外**：
- 盲聋哑人/精神病人
- 未成年人的法定代理人异议
- 其他不需要情形

## 五、一审审判阶段（第183-226条）

### 5.1 三种审判程序对比

| 维度 | 普通程序 | 简易程序 | 速裁程序 |
|------|----------|----------|----------|
| **适用条件** | 所有公诉/自诉案件 | 事实清楚+认罪+同意简易 | 事实清楚+**认罪认罚**+同意速裁 |
| **审判组织** | 合议庭（3人或7人） | 合议庭或独任制（≤3年可独任） | **独任审判** |
| **审理特点** | 完整法庭调查辩论 | 简化部分程序 | **一般不进行法庭调查辩论** |
| **审限** | 2-6个月（可延长） | **20日**（>3年可1.5月） | **10日**（>1年可15日） |
| **判决方式** | 定期/当庭 | — | **当庭宣判** |
| **不适用** | — | 盲聋哑/精神病/重大影响/共同犯罪部分不认罪/其他 | 盲聋哑/未成年/重大影响/共同犯罪部分不认罪/未达成和解/其他 |

### 5.2 庭审核心环节

```
开庭流程：
┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│ 开庭准备 │ → │ 法庭调查 │ → │ 法庭辩论 │ → │最后陈述  │ → │宣判     │
│ 查明身份  │   │ 出示质证  │   │ 控辩交锋  │   │被告人发言│   │当庭/定期│
│ 告知权利  │   │ 询问证人  │   │ 互相辩论  │   │(不可剥夺)│   │         │
└──────────┘   └──────────┘   └──────────┘   └──────────┘   └──────────┘
```

### 5.3 三种判决结果（第200条）

| 结果 | 条件 | 性质 |
|------|------|------|
| **有罪判决** | 事实清楚，证据确实充分，依法认定有罪 | 有罪 |
| **无罪判决** | 依据法律认定无罪 | 无罪 |
| **证据不足的无罪判决** | 证据不足，不能认定有罪 | **疑罪从无** |

> **"疑罪从无"** 是刑事司法的核心原则。证据不足就必须判无罪——宁可放过，不可冤枉。

### 5.4 审限规定（第208条）

| 案件类型 | 基础审限 | 可延长 |
|----------|----------|--------|
| 一般案件 | **2个月**，至迟**3个月** | 经上一级法院批准可**延长3个月** |
| 死刑案件/附带民诉 | 同上 | 再延长3个月后仍需延长 → 报**最高院批准** |
| 自诉案件（未羁押） | — | 受理后**6个月**以内 |
| 改变管辖 | — | 从改变后的法院收到案件重新计算 |
| 补充侦查 | — | 补侦完毕移送后重新计算 |

## 六、二审程序（第227-244条）

### 6.1 上诉 vs 抗诉

| 项目 | 上诉 | 抗诉 |
|------|------|------|
| 主体 | 被告人/自诉人/法定代理人/辩护人（经同意）/近亲属（经同意） | 地方各级人民检察院 |
| 对象 | 一审判决/裁定中的刑事部分（附民诉当事人可对民事部分上诉） | 本级法院一审判决/裁定 |
| 期限 | 判决**10日**，裁定**5日**（从接到次日起算） | 同左 |
| 效果 | 上诉**不加刑**（237条） | 抗诉**可以加刑** |

### 6.2 上诉不加刑原则（第237条）⭐

> **铁律**：只有被告一方上诉的，二审**不得加重刑罚**。

**具体含义**：
- 二审不能改判更重的刑种（如拘役改有期徒刑）
- 二审不能增加刑期
- 二审不得加重附加刑
- 发回重审时，原审法院也不得加重刑罚（除非有新犯罪事实+检察院补诉）

**例外**：
- 检察院提出抗诉 or 自诉人提出上诉的 → 不受限制
- 发回重审后有**新的犯罪事实** + 人民检察院**补充起诉**的 → 可以加重

### 6.3 二审处理方式（第236条）

| 处理方式 | 适用条件 |
|----------|----------|
| **驳回上诉/抗诉，维持原判** | 原判认定事实和适用法律正确、量刑适当 |
| **改判** | 适用法律错误 或 量刑不当 |
| **发回重审** | 事实不清/证据不足（可查清后改判也可发回） |
| **撤销原判，发回重审** | 违反法定程序（公开审判/回避/剥夺权利/审判组织不合法等） |

**重要限制**：发回重审只能**一次**！原审重审判决后被告再上诉/抗诉的，二审**不得再次发回**。

### 6.4 二审应当开庭审理的情形（第234条）

1. 被告人/自诉人/法定代理人对第一审认定的事实、证据有异议，可能影响定罪量刑的上诉案件
2. 被告人被判处**死刑**的上诉案件
3. **人民检察院抗诉**的案件
4. 其他应当开庭审理的案件

不开庭审理的 → 必须讯问被告人，听取各方意见。

## 七、死刑复核程序（第246-251条）

### 7.1 死刑核准权归属

| 刑种 | 核准机关 |
|------|----------|
| **死刑立即执行** | **最高人民法院**（唯一） |
| **死刑缓期二年执行** | **高级人民法院** |

### 7.2 死刑复核流程图

```
中院判处死立执
    ↓
高院复核（不同意→提审/发回重审）
    ↓ 同意
最高院核准
    ↓
院长签发执行命令
    ↓ 7日内执行
    ↓ 发现以下情形→停止执行：
    ├ 判决可能有错误
    ├ 揭发重大立功
    └ 罪犯正在怀孕
```

### 7.3 死刑执行（第262-263条）

- 下级法院接到命令后**7日以内**交付执行
- **停止执行的法定情形**（3种）：①判决可能有错误 ②揭发重大立功可能改判 ③罪犯正在怀孕
- 执行方法：**枪决**或**注射**
- **不得示众**

## 八、审判监督程序（再审）（第252-258条）

### 8.1 申诉引发再审的条件（第253条）

满足以下任一条件，法院**应当**重新审判：

1. **有新的证据**证明原判决认定的事实确有错误，可能影响定罪量刑
2. 据以定罪量刑的证据**不确实、不充分**，依法应当**排除**，或主要证据之间存在**矛盾**
3. 原判决、裁定**适用法律确有错误**
4. **违反法律规定的诉讼程序**，可能影响公正审判
5. 审判人员在审理该案件时有**贪污受贿、徇私舞弊、枉法裁判行为**

### 8.2 再审提起主体

| 主体 | 对象 | 方式 |
|------|------|------|
| 各级人民法院院长 | 本院生效裁判 | 提交审判委员会处理 |
| 最高人民法院 | 各级法院生效裁判 | 提审/指令下级再审 |
| 上级人民法院 | 下级法院生效裁判 | 提审/指令下级法院再审 |
| 最高人民检察院 | 各级法院生效裁判 | 按审判监督程序向同级法院抗诉 |
| 上级人民检察院 | 下级法院生效裁判 | 同上 |
| 当事人及近亲属 | 生效裁判 | 申诉（**不能停止执行**！）|

### 8.3 再审审理原则

- 原是一审案件 → 按**一审程序**（判决可上诉/抗诉）
- 原是二审案件/上级提审 → 按**二审程序**（终审判决）
- **另行组成合议庭**（原审人员回避）
- 再审一般**不得加重刑罚**（抗诉除外）
- 审限：**3个月**内审结，需延长**不得超过6个月**

## 九、执行程序（第259-276条）

### 9.1 各类刑罚的执行机关

| 刑种 | 执行机关 |
|------|----------|
| 死刑立即执行 | 人民法院（下级法院交付） |
| 死刑缓期/无期/有期徒刑 | **监狱**（剩余<3月的由看守所代执行） |
| 拘役 | **公安机关** |
| 管制/缓刑/假释/暂予监外执行 | **社区矫正机构** |
| 剥夺政治权利 | **公安机关** |
| 罚金/没收财产 | **人民法院** |

### 9.2 减刑与假释（第273条）

| 项目 | 减刑 | 假释 |
|------|------|------|
| 条件 | 确有悔改/立功表现 | 已执行一定刑期+假释后不致再危害社会 |
| 启动 | 执行机关提出建议书 | 执行机关提出建议书 |
| 审核 | 人民法院审核裁定 | 人民法院审核裁定 |
| 监督 | 检察院可在20日内提纠正意见 | 同左 |

### 9.3 暂予监外执行（第265-268条）

**适用条件**（符合其一）：
1. 有严重疾病需要保外就医
2. 怀孕或者正在哺乳自己婴儿的妇女
3. 生活不能自理，适用监外执行不致危害社会

**限制**：
- 无期徒刑只允许怀孕/哺乳妇女适用保外就医
- 有社会危险性 or 自伤自残的 → **不得保外就医**
- 保外就医由**省级人民政府指定的医院**诊断并开具证明

**终止收监的情形**：
1. 不符合条件
2. 严重违反监督管理规定
3. 监外执行情形消失但刑期未满

> ⚠️ 通过贿赂等非法手段获得监外执行的，**监外执行期间不计入刑期**！

## 十、特别程序（第五编，第277-307条）

### 10.1 五大特别程序总览

| 程序 | 适用对象 | 核心目的 | 关键条文 |
|------|----------|----------|----------|
| **未成年人程序** | <18周岁（犯罪时） | 教育、感化、挽救为主 | 第277-287条 |
| **当事人和解** | 特定公诉案件 | 达成谅解从宽处理 | 第288-290条 |
| **缺席审判** | 在境外逃匿/严重疾病/死亡 | 不让嫌疑人逃避审判 | 第291-297条 |
| **没收违法所得** | 逃匿≥1年或死亡的重大犯罪嫌疑人 | 追缴赃款赃物 | 第298-301条 |
| **强制医疗** | 不负刑责的精神病人 | 保护社会+治疗病人 | 第302-307条 |

### 10.2 当事人和解程序（第288-290条）

**适用范围**：
1. 因**民间纠纷**引起，涉嫌侵犯人身权利/财产权利犯罪，可能判**≤3年**有期徒刑
2. 除渎职外的**过失犯罪**，可能判**≤7年**有期徒刑

**排除条件**：5年内曾故意犯罪的，不适用

**法律效果**：
- 公安机关 → 向检察院提出从宽建议
- 检察院 → 提出从宽处罚建议 / 酌定不起诉
- 法院 → 依法从宽处罚

### 10.3 缺席审判（第291-297条）

**适用情形**：

| 类型 | 条件 |
|------|------|
| **贪污贿赂+境外逃匿** | 犯罪事实清楚+证据确实充分+依法追究 |
| **国安/恐怖活动+境外逃匿** | 经**最高人民检察院核准** |
| **严重疾病无法出庭** | 中止超6个月仍无法出庭+本人/法定代理人申请/同意 |
| **被告人死亡但有证据证明无罪** | 缺席审理确认无罪 |

**权利保障**：被告人有权委托辩护人，未委托的法院**应当通知法律援助指派律师**

### 10.4 违法所得没收程序（第298-301条）

**适用条件**：
- 贪污贿赂/恐怖活动等**重大犯罪**
- 犯罪嫌疑人**逃匿且通缉满1年**不能到案 OR **死亡**
- 依法应当追缴违法所得及其他涉案财产

**程序要点**：
- 由**中级人民法院**合议庭审理
- 公告期间**6个月**（利害关系人可参加诉讼）
- 利害关系人参加的 → **开庭审理**
- 对裁定可上诉/抗诉

### 10.5 强制医疗程序（第302-307条）

**适用条件（三者缺一不可）**：
1. 实施了**暴力行为**，危害公共安全或严重危害公民人身安全
2. 经法定程序鉴定**依法不负刑事责任的精神病人**
3. **有继续危害社会可能**

**程序**：
- 公安机关写意见书 → 检察院向法院申请 → 法院合议庭审理（**1个月内**决定）
- 被申请人无委托诉讼代理人的 → **法院通知法律援助指派律师**
- 不服决定 → 向**上一级法院申请复议**
- 解除：定期诊断评估 → 不需继续的 → 报法院批准解除

## 十一、常见犯罪辩护要点速查

### 11.1 常见高频犯罪

#### 盗窃罪（刑法第264条）

| 要素 | 内容 |
|------|------|
| 行为模式 | 秘密窃取公私财物 |
| 数额标准 | 一般1000-3000元以上入刑（各地不同）；多次盗窃/入户盗窃/携带凶器/扒窃不论数额 |
| 量刑幅度 | 3年以下/3-10年/10年以上至无期（数额巨大/特别巨大） |
| 辩护方向 | 数额认定是否准确、是否有从轻情节、是否构成其他盗窃类型 |

#### 诈骗罪（刑法第266条）

| 要素 | 内容 |
|------|------|
| 核心要件 | **虚构事实/隐瞒真相** → 被害人**陷入错误认识** → **自愿交付财物** |
| 数额标准 | 一般3000-10000元以上入刑（各地不同） |
| 量刑幅度 | 3年以下/3-10年/10年以上（数额特别巨大或其他特别严重情节） |
| 与民间借贷区分 | 是否有**非法占有目的**（借的时候就不打算还） |
| 辩护方向 | 是否有非法占有目的、数额计算是否重复、被害人有无过错 |

#### 故意伤害罪（刑法第234条）

| 要素 | 内容 |
|------|------|
| 分级 | 轻伤（3年以下）、重伤（3-10年）、致人死亡/特别残忍手段致重伤残疾（10年以上至死刑） |
| 鉴定标准 | 必须有**法医学鉴定意见**（轻伤一级/二级、重伤一级/二级） |
| 辩护方向 | 伤害程度鉴定是否准确、谁先动手、是否有防卫性质、是否达成赔偿谅解 |

#### 危险驾驶罪（刑法第133条之一）

| 要素 | 内容 |
|------|------|
| 行为类型 | ①醉酒驾驶 ②追逐竞驶情节恶劣 ③客运超载/超速 ④违规运输危险化学品 |
| 醉酒标准 | 血液酒精含量**≥80mg/100ml** |
| 量刑 | **拘役（1-6个月）+ 罚金** |
| 从重情节 | 无证驾驶/造成事故/逃避检查/在城市快速路驾驶等 |
| 辩护方向 | 酒精含量刚过线、行驶距离短、未造成后果、初犯偶犯、认罪认罚 |

#### 交通肇事罪（刑法第133条）

| 要素 | 内容 |
|------|------|
| 构成标准 | ①死亡1人以上或重伤3人以上+负主要/全部责任 ②死亡3人以上+负同等责任 ③造成公私财产直接损失60万以上+无力赔偿+主要/全部责任 等 |
| 逃逸加重 | 交通肇事后逃逸 → 3-7年；因逃逸致人死亡 → **7年以上** |
| 与危险驾驶区分 | 过失犯罪（非故意）vs 故意犯罪（明知醉酒仍驾驶） |

### 11.2 刑民交叉典型场景

| 场景 | 先刑后民？还是刑民并行？ |
|------|--------------------------|
| 借钱不还（合同诈骗嫌疑） | **牵连型**→可刑民并行：刑事追责+民事诉讼 |
| 打架受伤（治安→刑事转化） | 先走治安处罚，鉴定达轻伤级别→转为刑事 |
| 公司拖欠工资 | **劳动仲裁前置**→劳动争议→不涉及刑事除非拒不支付劳动报酬罪 |
| 离婚中发现对方转移财产隐匿犯罪所得 | 可同时进行：离婚诉讼+报案 |
| 购买到伪劣产品 | 民事索赔（消法555）+ 行政投诉（市监局）+ 刑事举报（生产销售伪劣产品罪） |

## 十二、刑事被害人权益保障

### 12.1 附带民事诉讼（第101-104条）

| 项目 | 内容 |
|------|------|
| 提起条件 | 被告人的犯罪行为导致**物质损失** |
| 提起期间 | 刑事诉讼过程中（一般在一审宣判前） |

| 审理原则 | **原则上与刑事案件一并审判**（防止过分迟延才分开） |

对于因案致贫的被害人，可向办案机关申请司法救助金，帮助解决紧急生活困难。

## 十三、刑事辩护实战策略框架

### 13.1 辩护思路四步法

```
Step 1: 程序辩护（程序是否合法？）
    ↓
Step 2: 证据辩护（证据是否充分？）
    ↓
Step 3: 定性辩护（此罪还是彼罪？）
    ↓
Step 4: 量刑辩护（如何从轻减轻？）
```

### 13.2 常见从轻减轻情节清单

| 类型 | 具体情节 | 法律效果 |
|------|----------|----------|
| **法定从轻** | 未成年/已满75周岁/尚未完全丧失辨认能力的精神病人/又聋又哑的人/盲人 | 应当从轻或减轻 |
| **自首** | 自动投案+如实供述 | 可以从轻或减轻；较轻的可免除 |
| **坦白** | 如实供述自己罪行（被动归案） | 可以从轻；如实供述避免特别严重后果的，可减轻 |
| **立功** | 提供线索/协助抓捕/重大立功 | 可以从轻或减轻/减轻或免除 |
| **认罪认罚** | 认罪+认罚+签具结书 | 一般应当从宽处理 |
| **退赔退赃** | 退还赃款/赔偿损失 | 可酌情从轻 |
| **取得谅解** | 获得被害人谅解（出具谅解书） | 可酌情从轻（尤其影响是否适用缓刑） |
| **初犯偶犯** | 第一次犯罪 | 酌情从轻 |
| **预备未遂中止** | 未完成犯罪/自动放弃 | 可以比照既遂犯从轻/减轻/免除 |
| **从犯** | 起辅助作用 | 应当从轻/减轻/免除 |
| **正当防卫/紧急避险** | 排除犯罪性 | 不负刑事责任 |

### 13.3 不同阶段的辩护重点

| 阶段 | 核心任务 | 关键动作 |
|------|----------|----------|
| **侦查阶段（黄金37天）** | 争取不批捕/取保候审 | 会见嫌疑人→了解案情→提交不予批捕法律意见→申请取保候审 |
| **审查起诉阶段** | 争取不起诉/有利量刑建议 | 全面阅卷→提出辩护意见→协商认罪认罚→参与量刑协商 |
| **一审阶段** | 争取无罪/罪轻/从轻判决 | 庭前会议→排除非法证据→法庭质证→发表辩护词→最后陈述 |
| **二审阶段** | 争取改判/发回重审 | 提交上诉状→全面阅卷→寻找一审程序/实体问题→庭审辩论 |

## 十五、与已有模块的联动关系

### 15.1 与刑法模块（已有）的关系

### 15.2 与证据规则模块（已有）的关系

- 证据规则模块提供通用理论（证明标准/举证责任/非法证据排除原理）

  - 刑事举证责任：**控方举证**（无罪推定）
  - 刑事证明标准：**排除合理怀疑**（高于民事的高度盖然性）

- 法域判断负责识别一个问题是刑事/民事/行政/劳动

- 特别注意**刑民交叉**案件的分流策略

### 15.4 与劳动者权益保护/公司经营实务的关系

- 劳动者权益保护中的**拒不支付劳动报酬罪**（刑法第276条之一）

## 十六、常见咨询场景速答模板

### 场景1："我被警察带走了，家属该怎么办？"

**L2定性**：这是【刑事强制措施】问题，适用《刑事诉讼法》关于拘留/逮捕的规定。

**第一步：确认状态**
- 问清楚是被传唤（≤24小时放回）、被拘留（进看守所）、还是被逮捕
- 问清楚涉嫌什么罪名、哪个派出所/公安局、承办民警是谁

**第二步：黄金时间内行动**
- **37天内是关键窗口期**：拘留后公安有3-30天侦查时间+检察院7天批捕时间=最长37天
- 尽快聘请**律师**：律师可以在37天内会见、了解案情、申请取保候审、提出不批捕意见
- 家属能做的：准备好身份证关系证明、准备保证金/找保证人

**第三步：配合但不盲目签字**
- 配合调查 ≠ 认罪
- 有权保持沉默（对与本案无关的问题）
- 签任何文件之前先问律师

### 场景2："朋友涉嫌XX罪，大概会判多久？"

**L1简单查询**：根据刑法分则对应条款给出法定刑幅度。

**但要强调**：最终判决取决于：
1. 具体金额/情节/后果（数额犯看金额，结果犯看后果）
2. 是否有自首/坦白/立功/退赃/谅解等从轻情节
3. 是否认罪认罚
4. 当地法院的量刑习惯
5. 辩护律师的专业能力

**回答格式**：先给法定刑区间 → 分析从重从轻因素 → 给出一个**合理的预估区间**（不要给确定数字）

### 场景3："能不能申请取保候审？"

**取保候审条件**（第67条，满足其一即可）：
1. 可能判管制/拘役/独立附加刑
2. 可能判徒刑以上但**取保不致发生社会危险性**
3. 患重病/生活不能自理/怀孕哺乳（取保不致发生危险性）
4. 羁押期限届满案件未结

**申请时机**：越早越好！**批捕前成功率最高**，批捕后难度加大但仍可申请。

**申请书核心内容**：
- 涉嫌罪名和事实简述
- 符合取保条件的理由
- 保证人信息或保证金额度
- 承诺遵守取保规定
- 联系方式
