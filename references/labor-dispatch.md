# 劳务派遣专项知识模块

> 版本：v1.0.0 | 更新日期：2026-04-08
> 法律依据：《劳动合同法》第五章第二节（第57-67条）、《劳务派遣暂行规定》（人社部令第22号，2014年）

## 一、劳务派遣三方关系

### 1.1 基本架构

```
┌─────────────┐        劳动合同        ┌─────────────┐
│  劳动者     │◄─────────────────────►│  派遣单位    │
│  (被派遣人) │                        │  (用人单位)  │
└──────┬──────┘                        └──────┬──────┘
       │                                      │
       │ 实际用工关系                          │ 劳务派遣协议
       │ (指挥、管理、报酬支付)                 │
       │                                      │
       ▼                                      ▼
┌─────────────────────────────────────────────────┐
│                    用工单位                       │
│         (实际使用被派遣劳动者的单位)              │
└─────────────────────────────────────────────────┘
```

### 1.2 三方主体的法律地位

| 主体 | 法律地位 | 主要职责 |
|------|----------|----------|
| **派遣单位** | 用人单位 | 签订劳动合同、支付工资、缴纳社保、提供培训 |
| **用工单位** | 实际管理方 | 提供劳动条件、执行劳动保护、告知规章制度 |
| **劳动者** | 被派遣员工 | 遵守双方规章制度、服从工作安排 |

## 二、"三性"岗位限制

### 2.1 法定"三性"要求（《劳动合同法》第66条）

劳务派遣用工是补充形式，**只能**在下列岗位实施：

| 特性 | 法定定义 | 实务判断标准 |
|------|----------|--------------|
| **临时性** | 存续时间**不超过6个月** | 从岗位设立到消失≤6个月，非固定/常设岗位 |
| **辅助性** | 为主营业务岗位**提供服务** | 经职代会/全体职工讨论确定并公示的辅助岗位 |
| **替代性** | 原员工因**脱产学习、休假**等暂时无法工作 | 有确定的被替代员工和替代期限 |

### 2.2 实务要点

1. **"三性"是并列关系**，只需满足其一即可
2. **违反"三性"限制的后果**：
   - 不直接导致劳动关系被认定（有争议）
   - 但可作为"假派遣真用工"的重要证据
   - 用工单位面临行政处罚（每人5000-10000元罚款）

## 三、派遣单位（用人单位）的法定义务

### 3.1 设立条件（第57条）

- 注册资本：**≥200万元**
- 资质要求：取得劳务派遣**行政许可**
- 经营范围：明确包含"劳务派遣"

### 3.2 核心义务

| 义务 | 法律依据 | 要点 |
|------|----------|------|
| **签订书面劳动合同** | 第58条 | 必须2年以上固定期限合同 |
| **告知派遣信息** | 第58条 | 告知被派遣劳动者的工作内容、地点、岗位、期限 |
| **无工作期间工资** | 第58条 | 按当地最低工资标准支付 |
| **同工同酬** | 第63条 | 与用工单位同类岗位劳动者相同的劳动报酬分配办法 |
| **跨地区派遣** | 第61条 | 按用工单位所在地标准执行劳动报酬和劳动条件 |

### 3.3 禁止行为

- ❌ 克扣被派遣劳动者的劳动报酬
- ❌ 向被派遣劳动者收取费用
- ❌ 再派遣（不得将被派遣劳动者再派遣到其他用人单位）

## 四、用工单位的法定义务

### 4.1 核心义务（第62条）

| 义务 | 具体内容 |
|------|----------|
| **劳动条件保障** | 执行国家劳动标准，提供相应劳动条件和劳动保护 |
| **工作要求告知** | 告知工作要求和劳动报酬 |
| **加班费支付** | 支付加班费、绩效奖金 |
| **福利待遇** | 提供与工作岗位相关的福利待遇 |
| **培训义务** | 提供必要的岗位培训 |
| **工资调整** | 连续用工的，实行正常的工资调整机制 |

### 4.2 用工比例限制（《劳务派遣暂行规定》第4条）

> **派遣用工比例 ≤ 用工总量的10%**

- 用工总量 = 订立劳动合同人数 + 派遣员工人数
- 超比例的：2年内（至2016年2月底）逐步降至10%以下

### 4.3 禁止行为

- ❌ 将被派遣劳动者再派遣到其他用人单位
- ❌ 设立劳务派遣单位向本单位或所属单位派遣劳动者（关联派遣禁止）
- ❌ 以非全日制用工形式招用被派遣劳动者

## 五、劳动合同解除与退回机制

### 5.1 劳动者主动解除

与普通劳动合同相同：
- 提前30日书面通知（试用期提前3日）
- 用人单位有过错的，随时解除（第38条）

### 5.2 派遣单位解除

**严格限制**，仅限以下情形：

| 解除情形 | 法律依据 | 条件 |
|----------|----------|------|
| 过失性解除 | 第39条 | 严重违纪、严重失职、被追究刑责等 |
| 医疗期满解除 | 第40条第1项 | 不能从事原工作也不能从事另行安排的工作 |
| 不胜任解除 | 第40条第2项 | 培训或调整岗位后仍不胜任 |

⚠️ **特别限制**：《劳动合同法》第40条第3项（客观情况重大变化）和第41条（经济性裁员）**不适用于**被派遣劳动者。

### 5.3 用工单位退回机制（第65条）

#### 可以退回的情形：

1. **法定退回**（第39条、第40条第1、2项情形）
2. **协议退回**（劳务派遣协议约定的退回情形）
3. **派遣协议期满退回**

#### 退回后的处理：

```
用工单位退回
     │
     ▼
派遣单位是否有新岗位？
     │
   ┌─┴─┐
   有   无
   │    │
   ▼    ▼
重新派遣  待岗
         │
         ▼
   按最低工资发放
   (派遣单位不得解除)
```

#### ⚠️ 退回≠解除

- 退回是用工单位将劳动者退还给派遣单位
- 退回后，劳动关系仍存续于派遣单位
- 派遣单位不得仅因被退回而解除劳动合同

## 六、"假派遣真用工"的穿透认定

### 6.1 什么是"假派遣"

指形式上采用劳务派遣形式，但实质上：
- 派遣单位不具备派遣资质
- 派遣单位与用工单位存在关联关系
- 实际用工管理完全由用工单位控制
- 劳务派遣仅为规避用工责任的工具

### 6.2 穿透认定的典型情形

| 情形 | 表现特征 | 法律后果 |
|------|----------|----------|
| **无资质派遣** | 派遣单位无劳务派遣许可证 | 认定劳动者与用工单位存在劳动关系 |
| **关联公司派遣** | 派遣单位为用工单位设立或控制 | 可认定为规避行为，穿透 |
| **长期固定岗位派遣** | 严重违反"三性"限制，长期派遣 | 作为认定事实劳动关系的证据 |
| **混同用工** | 工资、社保、管理主体混乱 | 连带责任或认定事实劳动关系 |

### 6.3 穿透认定的证据要点

劳动者主张与用工单位存在事实劳动关系时，可收集：

- [ ] 用工单位直接发放工资的记录
- [ ] 用工单位直接缴纳社保的记录
- [ ] 用工单位盖章的规章制度、工牌、工作证
- [ ] 用工单位管理人员的日常指挥管理记录（微信、钉钉）
- [ ] 派遣单位从未实际管理的证据
- [ ] 派遣单位与用工单位的关联关系证据（股权、人员、财务）
- [ ] 岗位"三性"不符的证据

## 七、派遣用工的责任划分

### 7.1 连带责任（第92条）

> 用工单位给被派遣劳动者造成损害的，劳务派遣单位与用工单位承担**连带赔偿责任**。

### 7.2 责任划分明细

| 责任事项 | 派遣单位 | 用工单位 | 备注 |
|----------|:--------:|:--------:|------|
| 工资支付 | ● | ○ | 派遣单位为第一责任主体 |
| 社保缴纳 | ● | ○ | 派遣单位缴纳，用工单位支付费用 |
| 加班费 | ○ | ● | 用工单位按实际加班支付 |
| 绩效奖金 | ○ | ● | 用工单位根据绩效支付 |
| 劳动保护 | ○ | ● | 用工单位提供劳动条件 |
| 工伤责任 | ● | ● | 连带责任 |
| 违法解除赔偿 | ● | ● | 连带责任 |
| 职业病责任 | ● | ● | 连带责任 |

## 八、劳动者维权路径

### 8.1 确定被申请人/被告

| 纠纷类型 | 建议被申请人 | 理由 |
|----------|--------------|------|
| 工资争议 | 派遣单位 + 用工单位 | 两者可能分别承担部分 |
| 社保争议 | 派遣单位 | 缴纳义务主体 |
| 工伤争议 | 派遣单位 + 用工单位 | 连带责任 |
| 违法解除 | 派遣单位 + 用工单位 | 连带责任 |
| 确认劳动关系 | 用工单位（穿透时）| 主张与用工单位存在事实劳动关系 |

### 8.2 仲裁请求项参考

**常规请求（劳务派遣框架下）**：
1. 裁决派遣单位支付工资差额 xxx 元
2. 裁决派遣单位补缴 x 年 x 月至 x 年 x 月社会保险
3. 裁决派遣单位、用工单位连带支付违法解除赔偿金 xxx 元

**穿透请求（主张事实劳动关系）**：
1. 确认申请人与用工单位自 x 年 x 月 x 日起存在事实劳动关系
2. 裁决用工单位支付未签订书面劳动合同二倍工资差额 xxx 元
3. 裁决用工单位支付违法解除赔偿金 xxx 元

## 九、司法实践要点

### 9.1 最高法指导意见

《第八次全国法院民事商事审判工作会议（民事部分）纪要》（2016年）：

> 劳动者请求确认与用工单位之间存在劳动关系的，一般不予支持。但劳务派遣单位不具有劳务派遣资质，或者劳务派遣协议无效的，可以认定用工单位与劳动者之间存在劳动关系。

### 9.2 地方实践差异提示

⚠️ 各地对"假派遣真用工"的认定标准存在差异：
- 部分地区（如上海）对关联公司派遣穿透相对积极
- 部分地区（如北京）对超比例派遣的处理相对宽松
- 具体案件建议咨询当地劳动仲裁委或律师

## 十、常见问题 FAQ

**Q1: 派遣单位破产或注销了怎么办？**
A: 可以直接向用工单位主张权利，尤其是工伤、违法解除等可主张连带责任的事项。

**Q2: 派遣合同到期，用工单位不续签，有补偿吗？**
A: 如果是派遣协议期满导致退回，派遣单位应重新安排工作或按最低工资支付待岗工资。终止劳动合同的经济补偿由派遣单位承担。

**Q3: 我实际在用工单位工作超过2年，能认定与用工单位有劳动关系吗？**
A: 单纯的工作年限不能直接穿透。但如果同时存在"假派遣"的其他证据（无资质、关联公司、混同用工等），可以综合主张。

**Q4: 用工单位以"不胜任"为由退回，派遣单位就可以解除合同吗？**
A: 不可以。派遣单位应先培训或调整岗位，仍不胜任的才能解除。直接解除属于违法解除。
