> 核心法律：行政处罚法（2021修订）、行政复议法（2023修订）、行政诉讼法（2017修正）、行政强制法（2011年）、国家赔偿法（2012修正）

## 一、行政法基础理论

### 1.1 行政行为的分类

| 分类标准 | 类型 | 典型例子 |
|----------|------|----------|
| 按对象 | 抽象行政行为 | 政府发布规范性文件（红头文件） |
| | 具体行政行为 | 对你罚款500元 |
| 按内容 | 授益行政行为 | 颁发许可证、发放补贴 |
| | 负担行政行为 | 行政处罚、行政强制 |
| 按约束程度 | 羁束行政行为 | 符合条件**必须**批准（如低保） |
| | 裁量行政行为 | 罚款200-2000元，具体金额由执法者裁量 |
| 按是否需申请 | 依职权行政行为 | 城管主动查处违建 |
| | 依申请行政行为 | 申请营业执照 |

### 1.2 行政行为的合法性要件（五要件）

一个行政行为合法，必须**同时满足**：

1. **主体合法**：由有法定职权的行政机关作出
2. **权限合法**：不超越职权、不滥用职权
3. **内容合法**：事实清楚、证据确凿、适用法律正确
4. **程序合法**：遵守法定程序（告知、听证、送达等）
5. **形式合法**：书面决定、载明必要事项

> **实务要点**：打行政官司不需要证明行政行为"全部违法"，只要证明**任何一个要件**有问题就可能胜诉。程序违法是最常见也最容易抓住的突破口。

### 1.3 行政法基本原则

| 原则 | 内涵 | 实务价值 |
|------|------|----------|
| **合法行政** | 法无授权不可为 | 政府做事必须有法律依据，没依据就是违法 |
| **合理行政** | 公平、公正、比例原则 | 处罚畸重（偷一根葱罚10万）可以挑战 |
| **程序正当** | 告知、听证、回避 | 没告知权利/没听证就处罚 = 程序违法 |
| **高效便民** | 不得不当迟延 | 行政机关拖着不办也是违法 |
| **诚实守信** | 信赖保护 | 政府批了许可又反悔撤销，要赔偿损失 |
| **权责统一** | 有权必有责 | 违法行政要承担赔偿责任 |

### 1.4 行政主体识别

| 主体类型 | 例子 | 能否被告 |
|----------|------|----------|
| 行政机关 | 市场监管局、公安局、城管局 | ✅ 当然可以 |
| 法律法规授权组织 | 高校（开除学籍）、村委会（征地补偿） | ✅ 以自己名义行使的 |
| 行政机关委托组织 | 受委托的检测机构 | ❌ 告委托的行政机关 |
| 行政机关内设机构 | 某局某处/科 | ❌ 告该行政机关 |
| 派出机构 | 公安派出所、税务所 | ⚠️ 有法律授权时可以 |

> **实务要点**：告错被告 = 白折腾。派出所作出拘留决定 → 告公安局；派出所作出500元以下罚款/警告 → 告派出所（《治安管理处罚法》授权）。

## 二、行政处罚法（2021年修订）

### 2.1 版本信息

- **法律全称**：《中华人民共和国行政处罚法》
- **最新版本**：2021年1月22日修订，2021年7月15日施行
- **条文数**：86条（原64条，新增22条）
- **重大修订亮点**：这是实施25年来的首次全面修订

### 2.2 行政处罚种类（第9条，新增4种）

| 序号 | 种类 | 说明 | 新/旧 |
|------|------|------|--------|
| 1 | 警告、通报批评 | 声誉罚 | 通报批评**新增** |
| 2 | 罚款、没收违法所得、没收非法财物 | 财产罚 | 旧有 |
| 3 | 暂扣/吊销许可证照 | 资格罚 | 旧有 |
| 4 | 限制开展生产经营活动 | 行为罚 | **新增** |
| 5 | 责令关闭 | 行为罚 | **新增** |
| 6 | 限制从业 | 资格罚 | **新增** |
| 7 | 行政拘留 | 人身罚（最严厉） | 旧有 |

> **记忆口诀**：声誉（警告/通报）+ 财产（罚款/没收）+ 资格（吊照/限业）+ 行为（限产/关闭）+ 人身（拘留）

### 2.3 行政处罚的设定权

| 设定主体 | 可设定的处罚种类 | 不可设定 |
|----------|-----------------|----------|
| **法律**（全国人大/常委会） | 全部种类 | — |
| **行政法规**（国务院） | 除限制人身自由外的全部 | ❌ 行政拘留 |
| **地方性法规** | 除限制人身自由、吊销营业执照外 | ❌ 拘留、吊照 |
| **部门规章** | 警告、通报批评、一定数额罚款 | ❌ 其他种类 |
| **地方政府规章** | 警告、通报批评、一定数额罚款 | ❌ 其他种类 |
| **其他规范性文件** | ❌ 不得设定行政处罚 | 全部 |

> **核心铁律**：**限制人身自由（拘留）只有法律可以设定。** 如果公安局依据某个"规定"拘留你，先查那个"规定"是不是法律——不是法律就没有设定权。

### 2.4 处罚程序（三种程序）

#### 简易程序（当场处罚）

- **适用条件**：违法事实确凿 + 有法定依据 + 处罚较轻
- **处罚限额**：
  - 对公民：罚款 **≤200元**（2021年修订提高）
  - 对法人/组织：罚款 **≤3000元**（2021年新增）
  - 警告
- **程序要求**：当场告知 → 听取陈述申辩 → 填写决定书 → 当场交付

#### 普通程序

- **适用**：不适用简易程序的其他案件
- **核心流程**：
  ```
  立案 → 调查取证（不得少于2人）→ 告知（事实、理由、依据、权利）
       → 听取陈述申辩 → 审核 → 作出决定 → 送达
  ```
- **关键时间节点**：
  - 调查终结到作出决定：法律未统一规定，各部门规章一般30-90天
  - 送达决定书：当场/7日内邮寄/公告送达

#### 听证程序（第63-65条，重大修订）

- **应当告知听证权利的情形**（第63条）：
  1. 较大数额罚款
  2. 没收较大数额违法所得、没收较大价值非法财物
  3. 降低资质等级、吊销许可证件
  4. 责令停产停业、责令关闭、限制从业
  5. **其他较重的行政处罚**（2021年新增兜底条款）
  6. 法律、法规、规章规定的其他情形

- **听证程序要点**：
  - 当事人3日内提出申请
  - 行政机关7日前通知
  - 听证公开举行（涉密除外）
  - 听证由非本案调查人员主持
  - 当事人可委托代理人参加

> **实务攻防**：如果应当听证而未听证，处罚决定**程序违法**，在行政复议/诉讼中可以撤销。这是最有力的程序性武器。

### 2.5 处罚时效（第36条，重大修订）

| 时效 | 适用范围 | 起算 |
|------|----------|------|
| **2年** | 一般违法行为 | 违法行为发生之日起 |
| **5年** | 涉及公民生命健康安全、金融安全且有危害后果的 | 违法行为发生之日起 |
| 继续/连续状态 | 从行为终了之日起算 | 行为终了之日 |

> **2021年新增5年时效**：针对食品药品安全、金融诈骗等严重违法行为。

### 2.6 行政处罚的执行

| 规则 | 内容 |
|------|------|
| **罚缴分离** | 除当场收缴外，罚款必须缴到指定银行 |
| **当场收缴** | 不当场收缴事后难以执行的（20元以下/偏远地区） |
| **加处罚款** | 逾期不缴 → 每日按罚款数额的3%加处（第72条） |
| **强制执行** | 催告 → 仍不履行 → 申请法院强制执行/有强制执行权的自行执行 |
| **延期/分期** | 当事人经济困难可申请延期或分期缴纳 |

### 2.7 "首违不罚"制度（第33条，2021年新增）

**条件**（全部满足）：
1. 初次违法
2. 危害后果轻微
3. 及时改正

→ **可以不予行政处罚**

> **实务价值**：这是2021年修订最大的亮点之一。遇到首次被罚，先看是否符合"首违不罚"——特别是市场监管、税务、交通领域已大量适用。

### 2.8 行政处罚实务攻防

| 攻击点（当事人视角） | 具体操作 |
|---------------------|----------|
| **主体违法** | 查执法机关有无管辖权、执法人员有无执法证件 |
| **程序违法** | 查是否告知权利（陈述申辩/听证）、是否2人执法、是否出示证件 |
| **证据不足** | 查证据链是否完整、取证程序是否合法 |
| **适用法律错误** | 查处罚依据是否正确、处罚种类是否法定 |
| **处罚过重** | 比例原则——处罚与违法行为的性质/情节/社会危害程度相当 |
| **超过时效** | 查违法行为发生时间，是否超过2年/5年追责时效 |
| **首违不罚** | 初次+轻微+已改正 → 不应处罚 |

## 三、行政复议法（2023年修订）

### 3.1 版本信息

- **法律全称**：《中华人民共和国行政复议法》
- **最新版本**：2023年9月1日修订，**2024年1月1日施行**
- **条文数**：90条（原43条，翻倍式重写）
- **重大修订**：这是实施24年来首次全面修订，变化极大

### 3.2 2023年修订五大核心变化

#### 变化一：复议体制改革（县级以上政府统一管辖）

**旧制度**：上级主管部门 OR 本级政府（选择管辖）
**新制度**：**县级以上地方人民政府统一行使复议职责**

| 被申请人 | 复议机关 | 说明 |
|----------|----------|------|
| 县级政府部门 | 本级县政府 | 统一管辖 |
| 市级政府部门 | 本级市政府 | 统一管辖 |
| 省级政府部门 | 本省政府 | 统一管辖 |
| 县级政府 | 上一级市政府 | 政府告政府 |
| 国务院部门 | 国务院（最终裁决） | 特殊 |

> **实务影响**：以前"告工商局是找市监局还是市政府"的困惑没了——统一找本级政府。

#### 变化二：受案范围大幅扩大（第11条）

2023年修订新增可申请复议的行政行为：

| 新增范围 | 说明 |
|----------|------|
| 行政协议（签订/履行/变更/解除） | 以前不能复议 |
| 政府信息公开 | 以前不能复议 |
| 行政赔偿 | 以前不能复议 |
| 行政不作为 | 明确可以复议 |
| 不予受理信访事项 | 部分可以复议 |

#### 变化三：复议前置范围扩大（第23条）

以下情形**必须先复议，再诉讼**（不能直接告法院）：

| 复议前置情形 | 涉及法律 |
|-------------|----------|
| 对当场作出的行政处罚不服 | 行政处罚法 |
| 认为行政机关侵犯经营自主权的 | 多部法律 |
| 认为行政机关侵犯农村土地权益的 | 土地管理法 |
| 认为行政机关侵犯公平竞争权的 | 反垄断法等 |
| **对罚款、警告等4类行政处罚不服**（新增） | 2023年新增 |
| 对行政机关作出的不予许可决定不服（新增） | 2023年新增 |

> **实务要点**：前置情形越来越多 → 很多案件不能直接起诉 → 复议变成"必经之路"。

#### 变化四：审理程序改革

| 改革内容 | 旧制度 | 新制度 |
|----------|--------|--------|
| 审理方式 | 书面审理为主 | **听取意见为原则**（重大案件听证） |
| 调解和解 | 限于赔偿/补偿 | **扩大到行政裁量权范围内** |
| 附带审查 | 可以审查规范性文件 | **增加红头文件审查力度** |
| 审理期限 | 60日 | **60日，复杂延长30日** |
| 简易程序 | 无 | **新增**（小额罚款/警告/信息公开，30日内审结） |

#### 变化五：决定类型丰富

| 决定类型 | 适用情形 |
|----------|----------|
| **维持** | 行政行为合法合理 |
| **撤销/部分撤销** | 事实不清/证据不足/适用法律错误/违反程序/超越职权/明显不当 |
| **变更** | 行政处罚明显不当（可直接改轻） |
| **确认违法** | 不需要撤销/不可撤销但确属违法 |
| **责令履行** | 不履行法定职责/不作为 |

### 3.3 行政复议申请实务

#### 申请条件

1. **申请人**：认为行政行为侵犯其合法权益的公民/法人/组织
2. **被申请人**：作出行政行为的行政机关
3. **有具体的复议请求和事实理由**
4. **在法定期限内**：知道或应当知道行政行为之日起 **60日内**
5. **属于复议范围**

#### 申请材料清单

| 材料 | 说明 |
|------|------|
| 行政复议申请书 | 载明申请人、被申请人、复议请求、事实和理由 |
| 身份证明 | 身份证复印件/营业执照复印件 |
| 被诉行政行为的证据 | 处罚决定书/通知书等 |
| 授权委托书 | 委托代理人时 |
| 证据材料 | 证明行政行为违法的证据 |

#### 复议时限一览

| 环节 | 时限 |
|------|------|
| 申请期限 | 知道行政行为之日起 **60日** |
| 受理审查 | 收到申请之日起 **5日** |
| 被申请人答复 | 收到通知之日起 **10日** |
| 审理期限 | 受理之日起 **60日**（延长+30日） |
| 简易程序 | **30日**内审结 |
| 不服复议决定起诉 | 收到复议决定之日起 **15日** |
| 复议不受理后起诉 | 收到不予受理决定之日起 **15日** |
| 复议期满不决定 | 期满之日起 **15日**内起诉 |

### 3.4 行政复议实务攻防

| 攻防策略（申请人视角） | 具体操作 |
|----------------------|----------|
| **附带审查红头文件** | 认为规范性文件违法的，在对具体行政行为申请复议时一并提出审查请求 |
| **选择复议还是诉讼** | 前置情形必须先复议；非前置的比较：复议免费+60日出结果 vs 诉讼需缴费+6个月 |
| **复议期间不停止执行** | 原则上不停止，但申请人可申请停止执行（3种情形） |
| **调解结案** | 复议中主动提出调解，行政裁量权范围内可以调 |
| **不满复议结果** | 15日内向法院起诉（注意：复议维持→告原行政机关和复议机关） |

## 四、行政诉讼法（2017年修正）

### 4.1 版本信息

- **法律全称**：《中华人民共和国行政诉讼法》
- **最新版本**：2017年6月27日修正
- **条文数**：103条
- **俗称**："民告官"法

### 4.2 受案范围（第12条，正面清单）

以下行政行为可以起诉：

| 序号 | 可诉行政行为 | 典型场景 |
|------|-------------|----------|
| 1 | 行政处罚（罚款/拘留/吊照等） | 城管罚款、交警扣分 |
| 2 | 行政强制措施和执行 | 查封/扣押/冻结/强拆 |
| 3 | 行政许可（不予/撤销/变更） | 不给办营业执照 |
| 4 | 自然资源确权 | 土地/矿产权属纠纷 |
| 5 | 征收征用及补偿 | 拆迁补偿不合理 |
| 6 | 不履行法定职责 | 报警不出警、投诉不处理 |
| 7 | 侵犯经营自主权 | 行政垄断、不正当干预 |
| 8 | 滥用行政权力排除/限制竞争 | 地方保护主义 |
| 9 | 违法集资/摊派/征收财物 | 乱收费乱摊派 |
| 10 | 行政协议（不履行/违约/违法变更解除） | 特许经营协议 |
| 11 | 其他侵犯人身权/财产权的行政行为 | 兜底条款 |

### 4.3 不受理的案件（第13条，反面清单）

| 不予受理 | 说明 |
|----------|------|
| 国防、外交等国家行为 | 主权行为不可诉 |
| 行政法规/规章/规范性文件 | 抽象行政行为不可诉（但可附带审查） |
| 行政机关对工作人员的人事处理 | 内部行为 |
| 法律规定由行政机关最终裁决的 | 如部分行政复议终局决定 |

### 4.4 管辖规则

| 管辖类型 | 规则 | 说明 |
|----------|------|------|
| **一般管辖** | 被告所在地基层法院 | 最常见 |
| **特殊管辖** | 不动产 → 不动产所在地法院 | 拆迁/土地 |
| **经复议管辖** | 复议维持 → 原行政机关所在地法院；复议改变 → 可选原机关或复议机关所在地 | 2014年修改 |
| **共同被告** | 复议维持 → **原行政机关和复议机关是共同被告** | 2014年新增重大变化 |
| **跨区管辖** | 经最高院批准，可设跨行政区划法院管辖 | 防止地方保护 |

> **核心变化（2014年修改）**：复议维持 → 原机关和复议机关当共同被告。目的：倒逼复议机关认真审查，不能"维持了事"。

### 4.5 举证责任（核心！与民事完全不同）

| 规则 | 内容 | 说明 |
|------|------|------|
| **被告举证** | 行政机关对行政行为的合法性承担举证责任 | **行政诉讼最大特色** |
| **举证期限** | 被告应在收到起诉状副本15日内提供证据和规范性文件依据 | 逾期视为无证据 |
| **不得自行取证** | 诉讼过程中被告不得自行向原告/第三人收集证据 | 防止事后补证据 |
| **原告举证** | 起诉不作为 → 原告证明曾经申请过 | 例外情况 |
| **原告举证** | 行政赔偿 → 原告证明损害事实 | 例外情况 |

> **实务价值**：行政诉讼中，**原告只需要说"行政行为违法了"，具体"为什么合法"要行政机关自己证明。** 举证责任倒置是"民告官"最有力的制度武器。

### 4.6 审理规则

| 规则 | 内容 |
|------|------|
| **审查范围** | 审查行政行为的**合法性**（一般不审查合理性，但明显不当可以审） |
| **不适用调解** | 行政诉讼**原则上不适用调解**（例外：行政赔偿/补偿/行政协议可以调解） |
| **不停止执行** | 诉讼期间原则上不停止行政行为执行（但可申请停止） |
| **审理期限** | 一审6个月（延长需高院批准）；二审3个月 |
| **简易程序** | 基层法院审理一审案件，可独任审判，45日内审结 |

### 4.7 判决类型

| 判决类型 | 适用情形 | 效果 |
|----------|----------|------|
| **驳回诉讼请求** | 行政行为合法 | 维持原行政行为 |
| **撤销/部分撤销** | 主要证据不足/适用法律错误/违反程序/超越职权/滥用职权/明显不当 | 撤销+可责令重做 |
| **变更** | 行政处罚明显不当/涉及金额确定的 | 法院直接改判 |
| **确认违法** | 不需要撤销/不可撤销但确属违法 | 为国家赔偿铺路 |
| **确认无效** | 重大且明显违法 | 自始无效 |
| **责令履行** | 不履行/拖延履行法定职责 | 判令行政机关限期履行 |
| **责令给付** | 依法应支付赔偿金/最低生活保障/社保金等 | 判令限期支付 |

### 4.8 行政诉讼时效

| 情形 | 起诉期限 | 起算点 |
|------|----------|--------|
| 直接起诉 | **6个月** | 知道或应当知道行政行为之日 |
| 经过复议 | **15日** | 收到复议决定之日 |
| 复议不受理 | **15日** | 收到不予受理决定之日 |
| 复议逾期不决定 | **15日** | 复议期满之日 |
| 不知道行政行为内容 | 不动产20年/其他5年 | 行政行为作出之日 |
| 不作为 | **6个月** | 催告无果之日起（无法定期限的） |

> **实务要点**：6个月起诉期限是除斥期间，不是诉讼时效，不适用中止中断。**错过就错过了。**

### 4.9 行政诉讼实务攻防

| 攻击策略（原告视角） | 具体操作 |
|---------------------|----------|
| **先复议后起诉** | 利用复议免费+周期短，先试一轮；不行再诉讼 |
| **抓程序违法** | 执法程序是最容易出错的环节：没告知权利/没听证/送达不规范 |
| **举证责任在被告** | 原告不需要证明"行政行为违法"，行政机关需要证明"合法" |
| **附带审查红头文件** | 在诉具体行政行为时，一并请求审查所依据的规范性文件 |
| **申请停止执行** | 不停止执行会造成难以弥补的损害/明显无事实根据的 |
| **共同被告策略** | 复议维持 → 原机关和复议机关当共同被告，增加压力 |

| 防守策略（被告/行政机关视角） | 具体操作 |
|------------------------------|----------|
| **证据充分** | 15日内提供全部证据和法律依据 |
| **程序完整** | 确保执法程序每一步都有记录 |
| **不要事后补证** | 诉讼中不得自行收集新证据 |
| **主动纠错** | 发现确实有错，改变行政行为后原告撤诉 |

## 五、行政强制法（2011年）

### 5.1 版本信息

- **法律全称**：《中华人民共和国行政强制法》
- **版本**：2011年6月30日通过，2012年1月1日施行
- **条文数**：71条

### 5.2 行政强制措施种类（第9条）

| 种类 | 典型例子 | 性质 |
|------|----------|------|
| 限制人身自由 | 传唤、强制约束 | 最严厉 |
| 查封场所/设施/财物 | 查封违建、查封工厂 | 临时性 |
| 扣押财物 | 扣押违法物品 | 临时性 |
| 冻结存款/汇款 | 冻结银行账户 | 临时性 |
| 其他 | 法律规定的其他措施 | — |

### 5.3 行政强制执行方式（第12条）

| 方式 | 说明 |
|------|------|
| 加处罚款或滞纳金 | 逾期不缴罚款，每日3% |
| 划拨存款/汇款 | 直接从银行账户划走 |
| 拍卖/依法处理查封扣押财物 | 变现偿还 |
| 排除妨碍、恢复原状 | 拆除违建 |
| 代履行 | 行政机关代替执行，费用当事人承担 |

### 5.4 强制执行的核心程序

```
催告（书面，10日以上期限）
    ↓ 当事人可陈述申辩
听取意见
    ↓ 
作出强制执行决定
    ↓
送达（执行前）
    ↓
实施强制执行
```

> **实务要点**：没有催告就强制执行 = **程序违法**。催告是强制执行的必经前置程序。

### 5.5 代履行特别规则

| 规则 | 内容 |
|------|------|
| 适用范围 | 排除妨碍、恢复原状等可替代义务 |
| 费用承担 | 当事人承担（标准法定或合理） |
| 紧急代履行 | 道路/航道/环境等紧急情况可立即代履行 |
| 不得适用 | 对人身有强制的义务不得代履行 |

### 5.6 查封扣押规则

| 规则 | 内容 |
|------|------|
| 期限 | **≤30日**（特殊情况延长≤30日，即最长60日） |
| 范围 | 不得查封扣押与违法行为无关的财物 |
| 保管 | 行政机关妥善保管（损坏/灭失要赔偿） |
| 解除 | 超期未处理/违法查封的 → 解除查封 |

## 六、国家赔偿法（2012年修正）

### 6.1 版本信息

- **法律全称**：《中华人民共和国国家赔偿法》
- **最新版本**：2012年10月26日修正
- **条文数**：42条

### 6.2 行政赔偿范围

#### 侵犯人身权（第3条）

| 情形 | 典型场景 |
|------|----------|
| 违法拘留/限制人身自由 | 非法关押 |
| 违法采取强制措施 | 非法扣押 |
| 以殴打、虐待等行为造成身体伤害/死亡 | 暴力执法 |
| 违法使用武器/警械 | 开枪/电击致伤 |
| 其他违法行为 | 兜底 |

#### 侵犯财产权（第4条）

| 情形 | 典型场景 |
|------|----------|
| 违法罚款/吊销许可证/责令停产停业/没收 | 违法行政处罚 |
| 违法查封/扣押/冻结 | 违法行政强制 |
| 违法征收征用 | 违法拆迁 |
| 造成财产损害的其他违法行为 | 兜底 |

### 6.3 赔偿标准

#### 人身损害赔偿

| 赔偿项目 | 标准 |
|----------|------|
| **侵犯人身自由** | 日赔偿金 = 上年度职工日平均工资 |
| | **2025年标准：约 ¥486.79/日**（按2024年全国职工平均工资计算） |
| 造成身体伤害 | 医疗费 + 护理费 + 误工费 + 残疾赔偿金（部分丧劳→最高20年平均工资） |
| 造成死亡 | 死亡赔偿金（20年平均工资）+ 丧葬费 + 被抚养人生活费 |
| 精神损害 | 造成严重后果 → 精神损害抚慰金 |

#### 财产损害赔偿

| 情形 | 赔偿方式 |
|------|----------|
| 罚款/罚金/追缴/没收 | 返还财产 |
| 查封/扣押/冻结 | 解除 + 赔偿损失 |
| 财产已灭失 | 给付相应的赔偿金 |
| 财产已拍卖 | 给付拍卖所得价款；不足的补足差额 |
| 吊销许可/营业执照 | 赔偿停产停业期间必要的经常性费用 |

### 6.4 行政赔偿程序

```
确认行政行为违法（复议/诉讼确认，或行政机关自行确认）
    ↓
向赔偿义务机关提出赔偿申请（2年内）
    ↓
赔偿义务机关2个月内决定
    ↓
不满意/不答复 → 向法院提起赔偿诉讼（3个月内）
```

### 6.5 赔偿时效

| 时效 | 说明 |
|------|------|
| **2年** | 知道或应当知道国家机关及其工作人员行使职权侵犯其合法权益之日起 |
| 起算 | 行政行为被确认违法之日 |
| 申请赔偿 | 2年时效，自确认违法之日起 |

## 七、行政维权全流程实操指南

### 7.1 维权路径选择

```
行政行为侵犯权益
    ↓
┌─────────────────────────────────────────┐
│ 第一步：判断是否必须先复议              │
│   - 查复议前置清单（见 3.2 节）          │
│   - 前置 → 必须先复议                   │
│   - 非前置 → 可选复议或直接起诉          │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│ 第二步：复议 vs 诉讼，怎么选？           │
│                                          │
│   复议优势：                             │
│   ✅ 免费（不收费）                      │
│   ✅ 审理期限短（60日 vs 6个月）         │
│   ✅ 可以审查合理性（不限于合法性）       │
│   ✅ 简易程序30日内出结果                │
│                                          │
│   诉讼优势：                             │
│   ✅ 法院独立于行政体系（更公正）         │
│   ✅ 公开审理、对审制                    │
│   ✅ 判决执行力更强                      │
│   ✅ 可以上诉、申请再审                  │
│                                          │
│   建议策略：                             │
│   一般案件 → 先复议（免费试错）→ 不满意再诉 │
│   重大案件/地方保护严重 → 直接起诉        │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│ 第三步：申请国家赔偿（如有损害）          │
│   - 前提：行政行为已被确认违法            │
│   - 先向赔偿义务机关申请                  │
│   - 不满意 → 赔偿诉讼                   │
└─────────────────────────────────────────┘
```

## 八、常见行政纠纷场景速查

### 8.1 城管执法

| 场景 | 应对 |
|------|------|
| 城管没收物品 | 查是否有执法证件、是否出具扣押清单、是否告知权利 |
| 城管罚款 | 查处罚金额是否合法、是否告知听证权（较大数额） |
| 强拆违建 | 查是否有拆除决定书、是否催告、是否给合理期限 |

### 8.2 交通违法

| 场景 | 应对 |
|------|------|
| 认为交通罚单有误 | 60日内申请行政复议/6个月内起诉 |
| 电子眼抓拍有异议 | 查照片清晰度、测速设备检定证书 |
| 扣分有异议 | 扣分是记分不是行政处罚，不能单独复议/起诉 |

### 8.3 市场监管处罚

| 场景 | 应对 |
|------|------|
| 无照经营被罚 | 查处罚金额是否过重（比例原则）、是否首违不罚 |
| 食品安全处罚 | 注意5年追责时效；罚款起步5万元 |
| 虚假广告处罚 | 查广告内容认定是否正确、处罚幅度是否适当 |

### 8.4 税务处罚

| 场景 | 应对 |
|------|------|
| 认为税务处罚不当 | 必须先复议（税务争议复议前置） |
| 税务稽查 | 配合但保留异议权利，查稽查程序是否合规 |
| 滞纳金争议 | 滞纳金不得超过税款本身（行政强制法第45条） |

### 8.5 公安行政处罚

| 场景 | 应对 |
|------|------|
| 治安拘留 | 查是否告知听证权、是否有证据、是否超裁量基准 |
| 罚款 | 查金额是否在法定幅度内、是否适用减轻/从轻 |
| 强制传唤 | 查是否超过法定时间（一般≤8小时，复杂≤24小时） |

## 九、行政法与其他法域的交叉

### 9.1 行政与刑事交叉

| 交叉类型 | 处理原则 | 典型场景 |
|----------|----------|----------|
| 行政违法可能构成犯罪 | 行政机关应移送公安/检察 | 环境污染：行政处罚→达到标准→移送刑事 |
| 行政处罚与刑事处罚竞合 | 刑事优先，但行政罚款→刑事罚金可折抵 | 生产假药：行政罚款+刑事罚金 |
| 行政强制与刑事强制并存 | 各自独立，互不替代 | 查封厂房（行政）+拘留负责人（刑事） |

### 9.2 行政与民事交叉

| 交叉类型 | 处理原则 | 典型场景 |
|----------|----------|----------|
| 行政行为引发民事损害 | 先行政后民事（确认行政违法→民事赔偿） | 违法强拆 → 确认违法 → 赔偿损失 |
| 行政确认影响民事权利 | 行政确认是民事诉讼的前提 | 不动产登记错误 → 先行政撤登记 → 再民事确权 |
| 行政许可与合同效力 | 许可被撤销 → 可能影响合同效力 | 建设许可撤销 → 施工合同效力争议 |

### 9.3 行政与劳动交叉

| 交叉类型 | 处理原则 | 典型场景 |
|----------|----------|----------|
| 工伤认定（行政确认） | 不服工伤认定 → 行政复议/行政诉讼 | 单位不认可工伤 → 先行政后仲裁 |
| 劳动监察（行政执法） | 劳动监察是行政行为 → 对处理不服走行政路径 | 举报欠薪 → 劳动监察 → 不服处理 → 行政复议 |
| 社保征缴（行政征收） | 社保征缴属行政行为 → 争议走行政路径 | 单位未缴社保 → 向社保部门投诉 → 不处理 → 行政诉讼 |

| **486.79元/日** | 2025年国家赔偿日赔偿金标准 |

### 10.2 对比速记

| 对比项 | 行政复议 | 行政诉讼 |
|--------|----------|----------|
| 审查机关 | 行政机关（上级/本级政府） | 法院 |
| 审查范围 | 合法性 **+ 合理性** | 主要审合法性 |
| 费用 | **免费** | 50元 |
| 期限 | 60日（简易30日） | 6个月（简易45日） |

| 调解 | 可以（裁量权范围内） | 原则不调解（例外：赔偿/协议） |

| 结果不满 | 15日内起诉 | 上诉 |
