## 模板1：民事起诉状

```
                      民事起诉状

原告：[姓名]，[性别]，[民族]，[出生日期]，
身份证号码：[XXXXXXXXXXXXXXXXXX]，
住址：[详细地址]，联系电话：[手机号]。

被告：[姓名/公司全称]，
[个人：性别/民族/出生日期/身份证号/住址/电话]
[公司：统一社会信用代码/住所地/法定代表人/电话]

诉讼请求：
一、判令被告向原告支付[款项名称]人民币[XXX]元；
二、判令被告向原告支付自[起算日]至实际支付之日
    止的利息（按[利率标准]计算）；
三、判令被告承担本案全部诉讼费用。

事实与理由：

    [第一段：法律关系建立]
    [日期]，原告与被告[签订合同/建立关系]，
    约定[核心内容]。

    [第二段：违约/侵权事实]
    [日期]，被告[具体违约/侵权行为]。
    原告多次[催告/协商]，被告[拒绝/未回应]。

    [第三段：法律依据]
    原告认为，被告的上述行为违反了
    《[法律名称]》第[X]条之规定，
    依法应当[承担的责任]。

    综上，为维护原告的合法权益，
    特依据《民事诉讼法》之规定向贵院提起诉讼，
    恳请依法判决。

此致
[XX]人民法院

                              原告：[签名]
                              日期：[XXXX年XX月XX日]

附：
1. 本起诉状副本X份；
2. 证据材料及证据目录X份；
3. 原告身份证复印件1份；
4. [被告企业信息打印件1份]。
```

## 模板2：民事答辩状

```
                      民事答辩状

答辩人（被告）：[基本信息]
被答辩人（原告）：[基本信息]

案  号：[XXXX]X民初字第XXXX号

答辩人就被答辩人XXX诉答辩人[案由]纠纷一案，
现提出如下答辩意见：

一、关于被答辩人第一项诉讼请求

    被答辩人主张[概述对方请求]，答辩人认为
    该请求没有事实和法律依据。
    理由如下：[具体反驳+证据+法条]

二、关于被答辩人第二项诉讼请求

    [同上格式，逐条反驳]

三、关于被答辩人陈述的事实

    被答辩人在起诉状中称[引用对方原文]，
    与事实不符。实际情况是[陈述真实情况]。

综上所述，被答辩人的诉讼请求没有事实和法律依据，
恳请贵院依法驳回被答辩人的全部诉讼请求。

此致
[XX]人民法院

                        答辩人：[签名]
                        日  期：[XXXX年XX月XX日]

附：
1. 证据材料及证据目录X份。
```

## 模板3：民事上诉状

```
                      民事上诉状

上诉人（一审[原告/被告]）：[基本信息]
被上诉人（一审[被告/原告]）：[基本信息]

上诉人不服[XX区/XX市人民法院]于[日期]作出的
（[年]）X民初字第XXXX号民事判决，
现依法提起上诉。

上诉请求：
一、撤销（[年]）X民初字第XXXX号民事判决
    [第X项/全部]；
二、依法改判[具体诉求] / 发回重审；
三、一、二审诉讼费用由被上诉人承担。

上诉理由：

一、一审判决认定事实错误
    一审判决认定"[引用判决书原文]"，
    与客观事实不符。
    [具体分析+证据支持]

二、一审判决适用法律错误
    一审判决适用《[法律]》第[X]条，
    本案应当适用《[法律]》第[X]条。
    [具体分析]

三、[其他上诉理由]

综上，一审判决认定事实不清、适用法律错误，
恳请二审法院依法改判/发回重审。

此致
[XX中级人民法院]

                        上诉人：[签名]
                        日  期：[XXXX年XX月XX日]

附：
1. 一审判决书复印件1份；
2. 新证据材料X份（如有）。
```

## 模板4：强制执行申请书

```
                    强制执行申请书

申请执行人：[基本信息]
被执行人：[基本信息]

执行依据：[XX法院]（[年]）X民初字第XXXX号
          民事判决书/调解书

申请执行事项：
一、依法强制被执行人向申请人支付
    [款项]人民币[XXX]元；
二、依法强制被执行人向申请人支付迟延履行期间
    的债务利息（自[判决确定的履行届满之日]起
    至实际支付之日止）；
三、本案执行费由被执行人承担。

事实与理由：

    [案号]判决书/调解书已于[日期]生效，
    判决/裁定被执行人应于[履行期限]前
    [履行内容]。

    履行期限届满后，被执行人至今未履行
    上述义务。为维护申请人的合法权益，
    特依据《民事诉讼法》第二百四十三条之规定，
    申请贵院强制执行。

被执行人已知财产线索：
1. [银行账户/房产/车辆/股权等]
2. [其他财产线索]

此致
[XX]人民法院

                    申请执行人：[签名]
                    日  期：[XXXX年XX月XX日]

附：
1. 生效法律文书原件1份；
2. 生效证明1份；
3. 申请人身份证复印件1份。
```

## 模板5：财产保全申请书

```
                    财产保全申请书

申请人：[基本信息]
被申请人：[基本信息]

申请事项：
请求贵院依法冻结、查封、扣押被申请人名下
价值不超过人民币[XXX]元的财产。

具体包括：
1. 冻结被申请人在[银行名+支行]账号为
   [XXXXXXXX]的银行存款人民币[XXX]元；
2. 查封被申请人名下位于[地址]的房产
   （产权证号：[XXXXX]）；
3. [其他财产]。

事实与理由：

    申请人与被申请人因[案由]纠纷，
    [已向贵院提起诉讼/拟向贵院提起诉讼]。

    申请人有充分理由认为，被申请人可能
    [转移/隐匿/变卖]其财产，致使将来判决
    难以执行。具体表现为：[说明紧迫性理由]

    申请人愿以[担保方式：保险公司保函/现金/
    房产抵押]提供等额担保。

    恳请贵院依法采取保全措施。

此致
[XX]人民法院

                        申请人：[签名]
                        日  期：[XXXX年XX月XX日]

附：
1. 担保材料[保函/存单/房产证复印件]；
2. 被保全财产线索材料；
3. 申请人身份证复印件。
```

## 模板6：律师函

```
              [律所全称]
                律 师 函

[律所名称]律函字（[年]）第[XXX]号

致：[对方全称/姓名]
地址：[对方地址]

[律所名称]依法接受[委托人全称/姓名]
（以下简称"委托人"）的委托，
就[事项简述]事宜，郑重致函如下：

一、基本事实

    [日期]，委托人与贵方[建立法律关系]。
    [简述争议事实，客观描述]

二、法律分析

    贵方上述行为违反了《[法律]》第[X]条
    [引用法条内容]之规定。
    依据该法第[X]条，贵方应当[法律后果]。

三、律师意见

    鉴于上述事实和法律规定，本律师郑重通知
    贵方：请于收到本函之日起[X]个工作日内
    [具体要求]。

    逾期未履行的，委托人将保留依法采取进一步
    法律措施的权利，届时由此产生的一切法律后果
    均由贵方承担。

    敬请贵方审慎对待。

                        [律所全称]
                        经办律师：[签名]
                        日  期：[XXXX年XX月XX日]
                        联系电话：[律师电话]

附：授权委托书复印件1份
```

## 使用提示

1. 模板中 `[方括号]` 内容需根据实际情况填写
2. 所有金额精确到元，日期精确到日
3. 法条引用格式：《法律名称》第X条第X款第X项
4. 提交法院的文书：正本1份+副本按对方当事人人数
5. 建议同时保留电子版和纸质签名版
