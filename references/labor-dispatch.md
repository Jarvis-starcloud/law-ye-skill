# 劳务派遣专题深化

> law爷 v1.7.0 新增 | 劳务派遣是劳动争议中最复杂的场景之一
>
> 核心法律依据：《劳动合同法》第57-67条、《劳务派遣暂行规定》（人社部令第22号，2014年施行）、《劳动争议司法解释（二）》法释〔2025〕12号

---

## 一、劳务派遣基础架构

### 1.1 三方法律关系

```
┌──────────────┐    劳务派遣协议    ┌──────────────┐
│  派遣单位     │ ←─────────────→  │  用工单位     │
│（用人单位）   │                   │（实际使用方） │
│  签劳动合同   │                   │  日常管理     │
│  发工资       │                   │  工作指令     │
│  缴社保       │                   │  提供条件     │
└──────┬───────┘                   └──────┬───────┘
       │ 劳动关系                         │ 用工关系
       │（法律上的雇主）                   │（事实上的管理者）
       └──────────┬────────────────────────┘
                  ↓
          ┌──────────────┐
          │  劳动者       │
          │（派遣工）     │
          └──────────────┘
```

### 1.2 核心法律关系辨析

| 关系 | 主体 | 性质 | 关键义务 |
|------|------|------|----------|
| **劳动关系** | 派遣单位 ↔ 劳动者 | 劳动合同 | 签合同、发工资、缴社保、解除/终止劳动合同 |
| **用工关系** | 用工单位 ↔ 劳动者 | 管理/服从 | 日常管理、安全保护、同工同酬、提供劳动条件 |
| **商务关系** | 派遣单位 ↔ 用工单位 | 民事合同（劳务派遣协议） | 派遣费用、岗位说明、管理职责划分 |

> **核心铁律**：劳动者与**派遣单位**签劳动合同 ≠ 与**用工单位**有劳动关系。但这不意味着用工单位可以逃避责任——**用工单位和派遣单位对劳动者承担连带赔偿责任**（《劳动合同法》第92条）。

### 1.3 派遣单位资质要求（第57条）

| 要求 | 标准 |
|------|------|
| 注册资本 | **≥200万元** |
| 经营许可 | 须取得**劳务派遣经营许可证** |
| 固定经营场所 | 有 |
| 管理制度 | 有与开展业务相适应的固定的经营场所和设施、有符合法律规定的劳务派遣管理制度 |

> **没有许可证搞派遣 = 违法经营。** 劳动者可以此主张与用工单位直接建立劳动关系。

## 二、劳务派遣的法定限制

### 2.1 "三性"岗位限制（第66条）

劳务派遣**只能在**以下三类岗位使用：

| 性质 | 含义 | 典型岗位 |
|------|------|----------|
| **临时性** | 存续时间 **≤6个月** 的岗位 | 项目制短期用工 |
| **辅助性** | 为主营业务岗位提供服务的**非主营业务**岗位 | 保安、保洁、食堂 |
| **替代性** | 用工单位劳动者因休假等原因无法工作时，由派遣工替代的岗位 | 产假替代 |

### 2.2 派遣用工比例限制（暂行规定第4条）

| 限制 | 标准 |
|------|------|
| 比例上限 | 用工单位使用的被派遣劳动者数量 **≤用工总量的10%** |
| 用工总量 | = 直接用工人数 + 被派遣劳动者人数 |

### 2.3 劳动合同要求

| 要求 | 标准 | 法律依据 |
|------|------|----------|
| 合同类型 | 必须签订 **≥2年固定期限** 劳动合同 | 第58条 |
| 无工作期间 | 派遣单位应按**最低工资标准**按月支付 | 第58条 |
| 同工同酬 | 与用工单位同岗位劳动者**同等报酬** | 第63条 |
| 试用期 | 只有派遣单位可以约定试用期（用工单位不得再约） | 一般规则 |

> **关键保护**：派遣工在无工作期间（被退回后），派遣单位仍然要发最低工资。不是"没活就不给钱"。

## 三、"假派遣"穿透认定

### 3.1 什么是假派遣？

用工单位为规避劳动法义务，将本应直接雇佣的劳动者，通过虚假的劳务派遣安排"洗白"为派遣工，以逃避签无固定期限合同、经济补偿等责任。

### 3.2 假派遣的典型特征

| 特征 | 说明 | 穿透理由 |
|------|------|----------|
| 劳动者一直在用工单位工作，从未去过派遣单位 | 派遣单位只是个"盖章工具" | 实际用人单位 = 用工单位 |
| 派遣前就在该单位工作，被要求改签派遣合同 | "逆向派遣" | 规避连续工龄/无固定期限合同 |
| 派遣单位无实际经营、无办公场所 | 空壳公司 | 不具备派遣资质 |
| 岗位是用工单位的核心/主营业务岗位 | 违反"三性"限制 | 主营岗位不能用派遣 |
| 超过10%比例使用派遣工 | 违反比例限制 | 超比例=违法使用 |
| 用工单位直接发工资、缴社保 | 实际履行雇主义务 | 名为派遣实为直雇 |

### 3.3 穿透后的法律后果

一旦被认定为"假派遣"：

1. **劳动者与用工单位之间成立事实劳动关系**
2. 工龄从实际入职之日起算（不是签派遣合同之日）
3. 连续工作满10年/连续签订两次固定期限合同 → 可主张无固定期限合同
4. 解除时按实际工龄计算经济补偿/赔偿金

> **实务价值巨大**：假派遣穿透是劳动仲裁中的"核武器"。一旦穿透成功，劳动者的保护力度大幅提升。

### 3.4 穿透认定的证据准备

| 证据类型 | 具体内容 | 证明目的 |
|----------|----------|----------|
| 工作证/工牌 | 用工单位印制的工作证 | 证明实际工作场所 |
| 考勤记录 | 用工单位的打卡/签到记录 | 证明受用工单位管理 |
| 工资记录 | 工资条/银行流水（注意打款方是谁） | 如果是用工单位直接发放 = 实际雇主 |
| 社保缴纳记录 | 查缴费单位是谁 | 谁缴社保谁是雇主证据之一 |
| 工作邮件/微信/钉钉 | 用工单位直接下达工作指令 | 管理关系的直接证据 |
| 劳动合同变更记录 | 从直签变派遣的记录 | "逆向派遣"的直接证据 |
| 派遣单位工商信息 | 企查查/天眼查 | 查是否空壳、是否有许可证 |

## 四、退回 ≠ 解除（最重要的概念辨析）

### 4.1 退回 vs 解除

| | 退回 | 解除/终止 |
|--|------|----------|
| **主体** | **用工单位** → 将劳动者退回**派遣单位** | **派遣单位** → 与劳动者解除**劳动合同** |
| **性质** | 用工关系结束（不是劳动关系结束） | 劳动关系终结 |
| **法律关系** | 劳动者仍然是派遣单位的员工 | 劳动者不再是派遣单位的员工 |
| **后果** | 派遣单位应重新安排工作或按最低工资发放 | 需支付经济补偿/赔偿金（如违法解除） |

> **核心铁律**：**用工单位退回劳动者 ≠ 劳动合同解除**。用工单位只能退回，不能"解雇"派遣工——因为它不是雇主。

### 4.2 用工单位可以退回的法定情形（第65条）

| 序号 | 可退回情形 | 说明 |
|------|-----------|------|
| 1 | 劳动者有《劳动合同法》第39条情形 | 严重违纪/严重失职/被追究刑事责任等 |
| 2 | 劳动者有第40条第1、2款情形 | 医疗期满不能从事工作 / 不能胜任经培训调岗仍不行 |
| 3 | 用工单位经营困难等（第41条第1款） | 经济性裁员相关情形 |
| 4 | 派遣协议期满终止 | 正常到期 |

### 4.3 违法退回的认定与后果

| 情形 | 后果 |
|------|------|
| 不属于法定退回情形但强行退回 | 违法退回 |
| 退回后派遣单位违法解除劳动合同 | 违法解除，赔偿2N |
| 用工单位和派遣单位共同实施违法退回+解除 | **连带赔偿责任** |

### 4.4 退回后的法律处理链

```
用工单位退回劳动者
    ↓
派遣单位接收
    ↓
┌─────────────────────────────────┐
│ 选项1：重新安排其他用工单位      │ → 重新上岗
│ 选项2：无法安排新岗位            │ → 按最低工资标准按月支付（第58条）
│ 选项3：协商解除劳动合同          │ → 支付经济补偿（N）
│ 选项4：违法解除                  │ → 赔偿金（2N）
└─────────────────────────────────┘
```

## 五、劳务派遣中的工伤处理

### 5.1 工伤认定的特殊规则

| 规则 | 内容 |
|------|------|
| 工伤认定申请主体 | **派遣单位**（因为派遣单位是法律上的用人单位） |
| 用工单位义务 | 协助工伤认定（提供事故报告、见证材料等） |
| 工伤保险待遇 | 由**工伤保险基金**和**派遣单位**支付 |
| 用工单位连带责任 | 派遣单位不缴工伤保险的 → 用工单位承担连带赔偿责任 |

### 5.2 工伤后的退回限制

| 规则 | 说明 |
|------|------|
| **医疗期内不得退回** | 《劳动合同法》第42条保护期 |
| **工伤停工留薪期不得退回** | 工伤保险条例保护 |
| 鉴定为1-4级 | 保留劳动关系，退出工作岗位 |
| 鉴定为5-6级 | 保留劳动关系，安排适当工作 |
| 鉴定为7-10级 | 合同到期可终止，但需支付一次性就业补助金 |

## 六、劳务派遣争议的仲裁/诉讼要点

### 6.1 被告/被申请人的选择

| 情形 | 应当告谁 | 理由 |
|------|----------|------|
| 欠发工资/未缴社保 | **派遣单位**（可追加用工单位承担连带责任） | 派遣单位是劳动合同主体 |
| 工伤赔偿 | **派遣单位**（可追加用工单位） | 派遣单位是工伤保险参保主体 |
| 违法解除赔偿 | **派遣单位**（可追加用工单位承担连带责任） | 解除劳动合同的主体是派遣单位 |
| 同工不同酬 | **用工单位**（可追加派遣单位） | 同工同酬义务在用工单位 |
| 安全事故 | **用工单位**（可追加派遣单位） | 安全保护义务在用工单位 |
| 假派遣确认劳动关系 | **用工单位** | 穿透后建立劳动关系 |

> **实务策略**：大部分情况下，**两家都告**（派遣单位 + 用工单位），请求承担连带责任。这是最稳的策略。

### 6.2 常见仲裁请求写法

#### 请求1：确认劳动关系（假派遣穿透）

```
请求裁决：确认申请人与被申请人[用工单位名称]自____年__月__日起存在劳动关系。
```

#### 请求2：违法解除赔偿

```
请求裁决：
1. 确认被申请人[派遣单位名称]解除劳动合同的行为违法；
2. 被申请人[派遣单位名称]向申请人支付违法解除劳动合同赔偿金____元
   （计算：月工资____元 × 工龄____年 × 2 = ____元）；
3. 被申请人[用工单位名称]对上述赔偿金承担连带赔偿责任。
```

#### 请求3：工伤待遇

```
请求裁决：
1. 被申请人[派遣单位名称]向申请人支付一次性伤残补助金____元、
   一次性工伤医疗补助金____元、一次性伤残就业补助金____元；
2. 被申请人[用工单位名称]对上述工伤保险待遇承担连带赔偿责任。
```

### 6.3 举证要点

| 待证事实 | 举证方 | 关键证据 |
|----------|--------|----------|
| 劳动关系存在 | 申请人 | 劳动合同、工资记录、社保记录 |
| 假派遣（穿透） | 申请人 | 入职时间证明、工作证、考勤、岗位性质证据 |
| 退回/解除合法性 | 被申请人 | 退回理由、通知文件、培训调岗记录 |
| 工资标准 | 用工单位 | 同岗位劳动者工资记录 |
| 工伤事实 | 申请人+用工单位 | 事故报告、证人证言、就医记录 |

## 七、劳务派遣最新司法解释要点（法释〔2025〕12号）

### 7.1 与劳务派遣相关的新规则

| 规则 | 内容 | 实务影响 |
|------|------|----------|
| **混同用工穿透** | 关联企业交叉使用劳动者，共同承担责任 | 扩大到关联公司间"假派遣" |
| **放弃社保约定无效** | 即使签了"自愿放弃"声明 | 派遣单位不缴社保 = 劳动者可被迫辞职+经济补偿 |
| **竞业限制不知密** | 不知悉商业秘密 → 竞业限制不生效 | 保护基层派遣工 |

### 7.2 连带责任的最新司法实务趋势

| 趋势 | 说明 |
|------|------|
| 连带责任扩大化 | 法院越来越倾向于判用工单位承担连带责任 |
| 穿透认定标准降低 | 越来越多法院采用"实质重于形式"原则 |
| 同工同酬强化 | 差距过大法院会直接调整 |

## 八、劳务派遣实务攻防

### 8.1 劳动者（原告/申请人）攻击策略

| 策略 | 具体操作 |
|------|----------|
| **穿透假派遣** | 收集工作证、考勤、工资发放方、岗位性质等证据，证明"名为派遣实为直雇" |
| **查资质** | 企查查查派遣单位是否有《劳务派遣经营许可证》、是否注册资本达标 |
| **查三性** | 证明岗位是主营业务岗位（非临时/辅助/替代） → 违法使用派遣 |
| **查比例** | 了解用工单位派遣工比例是否超10% |
| **退回≠解除** | 被退回后，主张派遣单位应继续发工资安排工作，否则违法解除 |
| **连带责任** | 两家都告，请求连带赔偿 |
| **同工同酬** | 收集同岗位正式工薪资信息，主张差额 |

### 8.2 用人单位（被告/被申请人）防守策略

| 策略 | 具体操作 |
|------|----------|
| **规范派遣协议** | 派遣协议明确岗位属于辅助性/临时性/替代性 |
| **保留退回证据** | 退回必须有书面理由，符合法定情形 |
| **比例合规** | 控制派遣工比例在10%以内 |
| **独立管理** | 避免用工单位直接发工资、缴社保（防止被穿透） |
| **三性公示** | 辅助性岗位需经职工代表大会讨论确定并公示 |

## 九、劳务派遣 vs 劳务外包 vs 业务外包 区分

| 维度 | 劳务派遣 | 劳务外包 | 业务外包 |
|------|----------|----------|----------|
| 法律关系 | 三方（派遣单位-用工单位-劳动者） | 发包方-承包方（劳动者是承包方员工） | 同劳务外包 |
| 管理权 | **用工单位直接管理劳动者** | **承包方自行管理** | 承包方自行管理 |
| 用工风险 | 连带责任 | 发包方无用工风险 | 发包方无用工风险 |
| 限制 | 三性、10%比例、许可证 | 无特别限制 | 无特别限制 |
| 区分关键 | 谁管人？谁发指令？ | 承包方独立组织生产 | 承包方独立完成业务 |

> **最重要的区分标准**：**谁对劳动者有日常管理权和工作指令权。** 如果发包方/甲方直接管人、排班、考勤、发指令 → 不管合同叫什么名字，本质就是劳务派遣。

### "假外包真派遣"的穿透

越来越多企业把劳务派遣包装成"业务外包"来规避10%比例限制和三性限制。法院的穿透标准：

| 认定为假外包 | 认定为真外包 |
|-------------|-------------|
| 甲方直接管理劳动者的日常工作 | 乙方自主管理劳动者 |
| 甲方直接安排工作内容和排班 | 乙方自主组织生产 |
| 甲方对劳动者进行考核评价 | 乙方对劳动者考核 |
| 按人头结算费用 | 按工作成果/项目结算 |
| 劳动者混编在甲方团队中 | 劳动者独立编组 |
