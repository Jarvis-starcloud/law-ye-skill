# 法律基本概念知识体系

> 贾维斯学习笔记 | 2026-04-02
> 目标：建立法律基础认知框架，理解核心概念之间的关系

---

## 一、法律是什么？

### 1.1 法律的定义

**法律**是由国家制定或认可的、以国家强制力保证实施的、具有普遍约束力的社会行为规范的总和。

拆解几个关键词：
- **国家制定或认可**：法律不是谁说了算就算，必须经过法定程序（立法机关制定，或国家认可已有的习惯/判例）
- **国家强制力保证**：这是法律和道德的根本区别——道德靠舆论和良心，法律靠警察和法院
- **普遍约束力**：不管你是谁，法律面前人人平等
- **行为规范**：规范的是行为，不是思想（想犯罪不犯法，动手了才犯法）

### 1.2 法律与道德的区别

| 维度 | 法律 | 道德 |
|------|------|------|
| 产生方式 | 国家制定或认可 | 社会自发形成 |
| 实施方式 | 国家强制力（法院、警察） | 社会舆论、良心、习惯 |
| 适用范围 | 有管辖权的区域 | 可跨国界、无明确边界 |
| 表现形式 | 成文的法律条文 | 不成文的社会共识 |
| 调整范围 | 重要的社会关系 | 更广泛的日常行为 |
| 确定性 | 明确具体 | 模糊灵活 |

> 💡 一句话：法律是道德的底线。很多行为道德谴责但不违法（如不给老人让座），但违法的行为一定违反道德。

### 1.3 法的本质

法的本质有三个层面（由浅入深）：
1. **形式层**：法是国家意志的体现（写在纸上的规则）
2. **阶级层**：法是统治阶级意志的体现（谁掌权谁定规则）
3. **根本层**：法由物质生活条件所决定（经济基础决定上层建筑——生产力水平、经济关系决定了法律的内容）

## 二、法律的分类

### 2.1 按法的创制方式和表现形式

| 类型 | 说明 | 例子 |
|------|------|------|
| **成文法（制定法）** | 由立法机关按法定程序制定，以文字形式表现 | 《民法典》《刑法》 |
| **不成文法** | 未经立法机关制定，但被国家认可有法律效力 | 判例法、习惯法 |

### 2.2 按法的内容（公法 vs 私法）

这是大陆法系最经典的分类：

| 类型 | 调整对象 | 核心特征 | 例子 |
|------|---------|---------|------|
| **公法** | 国家与公民之间的关系 | 国家权力介入，强调服从 | 宪法、行政法、刑法 |
| **私法** | 公民与公民之间的关系 | 当事人平等自愿，强调意思自治 | 民法、商法 |
| **社会法** | 兼有公法和私法特征 | 国家适度干预，保护弱势群体 | 劳动法、社保法 |

> 💡 举个例子：你和别人签合同是私法领域（民法管）；你偷了别人东西是公法领域（刑法管）；你被公司克扣工资是社会法领域（劳动法管）。

### 2.3 按法律效力范围

| 类型 | 说明 |
|------|------|
| **一般法** | 对一般人、一般事、一般时间、一般地域有效 |
| **特别法** | 对特定人、特定事、特定时间或特定地域有效 |

> 原则：特别法优于一般法。比如《消费者权益保护法》（特别法）与《民法典》（一般法）冲突时，消保法优先。

### 2.4 按法的地位和功能

| 类型 | 说明 |
|------|------|
| **实体法** | 规定权利义务的内容（你有什么权利、该承担什么义务） |
| **程序法** | 规定如何实现权利义务（怎么打官司、怎么调查取证） |

> 实体法如：《民法典》《刑法》；程序法如：《民事诉讼法》《刑事诉讼法》

## 三、世界两大法系

### 3.1 大陆法系（民法法系/成文法系）

- **起源**：罗马法 → 法国《拿破仑法典》
- **核心特征**：以成文法典为主要法律渊源
- **代表国家**：法国、德国、日本、**中国**（混合型，以大陆法系为主）
- **法官角色**：严格依据法条裁判，"法律的嘴巴"
- **法律结构**：明确区分公法与私法
- **特点**：体系化、条理化、逻辑严密

### 3.2 英美法系（普通法系/判例法系）

- **起源**：英国普通法
- **核心特征**：以判例法为主要法律渊源（遵循先例原则）
- **代表国家**：英国、美国、加拿大、澳大利亚、香港地区
- **法官角色**：可以"造法"，创造新的判例
- **法律结构**：不严格区分公法与私法
- **特点**：灵活、实用、经验主义

### 3.3 两大法系对比

| 维度 | 大陆法系 | 英美法系 |
|------|---------|---------|
| 法律渊源 | 成文法为主 | 判例法+成文法 |
| 法官角色 | 适用法律 | 适用+创造法律 |
| 法典编纂 | 重视法典化 | 不追求法典化 |
| 诉讼模式 | 职权主义（法官主导） | 当事人主义（双方对抗） |
| 陪审制度 | 较少使用 | 广泛使用 |
| 公私法区分 | 严格区分 | 不严格区分 |

> 💡 现代趋势：两大法系逐渐融合。大陆法系国家也开始重视判例（如中国最高法的"指导性案例"），英美法系国家也在大量制定成文法。

## 四、中国法律体系

### 4.1 法律的效力层级（金字塔结构）

```
                    ┌─────────┐
                    │  宪 法   │  ← 根本大法，最高效力
                    ├─────────┤
                 ┌──┤  法 律   ├──┐  ← 全国人大及其常委会制定
                 │  ├─────────┤  │
              ┌──┤  │行政法规  │  ├──┐  ← 国务院制定
              │  │  ├─────────┤  │  │
           ┌──┤  │  │地方性法规│  │  ├──┐  ← 省级人大制定
           │  │  │  ├─────────┤  │  │  │
        ┌──┤  │  │  │  规 章   │  │  │  ├──┐  ← 部门/地方政府
        │  │  │  │  └─────────┘  │  │  │  │
        └──┴──┴──┴───────────────┴──┴──┴──┘
```

**效力规则**：
- 上位法优于下位法（宪法 > 法律 > 行政法规 > 地方性法规 > 规章）
- 同一层级：新法优于旧法、特别法优于一般法
- 所有法律都不得与宪法相抵触

### 4.2 七大法律部门

中国法律体系由**七大法律部门**构成，以宪法为统帅：

#### ① 宪法及宪法相关法
- **地位**：国家根本大法，一切法律的母法
- **内容**：国家基本制度、公民基本权利和义务、国家机构
- **核心法律**：《宪法》《选举法》《立法法》《国旗法》《民族区域自治法》
- **类比**：公司的"公司章程"，是一切规则的总纲

#### ② 民法商法
- **调整对象**：平等主体之间的人身关系和财产关系
- **核心原则**：平等、自愿、公平、诚信、守法
- **核心法律**：
  - 民法：《民法典》（含总则、物权、合同、人格权、婚姻家庭、继承、侵权责任七编）
  - 商法：《公司法》《合伙企业法》《证券法》《保险法》《票据法》《破产法》
- **类比**：管的是人和人之间"做买卖""搞关系"的事

#### ③ 行政法
- **调整对象**：国家行政管理活动中的各种社会关系
- **核心原则**：依法行政、合理行政、程序正当
- **核心法律**：《行政处罚法》《行政许可法》《行政强制法》《行政复议法》《行政诉讼法》《国家赔偿法》
- **类比**：管的是"政府怎么管老百姓"以及"老百姓怎么告政府"

#### ④ 经济法
- **调整对象**：国家宏观调控和市场规制关系
- **核心原则**：国家适度干预、维护公平竞争
- **核心法律**：《反垄断法》《反不正当竞争法》《消费者权益保护法》《税法》《预算法》《审计法》
- **类比**：政府当"裁判"，确保市场不作弊

#### ⑤ 社会法
- **调整对象**：劳动关系、社会保障、社会福利
- **核心原则**：保护弱势群体、社会公平
- **核心法律**：《劳动法》《劳动合同法》《社会保险法》《残疾人保障法》《未成年人保护法》《慈善法》
- **类比**：社会的"安全网"，兜底保障

#### ⑥ 刑法
- **调整对象**：犯罪和刑罚
- **核心原则**：罪刑法定（法无明文规定不为罪）、罪责刑相适应、法律面前人人平等
- **核心法律**：《刑法》及修正案
- **刑罚种类**：
  - 主刑：管制、拘役、有期徒刑、无期徒刑、死刑
  - 附加刑：罚金、剥夺政治权利、没收财产、驱逐出境
- **类比**：法律的"牙齿"，对最严重的违法行为施以最严厉的制裁

#### ⑦ 诉讼与非诉讼程序法
- **调整对象**：解决纠纷的程序和方法
- **核心法律**：
  - 诉讼法：《民事诉讼法》《刑事诉讼法》《行政诉讼法》
  - 非诉讼：《仲裁法》《人民调解法》
- **类比**："怎么打官司"的操作手册

## 五、法律的核心概念

### 5.1 法律关系

**定义**：法律规范在调整人们行为过程中形成的、以权利和义务为内容的社会关系。

法律关系由**三个要素**构成：

```
法律关系 = 主体 + 客体 + 内容
```

| 要素 | 说明 | 例子（买卖合同） |
|------|------|---------|
| **主体** | 法律关系的参加者（享有权利、承担义务的人） | 买方、卖方 |
| **客体** | 权利义务指向的对象 | 商品（物）、货款（行为） |
| **内容** | 主体之间的权利和义务 | 买方有权收货、义务付款；卖方有权收款、义务交货 |

**法律关系主体的类型**：
- **自然人**：有生命的人类个体
- **法人**：法律拟制的"人"（公司、政府机关、事业单位等）
- **非法人组织**：合伙企业、个人独资企业等

### 5.2 法律行为

**定义**：能够引起法律关系产生、变更或消灭的、以人的意志为转移的活动。

**分类**：

| 分类标准 | 类型 | 说明 |
|---------|------|------|
| 是否合法 | 合法行为 / 违法行为 | 合法的如签合同；违法的如盗窃 |
| 主体数量 | 单方行为 / 双方行为 / 多方行为 | 遗嘱=单方；合同=双方；合伙=多方 |
| 是否要式 | 要式行为 / 非要式行为 | 房产过户=要式（必须书面+登记）；买菜=非要式 |
| 是否有偿 | 有偿行为 / 无偿行为 | 买卖=有偿；赠与=无偿 |

**法律行为的有效要件**：
1. 行为人具有相应的民事行为能力
2. 意思表示真实
3. 不违反法律、行政法规的强制性规定
4. 不违背公序良俗

### 5.3 法律事实

**定义**：能够引起法律关系产生、变更或消灭的客观事实。

```
法律事实
├── 法律事件（不以人的意志为转移）
│   ├── 自然事件：地震、洪水、人的出生和死亡
│   └── 社会事件：战争、罢工
└── 法律行为（以人的意志为转移）
    ├── 合法行为：签合同、结婚
    └── 违法行为：犯罪、违约
```

### 5.4 法律权利与法律义务

**权利**：法律赋予主体可以做出某种行为或要求他人做出/不做出某种行为的资格。

**义务**：法律规定主体必须做出或不得做出某种行为的约束。

> 💡 没有无义务的权利，也没有无权利的义务——二者是对立统一的。

**权利的分类**：

| 类型 | 说明 | 例子 |
|------|------|------|
| 人身权 | 与人身不可分离 | 生命权、健康权、姓名权、肖像权、名誉权、隐私权 |
| 财产权 | 具有经济价值 | 物权（所有权、使用权）、债权（合同权利）、知识产权 |
| 政治权利 | 参与国家治理 | 选举权、被选举权、言论自由 |
| 社会权利 | 社会保障 | 受教育权、劳动权、社会保障权 |

## 六、法律责任

### 6.1 什么是法律责任？

**定义**：由于违反法律规定的义务（或约定的义务）而引起的、必须承担的不利法律后果。

简单说：做了不该做的事（或没做该做的事），就得承担后果。

### 6.2 三大法律责任对比

| 维度 | 民事责任 | 行政责任 | 刑事责任 |
|------|---------|---------|---------|
| **产生原因** | 违反民事义务（违约/侵权） | 违反行政法规 | 犯罪行为 |
| **追究主体** | 当事人（可起诉） | 行政机关 | 国家司法机关（检察院/法院） |
| **目的** | 补偿受害人损失 | 维护行政管理秩序 | 惩罚犯罪、保护社会 |
| **严厉程度** | 最轻 | 中等 | 最重 |
| **承担方式** | 赔偿损失、违约金、<br>恢复原状、停止侵害 | 警告、罚款、拘留、<br>吊销许可证 | 管制、拘役、有期徒刑、<br>无期徒刑、死刑 |
| **举例** | 撞坏别人的车→赔钱 | 闯红灯→罚款200扣分 | 醉驾→拘役+罚金 |

> ⚠️ 重要：三种责任可以同时承担！比如醉驾撞人 → 刑事责任（危险驾驶罪，拘役）+ 行政责任（吊销驾照）+ 民事责任（赔偿伤者医疗费）。

### 6.3 民事责任的两大类型

| 类型 | 说明 | 法律依据 |
|------|------|---------|
| **违约责任** | 不履行或不适当履行合同义务 | 《民法典》合同编 |
| **侵权责任** | 侵害他人人身权或财产权 | 《民法典》侵权责任编 |

### 6.4 归责原则

| 原则 | 说明 | 适用场景 |
|------|------|---------|
| **过错责任** | 有过错才担责（默认原则） | 一般侵权行为 |
| **过错推定** | 推定你有过错，除非你自证清白 | 建筑物倒塌致人损害 |
| **无过错责任** | 不管有没有过错，都要担责 | 产品缺陷致人损害、环境污染 |
| **公平责任** | 双方都没过错，按公平原则分担 | 见义勇为造成的损害 |

## 七、法律的基本原则

### 7.1 法治原则（最高原则）

- **法律至上**：一切组织和个人都必须在法律范围内活动
- **法律面前人人平等**：不分身份地位
- **依法行政**：政府权力要受法律约束
- **司法独立**：法院依法独立行使审判权

### 7.2 各部门法的核心原则

| 部门法 | 核心原则 |
|--------|---------|
| **宪法** | 人民主权、基本人权、法治、权力制约 |
| **民法** | 平等、自愿、公平、诚信、守法、绿色原则（生态保护） |
| **刑法** | 罪刑法定（法无明文规定不为罪）、罪责刑相适应、平等适用 |
| **行政法** | 依法行政、合理行政、程序正当、高效便民、诚实守信、权责统一 |
| **诉讼法** | 公开审判、辩护权保障、两审终审、回避制度 |

## 八、法律渊源

**定义**：法律的来源和表现形式，即法律从哪来、以什么形式存在。

### 中国法的正式渊源

| 渊源 | 制定机关 | 效力层级 | 例子 |
|------|---------|---------|------|
| 宪法 | 全国人大 | 最高 | 《中华人民共和国宪法》 |
| 法律 | 全国人大及常委会 | 第二 | 《民法典》《刑法》 |
| 行政法规 | 国务院 | 第三 | 《xxx条例》《xxx办法》 |
| 地方性法规 | 省级人大 | 第四 | 《xx省xxx条例》 |
| 部门规章 | 国务院各部委 | 第五 | 部门《xxx办法》 |
| 地方政府规章 | 省/市政府 | 第五 | 地方《xxx规定》 |
| 自治条例/单行条例 | 自治地方人大 | 特殊 | 民族自治地方立法 |

此外还有**非正式渊源**：
- **司法解释**：最高法/最高检对法律适用的解释
- **国际条约**：中国缔结或加入的国际条约
- **政策**：在一定条件下可作为裁判参考
- **习惯**：在法律没有规定时可适用

## 九、重要法律概念速查

| 概念 | 定义 | 关键点 |
|------|------|--------|
| **民事行为能力** | 独立实施民事法律行为的资格 | 18岁以上完全行为能力；8-18岁限制行为能力；8岁以下无行为能力 |
| **诉讼时效** | 权利人请求法院保护其权利的期限 | 一般3年；人身损害赔偿3年；不动产物权不适用 |
| **善意取得** | 从无权处分人处取得财产的合法方式 | 需满足：善意+合理价格+已交付/登记 |
| **不可抗力** | 不能预见、不能避免、不能克服的客观情况 | 地震、疫情等，可免除部分或全部责任 |
| **公序良俗** | 公共秩序和善良风俗 | 违反公序良俗的合同无效 |
| **举证责任** | "谁主张，谁举证"原则 | 部分情况下举证责任倒置（如医疗纠纷） |
| **正当防卫** | 对正在进行的不法侵害采取的防卫行为 | 不超过必要限度→不负刑事责任 |
| **紧急避险** | 为避免紧急危险而采取的损害较小利益的行为 | 损害不能大于所保护的利益 |

---

## 十、小结：法律知识的"思维导图"

```
法律
├── 是什么？ → 国家制定、强制力保障的行为规范
├── 从哪来？ → 法律渊源（宪法>法律>法规>规章）
├── 怎么分？
│   ├── 公法 vs 私法 vs 社会法
│   ├── 实体法 vs 程序法
│   ├── 成文法 vs 不成文法
│   └── 一般法 vs 特别法
├── 管什么？ → 七大法律部门
│   ├── 宪法（根本大法）
│   ├── 民商法（平等主体间关系）
│   ├── 行政法（政府与公民关系）
│   ├── 经济法（市场调控）
│   ├── 社会法（弱势群体保护）
│   ├── 刑法（犯罪与刑罚）
│   └── 诉讼法（纠纷解决程序）
├── 核心概念
│   ├── 法律关系 = 主体 + 客体 + 内容
│   ├── 法律行为 → 引起法律关系变动
│   ├── 法律事实 → 事件（天灾）+ 行为（人为）
│   └── 权利 ↔ 义务（对立统一）
├── 做错了怎么办？ → 法律责任
│   ├── 民事责任（赔钱、恢复原状）
│   ├── 行政责任（罚款、拘留）
│   └── 刑事责任（坐牢、罚金）
└── 全球视角 → 两大法系
    ├── 大陆法系（成文法为主，中国属此）
    └── 英美法系（判例法为主）
```

---

> 📌 **贾维斯的理解**：法律本质上是一套"游戏规则"——它告诉你什么能做、什么不能做、做错了怎么罚。和金融市场一样，理解规则是参与游戏的前提。这套知识体系是框架性的，具体到某个领域（如劳动法、知识产权法、税法）还需要深入学习。
