# 民事诉讼全流程实操指南

> 定位：从立案到执行，每一步怎么走的操作手册
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块1交付物
> 法律依据：《民事诉讼法》（2024修正）+相关司法解释

---

## 一、完整流程总览

```
起诉准备 → 立案 → 送达/答辩 → 证据交换 → 开庭审理 → 判决
                                                         ↓
                                                    不服 → 上诉(15日)
                                                         ↓
                                                    二审 → 终审判决
                                                         ↓
                                                    对方不履行 → 申请执行(2年)
```

---

## 二、管辖——去哪个法院起诉

### 2.1 级别管辖

| 法院 | 受理标的额（一般地区） | 受理标的额（北上广等） |
|------|----------------------|----------------------|
| 基层人民法院 | 一般民事案件 | 一般民事案件 |
| 中级人民法院 | 标的额较大/影响较大 | 5亿元以上 |
| 高级人民法院 | 标的额巨大 | 50亿元以上 |

> **实务提醒**：具体标的额标准各省不同，以当地高院发布的标准为准。普通人打的官司99%在基层法院。

### 2.2 地域管辖（最常用的规则）

| 案件类型 | 管辖法院 | 法条 |
|----------|----------|------|
| **一般原则** | **被告住所地** | 第22条 |
| 合同纠纷 | 被告住所地 或 合同履行地 | 第24条 |
| 侵权纠纷 | 被告住所地 或 侵权行为地 | 第29条 |
| 离婚案件 | 被告住所地（特殊情况原告住所地） | 第22条+第23条 |
| 不动产纠纷 | **不动产所在地**（专属管辖） | 第34条 |
| 继承纠纷 | 被继承人死亡时住所地或主要遗产所在地 | 第34条 |

### 2.3 协议管辖

合同双方可以书面约定管辖法院，但只能从以下5个中选：
1. 被告住所地
2. 合同履行地
3. 合同签订地
4. 原告住所地
5. 标的物所在地

> **实务提醒**：签合同时注意管辖条款！很多公司会约定在自己所在地法院管辖，对你不利要及时提出修改。

## 三、起诉准备

### 3.1 起诉材料清单

| 序号 | 材料 | 份数 | 说明 |
|------|------|------|------|
| 1 | **起诉状** | 正本1份+副本按被告人数 | 最核心文件 |
| 2 | **原告身份证复印件** | 1份 | 个人诉讼 |
| 3 | **被告信息** | 1份 | 身份证号/营业执照/企业信息打印件 |
| 4 | **证据材料** | 正本1份+副本按被告人数 | 附证据目录 |
| 5 | **诉讼费缴费凭证** | 1份 | 立案后缴费 |
| 6 | 授权委托书（有代理人时） | 1份 | 特别授权需明确列明 |
| 7 | 法人身份证明（公司起诉时） | 1份 | 营业执照+法人身份证 |

### 3.2 起诉状核心要素

```
                      民事起诉状

原告：[基本信息]
被告：[基本信息]

诉讼请求：
一、判令被告[具体请求]；
二、判令被告承担本案全部诉讼费用。

事实与理由：
[时间+地点+人物+事件经过+法律依据]

证据和证据来源：
[证据清单]

此致
[XX]人民法院

                              原告：[签名]
                              日期：[XXXX年XX月XX日]
```

### 3.3 诉讼请求撰写要点

| 原则 | 说明 | 示例 |
|------|------|------|
| **具体明确** | 金额精确到元，标的物描述清楚 | "判令被告支付借款本金50,000元" |
| **有法律依据** | 每项请求对应法律条文 | 依据《民法典》第675条 |
| **可执行** | 请求要能实际执行 | 不要写"判令被告道歉"（难执行） |
| **别漏诉讼费** | 最后一项固定写 | "判令被告承担本案诉讼费用" |

## 四、立案

### 4.1 立案方式

| 方式 | 操作 | 适用场景 |
|------|------|----------|
| **现场立案** | 去法院诉讼服务中心窗口 | 最传统、最稳妥 |
| **网上立案** | 中国移动微法院/各地法院网上立案平台 | 方便快捷，推荐 |
| **邮寄立案** | EMS寄到法院立案庭 | 跨地域立案 |

### 4.2 立案审查（7日）

- 法院收到起诉状后**7日内**决定是否受理
- 符合条件→《受理通知书》+《缴费通知书》
- 不符合条件→《不予受理裁定书》（可上诉）

### 4.3 诉讼费缴纳

**收到缴费通知后7日内缴费**，逾期不交视为撤诉。

**常见案件诉讼费速算**：

| 标的额 | 费率 | 速算 |
|--------|------|------|
| ≤ 1万 | 50元 | 固定50元 |
| 1万-10万 | 2.5% | 标的额×2.5%-200 |
| 10万-20万 | 2% | 标的额×2%+300 |
| 20万-50万 | 1.5% | 标的额×1.5%+1,300 |
| 50万-100万 | 1% | 标的额×1%+3,800 |
| 100万-200万 | 0.9% | 标的额×0.9%+4,800 |
| 200万-500万 | 0.8% | 标的额×0.8%+6,800 |
| 500万-1000万 | 0.7% | 标的额×0.7%+11,800 |
| 1000万-2000万 | 0.6% | 标的额×0.6%+21,800 |

> **减免**：经济困难可以申请**缓交、减交或免交**。申请书+经济困难证明（低保证/村委会证明等）。

## 五、送达与答辩

### 5.1 送达方式（法院→当事人）

| 方式 | 说明 |
|------|------|
| 直接送达 | 法院送到你手上，签收 |
| 留置送达 | 拒收的，放在住所即视为送达 |
| 电子送达 | 短信/微信/邮件（需当事人同意） |
| 邮寄送达 | EMS寄送 |
| 公告送达 | 找不到人，公告60日后视为送达 |

### 5.2 被告答辩（15日）

- 被告收到起诉状副本后**15日内**提交答辩状
- 不答辩不影响审理（缺席答辩≠缺席审判）

### 5.3 收到起诉状后该做什么（被告视角）

1. **立即看诉讼请求和事实理由**——搞清楚对方要什么
2. **15日内提交答辩状**——逐条反驳
3. **准备反驳证据**
4. **考虑是否提起反诉**——有反诉请求的可以在一审辩论终结前提出

## 六、证据交换与举证期限

### 6.1 举证期限

| 情形 | 期限 | 说明 |
|------|------|------|
| 法院指定 | 不少于**15日**（简易程序不少于**举证期限届满前**） | 以法院通知为准 |
| 当事人协商 | 双方协商确定 | 需法院准许 |
| 新证据 | 随时可以提交 | 但需说明未及时提交的理由 |

### 6.2 举证责任分配

**基本原则**：谁主张，谁举证（《民事诉讼法》第67条）

**举证责任倒置**（由被告举证）：

| 案件类型 | 说明 |
|----------|------|
| 医疗损害 | 医疗机构证明无过错 |
| 环境污染 | 排污者证明不存在因果关系 |
| 产品责任 | 生产者证明不存在缺陷 |
| 高度危险作业 | 经营者证明受害人故意 |
| 劳动争议（部分） | 用人单位掌握的证据由其提供 |

### 6.3 证据交换

法院可能组织庭前证据交换：
- 双方互相出示证据
- 确认无争议的事实
- 归纳争议焦点
- 好处：开庭时更高效

## 七、开庭审理

### 7.1 庭审流程

| 阶段 | 内容 | 时长 |
|------|------|------|
| **1. 庭前准备** | 核实身份、宣布纪律、告知权利 | 5-10分钟 |
| **2. 法庭调查** | 原告陈述→被告答辩→举证质证 | 30-60分钟 |
| **3. 法庭辩论** | 原告发言→被告发言→互相辩论 | 15-30分钟 |
| **4. 最后陈述** | 各方最后总结意见 | 5分钟 |
| **5. 调解** | 法官询问是否愿意调解 | 视情况 |
| **6. 签笔录** | 核对庭审笔录并签字 | 5-10分钟 |

### 7.2 开庭注意事项

| 必须做 | 不能做 |
|--------|--------|
| 准时到庭（提前15分钟） | 迟到超过15分钟可能按缺席处理 |
| 携带身份证原件 | 忘带证件不让进法庭 |
| 携带所有证据原件 | 只带复印件法官不认 |
| 称呼"审判长""审判员" | 不要叫"法官大人""青天大老爷" |
| 回答问题简明扼要 | 不要情绪化、不要长篇大论 |
| 仔细听对方陈述做笔记 | 不要打断对方发言 |
| 签笔录前仔细阅读 | 不要不看就签字 |

### 7.3 审理期限

| 程序 | 期限 | 可延长 |
|------|------|--------|
| **简易程序** | **3个月** | 经院长批准可延长1个月 |
| **普通程序** | **6个月** | 经院长批准可延长6个月 |
| **小额诉讼** | **2个月** | 一般不延长 |

**适用条件**：
- 简易程序：事实清楚、权利义务关系明确、争议不大（基层法院）
- 小额诉讼：标的额为各省年度上一年度就业人员年平均工资30%以下，一审终审
- 普通程序：其他案件

## 八、判决与上诉

### 8.1 判决送达

- 当庭宣判→10日内送达判决书
- 定期宣判→宣判后立即送达

### 8.2 上诉

| 要素 | 规则 |
|------|------|
| 上诉期限 | 判决书送达后**15日**内 |
| 上诉法院 | 上一级人民法院 |
| 上诉状提交 | 通过原审法院提交 |
| 上诉费 | 按一审标准缴纳 |
| 二审期限 | **3个月**（判决）/ **30日**（裁定） |

### 8.3 不上诉

- 15日上诉期满不上诉→一审判决**生效**
- 生效后即可申请执行

### 8.4 二审审理

- 二审为**终审**，判决后即生效
- 二审可能的结果：维持原判 / 改判 / 发回重审

## 九、执行

### 9.1 申请执行条件

1. 法律文书已生效
2. 义务人逾期未履行
3. 在申请执行时效内（**2年**）

### 9.2 申请执行材料

| 材料 | 说明 |
|------|------|
| 执行申请书 | 写明申请人/被执行人/执行依据/执行请求/已知财产线索 |
| 生效法律文书 | 判决书/调解书/裁定书原件 |
| 生效证明 | 法院出具（证明文书已生效） |
| 身份证复印件 | 申请人 |
| 被执行人财产线索 | 银行账号、房产、车辆、股权等 |

### 9.3 执行措施（法院可以采取）

| 措施 | 说明 |
|------|------|
| **冻结/划扣银行存款** | 最常用，直接扣钱 |
| **查封/拍卖/变卖财产** | 房、车、设备等 |
| **限制高消费** | 不能坐飞机、高铁商务座、住星级酒店 |
| **纳入失信名单** | "老赖"黑名单 |
| **拘留** | 有能力执行而拒不执行，15日以下 |
| **移送追究刑事责任** | 拒不执行判决裁定罪（情节严重） |

### 9.4 执行不能怎么办

- 被执行人确实没财产→法院裁定**终结本次执行程序**
- 不是"结案"！发现新财产可以随时恢复执行
- 申请人可以定期查询被执行人财产变动

## 十、特殊程序速查

### 10.1 财产保全

**什么时候用**：担心对方转移、隐匿、变卖财产

| 类型 | 时机 | 担保 |
|------|------|------|
| 诉前保全 | 起诉前（紧急情况） | 必须提供担保 |
| 诉中保全 | 诉讼过程中 | 法院可以要求提供担保 |

**操作**：向法院提交《财产保全申请书》+担保（保函/现金/房产）+财产线索

### 10.2 先予执行

追索劳动报酬、赡养费、医疗费等紧急情况，可以申请法院在判决前先执行。

### 10.3 申请再审

| 要素 | 规则 |
|------|------|
| 时限 | 判决/裁定生效后**6个月**内 |
| 向谁申请 | 上一级人民法院 |
| 法定事由 | 新证据/适用法律错误/程序违法/审判人员贪污受贿等（《民事诉讼法》第211条） |

## 十一、诉讼时效速查

| 时效 | 适用范围 | 法条 |
|------|----------|------|
| **3年** | 一般民事请求权（合同/侵权等） | 《民法典》第188条 |
| **1年** | 延付/拒付租金、寄存财物丢失损毁 | 特别规定 |
| **最长20年** | 从权利被侵害之日起算（特殊保护） | 第188条第2款 |
| 不适用 | 物权请求权（返还原物等）、人格权（停止侵害等） | 相关规定 |

**时效中断事由**（重新起算3年）：
1. 向对方主张权利（发催款函、打电话并录音）
2. 对方同意履行（书面承诺还款）
3. 提起诉讼/仲裁
4. 与提起诉讼等效的行为（申请调解、报案等）
