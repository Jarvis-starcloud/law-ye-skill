# 中华人民共和国反不正当竞争法（2019年修正版）

## 基本信息
- **通过/修订**：1993通过 → 2017第一次修订 → 2019修正
- **施行日期**：2018年1月1日起施行
- **条款总数**：共33条

---

## 第一章 总则（第1-5条）
- 立法目的：促进市场经济健康发展，鼓励公平竞争
- 基本原则：自愿、平等、公平、诚信
- 经营者定义：从事商品生产、经营或提供服务的自然人、法人、非法人组织

---

## 第二章 不正当竞争行为（第6-12条）

### 1. 混淆行为（第6条）
擅自使用他人有一定影响的标识、名称等（包括"傍名牌"）

### 2. 商业贿赂（第7条）
贿赂交易相关人员谋取交易机会

### 3. 虚假宣传（第8条）
虚假或引人误解的宣传，包括刷单炒信

### 4. 侵犯商业秘密（第9条）
以不正当手段获取/披露/使用商业秘密
- 商业秘密构成：秘密性（不为公众所知悉）、商业价值性（带来竞争优势）、保密性（采取保密措施）
- 侵权手段：盗窃/利诱/胁迫/披露/允许他人使用

### 5. 不当有奖销售（第10条）
信息不明确、谎称有奖、抽奖金额超5万元

### 6. 商业诋毁（第11条）
编造传播损害竞争对手商誉的信息

### 7. 互联网不正当竞争（第12条）
技术手段妨碍破坏网络产品/服务

---

## 第三章 调查（第13-16条）
- 检查权：进入场所、询问、查阅资料、查封扣押、查询银行账户
- 商业秘密保密义务

---

## 第四章 法律责任（第17-27条）

| 违法行为 | 处罚措施 |
|----------|----------|
| 一般民事责任 | 实际损失/获利赔偿；恶意+严重可处**1-5倍惩罚性赔偿**；难以确定则**500万以下** |
| 混淆行为 | 没收违法商品 + 罚款(5倍以下或25万以下) + 吊销执照 |
| 商业贿赂 | 没收违法所得 + 罚款10-300万 + 吊销执照 |
| 虚假宣传 | 罚款20-200万（严重可吊销） |
| 商业秘密侵权 | 责令停止 + 没收 + 罚款10-100万（严重50-500万） |
| 有奖销售违规 | 罚款5-50万 |
| 商业诋毁 | 消除影响 + 罚款10-50万（严重50-300万） |
| 互联网违规 | 罚款10-50万（严重50-300万） |

⚠️ **重要规则**：财产不足时优先承担民事责任（第27条）

**举证责任倒置**（第32条）：权利人初步证明后，涉嫌侵权人需证明不侵权。
