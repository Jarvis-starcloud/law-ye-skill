# 中华人民共和国民事诉讼法 学习笔记

> 学习时间：2026年4月2日 | 整理：贾维斯
> 版本：2023年9月1日第五次修正（1991年通过，历经五次修正）

---

## 一、法律概览

| 项目 | 内容 |
|------|------|
| 全称 | 中华人民共和国民事诉讼法 |
| 性质 | 程序法（规定民事纠纷如何打官司的法律） |
| 通过时间 | 1991年4月9日 |
| 修正历程 | 2007年第一次→2012年第二次→2017年第三次→2021年第四次→**2023年第五次** |
| 结构 | **4编27章306条** |
| 核心定位 | 民事诉讼的"操作手册"——从起诉到执行的全流程规则 |

### 四编结构一览

| 编 | 名称 | 章节范围 | 条文范围 | 核心内容 |
|---|------|---------|---------|---------|
| 第一编 | 总则 | 第1-11章 | 第1-121条 | 基本原则、管辖、证据、调解、保全等基础制度 |
| 第二编 | 审判程序 | 第12-18章 | 第122-234条 | 一审、二审、再审、特别程序、督促程序等 |
| 第三编 | 执行程序 | 第19-22章 | 第235-269条 | 执行申请、执行措施、执行中止和终结 |
| 第四编 | 涉外民事诉讼 | 第23-27章 | 第270-306条 | 涉外管辖、送达、仲裁、司法协助 |

## 二、基本原则体系

| 原则 | 条文 | 含义 |
|------|------|------|
| 审判独立 | 第6条 | 法院独立审判，不受行政机关、社会团体和个人干涉 |
| 以事实为根据，以法律为准绳 | 第7条 | 审判的根本依据 |
| 当事人平等 | 第8条 | 诉讼权利平等，适用法律平等 |
| 调解原则 | 第9条 | 自愿合法原则，调解不成及时判决 |
| 四大制度 | 第10条 | **合议制度+回避制度+公开审判制度+两审终审制度** |
| 辩论原则 | 第12条 | 当事人有权辩论 |
| 诚信原则 | 第13条 | **2012年新增**，民事诉讼遵循诚信原则 |
| 处分原则 | 第13条 | 当事人有权处分自己的民事权利和诉讼权利 |
| 检察监督 | 第14条 | 人民检察院有权对民事诉讼实行法律监督 |
| **在线诉讼** | **第16条** | **2021年新增**，经当事人同意可在线进行，与线下同等效力 |

## 三、管辖制度（第18-39条）

管辖 = 确定哪个法院来审理你的案件。

### 3.1 级别管辖

| 法院级别 | 管辖范围 | 条文 |
|---------|---------|------|
| 基层法院 | 第一审民事案件（一般规则） | 第18条 |
| 中级法院 | 重大涉外案件、本辖区重大影响案件、最高院确定的案件 | 第19条 |
| 高级法院 | 本辖区有重大影响的一审案件 | 第20条 |
| 最高法院 | 全国重大影响案件、认为应当审理的案件 | 第21条 |

### 3.2 地域管辖

**一般规则：原告就被告**（第22条）
- 对公民起诉 → 被告住所地/经常居住地法院
- 对法人起诉 → 被告住所地法院

**四种例外（原告住所地管辖）**（第23条）：
1. 对域外居住的人提起身份关系诉讼
2. 对下落不明/宣告失踪的人提起身份关系诉讼
3. 对被强制教育措施的人起诉
4. 对被监禁的人起诉

### 3.3 特殊地域管辖

| 案件类型 | 管辖法院 | 条文 |
|---------|---------|------|
| 合同纠纷 | 被告住所地/合同履行地 | 第24条 |
| 保险合同 | 被告住所地/保险标的物所在地 | 第25条 |
| 侵权纠纷 | 侵权行为地/被告住所地 | 第29条 |
| 交通事故 | 事故发生地/车辆最先到达地/被告住所地 | 第30条 |
| 公司纠纷 | 公司住所地 | 第27条 |

### 3.4 专属管辖（第34条，不得协议变更）

| 案件类型 | 管辖法院 |
|---------|---------|
| 不动产纠纷 | 不动产所在地 |
| 港口作业纠纷 | 港口所在地 |
| 继承遗产纠纷 | 被继承人死亡时住所地/主要遗产所在地 |

### 3.5 协议管辖（第35条）

当事人可以书面协议选择与争议有实际联系的法院，但**不得违反级别管辖和专属管辖**。

## 四、诉讼参加人

### 4.1 当事人

| 类型 | 说明 | 条文 |
|------|------|------|
| 一般当事人 | 公民、法人和其他组织 | 第51条 |
| 共同诉讼 | 诉讼标的共同或同一种类 | 第55条 |
| 代表人诉讼 | 人数众多的共同诉讼，推选代表人 | 第56-57条 |
| 公益诉讼 | 法定机关/组织+检察院可提起 | 第58条 |
| 第三人 | 有独立请求权/无独立请求权 | 第59条 |

### 4.2 诉讼代理人

| 类型 | 说明 |
|------|------|
| 法定代理人 | 无诉讼行为能力人的监护人（第60条） |
| 委托代理人 | 律师、近亲属、工作人员、社区推荐公民（第61条） |

**特别授权事项**（第62条）：承认、放弃、变更诉讼请求，和解，反诉，上诉 → 必须**特别授权**。

**离婚案件**（第65条）：即使有代理人，本人仍应出庭（除不能表达意思外）。

## 五、证据制度（第66-84条）

### 5.1 八种法定证据类型

| 序号 | 证据类型 | 说明 |
|------|---------|------|
| 1 | 当事人陈述 | 结合其他证据审查（第78条） |
| 2 | 书证 | 应提交原件（第73条） |
| 3 | 物证 | 应提交原物（第73条） |
| 4 | 视听资料 | 需辨别真伪（第74条） |
| 5 | **电子数据** | **2012年新增**，微信记录、邮件等 |
| 6 | 证人证言 | 知情者有义务出庭（第75条） |
| 7 | 鉴定意见 | 有异议→鉴定人应出庭（第81条） |
| 8 | 勘验笔录 | 需出示法院证件（第83条） |

### 5.2 举证责任

**核心原则**："谁主张，谁举证"（第67条）

- 当事人应及时提供证据（第68条）
- 逾期提供 → 法院可不予采纳或训诫、罚款
- 客观原因不能收集 → 法院应调查收集

### 5.3 证据保全（第84条）

- 诉讼中 → 当事人申请或法院主动
- 诉前 → 利害关系人可向法院申请（情况紧急时）

## 六、期间与送达（第85-95条）

### 6.1 期间规则

| 规则 | 内容 |
|------|------|
| 开始不算 | 期间开始的时和日不计入（第85条） |
| 节假日顺延 | 届满最后一日是休假日→顺延至第一个工作日 |
| 在途不算 | 诉讼文书期满前交邮→不算过期 |
| 可申请顺延 | 不可抗拒事由→障碍消除后10日内申请（第86条） |

### 6.2 送达方式

| 方式 | 说明 | 条文 |
|------|------|------|
| 直接送达 | 送交本人/同住成年家属/法人负责人 | 第88条 |
| 留置送达 | 拒收→留在住所+拍照录像 | 第89条 |
| **电子送达** | **经同意**，可电子方式送达 | **第90条** |
| 委托/邮寄送达 | 直接送达困难时 | 第91条 |
| 公告送达 | 下落不明/其他方式无法送达→**30日**视为送达 | 第95条 |

## 七、保全与先予执行（第103-111条）

### 7.1 保全制度

| 类型 | 时间 | 条件 | 关键时限 |
|------|------|------|---------|
| 诉讼保全 | 诉讼中 | 可能使判决难以执行 | **48小时内**作出裁定 |
| 诉前保全 | 起诉前 | 情况紧急，权益将受难以弥补的损害 | **48小时内**裁定+采取措施后**30日内**必须起诉 |

### 7.2 先予执行（第109条）

三种情形可先予执行：
1. 追索赡养费、扶养费、抚养费、抚恤金、医疗费用
2. 追索劳动报酬
3. 情况紧急需要先予执行

条件：权利义务关系明确 + 不先予执行严重影响生活/生产 + 被申请人有履行能力。

## 八、审判程序体系

### 8.1 第一审普通程序（第122-159条）

#### 起诉四条件（第122条）

| 条件 | 内容 |
|------|------|
| 适格原告 | 与本案有直接利害关系 |
| 明确被告 | 有明确的被告 |
| 具体请求 | 有具体的诉讼请求和事实、理由 |
| 管辖正确 | 属于法院受理范围和管辖 |

#### 关键时限

| 事项 | 时限 |
|------|------|
| 立案 | 符合条件→**7日内**立案 |
| 送达起诉状副本 | 立案之日起**5日内** |
| 答辩状提交 | 收到起诉状副本之日起**15日内** |
| 当庭宣判→发送判决书 | **10日内** |
| 普通程序审限 | **6个月**（院长批准可延长6个月，再延长报上级批准） |

#### 开庭审理流程

```
开庭准备（核对身份→宣布案由→告知权利→询问回避）
    ↓
法庭调查（陈述→证人作证→出示证据→鉴定意见→勘验笔录）
    ↓
法庭辩论（原告→被告→第三人→互相辩论→最后意见）
    ↓
调解/判决
```

#### 缺席判决

| 情形 | 后果 |
|------|------|
| 原告无故不到庭/中途退庭 | 按撤诉处理（被告反诉的→缺席判决） |
| 被告无故不到庭/中途退庭 | 缺席判决 |
| 必须到庭的被告 | 可拘传（经两次传票传唤） |

### 8.2 简易程序（第160-170条）

| 项目 | 内容 |
|------|------|
| 适用法院 | 基层法院及其派出法庭 |
| 适用条件 | 事实清楚、权利义务明确、争议不大 |
| 审判组织 | 审判员一人**独任审理** |
| 审限 | **3个月**（院长批准可延长1个月） |

### 8.3 小额诉讼程序（第165-169条）

| 项目 | 内容 |
|------|------|
| 适用条件 | 简单金钱给付+标的额≤上年度就业人员年平均工资**50%** |
| 约定适用 | 标的额50%-200%的，双方可约定适用 |
| 核心特征 | **一审终审**（不可上诉！） |
| 审限 | **2个月**（院长批准可延长1个月） |

> ⚠️ **不适用小额诉讼**：人身关系案件、涉外案件、需要鉴定的、一方下落不明、提出反诉

### 8.4 第二审程序（第171-183条）

| 项目 | 内容 |
|------|------|
| 上诉期限 | 判决：**15日** / 裁定：**10日** |
| 处理方式 | 维持/改判/发回重审（只能发回一次！） |
| 审限 | 判决上诉：**3个月** / 裁定上诉：**30日** |
| 效力 | 二审判决、裁定 = **终审**判决、裁定 |

**发回重审限制**（第177条第5款）：原审法院重审后当事人再上诉的，**二审法院不得再次发回重审**——防止案件来回踢皮球。

### 8.5 审判监督程序（再审）（第209-224条）

#### 再审启动三种途径

| 途径 | 发起人 | 条文 |
|------|--------|------|
| 法院自行再审 | 院长→审委会 / 上级法院 | 第209条 |
| 当事人申请 | 向上一级法院（公民案件也可向原审法院） | 第210条 |
| 检察院抗诉 | 检察院发现法定情形 | 第219条 |

#### 当事人申请再审的13种法定情形（第211条）

1. 新证据足以推翻原判
2. 基本事实缺乏证据证明
3. 主要证据是伪造的
4. 主要证据未经质证
5. 法院未调查收集当事人申请的证据
6. 适用法律确有错误
7. 审判组织不合法/应回避未回避
8. 无诉讼行为能力人未经代理/应参加未参加
9. 剥夺辩论权利
10. 未经传票传唤缺席判决
11. 遗漏/超出诉讼请求
12. 据以判决的法律文书被撤销/变更
13. 审判人员贪污受贿、徇私枉法

**申请期限**：判决生效后**6个月内**（第216条）

**特别规定**：已解除婚姻关系的判决/调解书→**不得申请再审**（第213条）

## 九、特别程序（第184-208条）

一审终审，处理非争议性事务：

| 案件类型 | 关键要点 |
|---------|---------|
| 选民资格案件 | 选举日前5日起诉，选举日前审结 |
| 宣告失踪 | 下落不明满**2年**，公告期**3个月** |
| 宣告死亡 | 下落不明满**4年**（意外事件满**2年**），公告期**1年**（意外+不可能生存→3个月） |
| **指定遗产管理人** | **2023年新增**，配合民法典继承编 |
| 认定无/限制行为能力 | 利害关系人/有关组织申请，可鉴定 |
| 认定财产无主 | 公告满**1年**无人认领→收归国家/集体 |
| 确认调解协议 | 调解协议生效之日起**30日内**申请 |
| 实现担保物权 | 向担保财产所在地/登记地基层法院 |

审限：**30日内**或公告期满后**30日内**审结。

## 十、督促程序与公示催告（第225-234条）

### 10.1 督促程序（支付令）

| 项目 | 内容 |
|------|------|
| 适用条件 | 请求给付金钱/有价证券+无其他债务纠纷+能送达 |
| 受理通知 | **5日内** |
| 发出支付令 | 受理后**15日内** |
| 债务人回应 | 收到后**15日内**清偿或提出书面异议 |
| 异议效果 | 异议成立→终结督促程序→转入诉讼 |

### 10.2 公示催告程序

适用于票据被盗/遗失/灭失的情况，公告期不少于**60日**。

## 十一、执行程序（第235-269条）

### 11.1 执行管辖

- 第一审法院执行
- 或与第一审法院同级的被执行财产所在地法院

### 11.2 执行申请时效

| 项目 | 内容 |
|------|------|
| 申请期限 | **2年**（适用诉讼时效中止、中断规定） |
| 起算点 | 履行期限最后一日/分期最后一期届满日/文书生效日 |

### 11.3 执行措施

| 措施 | 说明 |
|------|------|
| 查询财产 | 存款、债券、股票、基金等 |
| 扣押冻结划拨 | 不得超出应履行义务范围 |
| 扣留收入 | 保留生活必需费用 |
| 查封拍卖变卖 | 保留生活必需品 |
| 搜查 | 院长签发搜查令 |
| 限制出境 | 被执行人不履行义务 |
| **征信记录+媒体公布** | **信用惩戒** |

### 11.4 妨害诉讼的强制措施

| 措施 | 对个人 | 对单位 |
|------|--------|--------|
| 罚款 | ≤**10万元** | **5万-100万元** |
| 拘留 | ≤**15日** | 对主要负责人/直接责任人 |

## 十二、涉外民事诉讼（第270-306条）

### 12.1 涉外管辖特点

| 规则 | 内容 |
|------|------|
| 管辖连接点 | 合同签订地/履行地/标的物所在地/可供扣押财产所在地/侵权行为地/代表机构住所地（第276条） |
| 协议管辖 | 当事人可书面选择中国法院（第277条） |
| 应诉管辖 | 未提异议+应诉答辩/反诉→视为有管辖权（第278条） |
| 专属管辖 | 中国境内法人设立解散/知识产权有效性/中外合资合作合同（第279条） |
| **不方便法院** | **2023年新增**，满足五条件可裁定驳回（第282条） |

### 12.2 涉外送达（第283条）

十种方式（按序选择）：国际条约→外交→使领馆→代理人→代表机构→法定代表人→邮寄（3个月）→电子方式→其他同意方式→**公告（60日）**。

### 12.3 涉外期间特殊规定

| 事项 | 国内 | 涉外 |
|------|------|------|
| 答辩期 | 15日 | **30日** |
| 上诉期 | 判决15日/裁定10日 | **30日** |
| 审限 | 6个月/3个月 | **不受限制**（第287条） |
| 公告送达 | 30日 | **60日** |

### 12.4 外国判决承认与执行（2023年重大修改）

**不予承认的5种情形**（第300条）：
1. 外国法院无管辖权
2. 未合法传唤/未获陈述辩论机会
3. 通过欺诈取得
4. 中国法院已对同一纠纷作出判决
5. 违反中国法律基本原则/损害国家利益

## 十六、实践知识点

### 16.1 小明场景：买到假货怎么打官司？

> 小明在网上买了一台笔记本电脑，收到后发现是翻新机。卖家拒绝退款。

**诉讼路径分析**：

1. **起诉条件**（第122条）：小明是直接利害关系人✅，被告为卖家✅，诉讼请求为退款✅，基层法院管辖✅
2. **管辖法院**（第24条合同纠纷）：被告住所地或合同履行地法院
3. **证据准备**（第66条）：购物记录（电子数据）+聊天记录+鉴定意见
4. **程序选择**：如金额较小→可能适用小额诉讼（一审终审，2个月审结）
5. **先行调解**（第125条）：法院可先行调解
6. **举证**（第67条）：小明主张假货→应提供鉴定意见等证据
7. **判决后执行**：卖家不履行→申请强制执行（2年内）
