## ✨ v2.0.0 新特性：知识产权法

著作权(67条) + 专利(82条) + 商标(73条) + 反不正当竞争法(33条)，IP四支柱全覆盖。

### 🎨 著作权法
- **17项权利**：人身4项(发表/署名/修改/保护完整) + 财产13项
- **9类作品**：文字/音乐/美术/摄影/影视/软件等
- **归属规则**：11种常见场景（委托/职务/演绎/汇编等）
- **合理使用**：13种情形（个人学习/新闻报道/课堂教学等）
- **侵权赔偿**：**惩罚性赔偿 1-5倍**

### 🔬 专利法
- **三种类别**：发明(20年)/实用新型(10年)/外观设计(15年)
- **授权三要件**：新颖性 + 创造性 + 实用性
- **先申请原则**：同日申请→协商，协商不成均驳回
- **不视为侵权的5种例外**：先用权/临时过境/科研实验/Bolar例外/权利用尽

### ™️ 商标法
- **注册3原则**：自愿/注册在先/分类申请
- **有效期10年**，可无限续展（每次10年）
- **驰名商标特殊保护**：跨类保护 + 驳回/无效的绝对理由

### ⚖️ 刑事诉讼全流程

| 阶段 | 核心内容 |
|------|----------|
| **立案** | 立案条件/职能管辖/立案监督（检察院监督公安不立案） |

| **审查起诉** | 四类不起诉(法定/酌定/存疑/特殊)/**认罪认罚从宽制度**/补充侦查≤2次 |
| **一审** | 三种程序对比(普通/简易/速裁)/**疑罪从无**原则/三种判决结果 |
| **二审** | **上诉不加刑**铁律及其例外/四种处理方式/发回重审限一次 |
| **死刑复核** | 最高院唯一核准权/停止执行三情形 |
| **再审** | 五种申诉条件/另组合议庭/一般不加刑 |
| **执行** | 各刑罚执行机关分工/减刑假释/**暂予监外执行**/社区矫正 |

### 🔥 核心实务要点
- **"黄金37天"**：拘留后3-30日+7日批捕=最长37天，律师介入最佳窗口期
- **非法证据排除**：言词证据绝对排除，实物证据相对排除
- **刑民交叉**：竞合型先刑后民，牵连型刑民并行
- **常见犯罪辩护**：盗窃/诈骗/故意伤害/危险驾驶/交通肇事速查表
- **37个关键期限**一目了然

## ✨ v1.6.0 新特性：行政法与行政诉讼

"民告官"全领域覆盖——行政法基本原理、行政行为体系、行政诉讼、行政复议、国家赔偿。

### 🏛️ 行政法基本原理
- **六大基本原则**：合法行政（首要）+ 比例原则（"帝王条款"）+ 程序正当 + 信赖保护
- **行政主体理论**：行政机关 vs 授权组织 vs 委托组织，被告认定规则
- **行政行为体系**：许可/处罚/强制三大基石 + 行政协议 + 政府信息公开

### ⚖️ 行政诉讼法（2017年修正）
- **受案范围12项** + 不受理4项
- **核心特征**：被告举证（行政机关自证合法）
- **六种判决类型**：驳回→撤销→履行→变更→确认违法→确认无效
- **附带审查**：可一并审查规范性文件合法性
- **起诉期限**：6个月（不动产20年/其他5年）

### 📋 行政复议法（2023年大修，2024年施行）
- **管辖集中化**：统一由县级以上政府管辖
- **前置范围扩大**：当场处罚/不作为/信息公开必须先复议
- **双轨审理**：普通程序60日 + 简易程序30日
- **定位升级**：化解行政争议的"主渠道"

### 💰 国家赔偿法
- **违法→赔偿，合法→补偿**
- 人身自由：每日 = 国家上年度职工日平均工资
- 死亡/丧失劳动能力：最高20倍年均工资

### 🔨 行政处罚法（2021年修订）
- **首违不罚** + **无过错不罚**
- 新增处罚种类：通报批评、降低资质等级、限制从业
- 时效延长：一般2年，涉及生命健康/金融安全5年
