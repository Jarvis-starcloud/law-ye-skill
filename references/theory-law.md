## 三、法理学精要

### 3.1 法的概念与本质

#### 3.1.1 法的定义（马克思主义法学观）

```
法 = 国家制定/认可 + 国家强制力保障实施 + 行为规范 = 统治阶级意志的体现
```

**三个基本特征**：

| 特征 | 说明 | 对比其他规范 |
|------|------|-------------|
| **国家意志性** | 由国家机关制定或认可 | 区别于道德、宗教、习惯 |
| **国家强制性** | 以国家强制力为后盾 | 区别于纪律、团体章程 |
| **普遍适用性** | 在主权范围内对所有人有效 | 区别于单位内部规定 |

#### 3.1.2 法的本质（三层）

```
第一层（表象）：正式性/官方性 — 国家的行为
第二层（内容）：阶级性 — 统治阶级意志的体现
第三层（深层）：物质制约性 — 由物质生活条件决定
```

> ⚠️ 法考重点：**物质制约性是最深层本质**。法不是随心所欲创造的，而是由经济基础决定的。

#### 3.1.3 法的作用

```
法的规范作用（对人行为的直接影响）
│
├── 指引作用（对自己）— 确定的指引 vs 有选择的指引
├── 评价作用（对他人）— 判断他人行为是否合法
├── 教育作用（一般公众）— 通过法的实施影响人们的行为倾向
├── 预测作用（相互间）— 预期对方行为及后果
└── 强制作用（违法者）— 制裁违法犯罪行为

法的社会作用（对社会的影响）
│
├── 政治作用 — 维护统治秩序
├── 经济作用 — 服务经济建设
├── 文化作用 — 促进文化发展
└── 社会公共事务 — 环境保护、交通管理、公共卫生等
```

### 3.2 法的价值

#### 3.2.1 法的基本价值及其冲突解决

| 价值 | 含义 | 冲突时的处理原则 |
|------|------|------------------|
| **自由** | 法不禁止即为自由；法无明文规定不处罚 | 自由优先（但受限制）|
| **正义** | 分配正义（利益分配）+ 矫正正义（纠纷解决）| 实质正义 > 形式正义 |
| **秩序** | 社会关系的稳定性、一致性、连续性 | 秩序是基础价值 |
| **效率** | 以最小资源消耗获得最大产出 | 效率与公平平衡 |

#### 3.2.2 价值冲突的解决规则

```
优先顺序（一般情况）：秩序 → 正义 → 自由 → 效率
特殊情况下的调整：
  - 危机时刻：秩序优先
  - 平稳时期：自由和正义优先
  - 发展时期：效率权重上升
```

### 3.3 法的运行

#### 3.3.1 立法（法的制定）

**我国立法体制**：**一元两级多层次**

```
                    全国人大及其常委会（中央立法权）
                   /              \
        地方性法规（省级）       民族自治条例
             │                     │
        较大的市（设区市）       单行条例
```

**立法程序**（普通法律的"四步走"）：

| 步骤 | 内容 | 注意事项 |
|------|------|----------|
| **① 提出法案** | 人大代表/常委会委员/专门委员会/国务院等有权提案主体 | 并非任何人都能提案 |
| **② 审议法案** | 人大常委会审议（通常三审制）| 三读程序 |
| **③ 表决法案** | 过半数通过（宪法修改需三分之二以上代表同意）| 重要法律单独表决条款 |
| **④ 公布法律** | 国家主席签署主席令公布 | 未签署则未生效 |

#### 3.3.2 法的实施方式

| 方式 | 含义 | 举例 |
|------|------|------|
| **法的遵守（守法）** | 一切主体依法行使权利/履行义务 | 公民遵守交规 |
| **法的执行（执法）** | 国家行政机关执行法律的活动 | 工商局查处假冒伪劣 |
| **法的适用（司法）** | 国家司法机关依法处理案件 | 法院判决案件 |
| **法的监督** | 对法的运行全过程进行监察 | 人大监督、检察监督 |

#### 3.3.3 法律适用的原则

1. **上位法优于下位法**
2. **特别法优于一般法**（同一机关制定的）
3. **新法优于旧法**（同一机关制定的）
4. **国际条约保留除外**（中国参加/缔约的国际条约，除声明保留外，直接适用）

> ⚠️ **冲突规则的使用顺序**：先判断上下位关系 → 再判断新旧/特别关系 → 最后考虑国际条约

#### 3.3.4 法律推理方法

| 方法 | 内容 | 适用场景 |
|------|------|----------|
| **演绎推理（三段论）** | 大前提（法律规范）→ 小前提（事实）→ 结论 | 最常用 |
| **归纳推理** | 从个别到一般 | 判例法/类比参考 |
| **类比推理** | 相似案件相似处理 | 法律漏洞填补 |
| **辩证推理** | 综合考量多种因素 | 复杂/疑难案件 |

### 3.4 法的演进

#### 3.4.1 法的历史类型

| 类型 | 所有制 | 代表法系 | 特点 |
|------|--------|----------|------|
| 奴隶制法 | 奴隶主所有制 | 古罗马法、汉谟拉比法典 | 不平等、公开等级歧视 |
| 封建制法 | 封建所有制 | 《唐律疏议》、日耳曼法 | 等级森严、刑罚残酷 |
| 资本主义法 | 资本家私有制 | 大陆法系、英美法系 | 形式平等、私有财产神圣 |
| 社会主义法 | 生产资料公有制 | 社会主义法律体系 | 人民意志、实质平等 |

#### 3.4.2 世界主要法系

| 维度 | **大陆法系（民法法系）** | **英美法系（普通法法系）** |
|------|-------------------------|-------------------------|
| **起源** | 古罗马法（罗马法复兴运动）| 英国中世纪普通法院判例 |
| **代表** | 法国、德国、日本、中国 | 英国、美国、加拿大、澳大利亚 |
| **法律渊源** | 制定法为主（成文法典）| 判例法为主+制定法 |
| **思维方式** | 演绎推理（从规则到结论）| 归纳推理（从先例到规则）|
| **诉讼模式** | 职权主义（法官主导）| 当事人主义（对抗制）|
| **分类** | 公法与私法 | 普通法与衡平法 |
| **法官角色** | 适用者（不能造法）| 造法者（判例即法源）|

> 💡 我国属**大陆法系传统**，但近年来也逐步重视案例指导制度。

#### 3.4.3 法的发展趋势

1. **国际化趋势** — 国际法与国内法互动增强
2. **社会化趋势** — 从个人本位向社会本位转变
3. **科技化趋势** — 数字经济、AI、数据安全等新兴领域立法加速
4. **人权保障强化趋势** — 人权入宪、人权司法保障机制不断完善

### 3.5 法与社会的关系

| 关系维度 | 核心观点 |
|----------|----------|
| **法与经济** | 经济决定法，法反作用于经济（促进/阻碍）|
| **法与政治** | 政治主导法的产生和发展，法服务于政治目标 |
| **法与文化** | 文化影响法的价值观，法塑造社会文化环境 |
| **法与道德** | 互相渗透、互相促进，但二者有明确边界 |
| **法与宗教** | 政教分离是我国基本原则，宗教不得干预司法 |

> ⚠️ **法与道德的区别**（法考高频辨析点）：
> - 生成方式不同：法由国家制定/认可；道德在社会生活中自然形成
> - 表现形式不同：法以规范性文件表现；道德存在于意识/舆论中
> - 调整范围不同：法只调整人的行为；道德还调整思想
> - 实施方式不同：法靠国家强制力；道德靠内心信念+舆论压力
> - 内容不同：法侧重权利义务配置；道德重义务轻权利

### 4.2 审判制度（法院系统）

#### 4.2.1 人民法院组织体系

| 层级 | 名称 | 管辖范围 | 备注 |
|------|------|----------|------|
| **最高法院** | 最高人民法院 | 全国范围内有重大影响的第一审案件；二审案件；再审案件 | 设巡回法庭 |
| **高级法院** | 省/自治区/直辖市高级人民法院 | 本辖区内有重大影响的案件 | 不受理一审民事/商事案件（除破产等特殊情形）|
| **中级法院** | 设区的市中级人民法院 | 本辖区内重大案件；涉外案件；知识产权案件 | 可设派出法庭 |
| **基层法院** | 县/区/县级市人民法院 | 一般第一审案件 | 可设人民法庭 |
| **专门法院** | 海事/军事/知识产权法院/金融法院 | 特定类型案件 | 跨行政区划管辖 |

#### 4.2.2 法官任职条件（《法官法》第12条）

```
担任法官必须具备的条件（缺一不可）：
  ① 中国国籍
  ② 拥护宪法+拥护中共领导和社会主义制度
  ③ 良好的政治、业务素质和道德品行
  ④ 正常履行职责的身体条件
  ⑤ 学历要求（满足其一即可）:
     - 法学类本科 + 学士及以上学位
     - 或非法学类本科 + 法律硕士/法学硕士及以上学位
     - 或非法学类本科 + 其他学位 + 法律专业知识
  ⑥ 从事法律工作满5年（法律硕士放宽至4年，法学博士放宽至3年）
  ⑦ 通过国家统一法律职业资格考试取得资格

⚠️ 例外：困难地区经最高法院审核可放宽学历为本科毕业
```

#### 4.2.3 法官的回避制度（《法官法》第23-24条）

| 回避类型 | 具体情形 | 处理 |
|----------|----------|------|
| **职务回避** | 不得兼任人大常委会组成人员、行政机关/监察机关/检察机关职务、企业/事业单位职务、律师/仲裁员/公证员 | 自动回避 |
| **亲属回避（法官之间）**| 夫妻/直系血亲/三代旁系血亲/近姻亲 | 不得同时担任：(1)同一法院院长/副院长/审委委员；(2)院长/副院长+审判员；(3)同庭庭长/副庭长/审判员；(4)上下相邻两级法院院长/副院长 |
| **任职回避（配偶子女）**| 配偶/父母/子女在辖区内任律所合伙人/设立人或以律师身份代理诉讼 | 法官应主动申请回避 |

> ⚠️ 法官离任后**两年内**不得以律师身份担任诉讼代理人/辩护人；**永远不得**担任原任职法院办理案件的诉讼代理人/辩护人（监护人/近亲属代理除外）。被开除后**永久禁业**。

#### 4.2.4 法官的职业保障（《法官法》第七章）

| 保障内容 | 具体规定 |
|----------|----------|
| **履职保障** | 非因法定事由/法定程序，不被调离/免职/降职/辞退/处分 |
| **人身安全保障** | 人身/财产/住所安全受法律保护 |
| **拒绝权** | 任何单位/个人不得要求法官从事超出法定职责范围的事务 |
| **记录报告权** | 干涉办案行为应全面如实记录和报告 |
| **工资待遇** | 按等级享有国家规定工资，建立与公务员工资同步调整机制 |
| **退休保障** | 因公致残享受伤残待遇；因公牺牲/死亡/病故家属享受抚恤优待 |

### 4.3 检察制度（检察院系统）

#### 4.3.1 人民检察院组织体系

| 层级 | 名称 | 主要职能 |
|------|------|----------|
| **最高检察院** | 最高人民检察院 | 领导全国检察工作；对全国人大负责并报告工作 |
| **省级** | 省/自治区/直辖市人民检察院 | 领导辖区内检察工作 |
| **分院/市级** | 省/自治区检察院分院/设区的市级检察院 | 辖区内具体检察业务 |
| **基层** | 县/区/县级市人民检察院 | 直接办理各类案件 |
| **专门** | 军事/铁路运输/知识产权等专门检察院 | 特定领域检察 |

#### 4.3.2 检察官职责（《检察官法》第7条）

```
检察官的六大职责：
  ① 对法律规定由检察院直接受理的刑事案件进行侦查（自侦案件）
  ② 对刑事案件进行审查逮捕、审查起诉，代表国家公诉
  ③ 开展公益诉讼工作
  ④ 开展对刑事、民事、行政诉讼活动的监督工作
  ⑤ 法律规定的其他职责
  ⑥ 检察长/副检察长/检委委员还需履行与其职务相适应的管理职责
```

#### 4.3.3 检察官 vs 法官对比

| 维度 | 法官 | 检察官 |
|------|------|--------|
| **行使权力** | 审判权（居中裁判）| 检察权（主动监督）|
| **核心职能** | 定纷止争 | 追诉犯罪+法律监督 |
| **工作立场** | 中立客观 | 客观公正（既追诉犯罪也保障无罪的人不受追究）|
| **员额制** | ✅ 是（总量控制、动态管理）| ✅ 是 |
| **等级序列** | 12级（首席大法官→五级法官）| 12级（首席大检察官→五级检察官）|
| **遴选委员会** | 省级法官遴选委员会 | 省级检察官遴选委员会 |
| **离任限制** | 2年内不以律师身份代理；永远不代理原法院案件 | 2年内不以律师身份代理；永远不代理原检察院案件 |

### 4.4 律师制度

#### 4.4.1 律师执业许可（《律师法》第二章）

**申请条件（5项必备）**：

| 序号 | 条件 | 说明 |
|------|------|------|
| ① | 拥护中华人民共和国宪法 | 政治门槛 |
| ② | 通过国家统一法律职业资格考试 | 专业资质（含原司考合格证）|
| ③ | 在律师事务所实习满一年 | 实习期要求 |
| ④ | 品行良好 | 道德门槛 |
| ⑤ | 提交完整申请材料 | 程序要求 |

**不予颁发律师证的情形**（《律师法》第7条）：

| 情形 | 能否补救 |
|------|----------|
| 无/限制民事行为能力 | ❌ 永久排除 |
| **受过刑事处罚** | ⚠️ **过失犯罪除外**（故意犯罪的永久排除）|
| 被开除公职 | ❌ 永久排除 |
| 被吊销律师/公证员执业证书 | ❌ 永久排除 |

#### 4.4.2 律师事务所的类型（《律师法》第三章）

| 类型 | 条件 | 责任承担 |
|------|------|----------|
| **合伙律师事务所（普通合伙）**| 3名+合伙人，每人3年+执业经历 | 合伙人对所债务承担无限连带责任 |
| **合伙律师事务所（特殊的普通合伙）** | 同上 | 合伙人因故意/重大过失造成的债务承担无限连带责任；其他合伙人以其出资为限 |
| **个人律师事务所** | 设立人5年+执业经历 | 设立人对所债务承担**无限责任** |
| **国资律师事务所** | 国家出资设立 | 以全部资产承担责任 |

> ⚠️ **重要规则**：律师只能在一个律师事务所执业（第10条），不受地域限制。公务员不得兼任执业律师（第11条）。

#### 4.4.3 律师的业务范围（《律师法》第28条）

| 类别 | 具体业务 |
|------|----------|
| **非诉讼业务** | 法律顾问、代写文书、法律咨询、出具法律意见书 |
| **民事诉讼代理** | 代理当事人参加一审/二审/再审/执行程序 |
| **刑事诉讼辩护** | 犯罪嫌疑人/被告人的辩护人、自诉案件代理人、被害人代理人 |
| **行政代理** | 行政复议/诉讼代理 |
| **ADR参与** | 调解、仲裁活动 |
| **申诉代理** | 各类案件的申诉代理 |

#### 4.4.4 律师的权利（《律师法》第33-37条）

| 权利 | 具体内容 | 法律依据 |
|------|----------|----------|
| **阅卷权** | 自审查起诉之日起查阅/摘抄/复制案卷材料 | 第34条 |
| **会见通信权** | 持三证（执业证+证明+委托书/援助函），会见时不被监听 | 第33条 |
| **调查取证权** | 凭执业证+所证向单位和个人调查取证 | 第35条 |
| **申请取证权** | 申请法院/检察院收集证据、通知证人出庭 | 第35条 |
| **辩论/辩护权** | 辩论/辩护的权利受法律保障 | 第36条 |
| **人身权豁免** | 法庭上的代理/辩护意见不受法律追究（危害国安/恶意诽谤/严重扰乱秩序的除外）| 第37条 |
| **拒辩权** | 委托事项违法/利用服务从事违法/故意隐瞒重要事实时，有权拒绝辩护/代理 | 第32条 |

#### 4.4.5 律师执业禁止行为（《律师法》第40条）

```
律师绝对不能做的8件事（红线！）：

  ❶ 私自接受委托/收费/财物
  ❹ 利用服务便利牟取争议权益
  ❸ 收对方当事人财物/恶意串通侵害委托人
  ❻ 违规会见法官/检察官/仲裁员等
  ❼ 向司法人员行贿/介绍贿赂/诱导当事人行贿
  ❽ 故意提供虚假证据/威胁利诱他人提供假证据
  ❾ 煽动教唆非法手段解决争议
  ❿ 扰乱法庭/仲裁庭秩序
```

#### 4.4.6 律师的法律责任（《律师法》第六章）

**行政处罚阶梯**（由轻到重）：

| 行为 | 处罚 | 法律依据 |
|------|------|----------|
| 同时在两所执业/不正当承揽/双方代理/离任两年内代理/拒绝法律援助 | 警告 + ≤5000元罚款 + 没收违法所得；情节严重：停业≤3月 | 第47条 |
| 私自收费/无正当理由拒绝辩护/泄露隐私/利用服务牟利 | 警告 + ≤10000元罚款 + 没收违法所得；停业3-6月 | 第48条 |
| **违规会见/行贿/提供假证据/扰乱秩序/危害国家安全/泄密** | **停业6月-1年** + ≤50000元罚款 + 没收违法所得；**吊销执业证**；刑事责任 | 第49条 |

> ⚠️ **最严厉后果**：律师因**故意犯罪**受到刑事处罚的 → **吊销律师执业证书**（第49条第二款）。

### 4.5 公证制度

#### 4.5.1 公证机构与公证员

| 要素 | 内容 |
|------|------|
| **公证的定义** | 公证机构根据申请，依照法定程序对民事法律行为/有法律意义的事实/文书的真实性/合法性予以证明的活动 |
| **公证机构性质** | 依法设立、**不以营利为目的**、独立行使公证职能、承担民事责任的**证明机构** |
| **设立原则** | 统筹规划、合理布局，**不按行政区划层层设立** |
| **设立条件** | (1)名称 (2)固定场所 (3)**2名以上公证员** (4)必要资金 |
| **公证员的任职年龄** | **25周岁以上65周岁以下** |
| **公证员的学历要求** | 通过法考 + 在公证机构**实习2年以上**或有3年以上其他法律职业经历并在公证机构实习**1年以上** |

#### 4.5.2 公证的效力（《公证法》第五章）

| 效力类型 | 内容 | 法律依据 |
|----------|------|----------|
| **证据效力** | 经公证的事实/文书应当作为认定事实的根据（但有相反证据推翻的除外）| 第36条 |
| **强制执行力** | 经公证的给付债权文书（载明债务人愿意接受强制执行承诺），债权人可直接向法院申请执行 | 第37条 |
| **生效要件效力** | 法律行政法规规定未经公证的事项不具有法律效力的 | 第38条 |
| **更正/撤销** | 错误的公证书应更正；违法/不符事实的应撤销并公告（**自始无效**）| 第39条 |

#### 4.5.3 公证业务范围

**可以公证的事项**（《公证法》第11条，共11类）：

```
  ①合同  ②继承  ③委托/声明/赠与/遗嘱  ④财产分割
  ⑤招标投标/拍卖  ⑥婚姻状况/亲属关系/收养关系
  ⑦出生/生存/死亡/身份/经历/学历/学位/职务/职称/有无犯罪记录
  ⑧公司章程  ⑨保全证据
  ⑩签名/印鉴/日期/副本与原本相符  ⑪其他自愿申请的事项
```

**可以办理的事务**（《公证法》第12条，5类）：登记、提存、保管遗嘱/遗产/财产/物品、代写文书、法律咨询。

**不予公证的情形**（《公证法》第31条，9种）：
无行为能力无监护人代理、与事项无利害关系、专业技术鉴定评估事项、当事人有争议、虚构隐瞒事实/虚假材料、材料不充分且拒绝补充、事项不真实不合法、违背社会公德、拒绝付费。

### 4.6 仲裁制度（2025最新版）

#### 4.6.1 仲裁的基本原则

| 原则 | 内容 | 体现条文 |
|------|------|----------|
| **自愿原则** | 必须达成书面仲裁协议才能仲裁 | 第4条 |
| **独立仲裁原则** | 仲裁依法独立进行，不受行政机关/社会团体/个人干涉 | 第9条 |
| **一裁终局原则** | 裁决作出后就同一纠纷不能再申请仲裁或起诉 | 第10条 |
| **诚实信用原则** | 2025新增！当事人在仲裁活动中应诚信行事 | 第8条 |
| **在线仲裁** | 2025新增！可通过信息网络在线进行，与线下同等效力 | 第11条 |

#### 4.6.2 仲裁协议（《仲裁法》第三章）

**仲裁协议的有效要件**（第27条）：

```
有效的仲裁协议 = 三要素（缺一不可）：
  ① 请求仲裁的意思表示（明确选择仲裁）
  ② 仲裁事项（约定了哪些纠纷提交仲裁）
  ③ 选定的仲裁机构（明确了去哪个仲裁委）

⚠️ 默认承认规则：一方主张有仲裁协议，
   另一方在首次开庭前不予否认的 → 视为存在仲裁协议
```

**仲裁协议无效的五种情形**（第28条）：
1. 约定的仲裁事项超出法定仲裁范围
2. 无民事/限制民事行为能力人订立的
3. 一方采取**胁迫手段**迫使对方订立
4. 仲裁事项/机构没有约定或约定不明确，且无法补充协议
5. （隐含）口头仲裁协议无效（必须是**书面**形式）

**仲裁协议的独立性**（第30条）：🔑🔑🔑 **法考超级高频点**

> 仲裁协议**独立于主合同存在**！合同的变更/不生效/终止/被撤销/无效，**不影响已达成的仲裁协议的效力**。
> 仲裁庭有权确认合同效力！

#### 4.6.3 仲裁程序流程图

```
申请仲裁（有协议+有请求+在受理范围）
        │
        ▼
  机构受理（收到申请书后5日内决定）
        │
        ├─ 不予受理 → 书面说明理由 → 可另行提起诉讼
        │
        ▼
  组成仲裁庭
        │
        ├─ 独任仲裁庭（1人）— 当事人约定或主任指定
        └─ 合议仲裁庭（3人）— 双方各选1人 + 共同选/指定首席
        │
        ▼
  开庭审理
        │
        ├─ 原则：开庭进行（第51条）
        ├─ 原则：不公开进行（第52条）— 当事人协议公开的除外（涉密除外）
        ├─ 申请人不到庭/中途退庭 → 视为撤回申请（第54条）
        └─ 被申请人不到庭/中途退庭 → 缺席裁决（第54条）
        │
        ▼
  作出裁决（多数意见；形不成多数按首席意见）（第66条）
        │
        ├─ 裁决书自作出之日生效（第70条）
        ├─ 一裁终局（第10条）
        │
        ├─ 不服？→ 只能申请撤销（第71条，6种情形，3个月期限，中级法院）
        │           或申请不予执行（第76条）
        │
        └─ 执行 → 向法院申请（第75条，依民事诉讼法）
```

#### 4.6.4 2025年仲裁法修订要点（新增/变化）

| 变化领域 | 旧版 | 新版（2025修订，2026.3.1施行）| 影响 |
|----------|------|------|------|
| **机构性质** | 未明确 | 明确为**公益性非营利法人** | 与行政机关脱钩 |
| **在线仲裁** | 无明确规定 | 明确支持**线上仲裁**，与线下同等效力 | 数字化便民 |
| **诚信原则** | 无 | 新增第8条 | 打击虚假仲裁 |
| **仲裁地制度** | 无 | 新增第81条，允许约定仲裁地作为程序法和管辖依据 | 涉外仲裁重大利好 |
| **临时仲裁** | 无 | 第82条允许自贸试验区/海南自贸港等特定区域内**临时仲裁**（符合条件的人员组成仲裁庭）| 国际接轨 |
| **境外机构入境** | 无 | 第86条允许境外仲裁机构在自贸区设立业务机构 | 开放力度空前 |
| **仲裁员条件** | 从事仲裁工作/律师/曾任法官/教学研究满8年 | 同左 + 新增公职人员兼任的限制 | 更严格 |
| **仲裁员披露** | 仅规定回避 | 新增第45条，**可能导致独立性/公正性怀疑时应主动披露** | 透明度提高 |
| **恶意仲裁** | 无 | 新增第61条，发现捏造事实/恶意串通的**驳回仲裁请求** | 防范滥用 |
| **条文总数** | 80条 | **96条** | 体系更完整 |

### 4.7 人民调解制度

#### 4.7.1 基本原则（《人民调解法》第3条）

| 原则 | 含义 |
|------|------|
| **自愿平等** | 在当事人自愿、平等的基础上调解 |
| **合法合规** | 不违背法律法规和国家政策 |
| **尊重权利** | 不得因调解而阻止当事人依法通过仲裁/行政/司法途径维护权利 |

> ⭐ **免费原则**：人民调解**不收取任何费用**（第4条）！

#### 4.7.2 人民调解委员会

| 要素 | 内容 |
|------|------|
| **性质** | 依法设立的调解民间纠纷的**群众性组织**（第7条）|
| **设立地点** | 村委会/居委会（必须）；企事业单位（可选）（第8条）|
| **组成人数** | 3-9人，设主任1人，必要时副主任若干（第8条）|
| **委员任期** | 3年，可连选连任（第9条）|
| **人员结构** | 应有妇女成员；多民族地区应有少数民族成员（第8条）|

#### 4.7.3 调解程序

```
启动方式：
  ├─ 当事人申请调解（第17条）
  └─ 调解委员会主动调解（但一方明确拒绝的不得调解）

调解过程：
  ├─ 指定/选定调解员（第19条）
  ├─ 可邀请有关人员参与（第20条）
  ├─ 及时就地调解，防止矛盾激化（第21条）
  └─ 调解不成 → 终止 → 告知其他救济途径（第26条）

结果处理：
  ├─ 达成协议：
  │   ├─ 制作调解协议书（各方签名+盖章+指印+调解员签名+调委会印章）→ 生效
  │   └─ 口头协议 → 调解员记录 → 生效
  │
  └─ 司法确认（第33条）— 🌟🌟🌟 法考超级重点！
      → 协议生效后30日内共同向法院申请确认
      → 法院确认有效 → 拒绝履行可申请**强制执行**！
      → 法院确认无效 → 可重新调解 or 起诉
```

#### 4.7.4 人民调解 vs 其他ADR方式对比

| 维度 | 人民调解 | 仲裁 | 法院调解 |
|------|----------|------|----------|
| **主持者** | 调解委员会（群众性组织）| 仲裁庭 | 法院/法官 |
| **费用** | 免费 | 收费 | 诉讼费减半 |
| **效力** | 有约束力，需司法确认才可强执 | 一裁终局，可申请执行 | 调解书直接具有执行力 |
| **启动** | 申请或主动 | 必须有仲裁协议 | 诉讼中法官提议或当事人申请 |
| **保密性** | 较高 | 不公开（原则上）| 公开（涉及隐私除外）|

### 4.8 法律职业道德

#### 4.8.1 法律职业共同体道德准则

**所有法律职业人员的共同道德底线**：

| 准则 | 具体要求 |
|------|----------|
| **忠于宪法和法律** | 依法履职，维护法律尊严 |
| **忠于事实真相** | 客观公正，不伪造/篡改证据 |
| **恪守职业道德** | 清正廉明，不接受请托贿赂 |
| **保守职业秘密** | 国家秘密+商业秘密+个人隐私+当事人信息 |
| **尊重职业礼仪** | 文明执业，维护职业形象 |
| **勤勉尽责** | 认真对待每一起案件/每一项业务 |

#### 4.8.2 各职业特有道德要求

**法官职业道德**（"六个严禁"）：
1. 严禁接受当事人及其代理人请客送礼
2. 严禁违反规定与律师/当事人不正当接触交往
3. 严禁插手过问他人办理的案件
4. 严禁违规干预过问案件
5. 严禁泄露审判秘密
6. 严禁滥用职权/徇私枉法

**检察官职业道德**：
- 客观公正（既追诉犯罪也保障无辜者不受追究）
- 依法独立行使检察权
- 保守检察工作秘密
- 文明办案，尊重人权

**律师职业道德**：
- 维护当事人合法权益（但不得损害社会公共利益）
- 保守执业秘密（但准备/正在实施的严重危害国家安全/公共安全/人身安全的犯罪除外——**例外披露权**）
- 利益冲突审查（同一案件不得为双方代理）
- 诚实守信，勤勉尽责
- 不得低价恶性竞争

**公证员职业道德**：
- 客观公正，真实合法
- 保守执业秘密
- 不得同时在两个以上公证机构执业
- 不得从事有报酬的其他职业
- 不得为自己及近亲属办理公证

**仲裁员职业道德**：
- 独立公正（披露可能引起独立性怀疑的情形）
- 主动回避（与案件有利害关系/私自会见当事人等）
- 勤勉高效
- 保密义务

#### 4.8.3 法律职业道德的惩戒体系

| 职业 | 惩戒机构 | 惩戒种类 |
|------|----------|----------|
| 法官 | 法官惩戒委员会（省+最高两级）| 警告/记过/降级/撤职/开除 |
| 检察官 | 检察官惩戒委员会（省+最高两级）| 同上 |
| 律师 | 司法行政部门 + 律师协会 | 警告/罚款/停止执业/吊销执照 |
| 公证员 | 司法行政部门 | 警告/罚款/停止执业/吊销执照 |
| 仲裁员 | 仲裁机构 | 除名 |
