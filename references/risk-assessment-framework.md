# 案件风险评估框架

> 定位：评估一个案件值不值得打、能不能赢的系统方法
> 创建时间：2026-04-29
> 版本：v1.0 | Phase 2 模块5交付物

---

## 一、五维评估模型

### 1.1 评分维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **① 证据强度** | 30% | 核心证据是否充分、能否形成完整证据链 |
| **② 法律依据** | 25% | 法律支持是否明确、有无争议 |
| **③ 对方实力** | 20% | 对方有无律师、财力、反驳能力 |
| **④ 执行可能** | 15% | 赢了能不能拿到钱 |
| **⑤ 时间成本** | 10% | 预计耗时与当事人承受能力 |

### 1.2 各维度评分标准

#### ① 证据强度（0-100分）

| 分数 | 标准 |
|------|------|
| 80-100 | 核心证据齐全（书证原件+银行流水+录音/录像），形成完整证据链 |
| 60-79 | 主要证据有但不完整，部分关键事实只有间接证据 |
| 40-59 | 证据薄弱，核心事实依赖当事人陈述或证人证言 |
| 20-39 | 证据严重不足，主要靠对方举证不能来赢 |
| 0-19 | 几乎没有证据 |

#### ② 法律依据（0-100分）

| 分数 | 标准 |
|------|------|
| 80-100 | 法条明确支持，有利的司法解释或指导案例 |
| 60-79 | 法条支持但存在解释空间，主流观点有利 |
| 40-59 | 法律规定模糊，各地裁判口径不一 |
| 20-39 | 法律支持薄弱，需要突破性论证 |
| 0-19 | 法律明显不支持 |

#### ③ 对方实力（0-100分，分数越高对我方越有利）

| 分数 | 标准 |
|------|------|
| 80-100 | 对方无律师、证据薄弱、可能缺席 |
| 60-79 | 对方有律师但证据不强 |
| 40-59 | 对方有专业律师且有一定证据 |
| 20-39 | 对方实力强、律师专业、证据充分 |
| 0-19 | 对方为大型企业/机构，法务团队强大 |

#### ④ 执行可能（0-100分）

| 分数 | 标准 |
|------|------|
| 80-100 | 对方有明确可执行财产（房/车/银行账户/正常经营企业） |
| 60-79 | 对方有财产但可能需要查找 |
| 40-59 | 对方财产状况不明 |
| 20-39 | 对方可能是空壳公司或经济困难 |
| 0-19 | 对方明确无财产可供执行 |

#### ⑤ 时间成本（0-100分，分数越高越快）

| 分数 | 标准 |
|------|------|
| 80-100 | 预计3个月内结案（仲裁/简易程序） |
| 60-79 | 预计6个月内结案 |
| 40-59 | 预计6-12个月 |
| 20-39 | 预计1-2年（含二审/执行） |
| 0-19 | 预计超过2年 |

### 1.3 综合评分计算

```
总分 = 证据×30% + 法律×25% + 对方×20% + 执行×15% + 时间×10%
```

### 1.4 风险等级判定

| 总分 | 风险等级 | 建议 |
|------|----------|------|
| ≥ 75 | 🟢 **低风险** | 建议起诉/仲裁，胜诉概率高 |
| 60-74 | 🟡 **中低风险** | 可以打，但需充分准备 |
| 45-59 | 🟠 **中等风险** | 需权衡成本收益，建议先调解 |
| 30-44 | 🔴 **高风险** | 胜诉概率不高，慎重考虑 |
| < 30 | ⚫ **极高风险** | 不建议起诉，除非有特殊考虑 |

## 二、快速筛选清单

### 2.1 值得打的案件（满足3个以上）

- [ ] 有书面证据（合同/收据/通知/判决书等原件）
- [ ] 法律明确支持你的请求
- [ ] 对方有可执行的财产
- [ ] 标的额显著大于诉讼成本
- [ ] 你有时间和精力应对诉讼
- [ ] 仲裁时效/诉讼时效未过期

### 2.2 不建议打的案件（满足2个以上要警惕）

- [ ] 几乎没有书面证据，全靠口头约定
- [ ] 时效已过或即将过期
- [ ] 对方是空壳公司或明显无财产
- [ ] 标的额小于预计诉讼成本
- [ ] 法律不支持你的核心请求
- [ ] 你的核心诉求在法律上存在明显障碍

## 三、风险告知话术

### 3.1 低风险案件

```
根据您提供的材料，这个案件证据充分、法律依据明确，
胜诉把握较大。主要风险在于[具体风险点]。
建议尽快启动[诉讼/仲裁]程序。

预估成本约[XXX]元（含诉讼费+律师费），
预计耗时[X]个月，胜诉后诉讼费由对方承担。
```

### 3.2 中等风险案件

```
这个案件有一定胜算，但也存在以下风险：
1. [风险点1，如"关键证据只有复印件"]
2. [风险点2，如"法律规定存在解释空间"]

建议：先尝试协商/调解，调解不成再考虑诉讼。
如果决定诉讼，需要补充[XXX]方面的证据。

预估成本约[XXX]元。败诉情况下自行承担诉讼费和律师费。
```

### 3.3 高风险案件

```
坦率地说，这个案件目前面临较大困难：
1. [核心困难，如"缺乏直接证据"]
2. [核心困难，如"法律依据不明确"]

如果坚持要打，胜诉概率约[XX]%，
需要做好败诉的心理和经济准备。

建议：优先考虑协商解决，或者先补充证据后再评估。
```

## 四、特殊案件风险提示

### 4.1 劳动争议风险特点

| 有利因素 | 不利因素 |
|----------|----------|
| 仲裁免费 | 可能需要漫长的仲裁+诉讼两道程序 |
| 举证责任部分由公司承担 | 公司可能毁灭/伪造证据 |
| 一裁终局对劳动者有利 | 公司可能转移资产 |
| 法律对劳动者倾斜保护 | 维权期间无收入 |

### 4.2 执行风险评估

| 对方类型 | 执行风险 | 建议 |
|----------|----------|------|
| 国企/大型企业 | 🟢 低 | 有充分资产，通常会履行 |
| 中小企业 | 🟡 中 | 需提前查财产，考虑保全 |
| 个体工商户 | 🟠 中高 | 经营不稳定，建议保全 |
| 自然人 | 🔴 高 | 需查房/车/银行账户 |
| 空壳公司 | ⚫ 极高 | 考虑追加股东/实控人 |
