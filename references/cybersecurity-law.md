## 二、总则核心要点（第1-15条）

### 1. 立法目的（第1条）
保障网络安全 → 维护网络空间主权和国家安全 → 保护公民/法人/其他组织合法权益 → 促进经济社会信息化健康发展

### 2. 适用范围（第2条）
在**中华人民共和国境内**建设、运营、维护和使用网络，以及网络安全的监督管理。

### 3. 基本方针（第4条）
**十六字方针**：积极利用、科学发展、依法管理、确保安全

### 4. 监管体制（第9条）——"1+2+N"架构

| 层级 | 机构 | 职责 |
|------|------|------|
| 统筹协调 | **国家网信部门**（网信办） | 统筹协调网络安全工作和相关监管 |
| 主管部门 | 电信主管部门（工信部） | 电信和互联网行业网络安全 |
| 主管部门 | 公安部门 | 网络安全等级保护、打击网络犯罪 |
| 其他部门 | 各行业主管部门 | 各自职责范围内的网络安全保护 |

### 5. 核心原则汇总

| 原则 | 条款 | 内容 |
|------|------|------|
| 网络空间主权 | 第1条 | 维护网络空间主权（国际法新概念） |
| 总体国家安全观 | 第3条 | 统筹发展和安全 |
| 实名制 | 第26条 | 网络接入/域名/电话/信息发布/即时通讯均需实名 |
| 数据本地化 | 第39条 | CII运营者个人信息和重要数据境内存储 |
| 等级保护 | 第23条 | 国家实行网络安全等级保护制度 |

### 6. 公民权利与义务

**权利**：
- 依法使用网络的权利（第13条）
- 对危害网络安全行为的举报权（第15条）
- 举报人信息保密保护（第15条）

**义务/禁止行为**（第13条第二款）：
- 不得危害网络安全
- 不得利用网络从事危害国家安全等活动
- 不得传播暴力、淫秽色情信息
- 不得编造传播虚假信息扰乱秩序
- 不得侵害他人名誉、隐私、知识产权

## 三、网络安全支持与促进（第16-22条）

### 核心支持措施

| 方向 | 条款 | 具体措施 |
|------|------|----------|
| 标准体系 | 第16条 | 国家标准+行业标准，企业/高校/行业组织参与制定 |
| 技术创新 | 第17条 | 国务院+省级政府统筹规划，扶持重点产业，保护知识产权 |
| 安全服务 | 第18条 | 推进社会化服务（认证、检测、风险评估） |
| 数据利用 | 第19条 | 鼓励数据安全保护和利用技术，促进公共数据开放 |
| **AI支持** | **第20条** | **支持AI基础研究+关键技术研发+算力基建+伦理规范+风险监测+安全监管**（2025年修正新增） |
| 宣传教育 | 第21条 | 各级政府组织网络安全宣传教育 |
| 人才培养 | 第22条 | 支持企业和高校开展网络安全教育培训 |

> 💡 **2025年修正亮点**：第20条专门新增AI条款，体现国家对人工智能安全治理的重视。这是本次修法最具前瞻性的增补之一。

## 四、网络运行安全——一般规定（第23-32条）

### 1. 网络安全等级保护制度（第23条）——核心制度

网络运营者**五大义务**：

| 序号 | 义务 | 要点 |
|------|------|------|
| ① | 安全管理制度 | 制定制度和操作规程，确定网络安全负责人 |
| ② | 技术防护 | 防范病毒、网络攻击、网络侵入 |
| ③ | 监测记录 | 网络运行状态和安全事件技术监测，**日志留存≥6个月** |
| ④ | 数据保护 | 数据分类、重要数据备份和加密 |
| ⑤ | 其他法定义务 | 法律行政法规规定的 |

### 2. 网络产品和服务（第24条）

- 符合国家标准强制性要求
- **禁止设置恶意程序**
- 发现安全缺陷/漏洞 → 立即补救 + 及时告知用户 + 报告主管部门
- **持续提供安全维护**，约定期限内不得终止
- 收集用户信息 → 明示并取得同意

### 3. 网络关键设备和安全专用产品（第25条）
必须经**安全认证合格**或**安全检测符合要求**后方可销售/提供

### 4. 网络实名制（第26条）——重要制度

适用场景：
- 网络接入
- 域名注册
- 固定电话、移动电话入网
- 信息发布、即时通讯等服务

**用户不提供真实身份信息 → 运营者不得提供服务**

配套制度：网络可信身份战略 + 电子身份认证互认

### 5. 安全事件应急（第27条）
制定应急预案 → 及时处置风险 → 发生事件立即启动 → 补救 + 报告

### 6. 禁止网络攻击（第29条）——三大禁止

| 序号 | 禁止行为 |
|------|----------|
| ① | 非法侵入他人网络、干扰正常功能、窃取数据 |
| ② | 提供专门用于网络攻击的程序、工具 |
| ③ | 明知他人危害网络安全仍提供技术支持/广告/支付结算等帮助 |

### 7. 协助执法义务（第30条）
网络运营者**应当为公安机关、国家安全机关依法维护国家安全和侦查犯罪提供技术支持和协助**

### 8. 信息用途限制（第32条）
网信部门获取的信息**只能用于维护网络安全，不得用于其他用途**

## 五、关键信息基础设施(CII)保护（第33-41条）——重中之重

### 1. CII定义和范围（第33条）

**涵盖领域**：公共通信和信息服务、能源、交通、水利、金融、公共服务、电子政务等重要行业和领域

**判定标准**：一旦遭到破坏、丧失功能或数据泄露 → 可能**严重危害国家安全、国计民生、公共利益**

**保护原则**：在等级保护基础上实行**重点保护**

### 2. CII运营者额外义务（第36条）——在普通运营者义务之上

| 序号 | 额外义务 |
|------|----------|
| ① | 设置**专门安全管理机构和负责人** + 关键岗位**安全背景审查** |
| ② | 定期网络安全**教育培训和技能考核** |
| ③ | 重要系统和数据库**容灾备份** |
| ④ | 制定应急预案并**定期演练** |
| ⑤ | 其他法定义务 |

### 3. 国家安全审查（第37条）
CII运营者采购网络产品和服务 → 可能影响国家安全 → 必须通过**国家网信部门会同国务院有关部门组织的国家安全审查**

### 4. 数据本地化（第39条）——数据出境核心条款

```
CII运营者境内运营中收集和产生的个人信息和重要数据
                    ↓
              必须在境内存储
                    ↓
    确需向境外提供 → 安全评估（国家网信部门制定办法）
```

### 5. 年度安全检测（第40条）
CII运营者 → 自行或委托安全服务机构 → **每年至少一次**安全性检测评估 → 报送结果和改进措施

## 六、网络信息安全（第42-52条）

### 1. 个人信息保护体系

| 条款 | 规则 | 核心要求 |
|------|------|----------|
| 第42条 | 保密义务 | 严格保密 + 建立用户信息保护制度，遵守民法典和个保法 |
| 第43条 | 收集原则 | **合法、正当、必要** + 公开规则 + 明示目的方式范围 + **经被收集者同意** |
| 第43条 | 最小必要 | 不得收集与服务**无关**的个人信息 |
| 第44条 | 安全保障 | 不得泄露/篡改/毁损 + 未经同意不得向他人提供（匿名化除外） |
| 第44条 | 泄露响应 | 发生或可能发生泄露 → **立即补救 + 及时告知用户 + 报告主管部门** |
| 第45条 | 用户权利 | **删除权** + **更正权**（运营者应当配合） |
| 第46条 | 禁止非法获取 | 不得窃取/非法获取/非法出售/非法提供个人信息 |
| 第47条 | 监管者义务 | 监管部门工作人员对个人信息/隐私/商业秘密严格保密 |

### 2. 内容管理

| 条款 | 主体 | 义务 |
|------|------|------|
| 第48条 | 任何人和组织 | 不得设立违法犯罪网站/群组，不得发布违法犯罪信息 |
| 第49条 | 网络运营者 | 发现违法信息 → **立即停止传输 + 消除 + 防扩散 + 保存记录 + 报告** |
| 第50条 | 电子信息/APP提供者 | 不得设置恶意程序，不得含违法信息 |
| 第51条 | 网络运营者 | 建立投诉举报制度 + 配合监督检查 |
| 第52条 | 国家网信部门 | 发现违法信息 → 要求停止传输 + 消除 + 阻断境外来源 |

## 七、监测预警与应急处置（第53-60条）

### 应急响应机制

```
日常状态：网络安全监测预警和信息通报制度（第53条）
              ↓
风险增大：省级以上政府有关部门采取措施（第56条）
         ├── 要求收集报告信息
         ├── 分析评估预测
         └── 发布风险预警
              ↓
事件发生：启动应急预案（第57条）
         ├── 调查评估
         ├── 要求运营者采取技术措施
         ├── 消除安全隐患
         └── 向社会发布警示信息
              ↓
严重情况：约谈法定代表人/主要负责人（第58条）
              ↓
极端情况：经国务院决定/批准，特定区域限制网络通信（第60条）
```

### 关键制度

| 制度 | 条款 | 内容 |
|------|------|------|
| 监测预警 | 第53条 | 国家网信部门统一发布网络安全监测预警信息 |
| 行业监测 | 第54条 | CII保护部门建立本行业监测预警制度 |
| 应急预案 | 第55条 | 分级管理 — 按危害程度和影响范围分级 |
| 约谈制度 | 第58条 | 发现较大风险/安全事件 → 约谈运营者法人/负责人 |
| 网络限制 | 第60条 | 维护国安/公共秩序 + 处置重大突发事件 → 经**国务院决定/批准** → 特定区域限制网络通信 |

## 八、法律责任体系（第61-77条）

### 1. 罚款层级体系

| 违法严重程度 | 普通运营者罚款 | CII运营者罚款 | 直接责任人罚款 |
|-------------|---------------|---------------|---------------|
| 一般违法 | 1万-5万 | 5万-10万 | 1万-10万 |
| 拒不改正/造成后果 | 5万-50万 | 10万-100万 | 1万-10万 |
| **严重后果**（大量数据泄露/CII部分瘫痪） | **50万-200万** | **50万-200万** | **5万-20万** |
| **特别严重后果**（CII主要功能丧失） | **200万-1000万** | **200万-1000万** | **20万-100万** |

> 💡 **2025年修正重大变化**：第61条新增第三、四款，大幅提高严重/特别严重后果的罚款上限至**1000万**，直接责任人最高**100万**。力度倍增。

### 2. 各类违法行为的具体处罚

| 违法行为 | 条款 | 处罚 |
|----------|------|------|
| 不履行等保义务/应急预案 | 第61条 | 警告→罚款1万-50万→严重50万-1000万 |
| 设置恶意程序/不修漏洞/停止维护 | 第62条 | 警告→罚款5万-50万→严重50万-1000万 |
| 销售未认证网络关键设备 | 第63条 | 停止销售+警告+没收+罚款2万-违法所得5倍+吊照 |
| 未落实实名制 | 第64条 | 罚款5万-50万+停业整顿+吊照 |
| 违规发布安全信息 | 第65条 | 警告→罚款1万-100万+停业+吊照 |
| **从事网络攻击活动** | **第66条** | **公安机关：拘留5-15日+罚款5万-100万+从业禁止** |
| CII未经安全审查使用产品 | 第67条 | 采购金额**1-10倍**罚款 |
| 设立违法犯罪网站/群组 | 第68条 | 公安机关：拘留5-15日+罚款1万-50万+关闭 |
| 未处置违法信息 | 第69条 | 罚款5万-200万→特别严重200万-1000万+停业+吊照 |
| 拒绝监督检查/拒绝协助执法 | 第70条 | 罚款5万-50万 |

### 3. 从业禁止制度（第66条第三款）

| 情形 | 禁止期限 |
|------|----------|
| 受治安管理处罚 | **5年内**不得从事网络安全管理和网络运营关键岗位 |
| 受刑事处罚 | **终身**不得从事网络安全管理和网络运营关键岗位 |

### 4. 其他法律责任

| 类型 | 条款 | 内容 |
|------|------|------|
| 信用惩戒 | 第72条 | 违法行为记入信用档案并公示 |
| 从宽处理 | 第73条 | 符合行政处罚法从轻/减轻/不予处罚情形的依规从宽 |
| 政务网络 | 第74条 | 国家机关不履行义务 → 上级机关责令改正 + 处分 |
| 监管者责任 | 第75条 | 监管信息用于其他用途/玩忽职守/滥用职权 → 处分 |
| 民事责任 | 第76条 | 违法造成损害 → 依法承担民事责任 |
| 刑事责任 | 第76条 | 构成犯罪 → 依法追究刑事责任 |
| **境外制裁** | **第77条** | 境外主体危害中国网络安全 → 追究法律责任 + **冻结财产/其他制裁** |

## 九、附则——核心术语定义（第78条）

| 术语 | 定义 |
|------|------|
| **网络** | 由计算机或其他信息终端及相关设备组成的按照一定规则和程序对信息进行收集、存储、传输、交换、处理的系统 |
| **网络安全** | 防范网络攻击/侵入/干扰/破坏/非法使用及意外事故，使网络稳定可靠运行，保障数据**完整性、保密性、可用性**（CIA三性） |
| **网络运营者** | 网络的所有者、管理者和网络服务提供者 |
| **网络数据** | 通过网络收集、存储、传输、处理和产生的各种电子数据 |
| **个人信息** | 以电子或其他方式记录的能够单独或结合识别自然人个人身份的各种信息（姓名、出生日期、身份证号、生物识别信息、住址、电话等） |

> 💡 注意：网络安全的定义暗含了**CIA三要素**（Confidentiality机密性、Integrity完整性、Availability可用性），这是国际通用的信息安全基础模型。

## 十、2025年修正要点对比

| 修正内容 | 原条款 | 修正后 | 意义 |
|----------|--------|--------|------|
| 新增AI条款 | 无 | 第20条：支持AI基础研究+算力基建+伦理规范+风险监测+安全监管 | 应对AI时代安全挑战 |
| 新增AI赋能网安 | 无 | 第20条第二款：运用AI提升网络安全保护水平 | 技术手段现代化 |
| 加重处罚上限 | 第61条罚款上限50万/10万 | 严重后果50万-200万，特别严重200万-1000万；直接责任人5万-100万 | 大幅提高违法成本 |
| 新增分级处罚 | 第61条仅两档 | 新增第三、四款，按后果严重程度四档处罚 | 罚过相当，精细化 |
| CII激励条款 | 无 | 第33条第二款：鼓励非CII运营者自愿参与保护体系 | 扩大保护范围 |
| 总体国家安全观 | "国家坚持网络安全" | 第3条增加"贯彻总体国家安全观，统筹发展和安全，推进网络强国建设" | 上位法精神贯彻 |
| 处罚缓冲 | 无 | 第73条：具有行政处罚法规定从轻/减轻/不予处罚情形的，依规处理 | 罚教结合，避免一刀切 |
| 个人信息保护衔接 | 仅提及"本法" | 第42条第二款明确衔接"民法典""个人信息保护法" | 法律体系协调 |

| **1-10倍** | CII未经安全审查使用产品的罚款倍数（按采购金额） |
| **5年** | 受治安处罚后的从业禁止期（第66条） |
| **终身** | 受刑事处罚后的从业禁止期（第66条） |
| **5-15日** | 网络攻击行为的拘留期限（第66条） |

## 十二、与已学法律的交叉关联

### 1. 与《宪法》的关系
- 第1条"维护网络空间主权"→ 宪法序言和总纲中的国家主权原则
- 第3条"总体国家安全观"→ 宪法关于国家安全的规定
- 第13条"保障网络信息依法有序自由流动"→ 宪法第35条言论自由 + 第40条通信自由和通信秘密
- 第60条"特定区域限制网络通信"→ 宪法第51条权利行使不得损害国家/社会/集体利益

### 2. 与《民法典》的关系
- 第42条明确引用"遵守民法典"→ 民法典人格权编（隐私权、个人信息保护）
- 第43条"合法、正当、必要"原则 → 民法典第1035条个人信息处理原则
- 第44条"不得泄露、篡改、毁损"→ 民法典第1038条信息处理者安全保障义务
- 第45条"删除权、更正权"→ 民法典第1037条自然人的查阅、复制、更正、删除权
- 第76条"依法承担民事责任"→ 民法典侵权责任编

### 3. 与《刑法》的关系
- 第29条禁止网络攻击 → 刑法第285条非法侵入计算机信息系统罪、第286条破坏计算机信息系统罪
- 第46条禁止非法获取个人信息 → 刑法第253条之一侵犯公民个人信息罪
- 第48条禁止设立违法犯罪网站 → 刑法第287条之一非法利用信息网络罪、第287条之二帮助信息网络犯罪活动罪
- 第66条/第76条"构成犯罪的，依法追究刑事责任"→ 刑法相关条款
- 第77条境外制裁 → 刑法第8条保护管辖权

### 4. 与《劳动法》的关系
- 第36条CII关键岗位人员安全背景审查 → 劳动法第17条劳动合同可约定特殊条款
- 第36条定期安全教育培训 → 劳动法第68条用人单位培训义务
- 第66条从业禁止 → 劳动法第25条过失解除的延伸（因违法丧失从业资格）

### 2. "小明"案例的网络安全法视角

回顾之前的小明案例，如果小明的行为发生在网络空间（比如在线盗刷Apple账户购买手机）：

| 行为 | 适用条款 |
|------|----------|
| 非法侵入Apple系统 | 第29条 → 第66条（拘留5-15日 + 罚款5万-100万） |
| 窃取个人账户信息 | 第46条 → 第71条（公安机关依法处罚） |
| 如果使用攻击工具 | 第29条"提供/使用专门工具" → 第66条 + 刑法第285条 |
| 受刑事处罚后 | 第66条第三款 → **终身**不得从事网安管理和关键岗位 |

### 3. 网络安全法与数据安全法、个保法的"三驾马车"

```
     网络安全法（2017施行，2025修正）
            ↑ 基础性、框架性
            │
    ┌───────┴───────┐
    │               │
数据安全法        个人信息保护法
 (2021施行)        (2021施行)
  数据安全治理      个人信息权益保护
```

网络安全法是"三驾马车"中最早施行的基础法律，数据安全法和个保法在其框架上深化和细化。
