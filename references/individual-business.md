# 个体工商户法律实务

> 来源：律师成长计划阶段四 · 模块三
> 学习日期：2026年4月7日
> 核心法规：民法典 + 促进个体工商户发展条例 + 个体工商户登记管理规定(2025)

---

## 一、个体工商户的法律地位

### 1.1 法律定义

> **《民法典》第54条**：自然人从事工商业经营，经依法登记，为个体工商户。

### 1.2 核心特征

| 特征 | 说明 |
|------|------|
| 主体性质 | 自然人经营，**不具备法人资格** |
| 财产关系 | 经营者财产与个人/家庭财产绑定，不独立 |
| 责任承担 | **无限责任**——以个人全部财产承担债务 |
| 经营范围 | 工商业经营（较灵活） |
| 从业人数 | 无明确上限，可雇工 |

### 1.3 责任范围

| 经营形式 | 责任范围 |
|----------|----------|
| 个人经营 | 以**个人财产**承担 |
| 家庭经营 | 以**家庭财产**承担 |

---

## 二、登记注册

### 2.1 登记依据

- 《促进个体工商户发展条例》（国务院令第755号，2022年11月1日施行）
- 《个体工商户登记管理规定》（市场监管总局令第103号，2025年7月15日施行）

### 2.2 登记事项

| 登记项 | 说明 |
|--------|------|
| 经营者姓名 | 个人经营/家庭经营 |
| 组成形式 | 个人经营 or 家庭经营 |
| 经营范围 | — |
| 经营场所 | 固定场所 |
| 字号（可选） | 可以不用字号 |

### 2.3 登记流程

```
准备材料
  ├── 身份证
  ├── 经营场所证明
  └── 申请书
    ↓
向经营场所所在地市场监管部门申请
    ↓
审核通过 → 领取营业执照
    ↓
后续事项
  ├── 税务登记
  ├── 对公账户（可选）
  └── 行业许可（如食品经营许可证）
```

### 2.4 变更经营者（2022年新规）

> **重大便利**：个体工商户可以直接**变更经营者**，不再需要"先注销、后新设"。
> 但需结清税款，妥善处理原有债权债务。

## 三、税务制度

### 3.1 税种对比

| 税种 | 个体工商户 | 有限公司 |
|------|-----------|----------|
| 增值税 | ✅ 缴纳 | ✅ 缴纳 |
| 企业所得税 | ❌ 不缴 | ✅ 缴纳（25%/小微5%） |
| 个人所得税 | ✅ 经营所得（5%-35%累进） | ✅ 分红20% |
| 附加税 | ✅ | ✅ |

### 3.2 征税方式

| 方式 | 适用条件 | 说明 |
|------|----------|------|
| 查账征收 | 建账健全 | 2026年主流方向 |
| 核定征收 | 不能完整建账 | 逐步取消中 |
| 定期定额 | 微型经营 | 税务部门核定固定税额 |

### 3.3 税收优惠（2025-2026年）

| 优惠 | 内容 |
|------|------|
| 增值税免征 | 小规模纳税人月销≤10万（季销≤30万）→ 免征 |
| 个税减半 | 年应纳税所得额≤100万 → 减半征收 |
| "六税两费"减半 | 资源税/城建税/房产税等 → 减半 |

### 3.4 建账要求（2026年新规）

| 档次 | 标准 | 要求 |
|------|------|------|
| 复式账 | 注册资金≥20万 或 月销达标 | 完整的复式记账 |
| 简易账 | 注册资金10-20万 | 简化的复式账 |
| 收支凭证簿 | 未达上述标准 | 基本的收支记录 |

> ⚠️ 严禁不建账！2026年金税四期全面联网，不建账/零申报超6个月将被预警

## 四、雇工与劳动关系

### 4.1 法律性质

个体工商户与雇工之间构成**劳动关系**，适用《劳动法》《劳动合同法》。

### 4.2 个体户作为雇主的义务

| 义务 | 内容 | 法律依据 |
|------|------|----------|
| 签订劳动合同 | 用工之日起1个月内签书面合同 | 劳动合同法第10条 |
| 缴纳社保 | 必须为雇工缴纳社会保险 | 社会保险法第58条 |
| 支付工资 | 不低于最低工资标准，按时足额 | 劳动法第48条 |
| 劳动保护 | 提供劳动安全卫生条件 | 劳动法第54条 |
| 工时休假 | 遵守法定工时、休息休假制度 | 劳动法第36条 |
| 工伤责任 | 未缴工伤保险→工伤费用自行承担 | 工伤保险条例第62条 |

### 4.3 实务要点

1. **很多个体户不签合同不缴社保** — 违法，雇工随时可以仲裁
2. **工伤赔偿是大坑** — 不缴工伤保险，工伤全部费用自掏腰包
3. **无限责任叠加** — 劳动用工赔偿直接追到个人财产

## 五、经营风险与无限责任

### 5.1 无限责任——个体户最大的风险

开个体户亏了钱或欠了债，房子、车子、存款都可能被拿来抵债。不像公司有"有限责任"防火墙。

### 5.2 典型风险场景

| 场景 | 后果 |
|------|------|
| 经营亏损欠债 | 个人财产偿还 |
| 产品质量纠纷赔偿 | 个人财产赔偿 |
| 雇工工伤赔偿（未缴保险） | 个人财产赔偿 |
| 税务违规罚款 | 个人财产缴纳 |
| 消防/环保处罚 | 个人承担 |

## 六、权益保护与扶持政策

### 6.1 《促进个体工商户发展条例》核心保护

| 保护内容 | 具体规定 |
|----------|----------|
| 平等准入 | 政府不得制定歧视性政策 |
| 禁止违规收费 | 任何单位不得违法收费、摊派、集资 |
| 禁止拖欠账款 | 机关、企事业单位不得拖欠个体户账款 |
| 财产保护 | 经营自主权和财产权受法律保护 |
| 知识产权 | 加强字号、商标、专利等保护 |
| 禁止强迫登记 | 不得诱导、强迫劳动者登记为个体户 |

### 6.2 扶持政策

| 领域 | 政策 |
|------|------|
| 财税 | 有利于发展的财税政策 |
| 金融 | 建立信用评价体系，开发适配金融产品 |
| 场所 | 增加经营场所供给，降低成本 |
| 就业 | 公共就业机构提供招聘服务 |
| 数字化 | 引导线上线下一体化经营 |
| 分类培育 | 按行业和规模精准帮扶 |

## 七、个体户转型升级为企业

### 7.1 转型流程

```
决定转型
    ↓
结清税款
    ↓
处理原有债权债务
    ↓
办理企业设立登记
    ↓
办理个体工商户注销登记
    ↓
涉及行政许可 → 行政机关应简化手续
```

### 7.2 转型的利弊

| 维度 | 好处 | 代价 |
|------|------|------|
| 责任 | 有限责任，个人财产有防火墙 | — |
| 融资 | 更容易获得融资、合作 | — |
| 信用 | 企业信用高于个体户 | — |
| 管理 | 规范化治理结构 | 管理成本增加 |
| 税务 | 可享受企业税收优惠 | 税务合规成本增加 |
| 注销 | — | 企业注销比个体户复杂得多 |

## 八、个体户 vs 有限公司 综合对比

| 维度 | 个体工商户 | 有限责任公司 |
|------|-----------|-------------|
| 法律地位 | 自然人经营，非法人 | 法人主体，独立法律人格 |
| 设立门槛 | 低（一人即可，无最低资本） | 中（1-50人，5年内实缴） |
| 责任承担 | ⚠️ 无限责任 | ✅ 有限责任 |
| 税种 | 增值税 + 个人所得税 | 增值税 + 企所税 + 分红个税 |
| 低收入税负 | 较低（有减半优惠） | 较高（双重征税） |
| 高收入税负 | 较高（最高35%） | 较低（有优惠） |
| 建账要求 | 分档（可简易账） | 必须复式账 |
| 融资能力 | 弱 | 强 |
| 品牌信用 | 较弱 | 较强 |
| 注销难度 | 简单 | 较复杂 |
| 适合场景 | 小店/工作室/初创/轻资产 | 规模化/合伙/融资/品牌化 |

### 选择决策树

```
你打算做什么生意？
    │
    ├── 小本生意/自由职业/夫妻店
    │    └── 年收入预计 ＜50万？
    │         ├── 是 → 个体工商户（税负最低、最简单）
    │         └── 否 → 考虑转公司（35%个税太高）
    │
    ├── 需要合伙/融资/品牌
    │    └── 有限责任公司
    │
    ├── 经营有较大风险（制造业/食品/建筑）
    │    └── 有限责任公司（保护个人财产）
    │
    └── 一个人做，但业务金额较大
         └── 一人有限公司（必须能证明公私分开！）
```
